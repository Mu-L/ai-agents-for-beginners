## Naziv događaja: Multi-Agent API s LangGraph i Azure Cosmos DB

Asinkrone mogućnosti FastAPI-ja za održavanje responzivnosti i skalabilnosti interakcija
Do kraja sesije imat ćete jasnu skicu za izgradnju i implementaciju vlastitih skalabilnih multi-agentnih aplikacija u oblaku koje koriste moć moderne AI i infrastrukture u oblaku.
