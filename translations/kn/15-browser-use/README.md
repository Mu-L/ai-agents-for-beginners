# ಕಂಪ್ಯೂಟರ್ ಬಳಕೆ ಏಜೆಂಟ್ಸ್ (CUA) ನಿರ್ಮಾಣ

ಕಂಪ್ಯೂಟರ್ ಬಳಕೆ ಏಜೆಂಟ್ಸ್ ವ್ಯಕ್ತಿ ನಡೆಸುವ ಮೂಲಕ ವೆಬ್‌ಸೈಟ್‌ಗಳೊಂದಿಗೆ ಸಂವಹನ ಮಾಡಬಹುದು: ಬ್ರೌಸರ್ ತೆರೆಯುವುದರಿಂದ, ಪುಟ ಪರಿಶೀಲಿಸುವ ಮೂಲಕ, ಮತ್ತು ಅವರು ನೋಡಿದದ್ದರಿಂದ ಉತ್ತಮ ಮುಂದಿನ ಕ್ರಿಯೆಯನ್ನು ತೆಗೆದುಕೊಳ್ಳುವುದರಿಂದ. ಈ ಪಾಠದಲ್ಲಿ, ನೀವು Airbnb ಅನ್ನು ಹುಡುಕುವ, ರಚನೆಯಾಗಿರುವ ಲಿಸ್ಟಿಂಗ್ ಡೇಟಾವನ್ನು ಪಡೆಯುವ ಮತ್ತು ಸ್ಟಾಕ್‌ಹೋಲ್ಮ್‌ನಲ್ಲಿ ಅತ್ಯಂತ ಕಡಿಮೆ ದರದ ಉಳುವನ್ನು ಗುರುತಿಸುವ ಬ್ರೌಸರ್ ಸ್ವಯಂಚಾಲನೆ ಏಜೆಂಟ್ನ್ನು ನಿರ್ಮಿಸುತ್ತೀರಿ.

ಈ ಪಾಠವು AI ಸಂಚಾಲಿತ ನ್ಯಾವಿಗೇಶನ್‌ನಿಗಾಗಿ Browser-Use ಅನ್ನು, ಬ್ರೌಸರ್ ನಿಯಂತ್ರಣಕ್ಕಾಗಿ Playwright ಮತ್ತು Chrome DevTools ಪ್ರೋಟೋಕಾಲ್ (CDP) ಅನ್ನು, ದೃಷ್ಟಿ-ಚಾಲಿತ ತರ್ಕಕ್ಕಾಗಿ Azure OpenAI ಅನ್ನು ಮತ್ತು ರಚನೆಗೊಳ್ಳುವ ಎಕ್ಸ್ಟ್ರಾಕ್ಷನ್‌ಗೆ Pydantic ಅನ್ನು ಸಂಯೋಜಿಸುತ್ತದೆ.

## ಪರಿಚಯ

ಈ ಪಾಠದಲ್ಲಿ ಅವಲೋಕನ:

- API-ಮಾತ್ರ ಸ್ವಯಂಚಾಲನೆಯಿಗಿಂತ ಯಾವಾಗ ಕಂಪ್ಯೂಟರ್ ಬಳಕೆ ಏಜೆಂಟ್‌ಗಳು ಉತ್ತಮ ಹೊಂದಿಕೆಯಾಗುತ್ತವೆ ಎಂಬುದು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು
- ನಂಬನೀಯ ಬ್ರೌಸರ್ ಜೀವನ ಚಕ್ರ ನಿರ್ವಹಣೆಯಿಗಾಗಿ Browser-Use ಅನ್ನು Playwright ಮತ್ತು CDP ಜೊತೆಗೆ ಸಂಯೋಜಿಸುವುದು
- ಡೈನಾಮಿಕ್ ವೆಬ್ ಪುಟಗಳಿಂದ ಲಿಸ್ಟಿಂಗ್ ಡೇಟಾವನ್ನು ಹೊರತೆಗೆಸಲು Azure OpenAI ದೃಷ್ಟಿ ಮತ್ತು ರಚನೆಯ Pydantic ಔಟ್ಪುಟ್ ಅನ್ನು ಬಳಸುವುದು
- ಯಾವಾಗ ಏಜೆಂಟ್ನು ಮುಂಭಾಗ, ನಟನವನ್ನು ಮುಂಭಾಗ, ಅಥವಾ ಸಂಯೋಜಿತ ಬ್ರೌಸರ್ ಸ್ವಯಂಚಾಲನೆ ಕಾರ್ಯಪ್ರವಾಹವನ್ನು ಬಳಸಬೇಕೆಂದು ನಿರ್ಧರಿಸುವುದು

## ಕಲಿಕೆ ಗುರಿಗಳು

ಈ ಪಾಠವನ್ನು ಪೂರ್ಣಗೊಳಿಸಿದ ಮೇಲೆ, ನೀವು ತಿಳಿಯುತ್ತೀರಿ ಹೇಗೆ:

- Browser-Use ಅನ್ನು Azure OpenAI ಮತ್ತು Playwright ಜೊತೆಗೆ ಸಂರಚಿಸುವುದು
- ನೈಜ ವೆಬ್‌ಸೈಟ್ ಅನ್ನು ನ್ಯಾವಿಗೇಟ್ ಮಾಡುವ ಮತ್ತು ಡೈನಾಮಿಕ್ UI ಅಂಶಗಳನ್ನು ನಿರ್ವಹಿಸುವ ಬ್ರೌಸರ್ ಸ್ವಯಂಚಾಲನೆ ಕಾರ್ಯಪ್ರವಾಹವನ್ನು ನಿರ್ಮಿಸುವುದು
- ಗೋಚರಿಸುವ ಪುಟ ವಿಷಯದಿಂದ ಟೈಪ್ ಮಾಡಿದ ಫಲಿತಾಂಶಗಳನ್ನು ಹೊರತೆಗೆಯುವದು ಮತ್ತು ಅವುಗಳನ್ನು ನಂತರದ ವ್ಯವಹಾರ ತರ್ಕಕ್ಕೆ ತಿರುಗಿಸುವದು
- ಬ್ರೌಸರ್ ಕಾರ್ಯ ಸರ್ವನಿಯೋಜಕವೇ ಅನುವಾದಿಸಲು ಸಾಧ್ಯವೋ ಆಧರಿಸಿ ಏಜೆಂಟ್ನ ಮತ್ತು ನಟನ ಮಾದರಿಗಳ ನಡುವೆ ಆಯ್ಕೆ ಮಾಡುವುದು

## ಕೋಡ್ ಮಾದರಿ

ಈ ಪಾಠದಲ್ಲಿ ಒಂದು ನೋಟ್‌ಬುಕ್ ಪಾಠ್ಯಕ್ರಮವಿದೆ:

- [15-browser-user.ipynb](./15-browser-user.ipynb): CDP ಮೂಲಕ ಕ್ರೋಮ್ ಸೆಷನ್ ಅನ್ನು ಪ್ರಾರಂಭಿಸುತ್ತದೆ, ಸ್ಟಾಕ್‌ಹೋಲ್ಮ್ ಲಿಸ್ಟಿಂಗ್‌ಗಳನ್ನು Airbnb ನಲ್ಲಿ ಹುಡುಕುತ್ತದೆ, Browser-Use ದೃಷ್ಟಿಯಿಂದ ಬೆಲೆಯನ್ನು ಹೊರತೆಗೆಯುತ್ತದೆ, ಮತ್ತು ಅತ್ಯಂತ ಕಡಿಮೆ ಆಯ್ಕೆ ಆಯ್ಕೆಯನ್ನು ರಚನೆಯಾಗಿರುವ ಡೇಟಾ ರೂಪದಲ್ಲಿ ಹಿಂತಿರುಗಿಸುತ್ತದೆ.

## ಪೂರ್ವಾಪೇಕ್ಷಣಿಗಳು

- Python 3.12+
- ನಿಮ್ಮ ಪರಿಸರದಲ್ಲಿ Azure OpenAI ನಿಯೋಜನೆ ಸಂರಚಿಸಲಾಗಿದೆ
- ಕ್ರೋಮ್ ಅಥವಾ ಕ್ರೋಮಿಯಂ ಸ್ಥಳೀಯವಾಗಿ ಸ್ಥಾಪಿಸಲಾಗಿದೆ
- Playwright ಅವಲಂಬನೆಗಳು ಸ್ಥಾಪಿಸಲಾಗಿದೆ
- ಅಸಿಂಕ್ Python ಬಗ್ಗೆ ಮೂಲ ಪರಿಚಯ

## ಸೆಟಪ್

ನೋಟ್‌ಬುಕ್‌ನಲ್ಲಿ ಬಳಸುವ ಪ್ಯಾಕೇಜ್‌ಗಳನ್ನು ಸ್ಥಾಪಿಸಿ:

```bash
pip install browser_use playwright python-dotenv
playwright install chromium
```

ನೋಟ್‌ಬುಕ್ ಬಳಸುವ Azure OpenAI ಪರಿಸರ ಸ್ಥಿರಾಂಕಗಳನ್ನು ಹೊಂದಿಸಿ:

```bash
AZURE_OPENAI_ENDPOINT=...
AZURE_OPENAI_API_KEY=...
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=...
# ಆಯ್ಕೆಯಾಗಿ: ಬಿಟ್ಟುಹೋದಾಗ ಅತ್ಯಂತ ಇತ್ತೀಚಿನ API ಆವೃತ್ತಿಗೆ ಪೂರ್ವನಿಯೋಜಿತವಾಗಿ ಹೋಗುತ್ತದೆ
AZURE_OPENAI_API_VERSION=...
```

## ಮೂಲಭೂತ ವಿನ್ಯಾಸ ಅವಲೋಕನ

ನೋಟ್‌ಬುಕ್ ಸಂಯೋಜಿತ ಬ್ರೌಸರ್ ಸ್ವಯಂಚಾಲನೆ ಕಾರ್ಯಪ್ರವಾಹವನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ:

1. ಕ್ರೋಮ್ CDP ಸಕ್ರಿಯವಾಗಿರುವುದರಿಂದ Playwright ಮತ್ತು Browser-Use ಎರಡೂ ಅದೇ ಬ್ರೌಸರ್ ಸೆಷನ್ ಅನ್ನು ಹಂಚಿಕೊಳ್ಳಬಹುದು.
2. Browser-Use ಏಜೆಂಟ್ Airbnb ತೆರೆಯುವುದು, ಪಾಪ್-ಅಪ್ಗಳನ್ನು ರದ್ದು ಮಾಡುವುದು ಮತ್ತು ಸ್ಟಾಕ್‌ಹೋಲ್ಮ್‌ಗಾಗಿ ಹುಡುಕುವುದು ಮುಂತಾದ ತೆರೆಯುವ ಮುಕ್ತ ನ್ಯಾವಿಗೇಶನ್ ಕೆಲಸಗಳನ್ನು ನಿಭಾಯಿಸುತ್ತದೆ.
3. ಸಕ್ರಿಯ ಪುಟವನ್ನು ರಚನೆಗೊಳ್ಳುವ Pydantic ಸ್ಕೀಮಾ ಮೂಲಕ ಪರಿಶೀಲಿಸಲಾಗುತ್ತದೆ ಮತ್ತು ಲಿಸ್ಟಿಂಗ್ ಶೀರ್ಷಿಕೆಗಳು, ನೈಟ್‌ಲಿ ಬೆಲೆಗಳು, ರೇಟಿಂಗ್‌ಗಳು ಮತ್ತು URL ಗಳು ಹೊರತೆಗೆಯಲ್ಪಡುತ್ತವೆ.
4. Python ತರ್ಕವು ಹೊರತೆಗೆಯಲಾದ ಲಿಸ್ಟಿಂಗ್‌ಗಳನ್ನು ಹೋಲಿಸಿ ಅತ್ಯಂತ ಕಡಿಮೆ ದರದ ಫಲಿತಾಂಶವನ್ನು ಗುರುತಿಸುತ್ತದೆ.

ಈ ವಿಧಾನವು Browser-Use ಅನ್ನು ಉತ್ತಮಪಡುವ ಮತ್ತು ದೃಷ್ಟಿ-ಆಧಾರಿತ ತರ್ಕವನ್ನು ಅನುಸರಿಸುವುದನ್ನು ಇರಿಸುತ್ತದೆ, ಆದರೆ ನೀವು ಅದನ್ನು ಬೇಕಾದಾಗ ನಿಯತವಾದ ಬ್ರೌಸರ್ ನಿಯಂತ್ರಣವನ್ನು ನೀಡುತ್ತದೆ.

## ಪ್ರಮುಖ ಪಾಠಗಳು ಮತ್ತು ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು

### ಏಜೆಂಟ್ ಮತ್ತು ನಟನವನ್ನು ಯಾವಾಗ ಬಳಸುವುದು

| ಪರಿಸ್ಥಿತಿ | ಏಜೆಂಟ್ ಬಳಸಿ | ನಟನ ಬಳಸಿ |
|----------|-----------|-----------|
| ಡೈನಾಮಿಕ್ ವಿನ್ಯಾಸಗಳು | ಹೌದು, AI ಪುಟ ಬದಲಾವಣೆಗಳಿಗೆ ಹೊಂದಿಕೊಳ್ಳಬಹುದು | ಇಲ್ಲ, ಸುಲಭವಾಗುವ ಆಯ್ಕೆಗಳು ಕೆಡಬಹುದು |
| ತಿಳಿದಿರುವ ರಚನೆ | ಇಲ್ಲ, ಏಜೆಂಟ್ ನೇರ ನಿಯಂತ್ರಣಕ್ಕಿಂತ ನಿಧಾನ | ಹೌದು, ವೇಗವಾಗಿ ಮತ್ತು ಖಚಿತವಾಗಿ |
| ಅಂಶಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವದು | ಹೌದು, ಸಹಜ ಭಾಷೆ ಉತ್ತಮವಾಗಿ ಕೆಲಸ ಮಾಡುತ್ತದೆ | ಇಲ್ಲ, ಖಂಚ ಆಯ್ಕೆ ಅಗತ್ಯವಿದೆ |
| ಸಮಯ ನಿಯಂತ್ರಣ | ಇಲ್ಲ, ಕಡಿಮೆ ನಿರೀಕ್ಷಣೀಯ | ಹೌದು, ನಿರೀಕ್ಷೆ ಮತ್ತು ಮರಳಿಕೆಗಳನ್ನು ಸಂಪೂರ್ಣ ನಿಯಂತ್ರಿಸುತ್ತದೆ |
| ಜಟಿಲ ಕಾರ್ಯಪ್ರವಾಹಗಳು | ಹೌದು, ಅಪೇಕ್ಷಿತ ಬಾಹ್ಯ UI ಸ್ಥಿತಿಗಳನ್ನು ನಿಭಾಯಿಸುತ್ತದೆ | ಇಲ್ಲ, ಸ್ಪಷ್ಟ ಶಾಖೆ ಅಗತ್ಯವಿದೆ |

### Browser-Use ಉತ್ತಮ ಅಭ್ಯಾಸಗಳು

1. ಸಂಶೋಧನೆ ಮತ್ತು ಡೈನಾಮಿಕ್ ನ್ಯಾವಿಗೇಶನ್‌ಗಾಗಿ ಏಜೆಂಟ್‌ನಿಂದ ಪ್ರಾರಂಭಿಸಿ.
2. ವೈಚಾರಿಕವಾದಾಗ ನೇರ ಪುಟ ನಿಯಂತ್ರಣಕ್ಕೆ ಬದಲಾಯಿಸಿ.
3. ಹೊರತೆಗೆಯಲಾದ ಡೇಟಾವನ್ನು ಮಾನ್ಯ ಮತ್ತು ಪ್ರಕಾರ-ಭದ್ರವಾಗಿಸಲು ರಚನೆಯಾಗಿರುವ ಔಟ್ಪುಟ್ ಮಾದರಿಗಳನ್ನು ಬಳಸಿ.
4. ಗೋಚರಿಸುವ UI ಬದಲಾವಣೆಗಳನ್ನು ಉದ್ದೀಪನ ಮಾಡುವ ಕ್ರಿಯೆಗಳ ನಂತರ ತಾತ್ಕಾಲಿಕ ವಿಳಂಬಗಳನ್ನು ಸೇರಿಸಿ.
5. ಪುನರಾವೃತ್ತಿಯಲ್ಲಿನ ವಿಫಲತೆಗಳನ್ನು ಸುಲಭವಾಗಿ ಡಿಬಗ್ ಮಾಡಲು ಸ್ಕ್ರೀನ್‌ಶಾಟ್‌ಗಳನ್ನು ಹಿಡಿಯಿರಿ.
6. ವೆಬ್‌ಸೈಟ್ ಬದಲಾವಣೆಯನ್ನು ನಿರೀಕ್ಷಿಸಿ ಮತ್ತು ಪಾಪ್-ಅಪ್ಗಳು ಮತ್ತು ವಿನ್ಯಾಸ ಬದಲಾವಣೆಗಳಿಗೆ ಬ್ಯಾಕಪ್ ತಂತ್ರಗಳನ್ನು ವಿನ್ಯಾಸಗೊಳಿಸಿ.
7. ಹೂಡಿಕೆ ಮತ್ತು ಖಚಿತತೆಗೆ ಎರಡು ರೀತಿಯ ಮಾದರಿಗಳ ಸಂಯೋಜನೆಯನ್ನು ಮಾಡಿ.

### ಬ್ರೌಸರ್ ಏಜೆಂಟ್ಗಳ ಸುರಕ್ಷತಾ ಮಾರ್ಗದರ್ಶಿಗಳು

ಬ್ರೌಸರ್ ಏಜೆಂಟ್‌ಗಳು ನೇರ ಜೀವರಾಜಿಯಾದ ವೆಬ್‌ಸೈಟ್‌ಗಳಲ್ಲಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತವೆ, ಆದ್ದರಿಂದ ಅವುಗೆ ತಿಳಿದಿರುವ API ಕೇವಲ ಕರೆಮಾಡುವ ಸ್ಕ್ರಿಪ್ಟ್ ಗಿಂತ ಕಡಿಮೆಯಲ್ಲದ ಮಿತಿಗಳನ್ನು ಬೇಕಾಗಿರುತ್ತದೆ. ನೋಟ್‌ಬುಕ್ ಡೆಮೋದಿಂದ ನಿಜವಾದ ಕಾರ್ಯಪ್ರವಾಹಕ್ಕೆ ಹೋಗುವ ಮೊದಲು ಏಜೆಂಟ್ ನೋಡಬಹುದು, ಕ್ಲಿಕ್ ಮಾಡಬಹುದು ಮತ್ತು ಸಲ್ಲಿಸಬಹುದು ಎಂಬ ನಿಯಂತ್ರಣಗಳನ್ನು ನಿರ್ಧರಿಸಿ.

1. **ಬ್ರೌಸಿಂಗ್ ಪರಿಸರದ ವ್ಯಾಪ್ತಿಯನ್ನು ನಿರ್ಧರಿಸಿ.** ಏಜೆಂಟ್ ಅನ್ನು ವಿಶಿಷ್ಟ ಬ್ರೌಸರ್ ಪ್ರೊಫೈಲ್ ಅಥವಾ ಸ್ಯಾಂಡ್‌ಬಾಕ್ಸ್‌ನಲ್ಲಿ ಚಾಲನೆ ಮಾಡಿ ಮತ್ತು ಕಾರ್ಯಕ್ಕೆ ಅಗತ್ಯವಿರುವ ಡೊಮೇನ್‌ಗಳಷ್ಟಾಗಿಯೇ ನಿಯಮಿತರಿಸಿ.
2. **ಕ್ರಿಯೆಯಿಂದ ವೀಕ್ಷಣೆಯನ್ನು ವಿಭಜಿಸಿ.** ಮೊದಲಿಗೆ ಏಜೆಂಟ್ ಹುಡುಕಿ, ಓದಿ ಮತ್ತು ಡೇಟಾವನ್ನು ಹೊರತೆಗೆಸಲು ಬಿಡಿ; ಫಾರ್ಮ್ ಸಲ್ಲಿಸುವುದು, ಸಂದೇಶಗಳನ್ನು ಕಳುಹಿಸುವುದು, ಪ್ರಯಾಣ ಬುಕ್ ಮಾಡುವುದು, ಖರೀದಿ ಮಾಡುವುದು, ದಾಖಲೆಗಳನ್ನು ಅಳಿಸುವುದು, ಅಥವಾ ಖಾತೆ ಸೆಟ್ಟಿಂಗ್‍ಗಳನ್ನು ಬದಲಾಯಿಸುವುದಕ್ಕೂ ಮೊದಲು ಸ್ಪಷ್ಟ ಅನುಮೋದನೆ ಅವಶ್ಯಕ.
3. **ಪಾಸ್ವರ್ಡ್‌, ಹಣಪಾವತಿ ವಿವರಗಳು, ಸೆಷನ್ ಕುಕೀಸ್, ಅಥವಾ ಕಚ್ಚಾ ವೈಯಕ್ತಿಕ ಡೇಟಾವನ್ನು ಮಾದರಿ ಸನ್ನಿವೇಶದಲ್ಲಿ ಇರಿಸಬೇಡಿ.** ಬಳಕೆದಾರ ಯೋಗ್ಯತೆಗಾಗಿ ಹಸ್ತಾಂತರಿಸಿ ಮತ್ತು ಸಂವೇದನಾಶೀಲ ಕ್ಷೇತ್ರಗಳನ್ನು ಲಾಗ್‍ಗಳಿಂದ ಅಡಗಿಸಿ.
4. **ಪುಟದ ವಿಷಯವನ್ನು ಅಪರಿಚಿತ ಮಾಹಿತಿ ಎಂದು ಪರಿಗಣಿಸಿ.** ವೆಬ್‌ಸೈಟ್ ಏಜೆಂಟ್‌ಗೆ, ಬಳಕೆದಾರನಿಗೆ ಅಲ್ಲದ ಸೂಚನೆಗಳನ್ನು ಹೊಂದಿರಬಹುದು. ಏಜೆಂಟ್ ತನ್ನ ಗುರಿಯನ್ನು ಬದಲಾಯಿಸುವುದಾಗಿ, ಡೇಟಾವನ್ನು ಬಹಿರಂಗಪಡಿಸುವುದಾಗಿ, ಸುರಕ್ಷತಾ ಕ್ರಮಗಳನ್ನು ನಿಷ್ಕ್ರಿಯಗೊಳಿಸುವುದಾಗಿ, ಅಥವಾ ಸಂಬಂಧವಿಲ್ಲದ ತಾಣಗಳನ್ನು ಭೇಟಿ ಮಾಡುವುದಾಗಿ ಅಂತೆ ತಿರುವುಳ್ಳ ಪಠ್ಯವನ್ನು ನಿರ್ಲಕ್ಷಿಸಬೇಕು.
5. **ಸಂಕಟಪೂರ್ಣ ಹಂತಗಳ ಸುತ್ತಲೂ ಖಚಿತ ಪರಿಶೀಲನೆಗಳನ್ನು ಬಳಸಿ.** ಬಳಕೆದಾರ ಅನುವು ನೀಡುವ ಮೊದಲು ಪ್ರಸ್ತುತ URL, ಪುಟ ಶೀರ್ಷಿಕೆ, ಆಯ್ಕೆಮಾಡಲಾದ ಅಂಶ, ಬೆಲೆ, ಸ್ವೀಕರಿಸುವವರ ಮತ್ತು ಕ್ರಿಯಾ ಸಾರಾಂಶವನ್ನು ಕೋಡ್ ಮೂಲಕ ಪರಿಶೀಲಿಸಿ.
6. **ಬಜೆಟ್‌ಗಳು ಮತ್ತು ನಿಲ್ಲಿಸುವ ತತ್‍ವಗಳನ್ನು ಸೆಟ್ ಮಾಡಿ.** ಏಜೆಂಟ್ ಬಳಸಬಹುದಾದ ಕ್ರಿಯೆಗಳ, ಪುನರಾವರ್ತನೆಗಳ, ಟ್ಯಾಬ್‌ಗಳ ಮತ್ತು ನಿಮಿಷಗಳ ಸಂಖ್ಯೆಯನ್ನು ನಿಯಮದಿಂದ ನಿರ್ಧರಿಸಿ. ಪುಟ ಸ್ಥಿತಿ ಸ್ಪಷ್ಟವಾಗದಿದ್ದರೆ ಮುಂದುವರಿಯದೆ ನಿಲ್ಲಿಸಿ.
7. **ಅವಶ್ಯಕವಾದ ಸಾಕ್ಷ್ಯಗಳನ್ನು ದಾಖಲೆ ಮಾಡಿ, ಎಲ್ಲವನ್ನೂ ಅಲ್ಲ.** ಕಾರ್ಯ ಸಾರಾಂಶಗಳು, ಸಮಯದ ಟಿಪ್ಪಣಿಗಳು, URLಗಳು, ಆಯ್ಕೆಮಾಡಲಾದ ಅಂಶಗಳ ವಿವರಣೆಗಳು ಮತ್ತು ಸ್ಕ್ರೀನ್‌ಶಾಟ್ ಉಲ್ಲೇಖಗಳನ್ನು ಇರಿಸಿ, ಆದ್ದರಿಂದ ತಪ್ಪುಗಳನ್ನು ಪರಿಶೀಲಿಸಲು ಸುಲಭವಾಗುತ್ತದೆ ಆದರೆ ಅನಗತ್ಯ ಸಂವೇದನಾಶೀಲ ಪುಟ ವಿಷಯವನ್ನು ಸಂಗ್ರಹಿಸುವುದಿಲ್ಲ.

Airbnb ಮಾದರಿಯಲ್ಲಿ, ಸುರಕ್ಷಿತ ಪೂರ್ವನಿಯೋಜನೆ ಲಿಸ್ಟಿಂಗ್‌ಗಳನ್ನು ಹುಡುಕುವುದು ಮತ್ತು ಬೆಲೆಗಳನ್ನು ಹೊರತೆಗೆಸುವುದು. ಲಾಗಿನ್ ಆಗುವುದು, ಹೋಸ್ಟ್ ಸಂಪರ್ಕಿಸುವುದು ಅಥವಾ ಬುಕ್ಕಿಂಗ್ ಪೂರ್ಣಗೊಳಿಸುವುದು ಹೊಸಬಳಕೆದಾರ ಅನುಮೋದಿಸಿ ಕ್ರಮವಾಗಬೇಕು.

### ನಿಜ ಜೀವನದ ಅನ್ವಯಿಕೆಗಳು

- ಪ್ರಯಾಣ ಬುಕ್ಕಿಂಗ್ ಮತ್ತು ಬೆಲೆ ಸಂಶೋಧನೆ
- ಇ-ಕಾಮರ್ಸ್ ಬೆಲೆ ಹೋಲಿಕೆ ಮತ್ತು ಲಭ್ಯತೆ ಪರಿಶೀಲನೆಗಳು
- ಡೈನಾಮಿಕ್ ವೆಬ್‌ಸೈಟ್‌ಗಳಿಂದ ರಚನೆಯಾಗಿರುವ ಎಕ್ಸ್ಟ್ರಾಕ್ಷನ್
- ದೃಷ್ಟಿ-ಜಾಗರೂಕ UI ಪರೀಕ್ಷೆ ಮತ್ತು ಪರಿಶೀಲನೆ
- ವೆಬ್‌ಸೈಟ್ ಮಿತಿಗೋಳ ಮತ್ತು ಎಚ್ಚರಿಕೆ
- ಹಲಾಳಾ ಹಂತದ ಹರಿವುಗಳಾದ ಫಾರ್ಮ್ ಭರ್ತಿ ಮಾಡುವ ಜಾಣ್ಮೆ

## ನಿಜ ಜೀವನದ ಉದಾಹರಣೆ: Microsoft ಪ್ರಾಜೆಕ್ಟ್ Opal

ನೀವು ಈ ಪಾಠದಲ್ಲಿ ನಿರ್ಮಿಸುವ ಏಜೆಂಟ್ ಒಂದು ಸಣ್ಣ, ಸ್ಥಳೀಯ ಆವೃತ್ತಿ **ಕಂಪ್ಯೂಟರ್ ಬಳಕೆ ಏಜೆಂಟ್ (CUA)** — ವ್ಯಕ್ತಿಯಂತೆ ಬ್ರೌಸರ್ ಚಾಲನೆ ಮಾಡುವ ಪ್ರೋಗ್ರಾಂ. ಮೈಕ್ರೋಸಾಫ್ಟ್ ಇದೇ ಕಲ್ಪನೆಯನ್ನು ಸಂಸ್ಥೆಗೆ ತರುತ್ತಿದೆ **[Project Opal (Frontier)](https://support.microsoft.com/en-us/microsoft-365-copilot/get-started-with-project-opal-frontier)** ಮೂಲಕ, ಇದು Microsoft 365 Copilot ನಲ್ಲಿ ಲಭ್ಯವಿದೆ.

Project Opal ಮೂಲಕ, ನೀವು ಒಂದು ಕೆಲಸವನ್ನು ವಿವರಣೆ ಮಾಡಿ ಮತ್ತು ಏಜೆಂಟ್ ನಿಮ್ಮ ಪರವಾಗಿ **ಕಂಪ್ಯೂಟರ್ ಬಳಕೆಯನ್ನು ಸುರಕ್ಷಿತ Windows 365 ಕ್ಲೌಡ್ ಪಿಸಿಯಲ್ಲಿ** ಸಂಬಂಧಿತ ಅಪ್ಲಿಕೇಶನ್‌ಗಳು, ತಾಣಗಳು ಮತ್ತು ಡೇಟಾಗಳೊಳಗಿನ ಬ್ರೌಸರ್ ಆಧಾರಿತ ಕಾರ್ಯಗಳಲ್ಲಿ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ. ಇದು **ಅಸಿಂಕ್ರೋನಸ್ ಬ್ಯಾಕ್ಗ್ರೌಂಡ್‌ನಲ್ಲಿ** ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತದೆ ಮತ್ತು ನೀವು ಯಾವಾಗ ಬೇಕಾದರೂ ಕಾರ್ಯದ ನಿರ್ವಹಣೆಯನ್ನು ಅಥವಾ ನಿಯಂತ್ರಣವನ್ನು ತೆಗೆದುಕೊಳ್ಳಬಹುದು. ಉದಾಹರಣೆ ಕೆಲಸಗಳು:

- ಭದ್ರತಾ ಗುಂಪಿನ ಸದಸ್ಯತ್ವ ವಿನಂತಿಗಳನ್ನು ನಿರ್ವಹಿಸುವುದು
- ಪಾಲನಾ ಪರಿಶೀಲನೆಗಳಿಗಾಗಿ ಲೆಕ್ಕಪರಿಶೀಲನಾ ಸಾಕ್ಷ್ಯ ಸಂಗ್ರಹಿಸುವುದು ಮತ್ತು ಮಾನ್ಯಗೊಳಿಸುವುದು
- IT ಘಟನಾವಳಿಗಳನ್ನು ತ್ವರಿತಗೊಳಿಸುವುದು (ಟಿಕೆಟ್ ಸ್ಥಿತಿಯನ್ನು ನವೀಕರಿಸುವುದು, ಮಾಲೀಕರನ್ನು ನಿಯೋಜಿಸುವುದು, ನಕಲಿ ದಾಖಲೆಗಳನ್ನು ಮುಚ್ಚುವುದು)
