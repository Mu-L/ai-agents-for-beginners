# ಜ್ಞಾನ ಪರಿಶೀಲನೆ ಉತ್ತರಗಳು

## ಪ್ರಶ್ನೆ 1

**ಉತ್ತರ: A2.** ಕಾರ್ಯದಲ್ಲಿ ವಿಭಿನ್ನ ತಜ್ಞ ರೋಲ್‌ಗಳು, ಬೇರೆ ಬೇರೆ ಉಪಕರಣಗಳು ಅಥವಾ ಅನುಮತಿಗಳು, ಮತ್ತು ಸ್ಪಷ್ಟ ಹಸ್ತಾಂತರಗಳು ಸಂಯೋಜನೆಗೆ ಅಗತ್ಯವಿರುವಾಗ ಬಹು ಏಜೆಂಟ್‌ಗಳು ಉಪಯುಕ್ತವಾಗುತ್ತವೆ. ಸರಳ ಬೆಂಬಲ ಬಾಟ್ ಸಾಮಾನ್ಯವಾಗಿ ಏಕ ಏಜೆಂಟ್ ಆಗಿಯೇ ಕಾಪಾಡಬಹುದು. ಒಂದೇ ರೀತಿಯ ಹೆಚ್ಚಿನ ವಿನಂತಿಗಳ ಪ್ರಮಾಣ ಮುಖ್ಯವಾಗಿ ಸ್ಕೇಲಿಂಗ್ ವಿಚಾರವಾಗಿದ್ದು, ಅದೇ ಮೊದಲು ಬಹು ಏಜೆಂಟ್ ವಿನ್ಯಾಸವನ್ನು ಅವಶ್ಯಕವಲ್ಲ.

## ಪ್ರಶ್ನೆ 2

**ಉತ್ತರ: A1.** ಒಂದು ಏಜೆಂಟ್ ಸಾಮಾನ್ಯವಾಗಿ ಪರೀಕ್ಷಿಸಲು, ಗಮನಿಸುಗೆ ಮತ್ತು ನಿರ್ವಹಿಸಲು ಸುಲಭ, ಒಂದೇ ಸೂಚನೆ ಮತ್ತು ಉಪಕರಣಗಳ ಸಂಗ್ರಹ ಕಾರ್ಯವನ್ನು ನಿಭಾಯಿಸುವಾಗ. ಅನೇಕ ಉಪಕರಣಗಳಿಗೆ ಪ್ರವೇಶವು ಸಹಜವಾಗಿ ಅನೇಕ ಏಜೆಂಟ್‌ಗಳನ್ನು ಅಗತ್ಯವಿಲ್ಲ. ಬೇರೆ ಬೇರೆ ಅನುಮತಿಗಳು ಮತ್ತು ಪರಿಶೀಲನೆ ಅಗತ್ಯ ಹೊಂದಿರುವ ವಿಭಿನ್ನ ರೋಲ್‌ಗಳು ಬಹು ಏಜೆಂಟ್ ವಿನ್ಯಾಸ justify ಮಾಡಬಹುದು.
