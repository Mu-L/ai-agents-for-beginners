# ⏩ ಅಜೂರ್ ಓಪನ್‌ಎಐ (Responses API) (ಡಾಟ್‌ನೆಟ್) ಜೊತೆ ಕ್ರಮಾನುಗತ ಎಜೆಂಟ್ ವರ್ಕ್‌ಫ್ಲೋಗಳು

## 📋 ಸುಧಾರಿತ ಕ್ರಮಾನುಗತ ಪ್ರಕ್ರಿಯೆ ಟ್ಯುಟೋರಿಯಲ್

ಈ ನೋಟ್ಬುಕ್ ಡಾಟ್‌ನೆಟ್ ಮತ್ತು ಅಜೂರ್ ಓಪನ್‌ಎಐ (Responses API) ಗಾಗಿ ಮೈಕ್ರೋಸಾಫ್ಟ್ ಎಜೆಂಟ್ ಫ್ರೇಮ್‌ವರ್ಕ್ ಬಳಸಿ **ಕ್ರಮಾನುಗತ ವರ್ಕ್‌ಫ್ಲೋ ಮಾದರಿಗಳು** ಅನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ. ನೀವು ಪ್ರತಿಯೊಂದು ಹಂತವು ಹಿಂದಿನ ಹಂತದ ಫಲಿತಾಂಶಗಳ ಮೇಲೆ ಆಧಾರಿತವಾಗಿರುವ, ನಿರ್ದಿಷ್ಟ ಆದೇಶದಲ್ಲಿ ಎಜೆಂಟ್‌ಗಳು ಚಾಲನೆ ಮಾಡುತ್ತಿರುವ ಜಟಿಲ, ಹಂತ ಹಂತದ ಪ್ರಕ್ರಿಯೆ পাইಪ್ಲೈನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವುದನ್ನು ಕಲಿಯುತ್ತೀರಿ.

## 🎯 ಕಲಿಕೆಯ ಉದ್ದೇಶಗಳು

### 🔄 **ಕ್ರಮಾನುಗತ ಪ್ರಕ್ರಿಯೆ ವಸ್ತುಶಿಲ್ಪ**
- **ರೇಖೀಯ ವರ್ಕ್‌ಫ್ಲೋ ವಿನ್ಯಾಸ**: ಸ್ಪಷ್ಟ ಅವಲಂಬನೆಗಳೊಂದಿಗೆ ಹಂತ ಹಂತದ ಪ್ರಕ್ರಿಯೆ ಪೈಪ್ಲೈನ್‌ಗಳನ್ನು ರಚಿಸಿ
- **ಸ್ಥಿತಿ ನಿರ್ವಹಣೆ**: ಕ್ರಮಾನುಗತ ವರ್ಕ್‌ಫ್ಲೋ ಹಂತಗಳಾದ್ಯಂತ ಪ್ರಾಪಂಚಿಕ ಮತ್ತು ಡೇಟಾ ಹರಿವು ಕಾಯ್ದುಕೊಳ್ಳಿರಿ
- **ಅಜೂರ್ ಓಪನ್‌ಎಐ (Responses API)**: ಬਹੁ- ಹಂತ .NET ವರ್ಕ್‌ಫ್ಲೋಗಳಲ್ಲಿ ಅಜೂರ್ ಓಪನ್‌ಎಐ ಮಾದರಿಗಳನ್ನು ಉಪಯೋಗಿಸಿ
- **ಉದ್ಘಾಟನಾ ಪೈಪ್ಲೈನ್ ಮಾದರಿಗಳು**: ಉತ್ಪಾದನಾ-ಸಿದ್ಧ ಕ್ರಮಾನುಗತ ಪ್ರಕ್ರಿಯಾ ವ್ಯವಸ್ಥೆ ನಿರ್ಮಿಸಿ

### 🏗️ **ಸುಧಾರಿತ ಕ್ರಮಾನುಗತ ಮಾದರಿಗಳು**
- **ಹಂತ-ಗೇಟು ಪ್ರಕ್ರಿಯೆ**: ವರ್ಕ್‌ಫ್ಲೋ ಹಂತಗಳ ನಡುವೆ ಪರಿಶೀಲನಾ ನಿರ್ದಿಷ್ಟಾವಳಿಗಳನ್ನು ಅನುಷ್ಠಾನಗೊಳಿಸಿ
- **ಸಂದರ್ಭ ಸಂರಕ್ಷಣೆ**: ಎಲ್ಲಾ ಹಂತಗಳಾದ್ಯಂತ ಸ್ಥಿತಿಯನ್ನು ಮತ್ತು ಸಂಸ್ಕೃತ ಜ್ಞಾನವನ್ನು ಕಾಪಾಡಿರಿ
- **ದೋಷ ಪ್ರಸರಣ**: ಕ್ರಮಾನುಗತ ಪ್ರಕ್ರಿಯೆ ಸರಪಳಿಗಳಲ್ಲಿ ವೈಫಲ್ಯಗಳನ್ನು ಸೌಮ್ಯವಾಗಿ ಹ್ಯಾಂಡಲ್ ಮಾಡಿ
- **ಕಾರ್ಯಕ್ಷಮತೆ ಉದ್ಧಾರಣೆ**: ಕನಿಷ್ಟ ಓವರ್‌ಹೆಡ್‌ನೊಂದಿಗೆ ಪರಿಣಾಮಕಾರಿ ಕ್ರಮಾನುಗತ ನಿರ್ವಹಣೆ

### 🏢 **ಉದ್ಘಾಟನಾ ಕ್ರಮಾನುಗತ ಅಪ್ಲಿಕೇಶನ್ಗಳು**
- **ದಾಖಲೆ ಪ್ರಕ್ರಿಯೆ ಪೈಪ್ಲೈನ್**: ಬಹು ಹಂತದ ದಾಖಲೆ ವಿಶ್ಲೇಷಣೆ, ಪರಿವರ್ತನೆ ಮತ್ತು ಪರಿಶೀಲನೆ
- **ಗುಣಮಟ್ಟದ ಭದ್ರತೆಯ ವರ್ಕ್‌ಫ್ಲೋಗಳು**: ಕ್ರಮಾನುಗತ ಪರಿಶೀಲನೆ, ಪರಿಶೀಲನೆ ಮತ್ತು ಅನುಮೋದನೆ ಪ್ರಕ್ರಿಯೆಗಳು
- **ವಿಷಯ ಉತ್ಪಾದನಾ ಪೈಪ್ಲೈನ್**: ಸಂಶೋಧನೆ → ಬರವಣಿಗೆ → ಸಂಪಾದನೆ → ವಿಮರ್ಶೆ → ಪ್ರಕಟಣೆ
- **ವ್ಯವಸ್ಥಾಪನ ಪ್ರಕ್ರಿಯೆ ಸ್ವಯಂಚಾಲನೆ**: ಸ್ಪಷ್ಟ ಹಂತ ಅವಲಂಬನೆಗಳೊಂದಿಗೆ ಬಹು ಹಂತ ಕಾರ್ಯಪ್ರವಾಹಗಳು

## ⚙️ ಪೂರ್ವಾಪೇಕ್ಷಿತ사항ಗಳು ಮತ್ತು ಸೆಟಪ್

### 📦 **ಅಗತ್ಯ NuGet ಪ್ಯಾಕೇಜ್‌ಗಳು**

ಡಾಟ್‌ನೆಟ್ ಕ್ರಮಾನುಗತ ವರ್ಕ್‌ಫ್ಲೋಗಳಿಗಾಗಿ ಅಗತ್ಯ ಪ್ಯಾಕೇಜ್‌ಗಳು:

```xml
<!-- Core AI Framework -->
<PackageReference Include="Microsoft.Extensions.AI" Version="10.*" />

<!-- Azure OpenAI (Responses API) -->
<PackageReference Include="Azure.AI.OpenAI" Version="2.*" />

<!-- Azure Identity and Async LINQ Support -->
<PackageReference Include="Azure.Identity" Version="1.15.0" />
<PackageReference Include="System.Linq.Async" Version="6.0.3" />

<!-- Local Agent Framework References -->
<!-- Microsoft.Agents.AI.dll - Core agent abstractions -->
<!-- Microsoft.Agents.AI.OpenAI.dll - Azure OpenAI (Responses API) integration -->
```

### 🔑 **ಅಜೂರ್ ಓಪನ್‌ಎಐ ಸಂರಚನೆ**

**ಪರಿಸರ ಸೆಟಪ್ (.env ಫೈಲ್):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-4.1-mini
```

**ಕಾನ್ಫಿಗರೇಶನ್ ನಿರ್ವಹಣೆ:**
```csharp
// Load environment variables securely
Env.Load("../../../.env");
var azureEndpoint = Environment.GetEnvironmentVariable("AZURE_OPENAI_ENDPOINT");
var deployment = Environment.GetEnvironmentVariable("AZURE_OPENAI_DEPLOYMENT");
```

- **ಸಂಪನ್ಮೂಲ ನಿರ್ವಹಣೆ**: ಹಂತಗಳಾದ್ಯಂತ AI ಮಾದರಿ ಸಂಪನ್ಮೂಲಗಳ ಪರಿಣಾಮಕಾರಿ ಹಂಚಿಕೆ

### 🔒 **ಭದ್ರತೆ ಮತ್ತು ಅನುಕೂಲತೆ**

### 🎯 ವಾಸ್ತವಿಕ ಜಗತ್ತಿನ ಅಪ್ಲಿಕೇಶನ್ಗಳು

- **ಇ-ಕಾಮರ್ಸ್**: ಉತ್ಪನ್ನ ವಿಶ್ಲೇಷಣೆ → ಬೆಲೆ ನಿರ್ಣಯ → ಕೋಟೇಶನ್ ರಚನೆ

- **ರಕ್ಷಣೆ**: ದಾವೆ ವಿಶ್ಲೇಷಣೆ → ಮೌಲ್ಯಮಾಪನ → ಕೋಟೇಶನ್ ರಚನೆ
- **ವಿಷಯ ಸೃಷ್ಟಿ**: ಸಂಶೋಧನೆ → ಬರವಣಿಗೆ → ಸಂಪಾದನೆ → ಪ್ರಕಟಣೆ
