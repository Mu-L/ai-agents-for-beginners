# 🤝 ಎಂಟರ್‌ಪ್ರೈಸ್ ಬಹು-ಏಜೆಂಟ್ ಕೆಲಸದ ಪ್ರಕ್ರಿಯೆ ವ್ಯವಸ್ಥೆಗಳು (.NET)

## 📋 ಕಲಿಕೆಯ ಗುರಿಗಳು

ಈ ನೋಟ್ಬುಕ್ Microsoft Agent Framework ಅನ್ನು .NET ನಲ್ಲಿ Azure OpenAI (Responses API) ಸಹಿತ ಬಳಸಿಕೊಂಡು ಸುಧಾರಿತ ಎಂಟರ್‌ಪ್ರೈಸ್-ಗ್ರೇಡ್ ಬಹು-ಏಜೆಂಟ್ ವ್ಯವಸ್ಥೆಗಳನ್ನು ಹೇಗೆ ರಚಿಸಬೇಕು ಎಂಬುದನ್ನು ತೋರಿಸುತ್ತದೆ. ನೀವು ರಚನಾತ್ಮಕ ಕೆಲಸದ ಪ್ರಕ್ರಿಯೆಗಳ ಮೂಲಕ ಒಟ್ಟಾಗಿ ಕೆಲಸ ಮಾಡುವ ಹಲವಾರು ವಿಶೇಷೀಕೃತ ಏಜೆಂಟ್‌ಗಳನ್ನು ಹೇಗೆ ಸಂಯೋಜಿಸಬಹುದು ಮತ್ತು .NET ನ ಎಂಟರ್‌ಪ್ರೈಸ್ ವೈಶಿಷ್ಟ್ಯಗಳನ್ನು ಪ್ರೋತ್ಸಾಹ-ಸಿದ್ಧ ಪರಿಹಾರಗಳಿಗಾಗಿ ಹೇಗೆ ಉಪಯೋಗಿಸಬಹುದು ಎಂದು ಕಲಿಯುತ್ತೀರಿ.

**ನೀವು ರಚಿಸುವ ಎಂಟರ್‌ಪ್ರೈಸ್ ಬಹು-ಏಜೆಂಟ್ ಸಾಮರ್ಥ್ಯಗಳು:**
- 👥 **ಏಜೆಂಟ್ ಸಹಯೋಗ**: ಸಂಚಯ-ಕಾಲದಲ್ಲಿ ದೃಢೀಕರಣದೊಂದಿಗೆ ಪ್ರಕಾರ-ಸುರಕ್ಷಿತ ಏಜೆಂಟ್ ಸಂಯೋಜನೆ
- 🔄 **ಕೆಲಸದ ಪ್ರಕ್ರಿಯೆ ಸಂಯೋಜನೆ**: .NET ಅವರ ಅಸಿಂಕ್ರೋನಸ್ ಮಾದರಿಗಳೊಂದಿಗೆ ಘೋಷಣಾತ್ಮಕ ಕೆಲಸದ ಪ್ರಕ್ರಿಯೆ ವ್ಯಾಖ್ಯಾನ
- 🎭 **ಪಾತ್ರ ವಿಶೇಷೀಕರಣ**: ಬಲವಾಗಿ-ಟೈಪ್ ಮಾಡಿದ ಏಜೆಂಟ್ ವೈಯಕ್ತಿಕತೆಗಳು ಮತ್ತು ಪರಿಣತಿ ಕ್ಷೇತ್ರಗಳು
- 🏢 **ಎಂಟರ್‌ಪ್ರೈಸ್ ಏಕೀಕರಣ**: ಪರಿಶೀಲನೆ ಮತ್ತು ದೋಷ ನಿರ್ವಹಣೆ ಸೇರಿದಂತೆ ಉತ್ಪಾದನಾ-ಸಿದ್ಧ ಮಾದರಿಗಳು

## ಈ ಮಾದರಿ ಏನು ತೋರಿಸುತ್ತದೆ

ಈ ಬಹು-ಏಜೆಂಟ್ ಕೆಲಸದ ಪ್ರಕ್ರಿಯೆ ವ್ಯವಸ್ಥೆ ಎರಡು ವಿಶೇಷೀಕೃತ ಏಜೆಂಟ್‌ಗಳೊಂದಿಗೆ ಹೋಟೆಲ್ ಪ್ರಯಾಣ ಶಿಫಾರಸು ಸೇವೆಯನ್ನು ರಚಿಸುತ್ತದೆ:

1. **ಮುಂಭಾಗದ ಏಜೆಂಟ್**: ಚಟುವಟಿಕೆ ಮತ್ತು ಸ್ಥಳ ಶಿಫಾರಸುಗಳನ್ನು ನೀಡುವ ಪ್ರವಾಸಿ ಏಜೆಂಟ್
2. **ಕನ್ಸಿಯರ್ಜ್ ಏಜೆಂಟ್**: ಶಿಫಾರಸುಗಳನ್ನು ಪರಿಶೀಲಿಸುತ್ತಾ ಪ್ರಾಮಾಣಿಕ ಮತ್ತು ಪ್ರವಾಸಿಗರಲ್ಲದ ಅನುಭವಗಳನ್ನು ಖಚಿತಪಡಿಸುವುದು

ಏಜೆಂಟ್‌ಗಳು ಈ ರೀತಿಯಾಗಿ ಕೆಲಸ ಮಾಡುತ್ತವೆ:
- ಮುಂಭಾಗದ ಏಜೆಂಟ್ ಪ್ರಾರಂಭಿಕ ಪ್ರಯಾಣ ವಿನಂತಿಯನ್ನು ಸ್ವೀಕರಿಸುತ್ತದೆ
- ಕನ್ಸಿಯರ್ಜ್ ಏಜೆಂಟ್ ಶಿಫಾರಸುಗಳನ್ನು ಪರಿಶೀಲಿಸಿ ಸ್ವಚ್ಚಗೊಳಿಸುತ್ತದೆ
- ಕೆಲಸದ ಪ್ರಕ್ರಿಯೆ ನೇರ ಸಮಯದಲ್ಲಿ ಉತ್ತರಗಳನ್ನು ಸ್ರೋತ ಮಾಡುತ್ತದೆ

## ಮುಖ್ಯ ಕಲ್ಪನೆಗಳು

### ಏಜೆಂಟ್ ಸಂಯೋಜನೆ
ಈ ಮಾದರಿ Microsoft Agent Framework ನೊಂದಿಗೆ ಪ್ರಕಾರ-ಸುರಕ್ಷಿತ ಏಜೆಂಟ್ ಸಂಯೋಜನೆಯನ್ನು ಸಂಚಯ-ಕಾಲ ದೃಢೀಕರಣದೊಂದಿಗೆ ತೋರಿಸುತ್ತದೆ.

### ಕೆಲಸದ ಪ್ರಕ್ರಿಯೆ ಸಂಯೋಜನೆ
.NET ಅವರ ಅಸಿಂಕ್ರೋನಸ್ ಮಾದರಿಗಳೊಂದಿಗೆ ಘೋಷಣಾತ್ಮಕ ಕೆಲಸದ ಪ್ರಕ್ರಿಯೆ ವ್ಯಾಖ್ಯಾನ ಬಳಸಿಕೊಂಡು ಹಲವಾರು ಏಜೆಂಟ್‌ಗಳನ್ನು ಪೈಪ್‌ಲೈನ್‌ನಲ್ಲಿ ಸಂಪರ್ಕಿಸುತ್ತದೆ.

### ಸ್ರೋತ ಉತ್ತರಗಳು
ಅಸಿಂಕ್ರೋನಸ್ ಎನ್ಯೂಮರೇಬಲ್ಸ್ ಮತ್ತು ಘಟನಾಕ್ರಮಾತ್ಮಕ ಆರ್ಕಿಟೆಕ್ಷರ್ ಬಳಸಿ ಏಜೆಂಟ್ ಉತ್ತರಗಳ ನೇರ-ಸಮಯ ಸ್ರೋತವನ್ನು ಜಾರಿಗೆ ತರುತ್ತದೆ.

### ಎಂಟರ್‌ಪ್ರೈಸ್ ಏಕೀಕರಣ
ಇದಕ್ಕೆ ಉತ್ಪಾದನಾ-ಸಿದ್ಧ ಮಾದರಿಗಳು ಸೇರಿವೆ:
- ಪರಿಸರ ಚರ ಸ್ಥಾಪನೆ
- ಸುರಕ್ಷಿತ ಕ್ರೆಡನ್ಶಿಯಲ್ ನಿರ್ವಹಣೆ
- ದೋಷ ನಿರ್ವಹಣೆ
- ಅಸಂಕ್ರೋನಸ್ ಘಟನಾ ಪ್ರಕ್ರಿಯೆ
