## Arrangementnavn: Multi-Agent API med LangGraph og Azure Cosmos DB

## Beskrivelse

Fremveksten av multi-agent AI-applikasjoner transformerer hvordan vi bygger intelligente systemer — men hvordan arkitekterer du dem for skalerbarhet og ytelse i den virkelige verden? I denne økten dykker vi dypt inn i en produksjonsklar multi-agent applikasjon bygget med LangGraph for agentorkestrering, FastAPI for et API-lag, og Azure Cosmos DB som ryggraden for tilstandshåndtering, vektorlager og transaksjonsdata.

Gjennom en detaljert kodegjennomgang vil du se hvordan du designer og implementerer en agent-drevet arbeidsflyt som sømløst integrerer retrieval-augmented generation (RAG), minneoppbevaring og dynamiske tilstandsoverganger. Vi vil dekke:

Agent-samarbeid med LangGraph for strukturert resonnement
Sanntidslagring av chatthistorikk ved bruk av Azure Cosmos DB – den samme databasen som driver chatthistorikken i ChatGPT, den raskest voksende AI-agent applikasjonen i historien
Vektorsøk for kunnskapsinnhenting med Cosmos DBs native embeddings-støtte
FastAPIs asynkrone evner for å holde interaksjoner responsive og skalerbare
Ved slutten av denne økten vil du ha en klar plan for å bygge og distribuere dine egne skalerbare, sky-native multi-agent applikasjoner som utnytter kraften fra moderne AI og skyinfrastruktur. Enten du er AI-ingeniør, skyarkitekt eller Python-utvikler, vil dette foredraget gi deg praktiske innsikter og dokumenterte mønstre for å bygge neste generasjon AI-drevne applikasjoner.
