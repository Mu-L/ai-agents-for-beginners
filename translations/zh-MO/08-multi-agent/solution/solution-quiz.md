# 知識檢測答案

## 問題 1

**答案：A2。** 當任務有明確的專家角色、不同的工具或權限，並且需要協調明確的交接時，多個代理人很有用。簡單的支援機械人通常可以保持為單一代理人。大量相同請求主要是規模問題，本身不需要多代理人設計。

## 問題 2

**答案：A1。** 當一套指令和工具能處理任務時，單一代理人通常較易測試、觀察和操作。多個工具的存取不代表必須有多個代理人。具有不同權限和審核要求的獨立角色可能需要多代理人設計。
