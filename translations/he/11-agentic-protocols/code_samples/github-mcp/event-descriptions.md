## שם האירוע: API רב-סוכנים עם LangGraph ו-Azure Cosmos DB

## תיאור

העלייה של אפליקציות AI רב-סוכנים משנה את הדרך בה אנו בונים מערכות אינטיליגנטיות - אך איך מגדירים אותן לסקלביליות וביצועים בעולם האמיתי? במפגש זה נצלול לעומק באפליקציה רב-סוכנים ברמת ייצור שנבנתה עם LangGraph לתיאום סוכנים, FastAPI לשכבת API, ו-Azure Cosmos DB כאבן שואבת לניהול מצב, אחסון וקטורי ונתונים טרנזקציוניים.

דרך סקירת קוד מפורטת, תראה כיצד לתכנן ולהוציא לפועל זרימת עבודה מונחית סוכן שמאחדת בשקיפות יצירה עם הגברה מבוססת אחזור (RAG), שמירת זיכרון, ומעברי מצב דינמיים. נסקור:

שיתוף פעולה של סוכנים עם LangGraph להסקת מסקנות מובנית
אחסון היסטוריית שיחות בזמן אמת באמצעות Azure Cosmos DB - אותו מסד נתונים שמפעיל את ההיסטוריה בשיחה ב-ChatGPT, אפליקציית הסוכן AI המהירה ביותר בהיסטוריה
חיפוש וקטורי לאחזור ידע עם תמיכה בשיבוצים טבעיים של Cosmos DB
יכולות async של FastAPI לשמור על תגובה וסקלביליות באינטראקציות
בסוף המפגש, תהיה לך תבנית ברורה לבניית והפצת אפליקציות רב-סוכנים סקיילביליות ונייטיב-ענן שמשלבות את כוחו של AI מודרני ותשתיות ענן. בין אם אתה מהנדס AI, ארכיטקט ענן או מפתח Python, ההרצאה הזו תספק תובנות מעשיות ותבניות מוכחות לבניית הדור הבא של אפליקציות מונחות AI.

## תיאור

jQuery פעם הפכה את פיתוח האינטרנט לקל יותר על ידי הפשטת סיבוכיות, ואפשרה למפתחים להתמקד בבנייה במקום להתמודד עם בעיות דפדפן. היום, פיתוח AI מתמודד עם אתגר דומה. תבניות חדשות מתהוות כל הזמן וההסתגלות להן יכולה להיות מתישה, במיוחד כשהכלים הסוכניים הופכים עוצמתיים ומורכבים יותר. מה אם תוכל להשתמש ביכולות AI מתקדמות לאוטומציה של משימות עם הפשטות וההיכרות של JavaScript? הכירו את GenAIScript - דרך לשלב AI בקלות בזרימת העבודה שלך, תוך טיפול בפרומפטים כמו קטעי קוד שניתן להשתמש בהם שוב. בהרצאה זו נחקור כיצד GenAIScript עושה את סוכני האוטומציה של AI אינטואיטיביים וקליטים כמו כתיבת JavaScript, ועוזר לייעל עבודות חוזרות دون צורך במומחיות עמוקה ב-AI.
