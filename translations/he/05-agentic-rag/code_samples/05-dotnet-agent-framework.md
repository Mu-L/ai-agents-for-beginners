# 🔍 RAG ארגוני עם Microsoft Foundry (.NET)

## 📋 יעדי למידה

פנקס רשימות זה מדגים כיצד לבנות מערכות ייצור-בתחום של Retrieval-Augmented Generation (RAG) באמצעות Microsoft Agent Framework ב-.NET עם Microsoft Foundry. תלמדו ליצור סוכנים מוכנים לייצור שיכולים לחפש במסמכים ולספק תגובות מדויקות המותאמות להיבטים הקונטקסטואליים, עם אבטחה וסקלאביליות ארגונית.

**יכולות RAG ארגוניות שתבנו:**
- 📚 **אינטליגנציה למסמכים**: עיבוד מסמכים מתקדם עם שירותי AI של Azure 
- 🔍 **חיפוש סמנטי**: חיפוש וקטורי מהיר עם תכונות ארגוניות
- 🛡️ **שילוב אבטחה**: בקרת גישה מבוססת תפקיד ותבניות הגנת נתונים
- 🏢 **ארכיטקטורה סקלאבילית**: מערכות RAG מוכנות ייצור עם ניטור

## 🏗️ ארכיטקטורה טכנית

### צנרת RAG ארגונית
```
Document Upload → Security Validation → Vector Processing → Index Creation
                      ↓                    ↓                  ↓
User Query → Authentication → Semantic Search → Context Ranking → AI Response
```

### רכיבי .NET מרכזיים
- **Azure.AI.Agents.Persistent**: ניהול סוכנים ארגוני עם שימור מצב
- **Azure.Identity**: אימות משולב לגישה מאובטחת לשירותי Azure
- **Microsoft.Agents.AI.AzureAI**: מימוש מסגרת סוכנים מותאמת ל-Azure
- **System.Linq.Async**: פעולות LINQ אסינכרוניות בעלות ביצועים גבוהים

**סביבת פיתוח:**
- ערכת פיתוח SDK .NET 9.0 ומעלה
- Visual Studio 2022 או VS Code עם תוסף C#

**חבילות NuGet נדרשות:**
