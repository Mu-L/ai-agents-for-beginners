# 🤝 एंटरप्राइज़ मल्टी-एजेंट वर्कफ़्लो सिस्टम (.NET)

## 📋 सीखने के उद्देश्य

यह नोटबुक दिखाता है कि कैसे .NET में Microsoft Agent Framework का उपयोग करके Azure OpenAI (Responses API) के साथ परिष्कृत एंटरप्राइज़-ग्रेड मल्टी-एजेंट सिस्टम बनाएँ। आप कई विशेषज्ञ एजेंटों को संरचित वर्कफ़्लो के माध्यम से एक साथ काम करने के लिए समन्वयित करना सीखेंगे, जो उत्पादन-तैयार समाधानों के लिए .NET की एंटरप्राइज़ विशेषताओं का लाभ उठाता है।

**आप जो एंटरप्राइज़ मल्टी-एजेंट क्षमताएँ बनाएंगे:**
- 👥 **एजेंट सहयोग**: संकलन-समय सत्यापन के साथ टाइप-सेफ एजेंट समन्वय
- 🔄 **वर्कफ़्लो समन्वयन**: .NET के async पैटर्न के साथ घोषितात्मक वर्कफ़्लो परिभाषा
- 🎭 **भूमिका विशेषज्ञता**: मजबूत टाइप एजेंट व्यक्तित्व और विशेषज्ञता डोमेन
- 🏢 **एंटरप्राइज़ एकीकरण**: निगरानी और त्रुटि प्रबंधन के साथ उत्पादन-तैयार पैटर्न

## कोड नमूना

इस पाठ के लिए पूरा कार्यशील कोड साथ में दिए गए C# फ़ाइल में उपलब्ध है: [`08-dotnet-agent-framework.cs`](../../../../08-multi-agent/code_samples/08-dotnet-agent-framework.cs)

नमूना चलाने के लिए:

```bash
# फाइल को निष्पादन योग्य बनाएं (Linux/macOS)
chmod +x 08-dotnet-agent-framework.cs

# नमूना चलाएं
./08-dotnet-agent-framework.cs
```

या .NET CLI का उपयोग करके:

```bash
dotnet run 08-dotnet-agent-framework.cs
```

## यह नमूना क्या दर्शाता है

यह मल्टी-एजेंट वर्कफ़्लो सिस्टम होटल यात्रा अनुशंसा सेवा बनाता है जिसमें दो विशेषज्ञ एजेंट होते हैं:

1. **फ्रंटडेस्क एजेंट**: एक यात्रा एजेंट जो गतिविधि और स्थान सुझाव प्रदान करता है
2. **कॉनसियर्ज एजेंट**: अनुशंसाओं की समीक्षा करता है ताकि वास्तविक, गैर-पर्यटक अनुभव सुनिश्चित हो सकें

ये एजेंट एक वर्कफ़्लो में साथ काम करते हैं जहाँ:
- फ्रंटडेस्क एजेंट प्रारंभिक यात्रा अनुरोध प्राप्त करता है
- कॉनसियर्ज एजेंट अनुशंसा की समीक्षा और संशोधित करता है
- वर्कफ़्लो रियल-टाइम में प्रतिक्रियाएँ स्ट्रीम करता है

## मुख्य अवधारणाएँ

### एजेंट समन्वय
नमूना Microsoft Agent Framework का उपयोग करके संकलन-समय सत्यापन के साथ टाइप-सेफ एजेंट समन्वय प्रदर्शित करता है।

### वर्कफ़्लो समन्वयन
कई एजेंटों को पाइपलाइन में जोड़ने के लिए .NET के async पैटर्न के साथ घोषितात्मक वर्कफ़्लो परिभाषा का उपयोग करता है।

### स्ट्रीमिंग प्रतिक्रियाएँ
async enumerable और इवेंट-प्रेरित आर्किटेक्चर का उपयोग करके एजेंट प्रतिक्रियाओं का रियल-टाइम स्ट्रीमिंग कार्यान्वित करता है।

### एंटरप्राइज़ एकीकरण
उत्पादन-तैयार पैटर्न को दिखाता है जिनमें शामिल हैं:
- पर्यावरण चर विन्यास
- सुरक्षित क्रेडेंशियल प्रबंधन
- त्रुटि प्रबंधन
- असमय इवेंट प्रोसेसिंग
