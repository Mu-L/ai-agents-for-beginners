विस्तार से कोड वॉकथ्रू के माध्यम से, आप सीखेंगे कि कैसे एक एजेंट-चालित वर्कफ़्लो डिज़ाइन और लागू करें जो Retrieval-Augmented Generation (RAG), मेमोरी पर्सिस्टेंस, और डायनामिक स्टेट ट्रांज़िशंस को सहजता से एकीकृत करता है। हम कवर करेंगे:

LangGraph के साथ एजेंट सहयोग संरचित तर्क के लिए
रीयल-टाइम चैट इतिहास भंडारण के लिए Azure Cosmos DB - वही डेटाबेस जो इतिहास में ChatGPT को शक्ति प्रदान करता है, जो अब तक का सबसे तेजी से बढ़ने वाला AI एजेंट एप्लिकेशन है
Cosmos DB के नेटिव एम्बेडिंग सपोर्ट के साथ ज्ञान पुनः प्राप्ति के लिए वेक्टर खोज
FastAPI की असिंक्रोनस क्षमताएं जो इंटरैक्शन को प्रतिक्रियाशील और स्केलेबल बनाती हैं
इस सत्र के अंत तक, आपके पास क्लाउड-नेटिव मल्टी-एजेंट एप्लिकेशन बनाने और तैनात करने के लिए एक स्पष्ट ब्लूप्रिंट होगा जो आधुनिक AI और क्लाउड इन्फ्रास्ट्रक्चर की शक्ति का उपयोग करता है। चाहे आप AI इंजीनियर हों, क्लाउड आर्किटेक्ट हों, या Python डेवलपर, यह वार्ता आपको व्यावहारिक अंतर्दृष्टि और सिद्ध पैटर्न प्रदान करेगी ताकि आप अगली पीढ़ी के AI-संचालित एप्लिकेशन बना सकें।

## कार्यक्रम का नाम: प्रांप्टिंग नई स्क्रिप्टिंग है: GenAIScript से मिलें

## विवरण

jQuery ने कभी वेब विकास को आसान बनाया था, जटिलताओं को हटाकर डेवलपर्स को निर्माण पर ध्यान केंद्रित करने दिया, बजाय ब्राउज़र की समस्याओं से जूझने के। आज, AI विकास एक समान चुनौती का सामना कर रहा है। नए पैटर्न निरंतर उभरते रहते हैं और उनके साथ बने रहना भारी हो सकता है, खासकर जब AI उपकरण — विशेष रूप से एजेंटिक — अधिक शक्तिशाली और जटिल होते जा रहे हैं। क्या होगा अगर आप अत्याधुनिक AI क्षमताओं का उपयोग सरल, परिचित JavaScript अमूर्तताओं के साथ कार्यों को स्वचालित करने के लिए कर सकते हैं? पेश है GenAIScript — एक तरीका जो AI को आपके वर्कफ़्लो में आसानी से एकीकृत करता है, प्रांप्ट को पुन: उपयोगी कोड स्निपेट की तरह व्यवहार करता है। इस वार्ता में, हम देखेंगे कि कैसे GenAIScript AI ऑटोमेशन एजेंट्स को उतना सहज बनाता है जितना JavaScript लिखना, जिससे आप गहरे AI ज्ञान के बिना भी पुनरावृत्ति कार्यों को सरल बना सकते हैं।

## कार्यक्रम का नाम: Microsoft Foundry Agent Service के साथ अपना कोड-प्रथम ऐप बनाएं (.NET)

## विवरण

Microsoft Foundry Agent Service सेवा और SDK का एक सहज मिश्रण है जो मजबूत AI-चालित समाधानों के विकास को सरल बनाता है। इस सत्र में, आप सीखेंगे कि कैसे Azure और C# के साथ अपना खुद का कोड-प्रथम AI एजेंट बनाना है जो सवालों के जवाब दे सकता है, डेटा विश्लेषण कर सकता है, और बाहरी डेटा स्रोतों को एकीकृत कर सकता है। आप कई एजेंटों के साथ काम करने जैसी जटिल आर्किटेक्चर का भी अन्वेषण करेंगे।

## URL
<https://developer.microsoft.com/en-us/reactor/events/25370/>

## कार्यक्रम का नाम: AI एजेंट + .NET Aspire

## विवरण

इस सत्र में हम एजेंट्स के आसपास .NET प्लेटफ़ॉर्म पर सबसे रोमांचक विकास साझा करेंगे। .NET की वर्तमान स्थिति का पता लगाएं, जिसमें इसके नए फीचर्स और सुधार शामिल हैं। शक्तिशाली AI एजेंट क्षमताओं की खोज करें। और हम एजेंट्स और .NET Aspire के साथ कुछ लाइव कोडिंग भी करेंगे।

## URL
<https://developer.microsoft.com/en-us/reactor/events/25332/>

## कार्यक्रम का नाम: Semantic Kernel के साथ C# में मल्टी-एजेंट AI एप्लिकेशन बनाना जो Azure Cosmos द्वारा संचालित है

## विवरण

हम आपको एक मल्टी-एजेंट एप्लिकेशन के माध्यम से चलाएंगे जो C# में Semantic Kernel फ्रेमवर्क के ऊपर बनाया गया है। आप एजेंटिक एप्लिकेशनों के पीछे के विचारों को समझेंगे, कार्यान्वयन विवरणों और सूक्ष्मताओं को जानेंगे, और सीखेंगे कि विभिन्न उपयोग मामलों के लिए Azure Cosmos DB को डेटाबेस के रूप में कैसे एकीकृत करें।

## URL
<https://developer.microsoft.com/en-us/reactor/events/25455/>
