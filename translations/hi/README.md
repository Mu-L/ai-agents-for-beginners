> **स्थानीय रूप से क्लोन करना पसंद करते हैं?**
>
> इस रिपॉजिटरी में 50+ भाषा अनुवाद शामिल हैं जो डाउनलोड आकार को काफी बढ़ाते हैं। अनुवाद के बिना क्लोन करने के लिए, sparse checkout का उपयोग करें:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> यह आपको कोर्स पूरा करने के लिए सभी आवश्यक सामग्री देता है साथ ही डाउनलोड भी तेजी से होता है।

## 🌱 शुरू करना

इस कोर्स में AI एजेंट बनाने के मूल तत्वों को कवर किया गया है। प्रत्येक पाठ अपनी एक विषयवस्तु कवर करता है इसलिए जहाँ चाहें वहाँ से शुरू करें!

इस कोर्स के लिए बहुभाषी समर्थन उपलब्ध है। हमारी [उपलब्ध भाषाएँ यहाँ देखें](#-multi-language-support)।

यदि यह आपका जनरेटिव AI मॉडल के साथ निर्माण करने का पहला अनुभव है, तो हमारा [Generative AI For Beginners](https://aka.ms/genai-beginners) कोर्स देखें, जिसमें GenAI से निर्माण पर 21 पाठ शामिल हैं।

इस रिपॉजिटरी को [स्टार (🌟) करना](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) और [फोर्क करना](https://github.com/microsoft/ai-agents-for-beginners/fork) न भूलें ताकि आप कोड चला सकें।

### अन्य शिक्षार्थियों से मिलें, अपने सवालों का जवाब पाएं

यदि आप अटक जाएं या AI एजेंट बनाने के संबंध में कोई प्रश्न हो, तो हमारे समर्पित डिस्कॉर्ड चैनल में शामिल हों [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) पर।

### आपकी क्या आवश्यकता है

इस कोर्स के प्रत्येक पाठ में कोड उदाहरण शामिल हैं, जो code_samples फोल्डर में पाए जा सकते हैं। आप अपना खुद का कॉपी बनाने के लिए [इस रिपॉजिटरी को फोर्क कर सकते हैं](https://github.com/microsoft/ai-agents-for-beginners/fork)।  

इन अभ्यासों के कोड उदाहरण Microsoft Agent Framework के साथ Microsoft Foundry Agent Service V2 का उपयोग करते हैं:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure खाता आवश्यक

यह कोर्स Microsoft के निम्नलिखित AI एजेंट फ्रेमवर्क और सेवाओं का उपयोग करता है:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

कुछ कोड नमूने वैकल्पिक OpenAI-संगत प्रदाताओं का भी समर्थन करते हैं जैसे कि [MiniMax](https://platform.minimaxi.com/), जो बड़े संदर्भ मॉडल (204K टोकन तक) प्रदान करता है। कॉन्फ़िगरेशन विवरण के लिए [Course Setup](./00-course-setup/README.md) देखें।

इस कोर्स के कोड को चलाने के बारे में अधिक जानकारी के लिए [Course Setup](./00-course-setup/README.md) पर जाएं।
