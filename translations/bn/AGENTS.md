# AGENTS.md

## প্রকল্পের ওভারভিউ

এই রিপোজিটরিটি "শিখুন AI এজেন্টস ফর বিগিনারস" - একটি ব্যাপক শিক্ষামূলক কোর্স যা AI এজেন্ট তৈরি করার জন্য প্রয়োজনীয় সবকিছু শেখায়। কোর্সটি ১৮টি পাঠ নিয়ে গঠিত (সংখ্যায় ০০-১৮), যা মৌলিক বিষয়, ডিজাইন প্যাটার্ন, ফ্রেমওয়ার্ক, প্রোডাকশন ডেপ্লয়মেন্ট, স্থানীয়/ডিভাইস-ভিত্তিক এজেন্টস, এবং AI এজেন্টের সুরক্ষা বিষয়গুলো অন্তর্ভুক্ত করে।

**মূল প্রযুক্তিসমূহ:**
- পাইথন ৩.১২+
- ইন্টারেক্টিভ শিক্ষার জন্য জুপিটার নোটবুক
- AI ফ্রেমওয়ার্কস: মাইক্রোসফট এজেন্ট ফ্রেমওয়ার্ক (MAF)
- আজুর AI সেবা: মাইক্রোসফট ফাউন্ড্রি, মাইক্রোসফট ফাউন্ড্রি এজেন্ট সেবা V2

**স্থাপত্য:**
- পাঠভিত্তিক কাঠামো (০০-১৫+ ডিরেক্টরি)
- প্রতিটি পাঠে থাকে: README ডকুমেন্টেশন, কোড স্যাম্পল (জুপিটার নোটবুক), এবং ইমেজ
- স্বয়ংক্রিয় অনুবাদ ব্যবস্থার মাধ্যমে বহু-ভাষার সমর্থন
- মাইক্রোসফট এজেন্ট ফ্রেমওয়ার্ক ব্যবহার করে প্রতিটি পাঠে একটি পাইথন নোটবুক

## সেটআপ কমান্ডসমূহ

### প্রয়োজনীয়তাসমূহ
- পাইথন ৩.১২ বা তার উপরে
- আজুর সাবস্ক্রিপশন (মাইক্রোসফট ফাউন্ডরির জন্য)
- আজুর CLI ইনস্টল ও অথেনটিকেট করা (`az login`)

### প্রাথমিক সেটআপ

১. **রিপোজিটরি ক্লোন বা ফর্ক করুন:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # অথবা
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

২. **পাইথন ভার্চুয়াল পরিবেশ তৈরি ও সক্রিয় করুন:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # উইন্ডোজে: venv\Scripts\activate
   ```

৩. **ডিপেনডেন্সি ইনস্টল করুন:**
   ```bash
   pip install -r requirements.txt
   ```

৪. **পরিবেশ ভেরিয়েবল সেট করুন:**
   ```bash
   cp .env.example .env
   # আপনার API কী এবং এন্ডপয়েন্ট সহ .env সম্পাদনা করুন
   ```

### প্রয়োজনীয় পরিবেশ ভেরিয়েবলসমূহ

**মাইক্রোসফট ফাউন্ডরির** জন্য (প্রয়োজনীয়):
- `AZURE_AI_PROJECT_ENDPOINT` - মাইক্রোসফট ফাউন্ডরি প্রকল্প এন্ডপয়েন্ট
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - মডেল ডেপ্লয়মেন্টের নাম (যেমন, gpt-4.1-mini)

**আজুর AI সার্চের** জন্য (পাঠ ০৫ - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - আজুর AI সার্চের এন্ডপয়েন্ট
- `AZURE_SEARCH_API_KEY` - আজুর AI সার্চ API কী

অথেনটিকেশন: নোটবুক চালানোর আগে `az login` চালান (ব্যবহার হয় `AzureCliCredential`)।

   - নিশ্চিত করুন পাইথন ৩.১২+ ব্যবহার হচ্ছে
   - কিছু প্যাকেজ পুরোনো ভার্সন সমর্থন নাও করতে পারে
   - `python3 -m venv` দিয়ে স্পষ্টভাবে পাইথন ভার্সন উল্লেখ করুন

২. **পরিবেশ ভেরিয়েবল:**
   - সবসময় `.env.example` থেকে `.env` তৈরি করুন
   - `.env` ফাইল কমিট করবেন না (এটি `.gitignore` তে আছে)
   - কীলেস Entra ID অথেনটিকেশনের জন্য `az login` ব্যবহার করে সাইন ইন করুন

৩. **প্যাকেজ সংঘাত:**
   - নতুন ভার্চুয়াল পরিবেশ ব্যবহার করুন
   - পৃথক প্যাকেজের পরিবর্তে `requirements.txt` থেকে ইনস্টল করুন
   - কিছু নোটবুকে অতিরিক্ত প্যাকেজের প্রয়োজন হতে পারে যা তাদের মার্কডাউন সেলে উল্লেখ আছে

৪. **আজুর সেবা:**
   - আজুর AI সেবার জন্য সক্রিয় সাবস্ক্রিপশন প্রয়োজন
   - কিছু ফিচার নির্দিষ্ট অঞ্চলে সীমিত
   - নিশ্চিত করুন আপনার আজুর ওপেনএআই মডেল ডেপ্লয়মেন্ট Responses API সমর্থন করে
