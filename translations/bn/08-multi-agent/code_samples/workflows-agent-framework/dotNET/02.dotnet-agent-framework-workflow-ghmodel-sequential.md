# ⏩ Azure OpenAI (Responses API) এর সাথে ক্রমান্বয়ে এজেন্ট ওয়ার্কফ্লো (.NET)

## 📋 উন্নত ক্রমান্বয় প্রক্রিয়াকরণ টিউটোরিয়াল

এই নোটবুকটি Microsoft Agent Framework for .NET এবং Azure OpenAI (Responses API) ব্যবহার করে **ক্রমান্বয় ওয়ার্কফ্লো প্যাটার্ন** প্রদর্শন করে। আপনি শিখবেন কীভাবে একটি নির্দিষ্ট ক্রমে এজেন্টগুলি কার্যকর করে বিশেষজ্ঞ, ধাপে ধাপে প্রক্রিয়াকরণ পাইপলাইন তৈরি করতে হয়, যেখানে প্রতিটি ধাপ আগে ধাপের ফলাফলের উপর ভিত্তি করে তৈরি হয়।

## 🎯 শেখার লক্ষ্য

### 🔄 **ক্রমান্বয় প্রক্রিয়াকরণ আর্কিটেকচার**
- **রৈখিক ওয়ার্কফ্লো ডিজাইন**: স্পষ্ট নির্ভরতা সহ ধাপে ধাপে প্রক্রিয়াকরণ পাইপলাইন তৈরি করা
- **স্টেট ম্যানেজমেন্ট**: ক্রমান্বয় ওয়ার্কফ্লো পর্যায় জুড়ে প্রসঙ্গ এবং ডেটা প্রবাহ বজায় রাখা
- **Azure OpenAI (Responses API)**: বহু-পর্যায়ের .NET ওয়ার্কফ্লোতে Azure OpenAI মডেল ব্যবহার করা
- **এন্টারপ্রাইজ পাইপলাইন প্যাটার্নস**: প্রোডাকশন-রেডি ক্রমান্বয় প্রক্রিয়াকরণ সিস্টেম নির্মাণ

### 🏗️ **উন্নত ক্রমান্বয় প্যাটার্নস**
- **স্টেজ-গেট প্রসেসিং**: ওয়ার্কফ্লো পর্যায়গুলির মধ্যে যাচাই চেকপয়েন্ট প্রয়োগ করা
- **প্রসঙ্গ সংরক্ষণ**: সকল পর্যায় জুড়ে স্টেট ও সংগৃহীত জ্ঞান রক্ষা করা
- **ত্রুটি বিস্তার**: ক্রমান্বয় প্রক্রিয়াকরণ চেইনে বিফলতা সুষ্ঠুভাবে পরিচালনা করা
- **পারফরম্যান্স অপ্টিমাইজেশন**: নুন্যতম ওভারহেড সহ দক্ষ ক্রমান্বয় কার্যকরী

### 🏢 **এন্টারপ্রাইজ ক্রমান্বয় অ্যাপ্লিকেশন**
- **ডকুমেন্ট প্রক্রিয়াকরণ পাইপলাইন**: বহু-পর্যায়ের ডকুমেন্ট বিশ্লেষণ, রূপান্তর এবং যাচাই
- **কোয়ালিটি অ্যাশিওরেন্স ওয়ার্কফ্লো**: ক্রমান্বয় পর্যালোচনা, যাচাই এবং অনুমোদন প্রক্রিয়া
- **কনটেন্ট প্রোডাকশন পাইপলাইন**: গবেষণা → লেখা → সম্পাদনা → পর্যালোচনা → প্রকাশনা
- **বিজনেস প্রসেস অটোমেশন**: স্পষ্ট পর্যায় নির্ভরতা সহ বহু-ধাপের ব্যবসায়িক ওয়ার্কফ্লো

// Load environment variables securely

- **Azure OpenAI ক্লায়েন্ট**: সকল ওয়ার্কফ্লো ধাপে AI মডেলে সঙ্গতিপূর্ণ অ্যাক্সেস

- **রিয়েল এস্টেট**: সম্পত্তি বিশ্লেষণ → মূল্যায়ন → তালিকা তৈরি
- **ইনস্যুরেন্স**: দাবি বিশ্লেষণ → মূল্যায়ন → কোট তৈরি
- **কনটেন্ট ক্রিয়েশন**: গবেষণা → লেখা → সম্পাদনা → প্রকাশনা

### 🔍 স্টেট ফ্লো বোঝা

ক্রমান্বয় এজেন্টের প্রত্যেকে নেয়:
- **মূল ইনপুট**: প্রাথমিক ব্যবহারকারী বার্তা (ছবি + পাঠ্য)
- **পূর্বের এজেন্ট আউটপুট**: কথোপকথনের ইতিহাসে সকল পূর্বের এজেন্ট প্রতিক্রিয়া
- **সংগৃহীত প্রসঙ্গ**: ওয়ার্কফ্লো জুড়ে সম্পূর্ণ স্টেট সংরক্ষণ

এর ফলে জটিল বহু-পর্যায়ের প্রক্রিয়াকরণ সম্ভব হয় যেখানে প্রতিটি এজেন্ট সকল পূর্বের ধাপের ব্যাপক প্রসঙ্গের উপর ভিত্তি করে কাজ করে।
