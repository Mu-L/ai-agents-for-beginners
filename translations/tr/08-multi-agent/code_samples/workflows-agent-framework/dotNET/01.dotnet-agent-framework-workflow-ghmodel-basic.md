# 🔄 Azure OpenAI ile Temel Ajan İş Akışları (Yanıtlar API) (.NET)

## 📋 İş Akışı Orkestrasyon Eğitimi

Bu not defteri, Microsoft Agent Framework for .NET ve Azure OpenAI (Yanıtlar API) kullanarak nasıl sofistike **ajan iş akışları** oluşturulacağını göstermektedir. Yapay zeka ajanlarının yapılandırılmış orkestrasyon desenleri aracılığıyla iş birliği yaparak karmaşık görevleri gerçekleştirdiği çok adımlı iş süreçleri oluşturmayı öğreneceksiniz.

## 🎯 Öğrenme Hedefleri

### 🏗️ **İş Akışı Mimarisinin Temelleri**
- **İş Akışı Oluşturucu**: Karmaşık çok adımlı yapay zeka süreçlerini tasarlama ve orkestre etme
- **Ajan Koordinasyonu**: İş akışları içinde birden fazla uzmanlaşmış ajanı koordine etme
- **Azure OpenAI (Yanıtlar API)**: İş akışlarında Azure OpenAI Yanıtlar API’sını kullanma
- **Görsel İş Akışı Tasarımı**: Daha iyi anlam için iş akışı yapıları oluşturma ve görselleştirme

### 🔄 **Süreç Orkestrasyon Desenleri**
- **Sıralı İşleme**: Birden çok ajan görevini mantıksal sırayla zincirleme
- **Durum Yönetimi**: İş akışı aşamalarında bağlam ve veri akışını koruma
- **Hata Yönetimi**: Sağlam hata kurtarma ve iş akışı dayanıklılığı uygulama
- **Performans Optimizasyonu**: Kurumsal ölçekli operasyonlar için verimli iş akışları tasarlama

### 🏢 **Kurumsal İş Akışı Uygulamaları**
- **İş Süreci Otomasyonu**: Karmaşık örgütsel iş akışlarını otomatikleştirme
- **İçerik Üretim Hattı**: İnceleme ve onay aşamaları ile editoryal iş akışları
- **Müşteri Hizmetleri Otomasyonu**: Çok adımlı müşteri talebi çözümü
- **Veri İşleme İş Akışları**: Yapay zeka destekli dönüşümlü ETL iş akışları

## ⚙️ Ön Koşullar & Kurulum

### 📦 **Gerekli NuGet Paketleri**

Bu iş akışı gösterimi birkaç önemli .NET paketi kullanmaktadır:

```xml
<!-- Core AI Framework -->
<PackageReference Include="Microsoft.Extensions.AI" Version="10.*" />

<!-- Azure OpenAI (Responses API) -->
<PackageReference Include="Azure.AI.OpenAI" Version="2.1.0" />
<PackageReference Include="Azure.Identity" Version="1.15.0" />

<!-- Agent Framework (Local Development) -->
<!-- Microsoft.Agents.AI.dll - Core agent abstractions -->
<!-- Microsoft.Agents.AI.OpenAI.dll - Azure OpenAI (Responses API) integration -->

<!-- Configuration and Environment -->
<PackageReference Include="DotNetEnv" Version="3.1.1" />
```

### 🔑 **Azure OpenAI Yapılandırması**

**Ortam Ayarı (.env dosyası):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-4.1-mini
```

**Azure OpenAI Erişimi:**
1. Azure portalında bir Azure OpenAI kaynağı oluşturun
2. Bir model dağıtın (örneğin, `gpt-4.1-mini`) ve dağıtım adını not edin
3. `az login` ile oturum açın ve yukarıda gösterildiği gibi ortam değişkenlerini yapılandırın

### 🏗️ **İş Akışı Mimarisi Genel Bakış**

```mermaid
graph TD
    A[İş Akışı Oluşturucu] --> B[Ajan Kaydı]
    B --> C[İş Akışı Yürütme Motoru]
    C --> D[Agent 1: İçerik Üreticisi]
    C --> E[Agent 2: İçerik İnceleyicisi] 
    D --> F[İş Akışı Sonuçları]
    E --> F
    G[Azure OpenAI (Yanıtlar API)] --> D
    G --> E
```

**Ana Bileşenler:**
- **WorkflowBuilder**: İş akışlarını tasarlamak için ana orkestrasyon motoru
- **AIAgent**: Belirli yeteneklere sahip bireysel uzman ajanlar
- **Azure OpenAI Client**: Azure OpenAI Yanıtlar API entegrasyonu
- **Yürütme Bağlamı**: İş akışı aşamaları arasında durum ve veri akışını yönetir

## 💻 Kodu Çalıştırma

Tam uygulama `01.dotnet-agent-framework-workflow-ghmodel-basic.cs` dosyasında mevcuttur. Bu dosya şunları gösterir:

1. **Ortam Yapılandırması** - Azure OpenAI yapılandırmasının `.env` dosyasından yüklenmesi
2. **Azure OpenAI İstemci Kurulumu** - Azure OpenAI Yanıtlar API’sını kullanmak için istemciyi yapılandırma
3. **Ajan Oluşturma** - Uzmanlaşmış ajanların tanımlanması (Resepsiyon ve Konsiyerj)
4. **İş Akışı Oluşturucu** - Sıralı işleme ile çok ajanlı iş akışının oluşturulması
5. **İş Akışı Yürütme** - Akışla birlikte sonuçların akarak yürütülmesi

### 🚀 Örneği Çalıştırma

```bash
# Betiği çalıştırılabilir yap (Unix/Linux/macOS)
chmod +x 01.dotnet-agent-framework-workflow-ghmodel-basic.cs

# İş akışını çalıştır
./01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```

Veya Windows üzerinde:
```powershell
dotnet run 01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```

### 📝 Beklenen Çıktı

İş akışı şunları yapacaktır:
