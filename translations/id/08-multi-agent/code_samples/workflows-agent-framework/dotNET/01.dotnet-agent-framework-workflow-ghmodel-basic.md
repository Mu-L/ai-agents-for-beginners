# 🔄 Alur Kerja Agen Dasar dengan Azure OpenAI (Responses API) (.NET)

## 📋 Tutorial Orkestrasi Alur Kerja

Notebook ini menunjukkan cara membangun **alur kerja agen** yang canggih menggunakan Microsoft Agent Framework untuk .NET dan Azure OpenAI (Responses API). Anda akan belajar membuat proses bisnis multi-langkah di mana agen AI berkolaborasi untuk menyelesaikan tugas kompleks melalui pola orkestrasi yang terstruktur.

## 🎯 Tujuan Pembelajaran

### 🏗️ **Dasar-Dasar Arsitektur Alur Kerja**
- **Workflow Builder**: Merancang dan mengorkestrasi proses AI multi-langkah yang kompleks
- **Agent Coordination**: Mengkoordinasikan beberapa agen khusus dalam alur kerja
- **Azure OpenAI (Responses API)**: Memanfaatkan Azure OpenAI Responses API dalam alur kerja
- **Visual Workflow Design**: Membuat dan memvisualisasikan struktur alur kerja untuk pemahaman lebih baik

### 🔄 **Pola Orkestrasi Proses**
- **Sequential Processing**: Menghubungkan beberapa tugas agen secara berurutan
- **State Management**: Mempertahankan konteks dan aliran data di seluruh tahap alur kerja
- **Error Handling**: Menerapkan pemulihan kesalahan yang tangguh dan ketahanan alur kerja
- **Performance Optimization**: Merancang alur kerja yang efisien untuk operasi skala perusahaan

### 🏢 **Aplikasi Alur Kerja Perusahaan**
- **Business Process Automation**: Mengotomatisasi alur kerja organisasi yang kompleks
- **Content Production Pipeline**: Alur kerja editorial dengan tahap tinjau dan persetujuan
- **Customer Service Automation**: Penyelesaian permintaan pelanggan multi-langkah
- **Data Processing Workflows**: Alur kerja ETL dengan transformasi berbasis AI

### 🔑 **Konfigurasi Azure OpenAI**

**Pengaturan Lingkungan (file .env):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-4.1-mini
```

**Akses Azure OpenAI:**
1. Buat sumber daya Azure OpenAI di portal Azure
2. Deploy model (misalnya, `gpt-4.1-mini`) dan catat nama deployment
3. Masuk dengan `az login` dan konfigurasi variabel lingkungan seperti di atas

## 💻 Menjalankan Kode

Implementasi lengkap tersedia di `01.dotnet-agent-framework-workflow-ghmodel-basic.cs`. File ini menunjukkan:

1. **Konfigurasi Lingkungan** - Memuat konfigurasi Azure OpenAI dari file `.env`
2. **Pengaturan Client Azure OpenAI** - Mengonfigurasi client untuk menggunakan Azure OpenAI Responses API
3. **Pembuatan Agen** - Mendefinisikan agen khusus (Front Desk dan Concierge)
4. **Workflow Builder** - Membuat alur kerja multi-agen dengan pemrosesan berurutan
5. **Eksekusi Alur Kerja** - Menjalankan alur kerja dengan hasil streaming

### 🚀 Menjalankan Contoh

```bash
# Jadikan skrip dapat dieksekusi (Unix/Linux/macOS)
chmod +x 01.dotnet-agent-framework-workflow-ghmodel-basic.cs

# Jalankan alur kerja
./01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```

Atau di Windows:
```powershell
dotnet run 01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```

### 📝 Output yang Diharapkan

Alur kerja akan:
1. Menerima permintaan tujuan perjalanan Anda ("Saya ingin pergi ke Paris")
2. Agen Front Desk memberikan rekomendasi awal
3. Agen Concierge meninjau dan menyempurnakan rekomendasi
4. Output akhir menampilkan alur percakapan lengkap
