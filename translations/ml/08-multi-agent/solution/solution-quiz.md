# വിജ്ഞാന പരിശോധന ഉത്തരം

## ചോദ്യം 1

**ഉത്തരം: A2.** ഒരു പ്രവൃത്തി വിഭിന്ന വിദഗ്ധ പങ്കുകളും, വേർതിരിച്ചിട്ടുള്ള ഉപകരണങ്ങളും അവകാശങ്ങളും, സുതാര്യമായ കൈമാറ്റങ്ങളും കോർഡിനേറ്റ് ചെയ്യേണ്ടവയും ഉള്ളപ്പോൾ ഒട്ടനവധി ഏജന്റുകൾ ഉപകാരപ്രദമാണ്. ഒരു ലളിതമായ പിന്തുണ ബോട്ട് സാധാരണയായി ഒരു ഏജന്റായിരിക്കാം. ഒരുപോലെ ഉള്ള അഭ്യർത്ഥനയുടെ വലിയ തോതിൽ പ്രധാനമായും സ്കെയിലിംഗ് പ്രശ്നമാണ്, അതിന് തന്നെ ഒട്ടനവധി ഏജന്റ് ഡിസൈൻ ആവശ്യമില്ല.

## ചോദ്യം 2

**ഉത്തരം: A1.** ഒരു ഏജന്റിനെയാണ് സാധാരണയായി പരീക്ഷിക്കുക, നിരീക്ഷിക്കുക, പ്രവർത്തിപ്പിക്കുക എളുപ്പം ആയിരിക്കുക, ഒരേയൊരു നിർദ്ദേശങ്ങളും ഉപകരണങ്ങളും പ്രവൃത്തി കൈകാര്യം ചെയ്യുമ്പോൾ. ഒരുപിടി ഉപകരണങ്ങൾ ലഭ്യമാകുന്നത് സ്വാഭാവികമായി പല ഏജന്റുകൾ ആവശ്യപ്പെടുന്നില്ല. വേറിട്ട പങ്കുകൾ വ്യത്യസ്ത അവകാശങ്ങളും ഓഡിറ്റ് ആവശ്യകതകളും ഉള്ളപ്പോൾ ഒട്ടന അഞ്ച് ഏജന്റ് ഡിസൈൻ ന്യായീകരിക്കാം.
