# ⚡ ആസ്യൂര്‍ ഒപണ്‍എഐ (Responses API) ഉപയോഗിച്ച് സഹപ്രവർത്തന ഏജന്റ് പ്രവൃത്തി പ്രവാഹങ്ങൾ (.NET)

## 📋 ഉയർന്ന പ്രകടന സമാന്തര പ്രോസസിംഗ് ടുട്ടോറിയൽ

ഈ നോട്ട് ബുക്ക് മൈക്രോസോഫ്റ്റ് ഏജന്റ് ഫ്രെയിംവർക്ക് ഫോർ .NET ഉം ആസ്യൂർ ഒപണ്‍എഐ (Responses API) ഉം ഉപയോഗിക്കുന്ന **സഹപ്രവൃത്തി പ്രവൃത്തി പ്രവാഹ മാതൃകകൾ** കാണിക്കുന്നു. നിങ്ങൾ ഉയർന്ന പ്രകടനമുള്ള, സമാന്തര പ്രോസസിംഗ് പ്രവൃത്തി പ്രവാഹങ്ങൾ എങ്ങനെ ഉണ്ടാക്കാമെന്ന് പഠിക്കും, സമയാനുരൂപ AI ഏജന്റുമാര്‍ ഒരിക്കലും പ്രവർത്തിപ്പിക്കുന്നതിനാൽ പാരംവാഹിക്ത്വം പരമാവധി വർദ്ധിപ്പിക്കുകയും കോർഡിനേഷൻ നിലനിര്‍ത്തുകയും ഡാറ്റ ഒത്തുചേരല്‍ ഉറപ്പുവരുത്തുകയും ചെയ്യുന്നു.

## 💻 കോഡ് പ്രവർത്തിപ്പിക്കൽ

പൂര്‍ണമായ നടപ്പാക്കല്‍ `03.dotnet-agent-framework-workflow-ghmodel-concurrent.cs` ഫയലിൽ ലഭ്യമാണ്. ഈ ഫയൽ യാത്രാ പദ്ധതിക്കായുള്ള **ഫാൻ-ഔട്ട്/ഫാൻ-ഇൻ സഹപ്രവൃത്തി പ്രവൃത്തി പ്രവാഹം** കാണിക്കുന്നു:

### 🏗️ **പ്രവൃത്തി പ്രവാഹ ആർക്കിടെക്ചർ**

```
User Request → ConcurrentStartExecutor → [Researcher Agent || Planner Agent] → ConcurrentAggregationExecutor → Final Output
```

**പ്രധാനംഘടകങ്ങൾ:**

1. **ConcurrentStartExecutor**: ഉപയോക്തൃ അഭ്യർത്ഥന എല്ലാ ഏജന്റുകൾക്കും ഒരുപോലിതൻ പ്രചരിപ്പിക്കുന്നു
2. **Researcher Agent**: സ്ഥാനങ്ങൾ, ആകർഷണങ്ങൾ എന്നിവ സമാന്തരമായി വിശകലനം ചെയ്യുന്നു
3. **Planner Agent**: വിശദമായ യാത്രാ പദ്ധതികൾ സമാന്തരമായി സൃഷ്ടിക്കുന്നു
4. **ConcurrentAggregationExecutor**: ഇരുപാടും ഏജന്റുമാരും ലഭിച്ച ഫലങ്ങൾ ശേഖരിക്കുകയും ലയിപ്പിക്കുകയും ചെയ്യുന്നു
