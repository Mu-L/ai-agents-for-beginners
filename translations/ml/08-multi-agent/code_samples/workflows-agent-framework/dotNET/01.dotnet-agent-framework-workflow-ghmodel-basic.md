# 🔄 ആസ്യൂർ ഓപ്പൺഎഐ (Responses API) ഉപയോഗിച്ചുള്ള അടിസ്ഥാന ഏജന്റ് വർക്ഫ്ലോകൾ (.NET)

## 📋 വർക്ഫ്ലോ ഓർക്കസ്ട്രേഷൻ ട്യൂട്ടോറിയൽ

മൈക്രോസോഫ്റ്റ് ഏജന്റ് ഫ്രെയിംവർക്ക് ഫോർ .NET, ആസ്യൂർ ഓപ്പൺഎഐ (Responses API) എന്നിവ ഉപയോഗിച്ച് എങ്ങനെ കൗശലമായ **ഏജന്റ് വർക്ഫ്ലോകൾ** നിർമ്മിക്കാമെന്ന് ഈ നോട്ട്ബുക്ക് കാണിക്കുന്നു. AI ഏജന്റുകൾ ഘടിതമായ ഓർക്കസ്ട്രേഷൻ മാതൃകകളിലൂടെ സഹകരിച്ച് സങ്കീർണമായ തങ്ങൾ പൂർത്തിയാക്കുന്ന ബഹു-പടി ബിസിനസ് പ്രക്രിയകൾ സൃഷ്ടിക്കാൻ നിങ്ങൾക്ക് പഠിക്കാം.

## 🎯 പഠന ലക്ഷ്യങ്ങൾ

### 🏗️ **വർക്ഫ്ലോ ആർക്കിടെക്ചർ അടിസ്ഥാനങ്ങൾ**
- **Workflow Builder**: സങ്കീർണമായ ബഹു-പടി AI പ്രക്രിയകൾ രൂപകൽപ്പന ചെയ്‌തു ഓർക്കസ്ട്രേറ്റ് ചെയ്യുക
- **Agent Coordination**: വർക്ഫ്ലോകളിൽ വിവിധ പ്രത്യേകജ്ഞതയുള്ള ഏജന്റുകൾ ഉള്പ്പെടെ ഏകോപിപ്പിക്കുക
- **Azure OpenAI (Responses API)**: വർക്ഫ്ലോകളിൽ ആസ്യൂർ ഓപ്പൺഎഐ Responses API ഉപയോഗിക്കുക
- **Visual Workflow Design**: നല്ല നിഗമനത്തിന് വർക്ഫ്ലോ ഘടനകൾ സൃഷ്ടിച്ചു ദൃശ്യവത്ക്കരിക്കുക

### 🔄 **പ്രോസസ് ഓർക്കസ്ട്രേഷൻ മാതൃകകൾ**
- **Sequential Processing**: നിരവധി ഏജന്റ് പ്രവർത്തനങ്ങളെ താർക്കികമായ ക്രമത്തിൽ ച chaîne ചെയ്യുക
- **State Management**: വർക്ഫ്ലോ ഘട്ടങ്ങളിലുടനീളം സന്ധർഭവും ഡാറ്റ ഒഴുക്കും കൈകാര്യം ചെയ്യുക
- **Error Handling**: ദൃഢമായ പിഴവ് പുനരുദ്ധാരണം, വർക്ഫ്ലോ പ്രതിരോധ ശേഷി നടപ്പിൽ കൊണ്ട് വരുക
- **Performance Optimization**: സംരംഭം തരം പ്രവർത്തനങ്ങൾക്കായി കാര്യക്ഷമമായ വർക്ഫ്ലോകൾ രൂപകൽപ്പന ചെയ്യുക

### 🏢 **സംരംഭ വർക്ഫ്ലോ ആപ്പ്ലിക്കേഷനുകൾ**
- **Business Process Automation**: സങ്കീർണമായ സ്ഥാപന വർക്ഫ്ലോകൾ ഓട്ടോമേറ്റുചെയ്യുക
- **Content Production Pipeline**: അവലോകനവും അംഗീകാര ഘട്ടങ്ങളും ഉൾക്കൊള്ളുന്ന എഡിറ്റോറിയൽ വർക്ഫ്ലോകൾ
- **Customer Service Automation**: ബഹു-പടി ഉപഭോ​ഗ്തൃ അന്വേഷണ പരിഹാരം
- **Data Processing Workflows**: AI കഴിവുള്ള ട്‌റാൻസ്ഫർമേഷനോടെ ETL വർക്ഫ്ലോകൾ

### 📦 **ആവശ്യമായ NuGet പാക്കേജുകൾ**


### 🔑 **ആസ്യൂർ ഓപ്പൺഎഐ കോൺഫിഗറേഷൻ**

**പരിസ്ഥിതി ക്രമീകരണ (.env ഫയൽ):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-5-mini
```

**ആസ്യൂർ ഓപ്പൺഎഐ ആക്സസ്:**
1. ആസ്യൂർ പോർട്ടലിൽ ആസ്യൂർ ഓപ്പൺഎഐ റിസോഴ്സ് സൃഷ്ടിക്കുക
2. ഒരു മോഡൽ വിന്യസിക്കുക (ഉദാ: `gpt-5-mini`) ആൻഡ് വിന്യാസ നാമം ശ്രദ്ധിക്കുക
3. `az login` ഉപയോഗിച്ച് സൈൻ ഇൻ ചെയ്യുക, മേൽപ്പറഞ്ഞ പ്രകാരം പരിസ്ഥിതി വേരിയബിൾസ് ക്രമീകരിക്കുക

### 🏗️ **വർക്ഫ്ലോ ആർക്കിടെക്ചർ അവലോകനം**

```mermaid
graph TD
    A[വർക്‌ഫ്ലോ ബിൽഡർ] --> B[ഏജന്റ് രജിസ്‌ട്രി]
    B --> C[വർക്‌ഫ്ലോ നിർവഹണ എഞ്ചിൻ]
    C --> D[Agent 1: ഉള്ളടക്ക നിർമ്മാതാവ്]

    D --> F[വർക്‌ഫ്ലോ ഫലങ്ങൾ]
    E --> F
```

**പ്രധാന ഘടകങ്ങൾ:**
- **WorkflowBuilder**: വർക്ഫ്ലോകൾ രൂപകൽപ്പന ചെയ്യുന്നതിനും ഓർക്കസ്ട്രേറ്റ് ചെയ്യുന്നതിനുമായ പ്രധാന എൻജിൻ
- **AIAgent**: വ്യക്തിഗത, പ്രത്യേക കഴിവുകളുള്ള ഏജന്റുകൾ
- **Azure OpenAI Client**: ആസ്യൂർ ഓപ്പൺഎഐ Responses API സംയോജനം
- **Execution Context**: വർക്ഫ്ലോ ഘട്ടങ്ങളിൽ സ്റ്റേറ്റ്, ഡാറ്റ ഒഴുക്ക് നിയന്ത്രിക്കുന്നു

## 🏢 **സംരംഭ ലാഭങ്ങൾ**

### 🎯 **വിശ്വസ്തതയും സ്കേലബിലിറ്റിയും**
- **Deterministic Execution**: സ്ഥിരതയും ആവർത്തനക്ഷമതയുമുള്ള വർക്ഫ്ലോ ഫലം
- **Error Recovery**: ഏത് വർക്ഫ്ലോ ഘട്ടത്തിലും പരാജയങ്ങൾ സുഖപ്രദമായി കൈകാര്യം ചെയ്യൽ
- **Performance Monitoring**: നടപ്പിലാക്കൽ മീറ്റ്രിക്‌സും ഓപ്റ്റിമൈസേഷൻ സാധ്യതകളും നിരീക്ഷിക്കുക
- **Resource Management**: AI മോഡൽ വിഭവങ്ങളുടെ കാര്യക്ഷമ ബഹുമുഖ വിനിയോഗം

### 🔒 **സുരക്ഷയും പാലനവും**
- **Secure Authentication**: `az login` മുഖേന മൈക്രോസോഫ്റ്റ് Entra ID സ്ഥിരീകരണം (AzureCliCredential)
- **Audit Trails**: വർക്ഫ്ലോ നടപ്പും തീരുമാന ബിന്ദുക്കളും പൂർണ്ണമായ രേഖപ്പെടുത്തൽ
- **Access Control**: വർക്ഫ്ലോ നടപ്പിലാക്കലിനും നിരീക്ഷണത്തിനും വിപുലമായ അനുമതികൾ
- **Data Privacy**: വർക്ഫ്ലോകളിലൂടെ നൈജ്ജിക വിവരങ്ങളുടെ സൂക്ഷ്മ കൈകാര്യം

### 📊 **പ്രതീക്ഷണപരവും മാനേജ്മെന്റും**
- **Visual Workflow Design**: പ്രോസസ് പ്രവാഹങ്ങൾക്കും ആശ്രിതങ്ങളുമായി വ്യക്തമായ പ്രതിനിധാനം
- **Execution Monitoring**: വർക്ഫ്ലോ പുരോഗതിയും ദക്ഷതയും യഥാർത്ഥ-സമയം നിരീക്ഷണം
- **Error Reporting**: തൽസമയ പിഴവ് വിശകലനവും ഡീബഗിങ്ങും
- **Performance Analytics**: ഓപ്റ്റിമൈസേഷനും ശേഷിയു പദ്ധതിയുവയ്ക്കലിനും മീറ്റ്രിക്‌സ്

നമുക്ക് നിങ്ങളുടെ ആദ്യ സംരംഭം-സജ്ജമായ AI വർക്ഫ്ലോ നിർമ്മിക്കാം! 🚀

## 💻 കോഡ് പ്രവർത്തിപ്പിക്കൽ

പൂര്‍ത്തിയായ നടപ്പാക്കല്‍ `01.dotnet-agent-framework-workflow-ghmodel-basic.cs`-ൽ ലഭ്യമാണ്. ഈ ഫയൽ കാണിക്കുന്നു:

1. **പരിസ്ഥിതി ക്രമീകരണം** - `.env` ഫയലിൽ നിന്നും ആസ്യൂർ ഓപ്പൺഎഐ കോൺഫിഗറേഷൻ ലോഡ് ചെയ്യൽ
2. **ആസ്യൂർ ഓപ്പൺഎഐ ക്ലയന്റ് സെറ്റപ്പ്** - ആസ്യൂർ ഓപ്പൺഎഐ Responses API ഉപയോഗിക്കാൻ ക്ലയന്റ് ക്രമീകരണം
3. **ഏജന്റ് സൃഷ്ടി** - പ്രത്യേകജ്ഞതയുള്ള ഏജന്റുകൾ നിർവ്വചിക്കൽ (Front Desk, Concierge)
4. **Workflow Builder** - അനുക്രമമായ പ്രോസസ്സിംഗോടെ ബഹു-ഏജന്റ് വർക്ഫ്ലോ സൃഷ്ടിക്കൽ
5. **Workflow Execution** - ലൈവ് ഫലങ്ങളോടെ വർക്ഫ്ലോ പ്രവർത്തിപ്പിക്കൽ
