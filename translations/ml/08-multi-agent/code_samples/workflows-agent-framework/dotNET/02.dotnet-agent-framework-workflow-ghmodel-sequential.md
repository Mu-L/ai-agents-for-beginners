# ⏩ Azure OpenAI (Responses API) (.NET) ഉപയോഗിച്ച് അനുക്രമണ ഏജന്റ് വർക്ക്‌ഫ്ലോകൾ

## 📋 ആധുനിക അനുക്രമണ പ്രോസസിംഗ് ട്യൂട്ടോറിയൽ

.NET-നും Azure OpenAI (Responses API)-നുമുള്ള Microsoft Agent Framework ഉപയോഗിച്ച് **അനുക്രമണ വർക്ക്‌ഫ്ലോ പാറ്റേണുകൾ** ഈ നോട്ട്‌ബുക്കിൽ കാണിക്കുന്നുണ്ട്. ഏജന്റുമാർ പ്രത്യേക ക്രമത്തിൽ പ്രവർത്തിക്കുന്ന, മുൻപത്തെ ഘടകത്തിന്റെ ഫലത്തിലെ അടിസ്ഥാനത്തിൽ ഓരോ ഘടകവും നിർമ്മിക്കുന്ന സുവിസ്തൃത, ഘട്ടതല പ്രോസസ്സിംഗ് പൈപ്പ്ലൈനുകൾ എങ്ങനെ നിർമ്മിക്കാമെന്ന് നിങ്ങൾക്ക് പഠിക്കാം.

- **സ്റ്റേറ്റ് മാനേജുമെന്റ്**: അനുക്രമണ വർക്ക്‌ഫ്ലോ ഘട്ടങ്ങളിലുടനീളം സੰਦਰഭവും ഡാറ്റാ പ്രവാഹവും നിലനിർത്തുക

## ⚙️ ആവശ്യമായ സജ്ജീകരണങ്ങൾ

### 📦 **ആവശ്യമായ NuGet പാക്കേജുകൾ**

.NET അനുക്രമണ വർക്ക്‌ഫ്ലോകൾക്കുള്ള അത്യാവശ്യം പാക്കേജുകൾ:

```xml
<!-- Core AI Framework -->
<PackageReference Include="Microsoft.Extensions.AI" Version="10.*" />

<!-- Azure OpenAI (Responses API) -->
<PackageReference Include="Azure.AI.OpenAI" Version="2.*" />

<!-- Azure Identity and Async LINQ Support -->
<PackageReference Include="Azure.Identity" Version="1.15.0" />
<PackageReference Include="System.Linq.Async" Version="6.0.3" />

<!-- Local Agent Framework References -->
<!-- Microsoft.Agents.AI.dll - Core agent abstractions -->
<!-- Microsoft.Agents.AI.OpenAI.dll - Azure OpenAI (Responses API) integration -->
```

### 🔑 **Azure OpenAI കോണ്ഫിഗറേഷൻ**

**Environment Setup (.env ഫയൽ):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-5-mini
```

**കോൺഫിഗറേഷൻ മാനേജുമെന്റ്:**
```csharp
// Load environment variables securely
Env.Load("../../../.env");
var azureEndpoint = Environment.GetEnvironmentVariable("AZURE_OPENAI_ENDPOINT");
var deployment = Environment.GetEnvironmentVariable("AZURE_OPENAI_DEPLOYMENT");
```

### 📝 പ്രതീക്ഷിക്കുന്ന ഔട്ട്പുട്ട്

വർക്ക്‌ഫ്ലോ:
1. **സെയിൽസ് ഏജന്റ്**: ചിത്രത്തിൽനിന്ന് ഫർണിച്ചർ ഇനങ്ങൾ തിരിച്ചറിയുകയും നിർദേശങ്ങൾ നൽകുകയും ചെയ്യുന്നു
2. **വില ഏജന്റ്**: ബഡ്ജെറ്റ് ടിയേഴ്സും ഷോപ്പിംഗ് നിർദ്ദേശങ്ങളും ഉൾപ്പെടെയുള്ള വിശദമായ വില വിശകലനം ചേർക്കുന്നു
3. **Quote Agent**: എല്ലാ വിവരങ്ങൾ സംയോജിപ്പിച്ച് ഫോർമാറ്റ് ചെയ്ത ക്വോട്ട് ഡോക്യുമെന്റ് നിർമ്മിക്കുന്നു

അന്തിമ ഔട്ട്പുട്ട് ചിത്രവിശകലനത്തെ അടിസ്ഥാനമാക്കിയും സമഗ്രവും പ്രൊഫഷണലുമായ ഫർണിച്ചർ ക്വോട് ആയിരിക്കും.

### 🎯 യാഥാർത്ഥ്യ ലോക പ്രയോഗങ്ങൾ

ഈ അനുക്രമണ പാറ്റേൺ അനുയോജ്യമാണ്:
- **ഇ-കൊമേഴ്സ്**: ഉൽപ്പന്ന വിശകലനം → വില നിർണ്ണയം → ക്വോട്ട് നിർമ്മാണം
- **റിയൽ എസ്റ്റേറ്റ്**: സ്വത്ത് വിശകലനം → മൂല്യനിർണ്ണയം → ലിസ്റ്റിംഗ് നിർമ്മാണം
- **ഇൻഷുറൻസ്**: ക്ലെയിം വിശകലനം → വിലയിരുത്തൽ → ക്വോട്ട് നിർമ്മാണം
- **കണ്ടന്റ് ക്രിയേഷൻ**: റിസർച്ച് → എഴുത്ത് → എഡിറ്റിംഗ് → പ്രസിദ്ധീകരിക്കല്‍

### 🔍 സ്റ്റേറ്റ് പ്രവാഹം മനസ്സിലാക്കൽ

പരമ്പരയിലെ ഓരോ ഏജന്റും സ്വീകരിക്കുന്നവ:
- **ഒറിജിനൽ ഇൻപുട്ട്**: തുടക്കത്തിലെ യൂസർ സന്ദേശം (ചിത്രം + ടെക്സ്റ്റ്)
- **മുൻ ഏജന്റ് ഔട്ട്പുട്ടുകൾ**: സംവാദ ചരിത്രത്തിലെ മുൻ ചോദ്യങ്ങൾ
- **അക്യൂമുലേറ്റഡ് കോൺടെക്സ്റ്റ്**: മുഴുവൻ വർക്ക്‌ഫ്ലോയിൽ നിലനിർത്തുന്ന സ്റ്റേറ്റ്

ഇത് സങ്കീർണ ബഹുഘട്ട പ്രോസസ്സിംഗ് സാദ്ധ്യമാക്കുന്നു, ഓരോ ഏജന്റും മുൻഘട്ടങ്ങളിൽ നിന്നുള്ള സമ്പൂർണ്ണ കോൺടെക്സ്റ്റിൽ ভিত্তി ചേർത്ത് പ്രവർത്തിക്കുന്നു.
