# 🔄 របៀបធ្វើការងារជា Agent យ៉ាងមូលដ្ឋានជាមួយ Azure OpenAI (Responses API) (.NET)

## 📋 មេរៀនណែនាំលីលានធ្វើការងារ

សៀវភៅកំណត់ត្រានេះបង្ហាញពីរបៀបសាងសង់ **លីលានធ្វើការងារជា agent** យ៉ាងស្មុគស្មាញដោយប្រើ Microsoft Agent Framework សម្រាប់ .NET និង Azure OpenAI (Responses API) ។ អ្នកនឹងរៀនបង្កើតដំណើរការអាជីវកម្មជាច្រើនជំហានដែល agent បែប AI ប្រតិបត្តិការជាមួយគ្នាដើម្បីសម្រេចបានភារកិច្ចស្មុគស្មាញតាមរយៈគំរូលានធ្វើការងារដែលបានរៀបចំយ៉ាងម៉ត់ចត់។

## 🎯 គោលបំណងការសិក្សា

### 🏗️ **មូលដ្ឋានស្ថាបត្យកម្មលីលានធ្វើការងារ**
- **អ្នកសាងសង់លីលានធ្វើការងារ**: រចនា និងរៀបចំដំណើរការអាជីវកម្ម AI ដែលមានជាច្រើនជំហានស្មុគស្មាញ
- **ការសម្របសម្រួល agent**: សម្របសម្រួល agent ជាច្រើនដែលមានជំនាញជាក់លាក់នៅក្នុងលីលានធ្វើការងារ
- **Azure OpenAI (Responses API)**: ប្រើប្រាស់ Azure OpenAI Responses API ក្នុងលីលានធ្វើការងារ
- **រចនាសម្ព័ន្ធលីលានធ្វើការងារជារូបភាព**: បង្កើត និងមើលឃើញរចនាសម្ព័ន្ធលីលានធ្វើការងារដើម្បីយល់ដឹងបានច្បាស់លាស់

### 🏢 **ការដាក់ពាក្យលីលានធ្វើការងារសម្រាប់សហគ្រាស**
- **ស្វ័យប្រតិបត្តិដំណើរការអាជីវកម្ម**: ស្វ័យប្រតិបត្តិលីលានធ្វើការងារសមុទ្រយសហគ្រាសស្មុគស្មាញ
- **បណ្តាញផលិតមាតិកា**: លីលានធ្វើការអតិថិជនដែលមានវគ្គពិនិត្យ និងអនុម័ត
- **ស្វ័យប្រតិបត្តិការប្រតិបត្តិការជូនសេវាអតិថិជន**: ដោះស្រាយការសួរអតិថិជនជាច្រើនជំហាន

### 🔑 **ការកំណត់រចនាសម្ព័ន្ធ Azure OpenAI**

**ការតំឡើងបរិស្ថាន (.env file):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-5-mini
```

**ការចូលប្រើ Azure OpenAI:**
1. បង្កើតធនធាន Azure OpenAI នៅក្នុងប្រអប់ Azure
2. ចាប់ផ្ដើមដាក់ម៉ូដែល (ឧទាហរណ៍, `gpt-5-mini`) ហើយចំណាំឈ្មោះដោតម៉ូដែល
3. ចូលប្រើដោយ `az login` ហើយកំណត់អថេរបរិស្ថានដូចបានបង្ហាញខាងលើ

### 🏗️ **ទិដ្ឋភាពទូទៅស្ថាបត្យកម្មលីលានធ្វើការងារ**

```mermaid
graph TD
    A[កម្មវិធីសង់សង់ស្របការ] --> B[ប្រព័ន្ធចុះបញ្ជីភ្នាក់ងារ]
    B --> C[ម៉ាស៊ីនអនុវត្តឯកសារស្របការ]
```

**សមាសធាតុសំខាន់ៗ៖**
- **WorkflowBuilder**: ម៉ាស៊ីនរៀបចំការសម្រាប់រចនាលីលានធ្វើការងារ
- **AIAgent**: agent ឯករាជ្យដែលមានជំនាញពិសេស
- **Azure OpenAI Client**: ការចងក្រង API Azure OpenAI Responses
- **Execution Context**: គ្រប់គ្រងស្ថានភាព និងចរន្តទិន្នន័យរវាងវគ្គលីលានធ្វើការងារ

### 📝 លទ្ធផលដែលរំពឹង
