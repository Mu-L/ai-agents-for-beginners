# рҹҺЁ бһӣбҹҶбһ“бһ¶бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ Agentic бһҮбһ¶бһҳбһҪбһҷ Azure OpenAI (Responses API) (.NET)

## рҹ“Ӣ бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһӣбҹҶбһ“бһ¶бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҹбһ¶бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһүбҹ’бһүбһ¶бһҮбҹҶбһ“бһ¶бһүбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Microsoft Agent Framework бһҖбҹ’бһ“бһ»бһ„ .NET бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Azure OpenAI (Responses API)бҹ” бһўбҹ’бһ“бһҖбһ“бһ·бһ„бһҡбҹҖбһ“бһӣбҹҶбһ“бһ¶бҹҶбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲ бһ“бһ·бһ„бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһўбһ¶бһ…бһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһ”бҹҶбһ–бһ¶бһҖбҹӢбһңбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹҶбһ бҹҶбһ”бһ¶бһ“бҹ”

### бһӣбҹҶбһ“бһ¶бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

- рҹҸӯ **бһӣбҹҶбһ“бһ¶бҹҶбһҡбҹ„бһ„бһ…бһҖбҹ’бһҡ**: бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбҹ’бһҠбһ¶бһҡбһ…бһ¶бһҖбҹӢбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ (dependency injection)
- рҹ”§ **бһӣбҹҶбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһҹбһ¶бһ„бһҹбһ„бҹӢ**: бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҷбҹүбһ¶бһ„бһҡбһӣбһјбһ“
- рҹ§ө **бһӣбҹҶбһ“бһ¶бҹҶ Thread-Safe**: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶ
- рҹ“Ӣ **бһӣбҹҶбһ“бһ¶бҹҶ Repository**: бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–

## рҹҺҜ бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ .NET

### бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

- **бһҖбһ¶бһҡбһҸбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҳбһәбһ“бҹ—**: бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ“бҹ…бһ–бҹҒбһӣбһ”бҹҶбһӣбҹӮбһ„бһҖбһјбһҠ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡ IntelliSense
- **бһҖбһүбҹ’бһ…бһ”бҹӢбһҖбҹ’бһҠбһ¶бһҡбһ…бһ¶бһҖбҹӢ**: бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ DI бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„вҖӢбһӮбҹ’бһҡбһҝбһ„бһ•бҹ’бһҹбҹҶвҖӢбһ‘бһ¶бҹҶбһ„бһҖбҹ’бһ“бһ»бһ„
- **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’**: IConfiguration бһ“бһ·бһ„бһӣбҹҶбһ“бһ¶бҹҶ Options
- **Async/Await**: бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһ¶бһҹбһ“бҹ’бһ“бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҠбҹҶбһ”бһјбһ„

### бһӣбҹҶбһ“бһ¶бҹҶбһҸбҹ’бһҡбҹҖбһҳбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ

- **бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»**: ILogger бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’
- **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһҒбһ—бһ¶бһ–**: бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһңбһ¶бһҹбҹӢбһңбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ
- **бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’**: бһҖбһ¶бһҡбһҸбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҒбҹ’бһӣбһ¶бҹҶбһ„бһҮбһ¶бһҳбһҪбһҷбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ
- **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһ бһ»бһҹ**: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ·бһҸбһ·бһ…бһӣбҹҗбһҸбһ“бҹғбһҖбҹҶбһ бһ»бһҹбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

## рҹ”§ бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ

### бһ’бһ¶бһҸбһ» .NET бһҸбһҳбҹ’бһҡбһ„бҹӢбһҒбҹ’бһҹбҹӮбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“

- **Microsoft.Extensions.AI**: бһҖбһ¶бһҡбһҸбҹҶбһҺбһ¶бһ„бһҹбҹҒбһңбһ¶ AI бһҹбһҡбһ»бһ”
- **Microsoft.Agents.AI**: бһҹбҹҶбһҺбһ»бҹҶбһҹбҹ’бһҸбһҫбһ„бһ бҹ’бһӮбҹ’бһҡбһ¶бҹҶбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ
- **Azure OpenAI (Responses API)**: бһӣбҹҶбһ“бһ¶бҹҶбһўбһҸбһ·бһҗбһ·бһҮбһ“ API бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢ
- **бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’**: appsettings.json бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ

## рҹҸ—пёҸ бһӣбҹҶбһ“бһ¶бҹҶбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһ”бһ„бҹ’бһ бһ¶бһү

### 1. **бһӣбҹҶбһ“бһ¶бҹҶвҖӢбһ”бһ„бҹ’бһҖбһҫбһҸ**

- **бһҡбҹ„бһ„бһ…бһҖбҹ’бһҡвҖӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ**: бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһҮбҹ’бһҲбһ·бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбҹҠбһёбһҮбһҳбҹ’бһҡбҹ…
- **бһӣбҹҶбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһҹбһ¶бһ„бһҹбһ„бҹӢ**: API бһҡбһӣбһјбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ–бһ·бһ”бһ¶бһҖ
- **бһӣбҹҶбһ“бһ¶бҹҶ Singleton**: бһҖбһ¶бһҡбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’
- **бһҖбһүбҹ’бһ…бһ”бҹӢбһҖбҹ’бһҠбһ¶бһҡбһ…бһ¶бһҖбҹӢ**: бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бһ¶бһ“бҹӢбһ–бҹҒбһӣ бһ“бһ·бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһ¶бһ“бһ„бһ¶бһҷ

### 2. **бһӣбҹҶбһ“бһ¶бҹҶбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶**

- **бһӣбҹҶбһ“бһ¶бҹҶбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ**: бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ¶бһ…бһ”бҹ’бһҠбһјбһҡбһ”бһ¶бһ“

- **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ**: бһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһ¶бһҹбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ“бһ·бһ„бһӮбһҺбҹҲбһңбһ·бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
- **бһҖбһ¶бһҡбһ”бҹ’бһҡбһЎбһ„**: бһҸбҹҒбһҹбҹ’бһҸбһҜбһҖбһҸбһ¶, бһҸбҹҒбһҹбҹ’бһҸбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ, бһ“бһ·бһ„бһӣбҹҶбһ“бһ¶бҹҶбһҸбҹҒбһҹбҹ’бһҸбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ”бһ“бҹ’бһ‘бһ»бһҖ

бһҡбҹҖбһ”бһ…бҹҶбһҠбһҫбһҳбҹ’бһ”бһёбһҖбһҹбһ¶бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһүбҹ’бһүбһ¶бһҮбҹҶбһ“бһ¶бһүбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҮбһ¶бһҳбһҪбһҷ .NET бһ‘бҹҒ? бһ…бһјбһҡбһҡбһ…бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҒбҹ’бһӣбһ¶бҹҶбһ„бһҳбһҪбһҷ! рҹҸўвңЁ

бһҳбһҫбһӣ[`03-dotnet-agent-framework.cs`](../../../../03-agentic-design-patterns/code_samples/03-dotnet-agent-framework.cs) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһјбһҠбһ–бҹҒбһүбһӣбҹҒбһүбҹ”

```csharp
#!/usr/bin/dotnet run

#:package Microsoft.Extensions.AI@10.*
#:package Microsoft.Agents.AI.OpenAI@1.*-*
#:package Azure.AI.OpenAI@2.1.0
#:package Azure.Identity@1.13.1

using System.ComponentModel;

using Microsoft.Agents.AI;
using Microsoft.Extensions.AI;

using Azure.AI.OpenAI;
using Azure.Identity;

// Tool Function: Random Destination Generator
// This static method will be available to the agent as a callable tool
// The [Description] attribute helps the AI understand when to use this function
// This demonstrates how to create custom tools for AI agents
[Description("Provides a random vacation destination.")]
static string GetRandomDestination()
{
    // List of popular vacation destinations around the world
    // The agent will randomly select from these options
    var destinations = new List<string>
    {
        "Paris, France",
        "Tokyo, Japan",
        "New York City, USA",
        "Sydney, Australia",
        "Rome, Italy",
        "Barcelona, Spain",
        "Cape Town, South Africa",
        "Rio de Janeiro, Brazil",
        "Bangkok, Thailand",
        "Vancouver, Canada"
    };

    // Generate random index and return selected destination
    // Uses System.Random for simple random selection
    var random = new Random();
    int index = random.Next(destinations.Count);
    return destinations[index];
}

// Azure OpenAI with the Responses API (stable v1 endpoint). Sign in with `az login`.
var azureEndpoint = Environment.GetEnvironmentVariable("AZURE_OPENAI_ENDPOINT")
    ?? throw new InvalidOperationException("AZURE_OPENAI_ENDPOINT is not set.");
var deployment = Environment.GetEnvironmentVariable("AZURE_OPENAI_DEPLOYMENT") ?? "gpt-4.1-mini";

var azureClient = new AzureOpenAIClient(new Uri(azureEndpoint), new AzureCliCredential());

// Define Agent Identity and Comprehensive Instructions
// Agent name for identification and logging purposes
var AGENT_NAME = "TravelAgent";

// Detailed instructions that define the agent's personality, capabilities, and behavior
// This system prompt shapes how the agent responds and interacts with users
var AGENT_INSTRUCTIONS = """
You are a helpful AI Agent that can help plan vacations for customers.

Important: When users specify a destination, always plan for that location. Only suggest random destinations when the user hasn't specified a preference.

When the conversation begins, introduce yourself with this message:
"Hello! I'm your TravelAgent assistant. I can help plan vacations and suggest interesting destinations for you. Here are some things you can ask me:
1. Plan a day trip to a specific location
2. Suggest a random vacation destination
3. Find destinations with specific features (beaches, mountains, historical sites, etc.)
4. Plan an alternative trip if you don't like my first suggestion

What kind of trip would you like me to help you plan today?"

Always prioritize user preferences. If they mention a specific destination like "Bali" or "Paris," focus your planning on that location rather than suggesting alternatives.
""";

// Create AI Agent with Advanced Travel Planning Capabilities
// Get the Responses client for the deployment and create the AI agent
// Configure agent with name, detailed instructions, and available tools
// This demonstrates the .NET agent creation pattern with full configuration
AIAgent agent = azureClient
    .GetChatClient(deployment)
    .AsAIAgent(
        name: AGENT_NAME,
        instructions: AGENT_INSTRUCTIONS,
        tools: [AIFunctionFactory.Create(GetRandomDestination)]
    );

// Create New Conversation Session for Context Management
// Initialize a new conversation session to maintain context across multiple interactions
// Sessions enable the agent to remember previous exchanges and maintain conversational state
// This is essential for multi-turn conversations and contextual understanding
var session = await agent.CreateSessionAsync();

// Execute Agent: First Travel Planning Request
// Run the agent with an initial request that will likely trigger the random destination tool
// The agent will analyze the request, use the GetRandomDestination tool, and create an itinerary
// Using the session parameter maintains conversation context for subsequent interactions
await foreach (var update in agent.RunStreamingAsync("Plan me a day trip", session))
{
    await Task.Delay(10);
    Console.Write(update);
}

Console.WriteLine();

// Execute Agent: Follow-up Request with Context Awareness
// Demonstrate contextual conversation by referencing the previous response
// The agent remembers the previous destination suggestion and will provide an alternative
// This showcases the power of conversation sessions and contextual understanding in .NET agents
await foreach (var update in agent.RunStreamingAsync("I don't like that destination. Plan me another vacation.", session))
{
    await Task.Delay(10);
    Console.Write(update);
}
```
