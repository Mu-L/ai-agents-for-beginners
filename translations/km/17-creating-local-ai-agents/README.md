# бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Microsoft Foundry Local бһ“бһ·бһ„ Qwen

бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“бһ“бҹҒбҹҮбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһ“бһ‘бҹҶбһ бҹҶбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҡбһјбһ„бһ…бһјбһӣбһҳбҹҒбһғбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ“бһ¶бҹҶбһ–бһҪбһҖбһңбһ¶бһ…бһ»бҹҮбһ‘бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҸбҹӮбһҳбһҪбһҷбҹ” бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһ¶бһ“бһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“ бһ“бһ№бһҖбһғбһҫбһү бһҹбһҳбҹ’бһҡбһ„бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ бһўбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҜбһҖбһҹбһ¶бһҡбһҮбҹҶбһ“бһҪбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ вҖ” **бһҠбҹ„бһҷбһҳбһ·бһ“бһҳбһ¶бһ“бһҖбһ¶бһҡбһ бҹ…бһҖбһҡбһҺбһёбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸбһҳбҹҒбһғбһҮбһ¶бһҠбһҫбһҳбҹ”**

бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶бһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бһ¶бһ“бһ”бҹӮбһ”бһ“бҹҒбҹҮ? бһҳбһ¶бһ“бһ”бһёбһ бҹҒбһҸбһ»бһ•бһӣбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһЎбһҫбһ„бһҮбһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ„бһ¶бһҡбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ–

- **бһ—бһ¶бһ–бһҜбһҖбһҮбһ“бҹ”** бһҖбһјбһҠ бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡбһҳбһ·бһ“бһ’бҹ’бһӣбһ¶бһҖбҹӢбһ…бҹҒбһүбһ–бһёбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбҹ” бһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһ“бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ бһҹбҹ’бһ“бһ¶бһҠбҹғ бһ¬бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһ”бһҺбҹ’бһҠбһ¶бһүбһ‘бҹҒбҹ”
- **бһҗбҹ’бһӣбҹғбһ…бҹҶбһҺбһ¶бһҷбҹ”** бһҖбһ¶бһҡбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҳбһ·бһ“бһҳбһ¶бһ“бһңбһ·бһӣбҹҗбһҷбһ–бһёбһҡ Token бһ‘бҹҒбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһ¶бһ“бһҡбһ¶бһӣбҹӢбһҗбҹ’бһ„бҹғбһҠбҹ„бһҷбһҗбҹ’бһӣбҹғбһ”бһјбһҖбһӮбһ·бһҸбһҸбһ¶бһҳбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһёбҹ”
- **бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ”бһ·бһ‘бһ—бҹ’бһҮбһ¶бһ”бҹӢбҹ”** бһ“бҹ…бһӣбһҫбһҷбһ“бҹ’бһҸбһ бҹ„бҹҮ бһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ¬бһ–бҹҒбһӣбһҖбһ¶бһҡбһ•бҹ’бһўбһ¶бһҖбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ”бһ»бһӮбҹ’бһӮбһӣбһӣбҹ„бһ бҹҲбһ“бҹ…бһҸбҹӮбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ”

бһ”бһүбҹ’бһ бһ¶бһӮбһәбһҗбһ¶бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҸбһјбһҡбһӮбҹҶбһҡбһјбһ”бһ„бҹ’бһ бһ¶бһүбһҳбҹҒбһғбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ **бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһҸбһјбһ… (SLM)** бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫ CPU, GPU, бһ¬ NPU бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһӮбһәбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһ“бҹ„бҹҮбһҮбһ¶бһ„бһҸбҹӮбһ’бҹ’бһңбһҫбһҳбһҫбһӣбһҗбһ¶бһҖбҹҶбһҺбһҸбҹӢбһ“бҹ„бҹҮбһҳбһ·бһ“бһҳбһ¶бһ“бҹ”

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбһјбһ…бһҸбһ‘бҹ…бҹ–

- **бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһҸбһјбһ… (SLMs)** вҖ” бһҮбһ¶бһўбҹ’бһңбһё бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣбһ–бһҪбһҖбһңбһ¶бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“ бһ“бһ·бһ„бһҳбһ·бһ“бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бҹ”
- **Microsoft Foundry Local** вҖ” бһӮбҹ’бһҡбһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹӮбһӣбһ‘бһ¶бһүбһҷбһҖ бһ“бһ·бһ„бһ”бһҳбҹ’бһҡбһҫбһӮбҹҶбһҡбһјбһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹ„бһҷбһҡбһҷбҹҲбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҸбҹӮбһҳбһҪбһҷ бһҸбһ¶бһҳбһҡбһҷбҹҲ **API бһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷбһҮбһ¶бһҳбһҪбһҷ OpenAI**бҹ”
- **бһӮбҹҶбһҡбһјбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ Qwen** вҖ” SLMs бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһ–бһӣбһ—бһ¶бһ–бһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһӣбҹ„бһҖбһҮбһ¶бһҸбһ· *бһ“бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ* бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“ (бһҳбһ·бһ“бһҳбҹӮбһ“бһҸбҹӮбһҮбһ¶бһ…бһ¶бһҸбҹӢбһҸбҹӮбһҹбһ“бҹ’бһ‘бһ“бһ¶)бҹ”
- **бһ§бһ”бһҖбһҡбһҺбҹҚбһҖбһ¶бһҡбһ„бһ¶бһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ RAG бһ“бһ·бһ„ MCP** вҖ” бһ•бҹ’бһҸбһӣбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһІбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбҹҒбһғбҹ”
- **бһӮбҹҶбһҡбһјбһ”бҹӮбһ”бһ”бһүбҹ’бһ…бһјбһӣ** вҖ” бһ–бҹҒбһӣбһҺбһ¶бһӮбһҪбһҡбһҸбҹӮбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһ–бҹҒбһӣбһҺбһ¶бһӮбһҪбһҡбһҸбҹӮбһ‘бҹ…бһҡбһҖбһҳбҹҒбһғбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҠбһ№бһ„бһҗбһ¶бһўбһ¶бһ…бҹ–

- бһ–бһ“бҹ’бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҮбһҪбһүбһҠбһјбһҡбһҹбһҳбҹ’бһҡбһҪбһӣбһҡбһңбһ¶бһ„ SLMs бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһІбҹ’бһҷбһҹбһҳбһҡбһҳбҹ’бһҷбҹ”
- бһ”бһҳбҹ’бһҡбһҫбһӮбҹҶбһҡбһј Qwen бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҮбһ¶бһҳбһҪбһҷ Foundry Local бһ бһҫбһҷбһ—бҹ’бһҮбһ¶бһ”бҹӢбһңбһ¶бһҸбһ¶бһҳбһҡбһҷбҹҲбһ…бҹҶбһҺбһ»бһ…бһ…бһјбһӣбһҹбһҳбһҡбһҳбҹ’бһҷбһҮбһ¶бһҳбһҪбһҷ OpenAIбҹ”
- бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бҹҶбһ–бҹ„бҹҮбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹӮбһҳбһҪбһҷбҹ”
- бһ”бһ“бҹ’бһҗбҹӮбһҳ Local RAG бһӣбһҫбһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһңбҹӮбһңбһ‘бҹҗбһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҡбһ”бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ (Chroma)бҹ”
- бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ MCP бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ бһҫбһҷбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ”бҹӮбһ”бһ”бһүбҹ’бһ…бһјбһӣбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“/бһҳбҹҒбһғбҹ”

## бһӮбҹҶбһҡбһјбһ—бһ¶бһҹбһ¶бһҸбһјбһ…бҹ– бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“

бһӮбҹҶбһҡбһјбһҳбҹҒбһғбһҮбһ¶бһӮбҹҶбһҡбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбҹ–бһҡбһ¶бһ”бҹӢбһҡбһҷбһ–бһ¶бһ“бҹӢбһӣбһ¶бһ“ бһ“бһ·бһ„бһҳбһ¶бһ“бһҳбһҮбҹ’бһҲбһҳбһҺбҹ’бһҢбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһҖбҹ’бһҡбҹ„бһҷбҹ” SLM бһҳбһ¶бһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ…бҹҶбһ“бһҪбһ“бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһ–бһ¶бһ“бҹӢбһӣбһ¶бһ“ бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҠбһ¶бһҖбҹӢбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„ RAM бһ“бҹғбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҷбһҪбһҡбһҠбҹғбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһ“бҹҒбҹҮбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбҹ”

**SLMs бһӣбҹ’бһўбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбҹ–**

## Microsoft Foundry Local

**Microsoft Foundry Local** бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһӣбһҠбҹӮбһӣбһ‘бһ¶бһүбһҷбһҖ бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһ”бһҳбҹ’бһҡбһҫбһӮбҹҶбһҡбһјбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„бһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ”бҹҶбһ•бһ»бһҸбһҡбһ”бһҹбҹӢбһңбһ¶бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҷбһҫбһ„бһӮбһәбһ”бһ„бҹ’бһ бһ¶бһү **бһ…бҹҶбһҺбһ»бһ…бһ…бһјбһӣ HTTP бһҹбһҳбһҡбһҳбҹ’бһҷбһҮбһ¶бһҳбһҪбһҷ OpenAI** вҖ” бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбһҡбһӮбһ»бһҺбһҗбһ¶ OpenAI SDK бһ“бһ·бһ„ Microsoft Agent Framework OpenAI client бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҮбһ¶бһҳбһҪбһҷбһңбһ¶бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ `base_url`бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ” бһўбҹ’бһңбһёбҹ—бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ“бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһ‘бҹҒбһҡбһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһ…бҹҶбһҺбһ»бһ…бһ…бһјбһӣбһ”бҹ’бһҸбһјбһҡбһ–бһёбһҳбҹҒбһғбһҳбһҖбһҖбһ¶бһ“бҹӢ `localhost`бҹ”

Foundry Local бһҖбҹҸбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ…бһ¶бһ”бҹӢбһҮбҹҶбһ“бһҪбһҹбһҖбҹҶбһҺбҹӮбһӮбҹҶбһҡбһјбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ№бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· вҖ” бһҖбҹҶбһҺбҹӮ CPU, бһҖбҹҶбһҺбҹӮ CUDA/GPU бһ¬ NPU вҖ” бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҠбҹ„бһҷбһҠбҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ“бһёбһҳбһҪбһҷбҹ—бһ‘бҹҒбҹ”

### бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„

бһҠбҹҶбһЎбһҫбһ„ Foundry Local (бһҳбһҫбһӣбһҜбһҖбһҹбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ [documentation](https://learn.microsoft.com/azure/ai-foundry/foundry-local/) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ) бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ–

```bash
# бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҸбһҳбҹ’бһӣбһҫбһ„ (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ; бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһҜбһҖбһҹбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбҹҒбһ‘бһ·бһҖбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ)
winget install Microsoft.FoundryLocal      # бһңбһёбһ“бһҠбһј
# brew install microsoft/foundrylocal/foundrylocal   # macOS

# бһ‘бһ¶бһүбһҷбһҖ бһ“бһ·бһ„бһҡбһҸбҹӢбһӮбҹҶбһҡбһј Qwen бһҳбһҪбһҷ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ
foundry model run qwen2.5-7b-instruct
foundry service status
```

бһ–бҹҒбһӣбһҠбҹӮбһӣбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ…бҹҶбһҺбһ»бһ…бһ…бһјбһӣ OpenAI-compatible local (бһ’бһҳбҹ’бһҳбһҸбһ¶ `http://localhost:PORT/v1`)бҹ” бһҹбҹҖбһңбһ—бҹ…бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҡбһҫ `foundry-local-sdk` бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбһ…бҹҶбһҺбһ»бһ…бһ…бһјбһӣбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһ…бһјбһӣбһӣбҹҒбһҒ Port бһҠбҹ„бһҷбһҠбҹғбһ‘бҹҒбҹ”

## Qwen Function Callingбҹ– бһҳбһјбһӣбһ бҹҒбһҸбһ»бһҠбҹӮбһӣбһңбһ¶бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢ

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҮбһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһ”бһҫбһңбһ¶бһўбһ¶бһ…бһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ¶бһ“бҹ” SLMs бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһўбһ¶бһ…бһҹбһ“бҹ’бһ‘бһ“бһ¶бһ”бһ¶бһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ·бһ“бһҳбһ¶бһ“бһҹбһ»бһ–бһӣбһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ” **бһӮбҹҶбһҡбһј Qwen** бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ бһ бһҫбһҷбһ”бһүбҹ’бһ…бҹҒбһүбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бҹӮбһ”бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҮбһ¶бһ”бҹ’бһҡбһ…бһ¶бҹҶ вҖ” бһҖбһ¶бһҡбһ“бҹҒбҹҮбһ бһҫбһҷбһ“бһ¶бҹҶбһІбҹ’бһҷбһҳбҹүбһјбһ…бһ¶бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ”бһҳбҹ’бһӣбҹӮбһ„бһ‘бҹ…бһҮбһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бҹ”

бһҮбҹҶбһ“бһҪбһҹбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһҡбһ„бҹ’бһңбһ„бҹӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҹбҹ’бһӮбһ¶бһӣбҹӢ бһҹбҹ’бһ‘бһҫбһҡбһҸбҹӮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбҹ–

## бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“

[MCP](../11-agentic-protocols/README.md) бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһҠбһ№бһҖбһҮбһүбҹ’бһҮбһјбһ“бһҳбһҪбһҷ бһҳбһ·бһ“бһҳбҹӮбһ“бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҳбҹҒбһғбһ‘бҹҒбҹ” бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӣбһҫ `stdio` бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһ бһҫбһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбҹ” бһ“бҹҒбҹҮбһўбһ¶бһ…бһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбһјбһ…бһҮбһ¶ бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҜбһҖбһҹбһ¶бһҡ бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ бҹ’бһӮбһёбһҸ бһҹбҹҶбһҺбһҪбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҳбҹҒбһғбҹ”

бһ‘бһҹбҹ’бһҹбһ“бҹҲбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҒбһ»бһҹбһ–бһёбһҳбҹҒбһғ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ·бһ“бһ”бһ¶бһҸбҹӢбһ”бһ„бҹӢбҹ– бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹ…бһҸбҹӮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҹбһ·бһ‘бҹ’бһ’бһ·бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ”бҹ’бһҡбһ¶бһ”бҹӢбһўбҹ„бһҷбһҖбҹҶбһҺбһҸбҹӢбһңбһ·бһҹбһ¶бһӣбһ—бһ¶бһ–бһўбҹ’бһңбһёбһҠбҹӮбһӣбһңбһ¶бһўбһ¶бһ…бһ”бҹүбҹҮбһҠбһӣбҹӢ (бһҗбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„ бһҳбһ·бһ“бһҳбҹӮбһ“бһҗбһҸбһ•бҹ’бһ‘бҹҮбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣ) бһ бһҫбһҷбһӮбһҪбһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһ”бһҹбҹӢбһңбһ¶бһ‘бҹ…бһҮбһ¶ бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһүбҹ’бһ…бһјбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбҹ”

## бһӮбҹҶбһҡбһјбһ”бһүбҹ’бһ…бһјбһӣбһҳбҹҒбһғбһ“бһ·бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“

бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҮбһ¶бһҳбһ»бһ“бһҳбһ·бһ“бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҮбһ¶ бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҸбҹӮбһҳбһҪбһҷбҹ” бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҳбһ¶бһ“бһңбҹҗбһҷбһңбһ„бҹ’бһңбҹҒбһ„бһӮбһәбһ”бһүбҹ’бһҮбһјбһ“бһҸбһ¶бһҳбһ—бһ¶бһ–бһҡбһҖбҹ’бһҹбһ¶бһҡбһ–бһ¶бһҸ бһ“бһ·бһ„бһҖбҹҶбһӣбһјбһӣбҹҲ

## бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ– бһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“

бһ”бһҫбһҖ [`code_samples/17-local-agent-foundry-local.ipynb`](./code_samples/17-local-agent-foundry-local.ipynb) бһ бһҫбһҷбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҸбһ¶бһҳбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸ **бһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“** бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бҹҶбһ–бҹ„бҹҮбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ бһҫбһҷбһўбһ¶бһ…бҹ–

1. **бһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ** вҖ” бһҸбһ¶бһҳбһҡбһҷбҹҲ Qwen function calling бһҸбһ¶бһҳ Foundry Localбҹ”
2. **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҜбһҖбһҹбһ¶бһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“** вҖ” бһ”бһүбҹ’бһҮбһё бһ“бһ·бһ„бһўбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡбһ–бһёбһҗбһҸбһӮбһҳбҹ’бһҡбҹ„бһ„бһҳбһҪбһҷбҹ”
3. **бһңбһ·бһ—бһ¶бһӮбһҖбһјбһҠ** вҖ” бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҳбһ¶бһҸбҹ’бһҡбһҠбҹ’бһӢбһ¶бһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӣбһҫбһҜбһҖбһҹбһ¶бһҡбһ”бҹ’бһҡбһ—бһ–бһҳбһҪбһҷбҹ”

бһ–бһ„бҹ’бһҡбһёбһҖбһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бҹ…бһҮбһ¶бһўбҹ’бһ“бһҖбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҜбһҖбһҹбһ¶бһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„бһҸбһјбһ…бһҳбһҪбһҷбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸ (бһ”бҹ’бһҡбһҫбһҗбһҸбһҳбҹҒбһҡбҹҖбһ“бһҳбһҪбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҺбҹ’бһҺбһҠбһ¶ repo бһ“бҹҒбҹҮ бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸ)бҹ”

бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҹбҹ’бһ“бһҫбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбҹ–

1. **бһҮбһҳбҹ’бһҡбҹҒбһёбһҹбһҗбһҸбһҜбһҖбһҹбһ¶бһҡ/бһҖбһјбһҠбһ–бһ·бһҸ** бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ Chroma (бһҷбҹүбһ¶бһ„бһ бҹ„бһ…бһҺбһ¶бһҹбҹӢ бҹҘ бһҜбһҖбһҹбһ¶бһҡ)бҹ”
2. **бһҠбһ¶бһҖбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ `find_todos`** бһҠбҹӮбһӣбһҹбҹ’бһҖбҹҒбһ“бһӮбһҳбҹ’бһҡбҹ„бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹҶбһҺбһ¶бҹҶ `TODO`/`FIXME` бһ бһҫбһҷбһ”бһүбҹ’бһ…бһјбһ“бһҮбһ¶бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮбһҜбһҖбһҹбһ¶бһҡ бһ“бһ·бһ„бһӣбҹҒбһҒбһҒбһҪбһ” вҖ” бһҡбһҖбҹ’бһҹбһ¶бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ sandbox бһҠбһјбһ… `read_file`бҹ”

3. **бһҹбһҪбһҡбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһёбһҹбҹҶбһҺбһҪбһҡ** бһҠбҹӮбһӣбһ”бһҺбҹ’бһҸбһ¶бһӣбһұбҹ’бһҷбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһ…бһјбһӣбһ§бһ”бһҖбһҡбһҺбҹҚ: бһҹбҹҶбһҺбһҪбһҡбһҳбһҪбһҷ RAG бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһҳбһҪбһҷ, бһҹбҹҶбһҺбһҪбһҡбһҳбһҪбһҷбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһўбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡбһ–бһ·бһҹбҹҒбһҹбһҳбһҪбһҷ, бһ“бһ·бһ„бһҹбҹҶбһҺбһҪбһҡбһҳбһҪбһҷбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ TODOsбҹ”
4. **бһңбһ¶бһҹбҹӢбһңбҹӮбһ„бһңбһ¶**: бһңбҹҒбһӣбһ¶бһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ‘бһ¶бҹҶбһ„бһ”бһёбһ“бҹ„бҹҮ бһ бһҫбһҷбһҖбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹ„бһҹбһ·бһҖбһ¶ markdown бһҳбһҪбһҷбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҷбһӣбҹӢбһҗбһ¶вҖӢбһ–бҹҒбһӣбһҷбһәбһҸбһ“бҹҒбҹҮвҖӢбһ‘бһ‘бһҪбһӣбһҷбһҖвҖӢбһ”бһ¶бһ“вҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ”бһ¶бһ“бһ‘бҹҒбҹ”

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҹбһҡбһҹбҹҒбһҡбһ”бһҗбҹ’бһ…бҹҒбһҖбһҒбҹ’бһӣбһёбһўбҹҶбһ–бһё **бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ–бһ–бһҖ бһ“бһ·бһ„бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ“бҹҒбҹҮ бһ бһҫбһҷбһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбҹ” бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҗбһ¶ бһ§бһ”бһҖбһҡбһҺбҹҚбһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢбһӮбҹ’бһ“бһ¶бһІбҹ’бһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһҸбһҫбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ…бһҳбҹ’бһҡбһ»бҹҮбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһӮбһҪбһҡбһұбҹ’бһҷбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸ бһ¬ бһҳбһ·бһ“бһҳбҹӮбһ“бһӣбһҫбһӮбһ»бһҺбһ—бһ¶бһ–бһҳбҹүбһјбһҢбҹӮбһӣбһ‘бҹҒбҹ”

## бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҹбһ„бҹ’бһҒбҹҒбһ”

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹҶбһ–бҹҒбһүбһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ:

- **SLMs** бһҮбһҪбһүбһҠбһјбһҡбһ–бҹ’бһҡбһёбһҠбҹ’бһӢбһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһҜбһҖбһҮбһ“, бһҗбҹ’бһӣбҹғбһҠбһҫбһҳ, бһ“бһ·бһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ“бҹ…бһҖбҹ’бһҡбҹ…бһ”бһҺбҹ’бһҸбһ¶бһү вҖ” бһ бһҫбһҷвҖӢбһ”бһ„бҹ’бһ бһ¶бһүбһ—бһ¶бһ–бһӣбҹ’бһўбһ–бҹҒбһӣбһҠбҹӮбһӣбһңбһ¶ **бһҡбҹҖбһ”бһ…бҹҶбһ§бһ”бһҖбһҡбһҺбҹҚ** бһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһ–бҹҒбһүбһӣбҹҒбһүбһ“бҹғбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бҹ”
- **Foundry Local** бһ”бһҳбҹ’бһҡбһҫбһҳбҹүбһјбһҢбҹӮбһӣбһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҖбҹ’бһҡбҹ„бһҷбһ–бҹҒбһӣбһ бҹ…бһ бһҫбһҷбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ”бһ„бҹ’бһ бһ¶бһүбһ“бһјбһң **бһ…бҹҶбһҺбһ»бһ…бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„ OpenAI** бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҖбһјбһҠбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ–бһ–бһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ”бһ¶бһ“бһҠбҹ„бһҷбһ”бһҳбҹ’бһӣбҹӮбһ„бһҳбһҪбһҷбһҮбһ¶бһ”бҹӢбҹ” 
- **бһҳбҹүбһјбһҢбҹӮбһӣбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ Qwen** бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҡбһ№бһ„бһ”бҹүбһ№бһ„ вҖ” бһ бһҫбһҷбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бҹ”
- **RAG бһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„** (Chroma) бһ“бһ·бһ„ **MCP бһҖбҹ’бһ“бһ»бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„** бһ•бҹ’бһҠбһӣбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҠбһӣбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бһ¶бһҖбһ…бҹҒбһүбһ–бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бҹ”
- **бһӮбҹҶбһҡбһјбһ…бһҳбҹ’бһҡбһ»бҹҮ** бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ•бҹ’бһүбһҫбһҡбһҸбһ¶бһҳбһ—бһ¶бһ–бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ“бһ·бһ„бһ—бһ¶бһ–бһӣбҹҶбһ”бһ¶бһҖ бһҠбҹ„бһҷбһҳбһ¶бһ“бһҖбһ“бҹ’бһӣбҹӮбһ„бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһҮбҹҶбһ“бһҪбһҹбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҳбһӣбҹ’бһ бҹӮбһҳбҹ”

бһ“бҹҒбҹҮбһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһ•бҹ’бһҹбһ¶бһҷбһ•бҹ’бһҹбһ„бһ”бһ“бҹ’бһҸбһҡбһҺбҹҚ бҹ– бһҳбҹҒбһҡбҹҖбһ“бһ‘бһёбҹЎбҹҰ бһ”бһ¶бһ“бһ–бһ„бҹ’бһҡбһёбһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢ Microsoft Foundry, бһ бһҫбһҷбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ”бһ¶бһ“бһҗбһҷбһ…бһ»бҹҮбһ‘бҹ…бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҸбҹӮбһҳбһҪбһҷбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ“бһ№бһ„бһ•бҹ’бһҠбҹ„бһҸбһ‘бҹ…бһӣбһҫбһҖбһ¶бһҡбһҮбһҪбһҷбһҖбһ¶бһҡбһ–бһ¶бһҡбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҹбһ¶бһҷбҹ”
