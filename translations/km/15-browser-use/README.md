# бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ»бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бһ¶бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ (CUA)

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бһ¶бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҠбһјбһ…бһҠбҹӮбһӣбһҳбһ“бһ»бһҹбҹ’бһҹбһ’бҹ’бһңбһҫбһ”бһ¶бһ“бҹ– бһҠбҹ„бһҷбһ”бһҫбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ, бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһ‘бҹҶбһ–бҹҗбһҡ, бһ бһҫбһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһӣбҹ’бһўбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҸбһҫбһ–бһҪбһҖбһӮбҹҒбһғбһҫбһүбҹ” бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбһҳбҹ’бһҡбһ»бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһҡбһ»бһҖбһҡбһҖбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ бһҠбҹӮбһӣбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ Airbnb, бһҠбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбҹҖбһ”бһ…бҹҶ бһ”бһүбҹ’бһҮбһёбһ‘бҹҶбһ“бһ·бһү, бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһ‘бһёбһҸбһ¶бҹҶбһ„бһҹбҹ’бһ“бһ¶бһҖбҹӢбһ“бҹ…бһҗбҹ„бһҖбһҮбһ¶бһ„бһӮбҹҒбһ“бҹ… Stockholmбҹ”

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ”бһүбҹ’бһ…бҹҒбһүбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҡбһңбһ¶бһ„ Browser-Use бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбһ»бһҖбһҡбһҖбһҠбҹ„бһҷ AI, Playwright бһ“бһ·бһ„ Chrome DevTools Protocol (CDP) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ, Azure OpenAI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҳбһҫбһӣбһғбһҫбһү, бһ“бһ·бһ„ Pydantic бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбһҖбһҹбҹ’бһҡбһ„бҹӢбһҠбҹ„бһҷбһҡбҹҖбһ”бһ…бҹҶбҹ”

## бһўбһ’бһ·бһ”бҹ’бһ”бһ¶бһҷбһ”бһҺбҹ’бһҸбһ¶бһӣ

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ“бһ№бһ„бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбҹ–

- бһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһ–бҹҒбһӣбһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бһ¶бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӣбҹ’бһўбһҮбһ¶бһ„бһҹбһҳбҹ’бһ—бһ¶бһҡбһҮбһ¶ API бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ
- бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Browser-Use бһҮбһ¶бһҳбһҪбһҷ Playwright бһ“бһ·бһ„ CDP бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһ”бһҡбһҮбһёбһңбһ·бһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҠбҹӮбһӣбһўбһ¶бһ…бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“
- бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Azure OpenAI vision бһ“бһ·бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбҹҖбһ”бһ…бҹҶбһҠбҹ„бһҷ Pydantic бһҠбһҫбһҳбҹ’бһ”бһёбһҠбһҖбһҹбҹ’бһҡбһ„бҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһүбҹ’бһҮбһёбһ–бһёбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҠбҹ„бһҷбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбһҖбһҳбҹ’бһҳ
- бһҖбҹҶбһҺбһҸбҹӢбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҠбҹӮбһӣбһӮбһҪбһҡбһ”бҹ’бһҡбһҫбһ“бһјбһңбһ”бҹӮбһ”бһ”бһ‘бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ-бһҠбҹҶбһ”бһјбһ„, бһҸбһҪбһўбһ„бҹ’бһӮ-бһҠбҹҶбһ”бһјбһ„, бһ¬бһ”бҹӮбһ”бһ”бһ‘бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҠбһ№бһ„бһ–бһёбһҡбһ”бҹҖбһ”бҹ–

- бһҖбҹҶбһҺбһҸбҹӢ Browser-Use бһҮбһ¶бһҳбһҪбһҷ Azure OpenAI бһ“бһ·бһ„ Playwright
- бһҹбһҳбҹ’бһҡбһ»бһҖбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҠбҹӮбһӣбһҡбһ»бһҖбһҡбһҖбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ–бһ·бһҸ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ¶бһҸбһ» UI бһҠбҹҸбһ…бһӣбҹҗбһҸ
- бһҠбһҖбһӣбһ‘бҹ’бһ’бһ•бһӣбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ–бһёбһҳбһ¶бһҸбһ·бһҖбһ¶бһ‘бҹҶбһ–бҹҗбһҡбһҠбҹӮбһӣбһҳбһҫбһӣбһғбһҫбһү бһ“бһ·бһ„бһ”бһҳбҹ’бһӣбҹӮбһ„бһңбһ¶бһ‘бҹ…бһҮбһ¶бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбһҖбҹ’бһҡбҹ„бһҳ
- бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҡбһңбһ¶бһ„бһ”бҹӮбһ”бһҹбҹҶбһҺбһ»бҹҶбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ“бһ·бһ„бһҸбһҪбһўбһ„бҹ’бһӮ бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ—бһ¶бһ–бһўбһ¶бһ…бһ…а№ҮаёҒбһ”бһ¶бһ“бһ“бҹғбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҡбһ»бһҖбһҡбһҖ

### бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҡбһ»бһҖбһҡбһҖ

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҡбһ»бһҖбһҡбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҹбһҖбһҳбҹ’бһҳ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ–бһҪбһҖбһӮбҹҒбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӮбҹ’бһҡбҹ„бһ„бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ№бһ„бһҳбһ¶бҹҶбһҮбһ¶бһ„бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҠбҹӮбһӣбһўбһ¶бһ…бһ бҹ… API бһҠбҹӮбһӣбһ”бһ¶бһ“бһҹбҹ’бһӮбһ¶бһӣбҹӢбһҸбҹӮбһҳбҹ’бһҠбһ„бҹ” бһҳбһ»бһ“бһ–бҹҒбһӣбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ–бһёбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹ’бһ“бһ»бһ„бһҹбҹҖбһңбһ—бҹ…бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ‘бҹ…бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбһ¶бһҡбһ„бһ¶бһҡбһ–бһ·бһҸ бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбҹҶбһ–бһёбһўбҹ’бһңбһёбһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһ¶бһ…бһҳбһҫбһӣбһғбһҫбһү, бһ…бһ»бһ… бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бҹ”

1. **бһҖбҹҶбһҺбһҸбҹӢбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһҡбһ»бһҖбһҡбһҖбҹ”** бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ”бҹ’бһҡбһјбһ бҹ’бһңбһ¶бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһ¬бһҹбҹҒбһ“бһ”бһ»бһҖ бһІбҹ’бһҷбһҳбһ¶бһ“бһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһңбһ”бҹ’бһ”бһ’бһҳбҹҢбһ‘бҹ…бһҸбҹӮ domain бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бҹ”
2. **бһ”бҹҶбһ”бҹӮбһҖбһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ–бһёбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бҹ”** бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ, бһўбһ¶бһ“, бһ“бһ·бһ„бһҠбһҖбһ–бҹҗбһҸбҹҚбһҳбһ¶бһ“бһҮбһ¶бһҳбһ»бһ“; бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбҹҶбһ бһ¶бһ“бһўбһ“бһ»бһҳбҹҗбһҸбһҒбһ¶бһ„бһҳбһ»бһҒ бһҳбһ»бһ“бһ–бҹҒбһӣбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһ‘бһҳбҹ’бһҡбһ„бҹӢ, бһ•бҹ’бһүбһҫбһҹбһ¶бһҡ, бһҖбһҖбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ, бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ‘бһ·бһү, бһӣбһ»бһ”бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶, бһ¬бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһӮбһҺбһ“бһёбҹ”
3. **бһҡбһҖбҹ’бһҹбһ¶бһҡбһҲбҹ’бһҳбҹ„бҹҮбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһІбҹ’бһҷбһ…бҹҒбһүбһ–бһёбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ бһ“бһ·бһ„бһ…бҹ’бһҡбһҖбһҸбһ¶бһҳбһҖбҹ’бһҡбҹ„бһҷбҹ”** бһҖбһ»бҹҶбһҠбһ¶бһҖбҹӢбһ–бһ¶бһҖбҹ’бһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ, бһ”бҹ’бһҡбһ¶бһҖбҹӢбһ”бһ„бҹӢ, бһӮбһ»бһӮбҹ’бһӮбһёбһҹбһҖбһҳбҹ’бһҳ, бһ¬бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҖбҹ’бһӣбҹӮбһ„бһҖбҹ’бһӣбһ¶бһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһҳбҹүбһјбһҠбҹӮбһӣбҹ” бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ бһ“бһ·бһ„бһӣбһ»бһ”бһңбһ¶бһӣбһҡбһ”бһҹбҹӢбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһ–бһёбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бҹ”
4. **бһ–бҹ’бһҷбһҪбһҡбһҳбһ¶бһҸбһ·бһҖбһ¶бһ‘бҹҶбһ–бҹҗбһҡбһҮбһ¶бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҳбһ·бһ“бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбҹ”** бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһўбһ¶бһ…бһҳбһ¶бһ“бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһҳбһ·бһ“бһҳбҹӮбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ” бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһӮбһҪбһҡбһҸбҹӮбһ—бҹ’бһӣбҹҒбһ…бһҒбҹ’бһӣбһ¶бһүбҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бһ‘бҹҶбһ–бҹҗбһҡбһҠбҹӮбһӣбһҹбҹ’бһ“бһҫбһІбҹ’бһҷбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһӮбҹ„бһӣбһҠбҹ…, бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, бһ”бһ·бһ‘бһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡ, бһ¬вҖӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”
5. **бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһҮбһ»бҹҶбһңбһ·бһүбһҮбҹҶбһ бһ¶бһ“бһ бһ¶бһ“бһ·бһ—бҹҗбһҷбҹ”** бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ URL бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“, бһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһ‘бҹҶбһ–бҹҗбһҡ, бһ’бһ¶бһҸбһ»бһҠбҹӮбһӣбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹ, бһҸбһҳбҹ’бһӣбҹғ, бһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣ, бһ“бһ·бһ„бһҹбһ„бҹ’бһҒбҹҒбһ”бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҮбһ¶бһҳбһҪбһҷбһҖбһјбһҠ бһҳбһ»бһ“бһҹбҹ’бһ“бһҫбһўбҹ„бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһўбһ“бһ»бһҳбҹҗбһҸбһҮбҹҶбһ бһ¶бһ“бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбҹ”
6. **бһҖбҹҶбһҺбһҸбҹӢбһҗбһңбһ·бһҖбһ¶ бһ“бһ·бһ„бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ”бһүбҹ’бһҲбһ”бҹӢбҹ”** бһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһ“бһҪбһ“бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–, бһҖбһ¶бһҡбһҹбһ¶бһҖбһ’бҹ’бһңбһҫбһҳбҹ’бһҠбһ„бһ‘бҹҖбһҸ, бһ•бҹ’бһ‘бһ¶бҹҶбһ„, бһ“бһ·бһ„бһ“бһ¶бһ‘бһёбһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бҹ” бһ”бһүбҹ’бһҲбһ”бҹӢбһ“бҹ…бһ–бҹҒбһӣбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ‘бҹҶбһ–бҹҗбһҡбһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһ”бһ“бҹ’бһҸбһ…бһ»бһ…бҹ”
7. **бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ…бҹҶбһҺбһҸбҹӢбһ—бһҹбҹ’бһҸбһ»бһҸбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ бһҳбһ·бһ“бһҳбҹӮбһ“бһӮбҹ’бһҡбһ”бҹӢбһҷбҹүбһ¶бһ„бһ‘бҹҒбҹ”** бһҡбһҖбҹ’бһҹбһ¶бһҹбһ„бҹ’бһҒбҹҒбһ”бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–, бһҳбҹүбҹ„бһ„, URL, бһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ’бһ¶бһҸбһ»бһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹ, бһ“бһ·бһ„бһҷбҹ„бһ„бһҡбһјбһ”бһ—бһ¶бһ– бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ…бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһҡбһҺбһёбһ”бһҡбһ¶бһҮбҹҗбһҷбһҠбҹ„бһҷбһҳбһ·бһ“бһ•бҹ’бһ‘бһ»бһҖбһҳбһ¶бһҸбһ·бһҖбһ¶бһ‘бҹҶбһ–бҹҗбһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹҶбһ„бһ¶бһҸбҹӢбҹ”

бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Airbnb, бһӣбҹҶбһ“бһ¶бҹҶбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһӮбһәбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бһүбҹ’бһҮбһё бһ“бһ·бһ„бһҠбһҖбһҸбһҳбҹ’бһӣбҹғбҹ” бһҖбһ¶бһҡбһ…бһјбһӣ, бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҳбҹ’бһ…бһ¶бһҹбҹӢбһ•бҹ’бһ‘бҹҮ, бһ¬бһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһҖбһҖбҹӢбһӮбһҪбһҡбһҸбҹӮбһҮбһ¶вҖӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһўбһ“бһ»бһҳбҹҗбһҸбһ…бҹӮбһҖбһ•бҹ’бһҹбҹҒбһ„бҹ”

- бһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһҸбһҳбҹ’бһӣбҹғ бһ“бһ·бһ„бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ—бһ¶бһ–бһўбһ¶бһ…бһҡбһҖбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„ E-commerce
- бһҖбһ¶бһҡбһҠбһҖбһҹбҹ’бһҡбһ„бҹӢбһҡбҹҖбһ”бһ…бҹҶбһ–бһёбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҠбҹ„бһҷбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбһҖбһҳбҹ’бһҳ
- бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ UI бһҠбҹӮбһӣбһҳбһ¶бһ“бһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ–бһёбһҳбһҫбһӣбһғбһҫбһү бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ
- бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„
- бһҖбһ¶бһҡбһ”бҹҶбһ–бҹҒбһүбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҸбһ¶бһҳбһҡбһҷбҹҲбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бһ¶бһ“

## бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ–бһ·бһ—бһ–бһ–бһ·бһҸбҹҲ Microsoft Project Opal

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡвҖӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҸбһјбһ… бһҸбҹҶбһ”бһ“бҹӢбһҳбһҪбһҷвҖӢбһ“бҹғ **бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бһ¶бһҹбҹӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ (CUA)** вҖ” бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһ”бһҫбһҖбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһҠбһјбһ…бһ“бҹҒбҹҮбһҠбһјбһ…бһҳбһ“бһ»бһҹбҹ’бһҹбһҳбҹ’бһ“бһ¶бһҖбҹӢбҹ” Microsoft бһҖбҹҶбһ–бһ»бһ„бһҷбһҖбһӮбҹҶбһ“бһ·бһҸбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ“бҹҒбҹҮбһ‘бҹ…бһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ **[Project Opal (Frontier)](https://support.microsoft.com/en-us/microsoft-365-copilot/get-started-with-project-opal-frontier)** бһҠбҹӮбһӣбһҮбһ¶бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„ Microsoft 365 Copilotбҹ”

бһҮбһ¶бһҳбһҪбһҷ Project Opal бһўбҹ’бһ“бһҖбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ–бһ·бһ’бһёбһҖбһ¶бһҡ бһ бһҫбһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ“бһ№бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбҹҶбһ“бһҪбһҹбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ **бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһӣбһҫ Windows 365 Cloud PC бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–**,бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бһҳбҹ’бһ„бһ¶бһҷбһӣбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ бһўбһ„бҹ’бһӮбһ—бһ¶бһ– бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһўбҹ’бһ“бһҖбҹ” бһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶ **бһўбһҹбҹҠбһ·бһ“бһҒбһҺбҹҲбһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһ–бһёбһҖбҹ’бһҡбҹ„бһҷ**, бһ бһҫбһҷбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбһ№бһҖбһ“бһ¶бҹҶ бһ¬бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹ…бһ–бҹҒбһӣбһҺбһ¶бһҳбһҪбһҷбһҖбҹҸбһ”бһ¶бһ“бҹ” бһҖбһ¶бһҡбһ„бһ¶бһҡбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹҶбһҺбһҫбһҡбһҹбһ»бҹҶбһҹбһҳбһ¶бһҮбһ·бһҖбһҖбҹ’бһҡбһ»бһҳбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–
- бһ”бҹ’бһҡбһҳбһјбһӣ бһ“бһ·бһ„бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ—бһҹбҹ’бһҸбһ»бһҸбһ¶бһ„бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ“бһ¶бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҶбҹ’бһӮбһ„бһҖбһ¶бһҡбһ„бһ¶бһҡ
- бһҡбҹҖбһ”бһ…бҹҶбһ”бһүбҹ’бһ бһ¶ IT (бһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҹбһ“бҹ’бһӣбһ№бһҖбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷ, бһ”бҹ’бһҡбһӮбһӣбҹӢбһҳбҹ’бһ…бһ¶бһҹбҹӢ, бһ”бһ·бһ‘бһ…бһҳбҹ’бһӣбһ„)
- бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ Excel бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹҖбһңбһ—бҹ…бһҹбҹ’бһҗбһ·бһҸбһ·бһ бһ·бһҡбһүбҹ’бһүбһңбһҸбҹ’бһҗбһ»

Opal бһҮбһ¶бһҖбһ¶бһҡбһҷбҹ„бһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһҫбһӣбһ–бһёбһңбһҸбҹ’бһҗбһ»бһ”бҹӮбһ”бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҳбһ¶бһ“ **бһҮбһјбһ“бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“** вҖ” бһ бһҫбһҷбһңбһ¶бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбҹ„бһ‘бһ“бһ—бһ¶бһ–бһҳбһҖбһ–бһёбһӮбҹҶбһ“бһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһҳбһ»бһ“бҹ—бҹ–

| **бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“ бһ“бһ·бһ„бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–** (бһҳбҹҒбһҡбҹҖбһ“ 06 бһ“бһ·бһ„ 18) | бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Windows 365 Cloud PC бһҠбҹӮбһӣбһ”бҹӮбһ„бһ…бҹӮбһҖ, бһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ (бһҖбһ¶бһҡвҖӢбһ…бһјбһӣвҖӢбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ¶бһҡбһ¶бҹҶбһ„ бһҸбһ¶бһҳбһҡбһҷбҹҲ Intune), бһ”бҹ’бһҡбһҫ *бһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ* бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹӮбһҸбҹҶбһ”бһ“бҹӢбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’, бһ“бһ·бһ„бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбҹ” |
| **бһ•бҹӮбһ“бһҖбһ¶бһҡбһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҒбһҪбһҡбһҖбҹ’бһ”бһ¶бһӣ** (бһҳбҹҒбһҡбҹҖбһ“ 07 бһ“бһ·бһ„ 09) | Opal бһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһҠбҹҶбһ”бһјбһ„, бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бһ“бҹ…бһҮбҹҶбһ бһ¶бһ“бҹ— бһ“бһ·бһ„бһ”бһүбҹ’бһҲбһ”бҹӢбһ”бһҫбһҡбһҖбһғбһҫбһүбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҹбһ„бҹ’бһҹбҹҗбһҷбҹ” |
| **бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–/бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһңбһ·бһү** (бһҳбҹҒбһҡбҹҖбһ“ 04) | **бһҮбҹҶбһ“бһ¶бһү** бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһҠбһҠбҹӮбһӣ (бһ“бһ¶бҹҶбһ…бһјбһӣбһ–бһёбһҜбһҖбһҹбһ¶бһҡ `.md` бһ¬бһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҳбһҪбһҷ Opal) бһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһңбһ·бһүбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бҹ” |

> **бһ—бһ¶бһ–бһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ–** Project Opal бһӮбһәбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бҹҒбһӣбһҠбҹҶбһ”бһјбһ„ [Frontier early access program](https://adoption.microsoft.com/copilot/frontier-program/) бһҮбһ¶бһҳбһҪбһҷбһҮбһ¶бһң Microsoft 365 Copilot бһ бһҫбһҷбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бҹ” бһҠбҹ„бһҷбһҹбһ¶бһҡбһңбһ¶бһҮбһ¶бһӣбһҖбҹ’бһҒбһҺбҹҲбһӣбһҖбҹ’бһҒбһҺбҹҲбһҡбһјбһ„бһҡбһјбһ„ Frontier бһҖбҹҶбһ бһјбһҡбһ“бҹҒбҹҮ бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҮбһ¶бһҳбһҪбһҷбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бҹ”
