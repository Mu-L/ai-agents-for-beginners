យើងនឹងដើរតួអ្នកឆ្ពោះទៅកម្មវិធី multi-agent ក្នុងភាសា C# ដែលបានសាងសង់លើគម្រោង Semantic Kernel។ អ្នកនឹងយល់ពីគំនិតពីក្រោយកម្មវិធីទ្រឹស្តី agentic យល់ពីលម្អិតនៃការអនុវត្ត និងរៀនរបៀបបញ្ចូល Azure Cosmos DB ជាឃ្លាំងទិន្នន័យសម្រាប់ករណីប្រើប្រាស់ជាច្រើន។
