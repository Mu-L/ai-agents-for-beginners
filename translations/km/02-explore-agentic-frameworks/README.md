# бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһӮбҹ’бһҡбҹ„бһ„бһҖбһ¶бһҡбһҸбҹҶбһҺбһ¶бһ„ AI

бһӮбҹ’бһҡбҹ„бһ„бһҖбһ¶бһҡбһҸбҹҶбһҺбһ¶бһ„ AI бһӮбһәбһҮбһ¶бһңбҹҒбһ‘бһ·бһҖбһ¶бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҡбһҪбһҳбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ…бҹҒбһү бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҸбҹҶбһҺбһ¶бһ„ AIбҹ” бһӮбҹ’бһҡбҹ„бһ„бһҖбһ¶бһҡбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ“бһјбһңбһҹбһҳбһ¶бһҹбһ—бһ¶бһӮ бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһҪбһ…бһ бһҫбһҷ бһҖбһ¶бһҡбһҸбҹҶбһҺбһ¶бһ„ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚ бһҠбҹӮбһӣбһҮбһҪбһҷбһҡбҹҶбһӣбһҸбҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбҹ”

бһӮбҹ’бһҡбҹ„бһ„бһҖбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҮбһҪбһҷбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҹбҹ’бһңбһёбһҸбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҸбҹӮбһҳбһҪбһҷбһҠбҹ„бһҷбһ•бҹ’бһҸбһӣбҹӢбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ•бҹ’бһҹбҹҶбһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҲбһҳбһҳбһ»бһҒбһ‘бһјбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҸбҹҶбһҺбһ¶бһ„ AIбҹ” бһңбһ¶бһ”бһ„бҹ’бһҖбһҫбһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ’бҹҶбһ‘бһјбһӣбһ¶бһҷ бһҹбһҳбҹ’бһҡбһ¶бһ…бһ¶бһҸбҹӢ бһ“бһ·бһ„бһ—бһ¶бһ–бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AIбҹ”

## бһҳбҹҒбһҡбҹҖбһ“бһ”бһҫбһҖбһ”бһҡ

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ“бһ№бһ„бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһўбҹҶбһ–бһёбҹ–

- бһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢбһҸбҹҶбһҺбһ¶бһ„ AI бһҮбһ¶бһўбҹ’бһңбһё бһ бһҫбһҷбһңбһ¶бһ’бҹ’бһңбһҫбһўбҹ„бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бһ¶бһ“бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ?
- бһҖбҹ’бһҡбһ»бһҳбһӮбһәбһ’бҹ’бһңбһҫбһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһңбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһҡбһј бһӣбҹҶбһ бһјбһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ· бһҖбҹҸбһҠбһјбһ…бһҮбһ¶бһҖбҹӮбһӣбһҳбҹ’бһўбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҸбҹҶбһҺбһ¶бһ„бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒ?
- бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣ Microsoft бһ”бһ„бҹ’бһҖбһҫбһҸ (<a href="https://aka.ms/ai-agents-beginners/ai-agent-service" target="_blank">Microsoft Foundry Agent Service</a> бһ“бһ·бһ„ <a href="https://learn.microsoft.com/azure/ai-services/openai/how-to/responses" target="_blank">Microsoft Agent Framework</a>)?
- бһҸбһҫбһҒбҹ’бһүбһ»бҹҶбһўбһ¶бһ…бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбҹҖбһңбһ—бҹ… Azure бһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ”бһ¶бһ“бһҸбҹ’бһҡбһ„бҹӢ бһҸбһ¶бһҳбһҡбһҷбҹҲбһҡбһ”бҹҖбһ”бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбһ‘бҹҒбһ¬?

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ“бҹғбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһӮбһәбһҮбһҪбһҷбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбҹ–

- бһҸбһҪбһ“бһ¶бһ‘бһёбһ“бҹғбһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢбһҸбҹҶбһҺбһ¶бһ„ AI бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ AIбҹ”
- бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢбһҸбҹҶбһҺбһ¶бһ„ AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҸбҹҶбһҺбһ¶бһ„бһҳбһ¶бһ“ бһ”бҹ’бһҡбһ¶бһҮбҹ’бһүбһ¶бҹ”
- бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҠбҹӮбһӣбһӮбһәбһ”бһ¶бһ“бһўбҹ„бһҷбһҠбҹ„бһҷбһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢбһҸбҹҶбһҺбһ¶бһ„ AIбҹ”
- бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„ Microsoft Agent Framework бһ“бһ·бһ„ Microsoft Foundry Agent Serviceбҹ”

## бһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢбһҸбҹҶбһҺбһ¶бһ„ AI бһҮбһ¶бһўбҹ’бһңбһё бһ бһҫбһҷбһңбһ¶бһўбһ¶бһ…бһўбҹ„бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ’бҹ’бһңбһҫбһўбҹ’бһңбһёбһ”бһ¶бһ“бһҒбҹ’бһӣбҹҮ?

бһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢ AI бһҠаёұа№үаёҮбһҠбһ№бһ„бһўбһ¶бһ…бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ”бһүбҹ’бһ…бһјбһӣ AI бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ”бҹ’бһҡбһҹбһҫбһҡбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

### бһ“бҹҒбҹҮбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бһ¶бһ“бһ®бһӮбһҪбһҡбһІбҹ’бһҷбһҖбһҸбҹӢбһ…бһ·бһҸбҹ’бһҸбһҸбһҫбһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢбһҸбҹҶбһҺбһ¶бһ„ AI?

``` python
import asyncio
import os

from agent_framework import tool
from agent_framework.foundry import FoundryChatClient
from azure.identity import AzureCliCredential


# бһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҖбҹӢбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡ
@tool(approval_mode="never_require")
def book_flight(date: str, location: str) -> str:
    """Book travel given location and date."""
    return f"Travel was booked to {location} on {date}"


async def main():
    provider = FoundryChatClient(
        project_endpoint=os.environ["AZURE_AI_PROJECT_ENDPOINT"],
        model=os.environ["AZURE_AI_MODEL_DEPLOYMENT_NAME"],
        credential=AzureCliCredential(),
    )
    agent = provider.as_agent(
        name="travel_agent",
        instructions="Help the user book travel. Use the book_flight tool when ready.",
        tools=[book_flight],
    )

    response = await agent.run("I'd like to go to New York on January 1, 2025")
    print(response)
    # бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһӣбһ‘бҹ’бһ’бһ•бһӣбҹ– бһҒбҹ’бһҷбһӣбҹӢбһ бҹ„бҹҮбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһ“бһ¶бһҷбҹүбһјбһңбһҷбҹүбһҖбһ“бҹ…бһҗбҹ’бһ„бҹғбһ‘бһё бҹЎ бһҒбҹӮбһҳбһҖбһҡбһ¶ бһҶбҹ’бһ“бһ¶бҹҶ бҹўбҹ бҹўбҹҘ бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһҖбҹӢбһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷбҹ” бһҹбһјбһҳбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҷбҹүбһ¶бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–! вңҲпёҸрҹ—Ҫ


if __name__ == "__main__":
```

## Microsoft Agent Framework (MAF)

Microsoft Agent Framework бһ•бҹ’бһҸбһӣбҹӢSDK бһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ”бҹҖбһ”бһҹбһ¶бһҳбһүбҹ’бһү бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҸбҹҶбһҺбһ¶бһ„ AI бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫ `FoundryChatClient`бҹ” бһңбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бһ„бҹ’бһҖбһҫбһҸбһҸбҹҶбһҺбһ¶бһ„бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ Azure OpenAI бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҖбҹ’бһ“бһ»бһ„бһҒбҹ’бһӣбһҪбһ“ бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ“бҹ’бһ‘бһ“бһ¶ бһ“бһ·бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ‘бҹҶбһ“бһҫбһ”бһҸбһ¶бһҳбһҡбһҷбҹҲбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ Azureбҹ”

**бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбҹҶбһҺбһ¶бһ„ AI бһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһҪбһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ бһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бһ¶бһ“ бһ“бһ·бһ„бһҹбһҳбһ¶бһҹбһ—бһ¶бһ–бһҡбһҪбһҳбһ•бҹ’бһҹбҹҶбһҮбһ¶бһҳбһҪбһҷбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ”

бһ“бҹҒбҹҮбһҮбһ¶бһӮбҹҶбһ“бһ·бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҡбһ”бһҹбҹӢ Microsoft Agent Frameworkбҹ–

- **бһҸбҹҶбһҺбһ¶бһ„**бҹ” бһҸбҹҶбһҺбһ¶бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһ¶бһҳбһҡбһҷбҹҲ `FoundryChatClient` бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһЎбһҫбһ„бһҮбһ¶бһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮ бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҺбҹӮбһ“бһ¶бҹҶ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбҹ” бһҸбҹҶбһҺбһ¶бһ„бһўбһ¶бһ…бҹ–
  - **бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбһ¶бһҡвҖӢвҖӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ** бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ Azure OpenAIбҹ”
  - **бһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ** бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ”бһҫбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бһҡбһ·бһ”бһ‘бһҹбһ“бҹ’бһ‘бһ“бһ¶бҹ”
  - **бһҡбһҖбҹ’бһҹбһ¶бһ—бһ¶бһ–бһҲбһҡбһӣбҹҶбһ“бһ№бһ„бһ“бҹғбһҹбһ“бҹ’бһ‘бһ“бһ¶** бһҡбһңбһ¶бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

  бһ“бҹҒбҹҮбһҮбһ¶ code snippet бһҳбһҪбһҷбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбҹҶбһҺбһ¶бһ„бҹ–

    ```python
    import os
    from agent_framework.foundry import FoundryChatClient
    from azure.identity import AzureCliCredential

    provider = FoundryChatClient(
        project_endpoint=os.environ["AZURE_AI_PROJECT_ENDPOINT"],
        model=os.environ["AZURE_AI_MODEL_DEPLOYMENT_NAME"],
        credential=AzureCliCredential(),
    )
    agent = provider.as_agent(
        name="my_agent",
        instructions="You are a helpful assistant.",
    )

    response = await agent.run("Hello, World!")
    print(response)
    ```

- **бһ§бһ”бһҖбһҡбһҺбҹҚ**бҹ” бһӮбҹ’бһҡбҹ„бһ„бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһҳбһ»бһҒбһ„бһ¶бһҡбһ—бһ¶бһҷбһҗбһ»бһ“бһҠбҹӮбһӣбһҸбҹҶбһҺбһ¶бһ„бһўбһ¶бһ…бһ бҹ…бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ” бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ“бҹ…бһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҸбҹҶбһҺбһ¶бһ„бҹ–

    ```python
    def get_weather(location: str) -> str:
        """Get the current weather for a location."""
        return f"The weather in {location} is sunny, 72\u00b0F."

    agent = provider.as_agent(
        name="weather_agent",
        instructions="Help users check the weather.",
        tools=[get_weather],
    ```

- **бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ…бҹ’бһҡбһҫбһ“бһҸбҹҶбһҺбһ¶бһ„**бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбҹҶбһҺбһ¶бһ„бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҮбҹҶбһ“бһ¶бһүбһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ¶бһҡбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбҹ–

    ```python
    planner = provider.as_agent(
        name="planner",
        instructions="Break down complex tasks into steps.",
    )

    executor = provider.as_agent(
        name="executor",
        instructions="Execute the planned steps using available tools.",
        tools=[execute_tool],
    )

    plan = await planner.run("Plan a trip to Paris")
    result = await executor.run(f"Execute this plan: {plan}")
    ```

- **бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ Azure**бҹ” бһӮбҹ’бһҡбҹ„бһ„бһҖбһ¶бһҡбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫ `AzureCliCredential` (бһ¬ `DefaultAzureCredential`) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ–бһёбһҡ бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһҠбҹ„бһҷбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ API keys бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ”

Microsoft Foundry Agent Service бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҗбҹ’бһҳбһёбһҳбһҪбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶбһ“бҹ… Microsoft Ignite 2024бҹ” бһңбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһҸбҹҶбһҺбһ¶бһ„ AI бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҢбҹӮбһӣбһҠбҹӮбһӣбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣ бһ”бһ¶бһ“бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“ бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ бҹ…бһ•бҹ’бһ‘бһ¶бһӣбҹӢ LLMs бһ•бҹ’бһ‘бһ¶бһӣбҹӢ (open-source) бһҠбһјбһ…бһҮбһ¶ Llama 3, Mistral бһ“бһ·бһ„ Cohereбҹ”

Microsoft Foundry Agent Service бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҳбҹҒбһҖбһ¶бһ“бһ·бһ…бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҠбҹҸбһҒбҹ’бһӣбһ¶бҹҶбһ„ бһ“бһ·бһ„бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһңбһ¶бһҹбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ”

бһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ Microsoft Agent Framework бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһҸбҹҶбһҺбһ¶бһ„бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбҹ”

бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ“бҹҒбҹҮбһҖбҹҶбһ–бһ»бһ„бһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҮбһ¶бһ”бҹ’бһҡбҹҖбһ”бһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҳбһ»бһ“ бһ“бһ·бһ„бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҸбҹҶбһҺбһ¶бһ„бһҮбһ¶бһҳбһҪбһҷ Python бһ“бһ·бһ„ C#бҹ”

бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Microsoft Foundry Agent Service Python SDK бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбҹҶбһҺбһ¶бһ„бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ–

```python
import asyncio
from azure.identity import DefaultAzureCredential
from azure.ai.projects import AIProjectClient

# бһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚ
def get_specials() -> str:
    """Provides a list of specials from the menu."""
    return """
    Special Soup: Clam Chowder
    Special Salad: Cobb Salad
    Special Drink: Chai Tea
    """

def get_item_price(menu_item: str) -> str:
    """Provides the price of the requested menu item."""
    return "$9.99"


async def main() -> None:
    credential = DefaultAzureCredential()
    project_client = AIProjectClient.from_connection_string(
        credential=credential,
        conn_str="your-connection-string",
    )

    agent = project_client.agents.create_agent(
        model="gpt-4.1-mini",
        name="Host",
        instructions="Answer questions about the menu.",
        tools=[get_specials, get_item_price],
    )

    thread = project_client.agents.create_thread()

    user_inputs = [
        "Hello",
        "What is the special soup?",
        "How much does that cost?",
        "Thank you",
    ]

    for user_input in user_inputs:
        print(f"# User: '{user_input}'")
        message = project_client.agents.create_message(
            thread_id=thread.id,
            role="user",
            content=user_input,
        )
        run = project_client.agents.create_and_process_run(
            thread_id=thread.id, agent_id=agent.id
        )
        messages = project_client.agents.list_messages(thread_id=thread.id)
        print(f"# Agent: {messages.data[0].content[0].text.value}")


if __name__ == "__main__":
    asyncio.run(main())
```

### бһӮбҹҶбһ“бһ·бһҸбһҹбҹ’бһ“бһјбһӣ

Microsoft Foundry Agent Service бһҳбһ¶бһ“бһӮбҹҶбһ“бһ·бһҸбһҹбҹ’бһ“бһјбһӣбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

- **бһҸбҹҶбһҺбһ¶бһ„**бҹ” Microsoft Foundry Agent Service бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷ Microsoft Foundryбҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Microsoft Foundry, бһҸбҹҶбһҺбһ¶бһ„ AI бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶ "microservice" бһҶбҹ’бһӣбһ¶бһҸ бһҠбҹӮбһӣбһўбһ¶бһ…бһҶбҹ’бһӣбһҫбһҷбһҹбҹҶбһҺбһҪбһҡ (RAG), бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳ бһ¬бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳбһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбҹ” бһңбһ¶бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ“бҹҒбҹҮбһҠбҹ„бһҷбһ”бһүбҹ’бһ…бһјбһӣбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҢбҹӮбһӣ AI бһҹбҹҶбһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ—бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҸбҹҶбһҺбһ¶бһ„бһҳбһҪбһҷбҹ–

    ```python
    agent = project_client.agents.create_agent(
        model="gpt-4.1-mini",
        name="my-agent",
        instructions="You are helpful agent",
    ```

    бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮ бһҸбҹҶбһҺбһ¶бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣ `gpt-4.1-mini`, бһҲбҹ’бһҳбҹ„бҹҮ `my-agent` бһ“бһ·бһ„бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҺбҹӮбһ“бһ¶бҹҶ `You are helpful agent`бҹ” бһҸбҹҶбһҺбһ¶бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ–бһ¶бһҖбҹӢбһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ”бһҖбһҹбҹ’бһҡбһ¶бһҷбһҖбһјбһҠбҹ”

- **Thread бһ“бһ·бһ„бһҹбһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“**бҹ” Thread бһӮбһәбһҮбһ¶бһӮбҹҶбһ“бһ·бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҳбһҪбһҷбһ‘бҹҖбһҸбҹ” бһңбһ¶бһҸбҹҶбһҺбһ¶бһ„бһІбҹ’бһҷбһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶ бһ¬бһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһҳбһҪбһҷбһҡбһңбһ¶бһ„бһҸбҹҶбһҺбһ¶бһ„ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ” Threads бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһ¶бһҳбһҠбһ¶бһ“бһҮбҹҶбһ бһ¶бһ“бһ“бҹғбһҹбһ“бҹ’бһ‘бһ“бһ¶ бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бһҡбһ·бһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ thread бһҳбһҪбһҷбҹ–

    ```python
    thread = project_client.agents.create_thread()
    ```
