# рҹ”Қ бһҖбһ¶бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҳбһҫбһӣбһӣбһҫ Microsoft Agent Framework - бһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ (.NET)

## рҹ“Ӣ бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹ’бһҡбһ№бҹҮбһ“бҹғ Microsoft Agent Framework бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„ .NETбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ–бһёбһӣбҹҶбһ“бһ¶бҹҶбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘ agentic бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҒбһ¶бһ„ бһҶбҹ’бһӣбһ¶бһҸ бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбҹ’бһҡбҹ„бһҳбһҶбһ¶бһҖбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ C# бһ“бһ·бһ„бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ .NETбҹ”

### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбһҖбһғбһҫбһү

- рҹҸ—пёҸ **бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҸбһ·бһ—бһ¶бһ¶бһҡбһҹбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ**: бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҡбһ”бһҹбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ .NET
- рҹӣ пёҸ **бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ§бһ”бһҖбһҡбһҺбҹҚ**: бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡбһ…бҹҒбһүбһҖбҹ’бһҡбҹ…бһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ„бҹ’бһҡбһёбһҖбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–
- рҹ’¬ **бһӣбҹҶбһ бһјбһҡбһ“бҹғбһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶**: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ”бҹӢбҹ—бһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһ”бһҡбһ·бһ”бһ‘бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҒбҹ’бһҹбҹӮ
- рҹ”§ **бһӣбҹҶбһ“бһ¶бҹҶбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’**: бһ‘бһҳбҹ’бһӣбһ¶бһ”бҹӢбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ .NET

## рҹҺҜ бһӮбҹҶбһ“бһ·бһҸбһҹбҹ’бһ“бһјбһӣбһҠбҹӮбһӣбһӮбҹҒбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ

### бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ Agentic

- **бһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбһ—бһ¶бһ–**: бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҠбҹ„бһҷбһҹбҹҒбһҡбһёбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҹбһ„бҹ’бһҒбҹҒбһ” AI бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ .NET
- **бһ—бһ¶бһ–бһҶбҹ’бһӣбһҫбһҷбһҸбһ”**: бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ“бһ№бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҠбҹ„бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ
- **бһ—бһ¶бһ–бһҹбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһ»бһ“**: бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҷбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһӮбҹ„бһӣбһҠбҹ… бһ“бһ·бһ„бһ”бһҡбһ·бһ”бһ‘
- **бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбһ„бҹ’бһӮбһҳ**: бһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһ—бһ¶бһҹбһ¶бһ”бҹ’бһҡбһ–бҹғбһҺбһёбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҒбҹ’бһҹбҹӮбһҹбһ“бҹ’бһ‘бһ“бһ¶

### бһ’бһ¶бһҸбһ»бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ

- **AIAgent**: бһҖбһ¶бһҡбһҸбһҳбҹ’бһӣбһҫбһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ“бҹ’бһ‘бһ“бһ¶ (.NET)
- **бһҳбһ»бһҒбһ„бһ¶бһҡ бһ§бһ”бһҖбһҡбһҺбҹҚ**: бһ–бһ„бҹ’бһҡбһёбһҖбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ C# бһ“бһ·бһ„бһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲ
- **бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Azure OpenAI**: бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһҸбһ¶бһҳбһҡбһҷбҹҲ API бһ…бҹҶбһӣбһҫбһҷ Azure OpenAI
- **бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–**: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ…бҹҶбһҺбһ»бһ…бһ…бҹҒбһүбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ

## рҹ”§ бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢ

### бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“

- Microsoft Agent Framework (.NET)
- бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Azure OpenAI (Responses API)
- бһӣбҹҶбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһҡбһ…бһ“бһ¶бһўбһҸбһ·бһҗбһ·бһҮбһ“ Azure.AI.OpenAI
- бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҮбһ¶бһҳбһҪбһҷ DotNetEnv

### бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ

- бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бһ¶бһҹбһ¶бһ”бҹ’бһҡбһ–бҹғбһҺбһё
- бһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһҳбһҪбһҷбһӮбһ»бһҺбһӣбһҖбҹ’бһҒбһҺбҹҲ C#
- бһ…бҹҶбһӣбһҫбһҷбһҳбһ¶бһ“бһ”бһҡбһ·бһ”бһ‘бһҮбһ¶бһҳбһҪбһҷбһҹбһҳбҹҗбһҷбһҹбһ“бҹ’бһ‘бһ“бһ¶
- бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ–бһ„бҹ’бһҡбһёбһҖбһ”бһ¶бһ“бһҮбһ¶бһҳбһҪбһҷбһӣбҹҶбһ“бһ¶бҹҶ dependency injection

бһҳбһҫбһӣ [`02-dotnet-agent-framework.cs`](../../../../02-explore-agentic-frameworks/code_samples/02-dotnet-agent-framework.cs) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһјбһҠбһ–бҹҒбһүбһӣбҹҒбһүбҹ”

```csharp
#!/usr/bin/dotnet run

#:package Microsoft.Extensions.AI@10.*
#:package Microsoft.Agents.AI.OpenAI@1.*-*
#:package Azure.AI.OpenAI@2.1.0
#:package Azure.Identity@1.13.1

using System.ComponentModel;

using Microsoft.Agents.AI;
using Microsoft.Extensions.AI;

using Azure.AI.OpenAI;
using Azure.Identity;

// Tool Function: Random Destination Generator
// This static method will be available to the agent as a callable tool
// The [Description] attribute helps the AI understand when to use this function
// This demonstrates how to create custom tools for AI agents
[Description("Provides a random vacation destination.")]
static string GetRandomDestination()
{
    // List of popular vacation destinations around the world
    // The agent will randomly select from these options
    var destinations = new List<string>
    {
        "Paris, France",
        "Tokyo, Japan",
        "New York City, USA",
        "Sydney, Australia",
        "Rome, Italy",
        "Barcelona, Spain",
        "Cape Town, South Africa",
        "Rio de Janeiro, Brazil",
        "Bangkok, Thailand",
        "Vancouver, Canada"
    };

    // Generate random index and return selected destination
    // Uses System.Random for simple random selection
    var random = new Random();
    int index = random.Next(destinations.Count);
    return destinations[index];
}

// Azure OpenAI with the Responses API (stable v1 endpoint). Sign in with `az login`.
var azureEndpoint = Environment.GetEnvironmentVariable("AZURE_OPENAI_ENDPOINT")
    ?? throw new InvalidOperationException("AZURE_OPENAI_ENDPOINT is not set.");
var deployment = Environment.GetEnvironmentVariable("AZURE_OPENAI_DEPLOYMENT") ?? "gpt-5-mini";

var azureClient = new AzureOpenAIClient(new Uri(azureEndpoint), new AzureCliCredential());

// Define Agent Identity and Comprehensive Instructions
// Agent name for identification and logging purposes
var AGENT_NAME = "TravelAgent";

// Detailed instructions that define the agent's personality, capabilities, and behavior
// This system prompt shapes how the agent responds and interacts with users
var AGENT_INSTRUCTIONS = """
You are a helpful AI Agent that can help plan vacations for customers.

Important: When users specify a destination, always plan for that location. Only suggest random destinations when the user hasn't specified a preference.

When the conversation begins, introduce yourself with this message:
"Hello! I'm your TravelAgent assistant. I can help plan vacations and suggest interesting destinations for you. Here are some things you can ask me:
1. Plan a day trip to a specific location
2. Suggest a random vacation destination
3. Find destinations with specific features (beaches, mountains, historical sites, etc.)
4. Plan an alternative trip if you don't like my first suggestion

What kind of trip would you like me to help you plan today?"

Always prioritize user preferences. If they mention a specific destination like "Bali" or "Paris," focus your planning on that location rather than suggesting alternatives.
""";

// Create AI Agent with Advanced Travel Planning Capabilities
// Get the Responses client for the deployment and create the AI agent
// Configure agent with name, detailed instructions, and available tools
// This demonstrates the .NET agent creation pattern with full configuration
AIAgent agent = azureClient
    .GetChatClient(deployment)
    .AsAIAgent(
        name: AGENT_NAME,
        instructions: AGENT_INSTRUCTIONS,
        tools: [AIFunctionFactory.Create(GetRandomDestination)]
    );

// Create New Session for Context Management.
// Initialize a new conversation session to maintain context across multiple interactions
// Sessions enable the agent to remember previous exchanges and maintain conversational state
// This is essential for multi-turn conversations and contextual understanding
AgentSession session = await agent.CreateSessionAsync();

// Execute Agent: First Travel Planning Request
// Run the agent with an initial request that will likely trigger the random destination tool
// The agent will analyze the request, use the GetRandomDestination tool, and create an itinerary
// Using the session parameter maintains conversation context for subsequent interactions
await foreach (var update in agent.RunStreamingAsync("Plan me a day trip", session))
{
    await Task.Delay(10);
    Console.Write(update);
}

Console.WriteLine();

// Execute Agent: Follow-up Request with Context Awareness
// Demonstrate contextual conversation by referencing the previous response
// The agent remembers the previous destination suggestion and will provide an alternative
// This showcases the power of conversation sessions and contextual understanding in .NET agents
await foreach (var update in agent.RunStreamingAsync("I don't like that destination. Plan me another vacation.", session))
{
    await Task.Delay(10);
    Console.Write(update);
}
```
