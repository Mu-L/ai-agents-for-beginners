# AGENTS.md

## សារណៈគម្រោង

គិតផលនេះមាន "ភ្នាក់ងារ AI សម្រាប់អ្នកចាប់ផ្តើម" - វគ្គសិក្សាយ៉ាងពេញលេញដែលបង្រៀនអ្វីគ្រប់យ៉ាងដែលត្រូវការ ដើម្បីសាងសង់ភ្នាក់ងារ AI ។ វគ្គសិក្សារួមមានមេរៀនចំនួន ១៨ (លេខ ០០-១៨) ដែលគ្របដណ្តប់ពីមូលដ្ឋាន សំណុំរចនាសម្ព័ន្ធ លក្ខណៈសាងសង់ វេទិកា ការផ្សាយក្នុងផលិតកម្ម ភ្នាក់ងារមូលដ្ឋាន/ក្នុងឧបករណ៍ និងសុវត្ថិភាពរបស់ភ្នាក់ងារ AI ។

**បច្ចេកវិទ្យាសំខាន់ៗ៖**
- Python 3.12+
- សៀវភៅកំណត់ត្រា Jupyter សម្រាប់ការរៀនអន្តរកម្ម
- ស៊ុមបញ្ជះ AI៖ Microsoft Agent Framework (MAF)
- សេវាកម្ម Azure AI៖ Microsoft Foundry, Microsoft Foundry Agent Service V2

**រចនាសម្ព័ន្ធ៖**
- រចនាសម្ព័ន្ធផ្អែកលើមេរៀន (ថតលេខ ០០-១៥+)
- មេរៀននីមួយៗមាន៖ ឯកសារ README ឧទាហរណ៍កូដ (សៀវភៅកំណត់ត្រា Jupyter) និងរូបភាព
- គាំទ្រភាសាជាច្រើនតាមប្រព័ន្ធប្រែសម្រួលស្វ័យប្រវត្តិ
- សៀវភៅកំណត់ត្រា Python មួយសម្រាប់មេរៀនមួយដោយប្រើ Microsoft Agent Framework

## ពាក្យបញ្ជា ការតំឡើង

### សេចក្តីត្រូវការ មុន
- Python 3.12 ឬខ្ពស់ជាង
- ការជាវ Azure (សម្រាប់ Microsoft Foundry)
- Azure CLI ត្រូវបានដំឡើង និង Authenticate (`az login`)

### ការតំឡើងដំបូង

1. **ចម្លង ឬ fork វៃហ្សិតនេះ៖**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # ឬ
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **បង្កើត និងបើកប្រើបរិយាកាសវីជេវ Python៖**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # លើ Windows: venv\Scripts\activate
   ```

3. **ដំឡើងផ្នែកគាំទ្រ៖**
   ```bash
   pip install -r requirements.txt
   ```

4. **កំណត់បរិយាកាសអថេរ៖**
   ```bash
   cp .env.example .env
   # កែសម្រួល .env ជាមួយគន្លង API និងចំណុចចូលរបស់អ្នក
   ```

### អថេរបរិយាកាស ត្រូវការ

សម្រាប់ **Microsoft Foundry** (ត្រូវការ):
- `AZURE_AI_PROJECT_ENDPOINT` - ចំណុចបញ្ចប់គម្រោង Microsoft Foundry
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - ឈ្មោះចាក់បញ្ជូនម៉ូដែល (ឧទាហរណ៍ gpt-4.1-mini)

សម្រាប់ **Azure AI Search** (មេរៀន ០៥ - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - ចំណុចបញ្ចប់ Azure AI Search
- `AZURE_SEARCH_API_KEY` - API key របស់ Azure AI Search

ការផ្ទៀងផ្ទាត់៖ រារត `az login` មុនពេលរត់សៀវភៅកំណត់ត្រា (ប្រើ `AzureCliCredential`)។
