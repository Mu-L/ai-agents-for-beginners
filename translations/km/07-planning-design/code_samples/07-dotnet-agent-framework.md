# рҹҺҜ бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡ бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһ”бһ‘бһҮбһ¶бһҳбһҪбһҷ Azure OpenAI (Responses API) (.NET)

## рҹ“Ӣ бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһҹбҹҖбһңбһ—бҹ…бһ…бҹҶбһҺбһ¶бҹҶбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡ бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһ”бһ‘бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Microsoft Agent Framework бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ .NET бһҮбһ¶бһҳбһҪбһҷ Azure OpenAI (Responses API)бҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹҶбһ”бҹӮбһҖбһ”бһүбҹ’бһ бһ¶бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү, бһҡбҹҖбһ”бһ…бҹҶбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ бһҳбһ¶бһ“бһҮбҹҶбһ бһ¶бһ“бһ…бҹ’бһҡбһҫбһ“, бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҜбһҖбһҹбһ¶бһҡбһӣбһҳбҹ’бһўбһ·бһҸбһҮбһ¶бһҳбһҪбһҷбһӣбһҖбҹ’бһҒбһҺбҹҲбһҡбһ”бһҹбҹӢ .NET бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ”

## бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһјбһҠ

бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ–бҹҒбһүбһӣбҹҒбһүбһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„ `07-dotnet-agent-framework.cs` бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбҹ–

- бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһҮбһ¶бһҳбһҪбһҷ DotNetEnv
- бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ Azure OpenAI client бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `GetChatClient().AsAIAgent()`
- бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһӮбҹҶбһҡбһјбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ (Plan бһ“бһ·бһ„ TravelPlan) бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„ JSON
- бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҮбһ¶бһҳбһҪбһҷбһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ JSON schema
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбҹҶбһҺбһҫбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ…бһҳбҹ’бһӣбһҫбһҷбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–

## бһҷбһӣбҹӢбһҠбһ№бһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—

### бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҮбһ¶бһҳбһҪбһҷбһӮбҹҶбһҡбһјбһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбһҫбһҗбҹ’бһ“бһ¶бһҖбҹӢ C# бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһ•бҹӮбһ“бһҖбһ¶бһҡбҹ–

```csharp
public class Plan
{
    [JsonPropertyName("assigned_agent")]
    public string? Assigned_agent { get; set; }

    [JsonPropertyName("task_details")]
    public string? Task_details { get; set; }
}

public class TravelPlan
{
    [JsonPropertyName("main_task")]
    public string? Main_task { get; set; }

    [JsonPropertyName("subtasks")]
    public IList<Plan> Subtasks { get; set; }
}
```

### JSON Schema бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһІбҹ’бһҷбһҸбҹ’бһҡбһЎбһ”бҹӢбһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһ‘бҹ…бһ“бһ№бһ„ schema TravelPlanбҹ–

```csharp
ChatClientAgentOptions agentOptions = new()
{
    Name = AGENT_NAME,
    Description = AGENT_INSTRUCTIONS,
    ChatOptions = new()
    {
        ResponseFormat = ChatResponseFormatJson.ForJsonSchema(
            schema: AIJsonUtilities.CreateJsonSchema(typeof(TravelPlan)),
            schemaName: "TravelPlan",
            schemaDescription: "Travel Plan with main_task and subtasks")
    }
};
```

### бһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡ

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ‘бҹ…бһҮбһ¶вҖӢбһҡбһҠбҹ’бһӢбһҹбһ—бһ¶ бһ”бҹ’бһҡбһӮбһӣбҹӢбһ…бҹҶбһ“бһҪбһ“бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбҹ–

- FlightBooking: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҖбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҷбһ“бҹ’бһҸбһ бҹ„бҹҮ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ бҹ„бҹҮбһ бһҫбһҡ
- HotelBooking: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҖбҹӢбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡ
- CarRental: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҖбҹӢбһҡбһҗбһҷбһ“бҹ’бһҸ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҮбһҪбһӣбһҡбһҗбһҷбһ“бҹ’бһҸ
- ActivitiesBooking: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҖбҹӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ•бҹ’бһҹбҹҒбһ„бҹ—
- DestinationInfo: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһӮбҹ„бһӣбһҠбҹ…
- DefaultAgent: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбҹҶбһҺбһҫбһ‘бһјбһ‘бҹ…

## бһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖ

бһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҹбҹҶбһҺбһҫбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡ бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ“бһ№бһ„бһңбһ·бһ—бһ¶бһӮбһҹбҹҶбһҺбһҫ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһ–бһ·бһҹбҹҒбһҹ бһҸбһ¶бһҳбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ JSON бһҠбҹӮбһӣбһҹбҹ’бһҡбһ”бһҸбһ¶бһҳ schema TravelPlanбҹ”
