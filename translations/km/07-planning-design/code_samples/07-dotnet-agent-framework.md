# рҹҺҜ бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡ бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҮбһ¶бһҳбһҪбһҷ Azure OpenAI (Responses API) (.NET)

## рҹ“Ӣ бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһҹбҹҖбһңбһ—бҹ…бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡ бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҮбһ¶бһҹбҹ’бһҡбҹҒбһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбһ¶бһҮбҹ’бһүбһ¶ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Microsoft Agent Framework бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ .NET бһҮбһ¶бһҳбһҪбһҷ Azure OpenAI (Responses API)бҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ’бҹ’бһңбһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹҶбһ”бҹӮбһҖбһ”бһүбҹ’бһ бһ¶бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү, бһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһҮбҹҶбһ бһ¶бһ“, бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһӣбҹҶбһ“бһ¶бҹҶбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶бһҳбһҪбһҷбһҳбһ»бһҒбһ„бһ¶бһҡбһҮбһ¶бһҹбҹ’бһҡбҹҒбһ…бһҡбһ”бһҹбҹӢ .NETбҹ•

## бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһјбһҠ

бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ–бҹҒбһүбһӣбҹҒбһүбһҳбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„ `07-dotnet-agent-framework.cs` бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбҹ–

- бһҖбһ¶бһҡбһӣбҹ„бһҸбһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҮбһ¶бһҳбһҪбһҷ DotNetEnv
- бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһўбһҸбһ·бһҗбһ·бһҮбһ“ Azure OpenAI бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `GetChatClient().AsAIAgent()`
- бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһҡбһјбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ (Plan бһ“бһ·бһ„ TravelPlan) бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҹбҹҠбҹҒбһҡбһёбҹҗбһҷ JSON
- бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҮбһ¶бһҳбһҪбһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ JSON
- бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбҹҶбһҺбһҫбһҡвҖӢбһ•бҹӮбһ“бһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ…бһҳбҹ’бһӣбһҫбһҷбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–

## бһӮбҹҶбһ“бһ·бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—

### бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡбһҮбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҮбһ¶бһҳбһҪбһҷбһӮбҹҶбһҡбһјбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡвҖӢбһ”бҹ’бһҡбһҫбһҗбҹ’бһ“бһ¶бһҖбҹӢ C# бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһӣбһ‘бҹ’бһ’бһ•бһӣбһ•бҹӮбһ“бһҖбһ¶бһҡбҹ–

```csharp
public class Plan
{
    [JsonPropertyName("assigned_agent")]
    public string? Assigned_agent { get; set; }

    [JsonPropertyName("task_details")]
    public string? Task_details { get; set; }
}

public class TravelPlan
{
    [JsonPropertyName("main_task")]
    public string? Main_task { get; set; }

    [JsonPropertyName("subtasks")]
    public IList<Plan> Subtasks { get; set; }
}
```

### бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ JSON бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһұбҹ’бһҷбһҸбһ”бһ”бһүбҹ’бһҮбһјбһ“бһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһҸбһ¶бһҳбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ TravelPlanбҹ–

```csharp
ChatClientAgentOptions agentOptions = new()
{
    Name = AGENT_NAME,
    Description = AGENT_INSTRUCTIONS,
    ChatOptions = new()
    {
        ResponseFormat = ChatResponseFormatJson.ForJsonSchema(
            schema: AIJsonUtilities.CreateJsonSchema(typeof(TravelPlan)),
            schemaName: "TravelPlan",
            schemaDescription: "Travel Plan with main_task and subtasks")
    }
};
```

### бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҺбҹӮбһ“бһ¶бҹҶбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бҹӮбһ“бһҖбһ¶бһҡ

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбһҫбһҮбһ¶бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ бһҠбһ¶бһҖбҹӢбһ”бҹҒбһҹбһҖбһҖбһҳбҹ’бһҳбһ‘бҹ…бһ—бһ¶бһӮбһёбһҡбһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ–

- FlightBooking: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбһҖбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҷбһ“бҹ’бһҸбһ бҹ„бҹҮ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҷбһ“бҹ’бһҸбһ бҹ„бҹҮ
- HotelBooking: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбһҖбҹӢбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡ
- CarRental: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбһҖбҹӢбһҡбһҗбһҷбһ“бҹ’бһҸ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҮбһҪбһӣбһҡбһҗбһҷбһ“бҹ’бһҸ
- ActivitiesBooking: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбһҖбҹӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–
- DestinationInfo: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹҶбһҺбһҫбһҡбһҖбҹҶбһҹбһ¶бһ“бҹ’бһҸ
