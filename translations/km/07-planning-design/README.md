# бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһ•бҹӮбһ“бһҖбһ¶бһҡ

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ“бһ№бһ„бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҠбһ”бҹӢ

* бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбһҪбһҳбһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢ бһ“бһ·бһ„бһ”бҹҶбһ”бҹӮбһҖбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҸбһјбһ…бҹ—бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ¶бһ“бҹ”
* бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҠбҹӮбһӣбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“ бһ“бһ·бһ„бһўбһ¶бһ…бһ…бһ¶бһ”бҹӢбһҷбһҖбһҠбҹ„бһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһ¶бһ“бһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһҳбһ»бһ“бҹ”
* бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҡбһ”бҹҖбһ”бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҠбҹӮбһӣбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ”бһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҠбҹӮбһӣбһҳбһ·бһ“бһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһ¶бһ“бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбҹ–

* бһҖбҹҶбһҺбһҸбҹӢ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбһҪбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбҹғбһҹбһҺбһёбһҷбҹҚ AI бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһңбһ¶бһҠбһ№бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһҗбһ¶бһҸбҹ’бһҡбһјбһңбһҹбһҳбҹ’бһҡбҹҒбһ…бһўбҹ’бһңбһёбҹ”
* бһ”бҹҶбһ”бҹӮбһҖбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶ бһ•бҹ’бһ“бҹӮбһҖбһҸбһјбһ…бҹ—бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ¶бһ“ бһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһ–бһҪбһҖбһңбһ¶бһ‘бҹ…бһҮбһ¶бһ…бҹҶбһҺбһ„бһҮбһҫбһ„бһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҳбһ¶бһ“бһ бҹҒбһҸбһ»бһ•бһӣбҹ”
* бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ (бһ§. бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ¬бһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ) бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһ–бҹҒбһӣбһҺбһ¶ бһ“бһ·бһ„бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ–бһҪбһҖбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ“бһ·бһ„бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҳбһ·бһ“бһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһҠбҹӮбһӣбһҖбһҫбһҸбһЎбһҫбһ„бҹ”
* бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһӣбһ‘бҹ’бһ’бһ•бһӣбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһ…бһ»бҹҮ бһҸбһҳбҹ’бһӣбһ¶бһ—бһ¶бһ–бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ…бҹҶбһ–бҹ„бҹҮбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ”бҹ’бһҡбһҹбһҫбһҡбҹ”

бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһӣбҹҶбһҠбһ¶бһ”бҹӢбһ“бҹҒбҹҮбһўбһ¶бһ…бһұбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ¬бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ“бһёбһҳбһҪбһҷбҹ—бҹ” бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһҪбһҷбһўбһ¶бһ…бһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҮбһҳбҹ’бһҡбһҫбһҹбһҮбһ·бҹҮбһҷбһ“бҹ’бһҸбһ бҹ„бҹҮбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһҖбһҖбҹӢбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡ бһӣбҹ” бһ бһҫбһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹҶбһҺбһ¶бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһҪбһҷбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹӮбһ“бһҖбһ¶бһҡбһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбҹ”

бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ“бҹҒбҹҮбһҖбҹҸбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҠбҹ„бһҷбһЎбҹӮбһҖбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбҹҶбһ“бһ¶бһүбһ…бҹҶбһ–бҹ„бҹҮбһҖбһ¶бһҡбһ•бҹ’бһҠбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбһўбһ¶бһ бһ¶бһҡ бһ¬бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢбһ“бһ·бһ„бһӣбһҫбһҖбһҖбһҳбҹ’бһ–бһҹбҹӢбһ•бҹӮбһ“бһҖбһ¶бһҡбһҮбһ¶бһҗбҹ’бһҳбһёбһ“бҹ…бһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷбҹ”

### бһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶбҹ— (LLMs) бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ (бһ§. JSON) бһҠбҹӮбһӣбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ¬бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбҹ’бһҡбҹ„бһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһ·бһҹбҹҒбһҹбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһ—бһ¶бһ–бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһҷбһҫбһ„бһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһўбҹҶбһ–бһҫбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһ•бҹӮбһ“бһҖбһ¶бһҡбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Python бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ”бһ„бҹ’бһ бһ¶бһүбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ‘бҹ…бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҸбһјбһ…бҹ— бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ–

```python
from pydantic import BaseModel

    assigned_agent: AgentEnum  # бһҷбһҫбһ„бһ…бһ„бҹӢбһ•бҹ’бһҠбһӣбҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбһІбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ

class TravelPlan(BaseModel):
    main_task: str
    subtasks: List[TravelSubTask]
    is_greeting: bool

provider = FoundryChatClient(
    project_endpoint=os.environ["AZURE_AI_PROJECT_ENDPOINT"],

    credential=AzureCliCredential(),

# бһҖбҹҶбһҺбһҸбҹӢбһҹбһ¶бһҡвҖӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ
system_prompt = """You are a planner agent.
    Your job is to decide which agents to run based on the user's request.
    Provide your response in JSON format with the following structure:

    Below are the available agents specialised in different tasks:
    - FlightBooking: For booking flights and providing flight information
    - HotelBooking: For booking hotels and providing hotel information
    - CarRental: For booking cars and providing car rental information
    - ActivitiesBooking: For booking activities and providing activity information

user_message = "Create a travel plan for a family of 2 kids from Singapore to Melbourne"
```

### бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ—бһ¶бһ–бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“

бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮ бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҹбһҳбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ·бһҳбҹӮбһ“бһ‘бһ·бһҖбһ‘бһ‘бһҪбһӣбһҹбҹҶбһҺбһҫбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ (бһ§. "бһҒбҹ’бһүбһ»бҹҶбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ•бҹӮбһ“бһҖбһ¶бһҡбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбҹ”")бҹ”

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһ•бҹӮбһ“бһҖбһ¶бһҡбһ“бҹ„бҹҮбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ:

* бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ•бҹӮбһ“бһҖбһ¶бһҡбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡбҹ– бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡбһҷбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ (бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“) бһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ”бһ¶бһ“бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҷбһҫбһ„бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҒбҹ’бһҹбҹӮбһ•бҹӮбһ“бһҖбһ¶бһҡбһҠбҹӮбһӣбһўбһ¶бһ…бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҢбһёбһҺбһ¶бһҳбһ·бһ…бҹ” бһӣбһ‘бҹ’бһ’бһ•бһӣбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһ•бҹӮбһ“бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… бһ“бһ·бһ„бһ…бҹӮбһ„бһҸбҹҶбһҺбһ¶бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбҹ” бһӮбҹҒбһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ/бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бҹ” бһӣбһҫбһҹбһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣбһӣбҹҶбһ“бһ¶бҹҶбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһҶбҹ’бһӣбһ»бҹҮбһ”бһүбҹ’бһ…бһ¶бҹҶбһ„ бһўбҹ’бһ“бһҖбһҹбһ„бҹ’бһҒбҹҒбһ” бһ“бһ·бһ„бһҖбһ¶бһҡбһҮбһҮбҹӮбһҖбһңбҹҒбһ…бһҒбҹ’бһ…бһ”бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһңбһ„бҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһ”бҹҶбһҺбһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”

## бһ’бһ“бһ’бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳ

Magnetic One - бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһ“бһ·бһ„бһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбҹҒбһ…бһӣбһ‘бҹ’бһ’бһ•бһӣбһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбһӣбһҫбһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһҪбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҹбҹ’бһҗбһ¶бһ”бҹҗбһ“бҹ” бһҜбһҖбһҹбһ¶бһҡбҹ– <a href="https://www.microsoft.com/research/articles/magentic-one-a-generalist-multi-agent-system-for-solving-complex-tasks" target="_blank">Magnetic One</a>бҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ“бһ·бһ„бһ•бҹ’бһүбһҫбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ” бһӣбһҫбһҹбһ–бһёбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡ бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбҹҸбһ”бҹ’бһҡбһҫбһҖбһүбҹ’бһ…бһ”бҹӢбһҸбһ¶бһҳбһҠбһ¶бһ“бһҠбһҫбһҳбҹ’бһ”бһёбһҸбһ¶бһҳбһҠбһ¶бһ“бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһ“бҹғбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡбһЎбһҫбһ„бһңбһ·бһүбһ–бҹҒбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбҹ”
