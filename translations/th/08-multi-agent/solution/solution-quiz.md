# คำตอบแบบทดสอบความรู้

## คำถามที่ 1

**คำตอบ: A2.** ตัวแทนหลายตัวมีประโยชน์เมื่อภารกิจมีบทบาทเฉพาะทางที่แตกต่างกัน เครื่องมือหรือสิทธิ์ที่แยกออกจากกัน และการส่งต่อที่ชัดเจนซึ่งต้องการการประสานงาน บอทสนับสนุนอย่างง่ายมักจะคงเป็นตัวแทนตัวเดียวได้ ปริมาณคำร้องขอที่เหมือนกันจำนวนมากเป็นเรื่องของการปรับขนาดเป็นหลักและไม่จำเป็นต้องออกแบบแบบหลายตัวแทนด้วยตัวเอง

## คำถามที่ 2

**คำตอบ: A1.** ตัวแทนเดียวมักจะง่ายต่อการทดสอบ สังเกต และใช้งานเมื่อคำสั่งและเครื่องมือชุดเดียวสามารถจัดการกับงานได้ การเข้าถึงเครื่องมือหลายอย่างไม่ได้หมายความว่าจะต้องมีตัวแทนหลายตัว บทบาทแยกต่างหากที่มีสิทธิ์และข้อกำหนดการตรวจสอบต่างกันอาจทำให้ออกแบบหลายตัวแทนได้อย่างสมเหตุสมผล
