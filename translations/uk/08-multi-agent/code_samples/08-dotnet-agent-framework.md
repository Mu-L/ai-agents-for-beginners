# 🤝 Підприємницькі системи мультиагентних робочих процесів (.NET)

## 📋 Мета навчання

Цей нотатник демонструє, як створювати складні мультиагентні системи підприємницького рівня, використовуючи Microsoft Agent Framework у .NET з Azure OpenAI (Responses API). Ви навчитесь координувати кілька спеціалізованих агентів, які працюють разом через структуровані робочі процеси, використовуючи можливості .NET для підприємницьких рішень, готових до продуктивного використання.

**Підприємницькі можливості мультиагентних систем, які ви розробите:**
- 👥 **Співпраця агентів**: Типобезпечна координація агентів з перевіркою під час компіляції
- 🔄 **Оркестрація робочих процесів**: Декларативне визначення робочого процесу з використанням асинхронних патернів .NET
- 🎭 **Спеціалізація ролей**: Сильно типізовані особистості агентів і області експертизи
- 🏢 **Інтеграція для підприємств**: Готові до продуктивності патерни з моніторингом та обробкою помилок

## Що демонструє цей приклад

Ця мультиагентна система робочих процесів створює сервіс рекомендацій для готелів і подорожей із двома спеціалізованими агентами:

1. **Агент FrontDesk**: Туристичний агент, який надає рекомендації щодо заходів і місць
2. **Агент Concierge**: Переглядає рекомендації для забезпечення автентичного, нетуристичного досвіду

Агенти працюють разом у робочому процесі, де:
- Агент FrontDesk отримує початковий запит на подорож
- Агент Concierge переглядає та уточнює рекомендацію
- Робочий процес передає відповіді в реальному часі в потоці

## Ключові поняття

### Координація агентів
Приклад демонструє типобезпечну координацію агентів за допомогою Microsoft Agent Framework із перевіркою під час компіляції.

### Оркестрація робочих процесів
Використовує декларативне визначення робочого процесу з асинхронними патернами .NET для з’єднання кількох агентів у конвеєр.

### Потокова передача відповідей
Реалізує потокову передачу відповідей агентів у реальному часі за допомогою асинхронних перелічувачів і подієвої архітектури.

### Інтеграція для підприємств
Показує готові до продуктивності патерни, включно з:
- Налаштування через змінні середовища
- Безпечне управління обліковими даними
- Обробка помилок
- Асинхронна обробка подій
