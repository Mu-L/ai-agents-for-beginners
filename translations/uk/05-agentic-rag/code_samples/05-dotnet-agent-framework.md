# 🔍 Enterprise RAG з Microsoft Foundry (.NET)

## 📋 Навчальні цілі

Цей зошит демонструє, як створювати системи Retrieval-Augmented Generation (RAG) корпоративного рівня за допомогою Microsoft Agent Framework у .NET з Microsoft Foundry. Ви навчитеся створювати готових до виробництва агентів, які можуть шукати в документах і надавати точні, контекстно-залежні відповіді з корпоративною безпекою та масштабованістю.

**Можливості корпоративного RAG, які ви створите:**
- 📚 **Інтелект документів**: Розширена обробка документів за допомогою сервісів Azure AI
- 🔍 **Семантичний пошук**: Високопродуктивний векторний пошук з корпоративними функціями
- 🛡️ **Інтеграція безпеки**: Рольовий доступ і шаблони захисту даних
- 🏢 **Масштабована архітектура**: Готові до виробництва системи RAG з моніторингом

## 🎯 Архітектура корпоративного RAG

### Основні компоненти для підприємств
- **Microsoft Foundry**: Керована платформа корпоративного AI з безпекою та відповідністю
- **Стійкі агенти**: Стейтові агенти з історією розмов і керуванням контекстом
- **Управління векторним сховищем**: Індексація та пошук документів корпоративного рівня
- **Інтеграція ідентифікації**: Аутентифікація Azure AD і контроль доступу за ролями

### Переваги .NET для підприємств
- **Типобезпека**: Валідація часу компіляції для операцій RAG і структур даних
- **Асинхронна продуктивність**: Не блокуюча обробка документів і пошуку
- **Управління пам’яттю**: Ефективне використання ресурсів для великих колекцій документів
- **Шаблони інтеграції**: Рідна інтеграція сервісів Azure з ін'єкцією залежностей

## 🏗️ Технічна архітектура

### Конвеєр корпоративного RAG
```
Document Upload → Security Validation → Vector Processing → Index Creation
                      ↓                    ↓                  ↓
User Query → Authentication → Semantic Search → Context Ranking → AI Response
```

### Основні компоненти .NET
- **Azure.AI.Agents.Persistent**: Управління корпоративними агентами зі збереженням стану
- **Azure.Identity**: Інтегрована аутентифікація для безпечного доступу до сервісів Azure
- **Microsoft.Agents.AI.AzureAI**: Оптимізована під Azure реалізація агентської платформи
- **System.Linq.Async**: Високопродуктивні асинхронні операції LINQ
