Через докладний огляд коду ви побачите, як спроектувати та реалізувати робочий процес під керуванням агентів, який плавно інтегрує вибіркове доповнення до генерації (RAG), збереження пам’яті та динамічні переходи станів. Ми охопимо:

Співпраця агентів за допомогою LangGraph для структурованого розуміння
Зберігання історії чату в режимі реального часу з Azure Cosmos DB — тієї ж бази даних, що забезпечує історію чату в ChatGPT, найшвидше зростаючому застосунку агенту ШІ в історії
Пошук векторів для отримання знань з підтримкою вбудованих функцій Cosmos DB
Асинхронні можливості FastAPI для забезпечення чутливих і масштабованих взаємодій
Наприкінці сеансу ви отримаєте чіткий план створення та розгортання своїх масштабованих багатоагентних хмарних додатків, які використовують сучасний ШІ та хмарну інфраструктуру.
