# 🔄 Alap ügynök munkafolyamatok Azure OpenAI-val (Responses API) (.NET)

## 📋 Munkafolyamat-orkesztrációs útmutató

Ez a jegyzetfüzet bemutatja, hogyan lehet összetett **ügynök munkafolyamatokat** létrehozni a Microsoft Agent Framework .NET-hez és az Azure OpenAI (Responses API) segítségével. Megtanulhatod, hogyan készíts több lépésből álló üzleti folyamatokat, ahol az AI ügynökök együttműködnek komplex feladatok megvalósításában strukturált orkestrációs minták segítségével.

## 🎯 Tanulási célok

### 🏗️ **Munkafolyamat-architektúra alapjai**
- **Munkafolyamat-építő**: Összetett, többlépéses AI folyamatok tervezése és orkestrálása
- **Ügynök koordináció**: Több specializált ügynök összehangolása a munkafolyamatokon belül
- **Azure OpenAI (Responses API)**: Az Azure OpenAI Responses API kihasználása munkafolyamatokban
- **Vizualizált munkafolyamat-tervezés**: Munkafolyamat struktúrák létrehozása és vizualizálása a jobb megértés érdekében

### 🔄 **Folyamatorchesztrációs minták**
- **Szekvenciális feldolgozás**: Több ügynöki feladat logikus sorrendben való láncolása
- **Állapotkezelés**: Kontextus és adatáramlás fenntartása a munkafolyamat fázisai között
- **Hibakezelés**: Robusztus hibajavítás és munkafolyamat-rugalmasság megvalósítása
- **Teljesítményoptimalizálás**: Hatékony munkafolyamatok tervezése vállalati szintű műveletekhez

### 🏢 **Vállalati munkafolyamat alkalmazások**
- **Üzleti folyamat automatizálás**: Összetett szervezeti munkafolyamatok automatizálása
- **Tartalomgyártási pipeline**: Szerkesztési munkafolyamatok felülvizsgálati és jóváhagyási fázisokkal
- **Ügyfélszolgálati automatizálás**: Többlépéses ügyfélmegkeresés megoldás

## ⚙️ Előfeltételek és beállítás

### 📦 **Szükséges NuGet csomagok**

Ez a munkafolyamat bemutató több kulcsfontosságú .NET csomagot használ:

```xml
<!-- Core AI Framework -->
<PackageReference Include="Microsoft.Extensions.AI" Version="10.*" />

<!-- Azure OpenAI (Responses API) -->
<PackageReference Include="Azure.AI.OpenAI" Version="2.1.0" />
<PackageReference Include="Azure.Identity" Version="1.15.0" />

<!-- Agent Framework (Local Development) -->
<!-- Microsoft.Agents.AI.dll - Core agent abstractions -->
<!-- Microsoft.Agents.AI.OpenAI.dll - Azure OpenAI (Responses API) integration -->

<!-- Configuration and Environment -->
<PackageReference Include="DotNetEnv" Version="3.1.1" />
```

### 🔑 **Azure OpenAI konfiguráció**

**Környezeti beállítás (.env fájl):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-5-mini
```

**Azure OpenAI hozzáférés:**
1. Hozz létre egy Azure OpenAI erőforrást az Azure portálon
2. Telepíts egy modellt (például `gpt-5-mini`), és jegyezd fel a telepítés nevét
3. Jelentkezz be az `az login` segítségével, és állítsd be a környezeti változókat a fentiek szerint

### 🏗️ **Munkafolyamat-architektúra áttekintése**

```mermaid
graph TD
    A[Munkafolyamat-készítő] --> B[Ügynök nyilvántartó]
    B --> C[Munkafolyamat-végrehajtó motor]
    C --> D[Agent 1: Tartalomgenerátor
    C --> E[Agent 2: Tartalombíráló 
    D --> F[Munkafolyamat eredmények]
    E --> F
    G[Azure OpenAI (Válaszok API)] --> D
    G --> E
```

**Kulcskomponensek:**
- **WorkflowBuilder**: Fő orkestrációs motor a munkafolyamatok tervezéséhez
- **AIAgent**: Egyéni specializált ügynökök specifikus képességekkel
- **Azure OpenAI Client**: Azure OpenAI Responses API integráció
- **Execution Context**: Kezeli az állapotot és az adatáramlást a munkafolyamat szakaszai között

## 💻 A kód futtatása

A teljes implementáció elérhető a `01.dotnet-agent-framework-workflow-ghmodel-basic.cs` fájlban. Ez a fájl bemutatja:

1. **Környezeti konfiguráció** - Azure OpenAI konfiguráció betöltése `.env` fájlból
2. **Azure OpenAI kliens beállítása** - A kliens konfigurálása az Azure OpenAI Responses API használatához
3. **Ügynök létrehozása** - Specializált ügynökök definiálása (Front Desk és Concierge)
4. **Munkafolyamat-építő** - Többügynökös munkafolyamat létrehozása szekvenciális feldolgozással
5. **Munkafolyamat végrehajtása** - A munkafolyamat futtatása eredményfolyamattal

### 🚀 Példa futtatása

```bash
# Tegye a szkriptet futtathatóvá (Unix/Linux/macOS)
chmod +x 01.dotnet-agent-framework-workflow-ghmodel-basic.cs

# Futtassa a munkafolyamatot
./01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```

Vagy Windows rendszeren:
```powershell
dotnet run 01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```

### 📝 Várt eredmény

A munkafolyamat:
1. Elfogadja az utazási cél kérésedet („Szeretnék Párizsba menni”)
2. A Front Desk ügynök kezdeti ajánlást ad
3. A Concierge ügynök felülvizsgálja és finomhangolja az ajánlást
4. A végső output megjeleníti a teljes beszélgetési folyamatot
