# ⏩ Folyamatos Ügynök Munkafolyamatok Azure OpenAI-val (Responses API) (.NET)

## 📋 Haladó Szekvenciális Feldolgozás Útmutató

Ez a jegyzetfüzet bemutatja a **szekvenciális munkafolyamat mintákat** a Microsoft Agent Framework .NET-hez és az Azure OpenAI-hoz (Responses API). Megtanulhatod, hogyan építs összetett, lépésről lépésre haladó feldolgozási csővezetékeket, ahol az ügynökök meghatározott sorrendben hajtanak végre feladatokat, minden szakasz az előző szakasz eredményeire építve.

## 🎯 Tanulási Célok

- **Lineáris Munkafolyamat Tervezés**: Lépésről lépésre haladó feldolgozási csővezetékek létrehozása egyértelmű függőségekkel
- **Állapotkezelés**: Kontextus és adatáramlás fenntartása a szekvenciális munkafolyamat szakaszok között

- **Vállalati Csővezeték Minták**: Termelésre kész szekvenciális feldolgozó rendszerek építése

## 💻 A Kód Futattása

A teljes implementáció megtalálható a `02.dotnet-agent-framework-workflow-ghmodel-sequential.cs` fájlban. Ez a fájl bemutat egy **háromfázisú bútor elemzési munkafolyamatot**:

1. **1. szakasz - Értékesítési Ügynök**: Elemez bútor képeket és vásárlási javaslatokat ad
2. **2. szakasz - Árazási Ügynök**: Részletes árképzési bontásokat és költségvetési opciókat nyújt
3. **3. szakasz - Árajánlat Ügynök**: Professzionális árajánlat dokumentumot generál Markdown formátumban

### 🏗️ **Munkafolyamat Architektúra**

```
Image Input → Sales Analysis → Price Estimation → Quote Generation → Final Output
```

Minden ügynök:
- Megkapja az előző szakasz kimenetét mint kontextust
- Az előző elemzésre épít speciális szakértelemmel
- Fenntartja a munkafolyamat folytonosságát állapotkezeléssel

### 🚀 Példa Futattása

**Előfeltételek:**
- Helyezz egy bútor képet a `../imgs/home.png` helyre (vagy módosítsd az `imgPath` változót)
- Állítsd be az `.env` fájlodat az Azure OpenAI végpontoddal és telepítéseddel, majd jelentkezz be az `az login` paranccsal

```bash
# Tegyük a szkriptet futtathatóvá (Unix/Linux/macOS)
chmod +x 02.dotnet-agent-framework-workflow-ghmodel-sequential.cs

# Futtassa a szekvenciális munkafolyamatot
./02.dotnet-agent-framework-workflow-ghmodel-sequential.cs
```

Vagy Windows rendszeren:
```powershell
dotnet run 02.dotnet-agent-framework-workflow-ghmodel-sequential.cs
```

### 📝 Várt Kimenet

A munkafolyamat:
1. **Értékesítési Ügynök**: Azonosítja a bútorokat a képről és javaslatokat ad
2. **Árazási Ügynök**: Részletes árképzést ad költségvetési szintekkel és vásárlási ajánlásokkal
3. **Árajánlat Ügynök**: Formázott árajánlat dokumentumot generál az összes összegzett információ alapján

A végső kimenet egy átfogó, professzionális bútor árajánlat lesz az elemzés alapján.
