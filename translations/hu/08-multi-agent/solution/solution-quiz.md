# Tudásellenőrző Válaszok

## 1. kérdés

**Válasz: A2.** Több ügynök hasznos, ha a feladatnak elkülönült szakértői szerepei vannak, külön eszközök vagy jogosultságok szükségesek, és egyértelmű átadások vannak, amelyeket koordinálni kell. ## 2. kérdés
