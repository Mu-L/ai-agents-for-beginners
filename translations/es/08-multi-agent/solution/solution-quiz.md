# Respuestas para la Verificación de Conocimiento

## Pregunta 1

**Respuesta: A2.** Varios agentes son útiles cuando una tarea tiene roles especializados distintos, herramientas o permisos separados, y entregas claras que necesitan coordinación. Un bot de soporte simple generalmente puede permanecer como un solo agente. Un alto volumen de solicitudes idénticas es principalmente una cuestión de escala y no requiere por sí solo un diseño de múltiples agentes.

## Pregunta 2

**Respuesta: A1.** Usualmente es más fácil probar, observar y operar un solo agente cuando un conjunto de instrucciones y herramientas puede manejar la tarea. El acceso a varias herramientas no requiere automáticamente varios agentes. Roles separados con diferentes permisos y requisitos de auditoría pueden justificar un diseño de múltiples agentes.
