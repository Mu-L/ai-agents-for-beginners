# Antwoorden Kenniscontrole

## Vraag 1

**Antwoord: A2.** Meerdere agenten zijn nuttig wanneer een taak duidelijke specialistische rollen, aparte gereedschappen of permissies, en duidelijke overdrachten heeft die gecoördineerd moeten worden. Een eenvoudige ondersteuningsbot kan meestal een enkele agent blijven. Een hoog volume van identieke verzoeken is voornamelijk een schaalbaarheidsprobleem en vereist op zich geen ontwerp met meerdere agenten.

## Vraag 2

**Antwoord: A1.** Een enkele agent is meestal gemakkelijker te testen, te observeren en te bedienen wanneer één set instructies en gereedschappen de taak kan afhandelen. Toegang tot meerdere gereedschappen vereist niet automatisch meerdere agenten. Gescheiden rollen met verschillende permissies en auditvereisten kunnen een ontwerp met meerdere agenten rechtvaardigen.
