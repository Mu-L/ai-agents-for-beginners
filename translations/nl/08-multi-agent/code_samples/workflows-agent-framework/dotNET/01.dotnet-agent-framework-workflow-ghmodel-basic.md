# 🔄 Basisagentworkflows met Azure OpenAI (Responses API) (.NET)

## 📋 Workflow Orkestratie Tutorial

Deze notebook toont aan hoe je geavanceerde **agentworkflows** bouwt met het Microsoft Agent Framework voor .NET en Azure OpenAI (Responses API). Je leert meervoudige zakelijke processen maken waarbij AI-agenten samenwerken om complexe taken te volbrengen via gestructureerde orkestratiepatronen.

### 🔑 **Azure OpenAI Configuratie**

**Omgevingsinstelling (.env bestand):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-5-mini
```

**Azure OpenAI Toegang:**
1. Maak een Azure OpenAI-resource aan in de Azure-portal
2. Implementeer een model (bijvoorbeeld `gpt-5-mini`) en noteer de implementatienaam
3. Log in met `az login` en configureer omgevingsvariabelen zoals hierboven getoond

## 💻 De code uitvoeren

De volledige implementatie is beschikbaar in `01.dotnet-agent-framework-workflow-ghmodel-basic.cs`. Dit bestand demonstreert:

1. **Omgevingsconfiguratie** - Laden van Azure OpenAI-configuratie uit `.env`-bestand
2. **Azure OpenAI Client Setup** - Configureren van de client voor het gebruik van Azure OpenAI Responses API
3. **Agentaanmaak** - Definiëren van gespecialiseerde agenten (Front Desk en Concierge)
4. **Workflow Builder** - Creëren van een multi-agent workflow met sequentiële verwerking
5. **Workflowuitvoering** - Het uitvoeren van de workflow met streamingresultaten

### 🚀 Het voorbeeld uitvoeren

```bash
# Maak het script uitvoerbaar (Unix/Linux/macOS)
chmod +x 01.dotnet-agent-framework-workflow-ghmodel-basic.cs

# Voer de workflow uit
./01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```

Of op Windows:
```powershell
dotnet run 01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```

### 📝 Verwachte uitvoer

De workflow zal:
1. Je reisbestemming accepteren ("Ik wil graag naar Parijs")
2. De Front Desk agent geeft een eerste aanbeveling
3. De Concierge agent beoordeelt en verfijnt de aanbeveling
4. De definitieve uitvoer toont de volledige gesprekstroom
