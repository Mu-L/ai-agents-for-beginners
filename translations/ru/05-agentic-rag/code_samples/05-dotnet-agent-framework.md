# 🔍 Корпоративный RAG с Microsoft Foundry (.NET)

## 📋 Цели обучения

Этот ноутбук демонстрирует, как создавать корпоративные системы Retrieval-Augmented Generation (RAG) с помощью Microsoft Agent Framework в .NET и Microsoft Foundry. Вы научитесь создавать готовых к производству агентов, которые могут искать по документам и предоставлять точные, контекстно-зависимые ответы с корпоративной безопасностью и масштабируемостью.

**Возможности корпоративного RAG, которые вы создадите:**
- 📚 **Аналитика документов**: Расширенная обработка документов с помощью Azure AI сервисов
- 🔍 **Семантический поиск**: Высокопроизводительный векторный поиск с корпоративными функциями
- 🛡️ **Интеграция безопасности**: Ролевая модель доступа и шаблоны защиты данных
- 🏢 **Масштабируемая архитектура**: Готовые к производству RAG системы с мониторингом

## 🎯 Архитектура корпоративного RAG

- **Microsoft Foundry**: Управляемая корпоративная платформа ИИ с безопасностью и соответствием требованиям

### Преимущества .NET для корпоративных решений
- **Типобезопасность**: Проверка во время компиляции для операций RAG и структур данных

## 🏗️ Техническая архитектура

### Корпоративный RAG конвейер
```
Document Upload → Security Validation → Vector Processing → Index Creation
                      ↓                    ↓                  ↓
User Query → Authentication → Semantic Search → Context Ranking → AI Response
```

### Основные компоненты .NET
- **Azure.AI.Agents.Persistent**: Управление корпоративными агентами с сохранением состояния
- **Azure.Identity**: Интегрированная аутентификация для безопасного доступа к Azure сервисам
- **Microsoft.Agents.AI.AzureAI**: Реализация агентского фреймворка, оптимизированного под Azure
- **System.Linq.Async**: Высокопроизводительные асинхронные LINQ операции

## ⚙️ Требования и настройка

**Среда разработки:**
- .NET 9.0 SDK или выше
- Visual Studio 2022 или VS Code с расширением C#
- Подписка Azure с доступом к Microsoft Foundry

**Необходимые NuGet пакеты:**
```xml
<PackageReference Include="Microsoft.Extensions.AI" Version="9.9.0" />
<PackageReference Include="Azure.AI.Agents.Persistent" Version="1.2.0-beta.5" />
<PackageReference Include="Azure.Identity" Version="1.15.0" />
<PackageReference Include="System.Linq.Async" Version="6.0.3" />
<PackageReference Include="DotNetEnv" Version="3.1.1" />
```

**Настройка аутентификации Azure:**
```bash
# Установите Azure CLI и выполните аутентификацию
az login
az account set --subscription "your-subscription-id"
```

**Конфигурация среды:**
* Конфигурация Microsoft Foundry (автоматически через Azure CLI)
* Убедитесь, что вы аутентифицированы в нужной подписке Azure

## Реализация кода

Полный рабочий пример кода к этому уроку доступен в `05-dotnet-agent-framework.cs`.

Для запуска примера:

```bash
# Сделайте скрипт исполняемым (Linux/macOS)
chmod +x 05-dotnet-agent-framework.cs
```


1. **Установку пакетов**: Установка необходимых NuGet пакетов для Azure AI Agents
2. **Конфигурацию среды**: Загрузка конечной точки Microsoft Foundry и настроек моделей
3. **Загрузку документа**: Загрузка документа для обработки RAG
4. **Создание хранилища векторов**: Создание векторного хранилища для семантического поиска
5. **Настройку агента**: Конфигурирование AI агента с возможностями поиска по файлам
6. **Выполнение запросов**: Запуск запросов к загруженному документу
