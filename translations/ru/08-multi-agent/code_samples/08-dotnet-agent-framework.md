# 🤝 Корпоративные многоагентные системы рабочих процессов (.NET)

## 📋 Цели обучения

В этом блокноте показано, как создавать сложные многоагентные системы корпоративного уровня с использованием Microsoft Agent Framework в .NET с Azure OpenAI (Responses API). Вы научитесь организовывать работу нескольких специализированных агентов, взаимодействующих через структурированные рабочие процессы, используя корпоративные возможности .NET для решений, готовых к промышленному использованию.

**Возможности многоагентной корпоративной системы, которые вы создадите:**
- 👥 **Сотрудничество агентов**: Координация агентов с типобезопасностью и проверкой во время компиляции
- 🔄 **Оркестрация рабочих процессов**: Декларативное определение рабочих процессов с использованием асинхронных шаблонов .NET
- 🎭 **Специализация ролей**: Сильно типизированные личности агентов и области экспертизы
- 🏢 **Интеграция в корпоративную среду**: Промышленные шаблоны с мониторингом и обработкой ошибок

## Что демонстрирует этот пример

Эта многоагентная система рабочих процессов создаёт сервис рекомендаций для путешествий в отели с двумя специализированными агентами:

1. **Агент на стойке регистрации (FrontDesk)**: Тревел-агент, предоставляющий рекомендации по мероприятиям и локациям
2. **Агент-консьерж (Concierge)**: Проверяет рекомендации для обеспечения аутентичных, не туристических впечатлений

Агенты работают вместе в рабочем процессе, где:
- Агент FrontDesk получает первоначальный запрос на путешествие
- Агент-консьерж проверяет и уточняет рекомендации
- Рабочий процесс транслирует ответы в реальном времени

## Ключевые концепции

### Координация агентов
Пример демонстрирует типобезопасную координацию агентов с использованием Microsoft Agent Framework с проверкой во время компиляции.

### Оркестрация рабочих процессов
Используется декларативное определение рабочих процессов с асинхронными шаблонами .NET для связывания нескольких агентов в конвейер.

### Потоковая передача ответов
Реализует потоковую передачу ответов агентов в реальном времени с использованием асинхронных перечислителей и событийно-ориентированной архитектуры.

### Интеграция в корпоративную среду
Показывает промышленные шаблоны, включая:
- Конфигурацию через переменные окружения
- Безопасное управление учётными данными
- Обработку ошибок
- Асинхронную обработку событий
