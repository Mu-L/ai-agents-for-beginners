# 🔍 Ентерпрајс РАГ са Microsoft Foundry (.NET)

## 📋 Циљеви учења

Овај нотебоок демонстрира како изградити системе за генерацију допуњену претрагом на ентерпрајс нивоу (RAG) користећи Microsoft Agent Framework у .NET уз Microsoft Foundry. Учићете како да креирате агенте спремне за производњу који могу претраживати документе и пружати тачне, контекстуално-свесне одговоре са ентерпрајс безбедношћу и скалабилношћу.

**Ентерпрајс РАГ могућности које ћете развити:**
- 📚 **Интелигенција докумената**: Напредна обрада докумената са Azure AI услугама
- 🔍 **Семантичка претрага**: Високопродуктивна векторска претрага са ентерпрајс функцијама
- 🛡️ **Интеграција безбедности**: Контрола приступа заснована на улогама и патерни заштите података
- 🏢 **Скалабилна архитектура**: Системи за RAG спремни за производњу са надзором

## 🎯 Ентерпрајс РАГ архитектура

### Кључне компоненте ентерпрајса
- **Microsoft Foundry**: Управљана платформа за ентерпрајс вештачку интелигенцију са безбедношћу и усаглашеношћу
- **Стални агенти**: Агенти са стањем, са историјом разговора и управљањем контекстом
- **Управљање векторском радњом**: Индексирање и претрага докумената на ентерпрајс нивоу
- **Интеграција идентитета**: Аутентификација Azure AD и контрола приступа заснована на улогама

### .NET предности за ентерпрајс
- **Врста безбедности**: Валидација у време компајлирања за RAG операције и структуре података
- **Асинхрона продуктивност**: Некомплетирајућа обрада докумената и претраживање
- **Управљање меморијом**: Ефикасна употреба ресурса за велике колекције докумената
- **Патерни интеграције**: Нативна интеграција Azure услуга са dependency injection

## 🏗️ Техничка архитектура

### Пипелајн за ентерпрајс RAG
```
Document Upload → Security Validation → Vector Processing → Index Creation
                      ↓                    ↓                  ↓
User Query → Authentication → Semantic Search → Context Ranking → AI Response
```

### Кључне .NET компоненте
- **Azure.AI.Agents.Persistent**: Управљање агената са перзистентношћу стања
- **Azure.Identity**: Интегрисана аутентификација за безбедан приступ Azure услугама
- **Microsoft.Agents.AI.AzureAI**: Оквир за агенте оптимизован за Azure
- **System.Linq.Async**: Високопродуктивне асинхроне LINQ операције

## ⚙️ Захтеви и подешавање

**Развојно окружење:**
- .NET 9.0 SDK или новији
- Visual Studio 2022 или VS Code са C# екстензијом
- Azure претплата са приступом Microsoft Foundry-у

**Потребни NuGet пакети:**
```xml
<PackageReference Include="Microsoft.Extensions.AI" Version="9.9.0" />
<PackageReference Include="Azure.AI.Agents.Persistent" Version="1.2.0-beta.5" />
<PackageReference Include="Azure.Identity" Version="1.15.0" />
<PackageReference Include="System.Linq.Async" Version="6.0.3" />
<PackageReference Include="DotNetEnv" Version="3.1.1" />
```

**Подешавање аутентификације у Azure-у:**
```bash
# Инсталирајте Azure CLI и аутентификујте се
az login
az account set --subscription "your-subscription-id"
```

**Конфигурација окружења:**
* Конфигурација Microsoft Foundry (аутоматски управљана преко Azure CLI)
* Проверите да сте аутентификовани на исправну Azure претплату
