# AGENTS.md

## Преглед пројекта

Ово складиште садржи "AI агенте за почетнике" - свеобухватни образовни курс који учи све што је потребно за изградњу AI агената. Курс се састоји из 18 лекција (бројаних 00-18) које покривају основе, образце дизајна, оквире, производну имплементацију, локалне/уређајне агенте и безбедност AI агената.

**Кључне технологије:**
- Python 3.12+
- Jupyter бележнице за интерактивно учење
- AI оквири: Microsoft Agent Framework (MAF)
- Azure AI услуге: Microsoft Foundry, Microsoft Foundry Agent Service V2

**Архитектура:**
- Структура заснована на лекцијама (00-15+ директоријуми)
- Свакa лекција садржи: README документацију, примере кода (Jupyter бележнице) и слике
- Подршка за више језика путем аутоматизованог система превода
- Једна Python бележница по лекцији користећи Microsoft Agent Framework

## Команде за припрему окружења

### Захтеви
- Python 3.12 или новији
- Azure претплата (за Microsoft Foundry)
- Azure CLI инсталиран и аутентификован (`az login`)

### Почетна припрема

1. **Клонирајте или направите форк репозиторијума:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # ИЛИ
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Креирајте и активирајте Python виртуелно окружење:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # На Windows: venv\Scripts\activate
   ```

3. **Инсталирајте зависности:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Подесите променљиве окружења:**
   ```bash
   cp .env.example .env
   # Измените .env са вашим API кључевима и крајним тачкама
   ```

### Обавезне променљиве окружења

За **Microsoft Foundry** (обавезно):
- `AZURE_AI_PROJECT_ENDPOINT` - крајња тачка Microsoft Foundry пројекта
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - име имплементације модела (нпр. gpt-5-mini)

За **Azure AI Search** (Лекција 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - крајња тачка Azure AI Search
- `AZURE_SEARCH_API_KEY` - Azure AI Search API кључ

Аутентикација: Покрените `az login` пре покретања бележница (користи `AzureCliCredential`).

Свакa лекција садржи више Jupyter бележница за различите оквире:

   - `*-python-agent-framework.ipynb` - Користећи Microsoft Agent Framework (Python)

- Захтева Azure претплату
- Користи `FoundryChatClient` за Agent Service V2 (агенти видљиви у Foundry порталу)
- Спремно за производњу са уграђеним надзором
- Образац имена датотека: `*-python-agent-framework.ipynb`

### Smoke-тестирање имплементираних агената

За лекције где је агент имплементиран као Microsoft Foundry хостирани агент (01, 04, 05, 16), репозиторијум садржи smoke-тест каталоге у `tests/` који се извршавају од стране `.github/workflows/smoke-test.yml` workflow-а преко [AI Smoke Test](https://github.com/marketplace/actions/ai-smoke-test) акције. Ово је лагана провера после имплементације (да ли је агент доступан и прати основна очекивања упита?), као допуна процесу процене у лекцијама 10 и 16. Погледајте [tests/README.md](./tests/README.md) за карту каталога-према-лекцији-према-агенту. Лекција 17 се покреће локално са Foundry Local и нема хостирану крајњу тачку, па се проверава непосредним покретањем бележнице.

## Додатне напомене

### Чести проблеми

1. **Неслагање верзије Python-а:**
   - Осигурајте да користите Python 3.12+
   - Неке пакете можда неће радити са старијим верзијама

2. **Променљиве окружења:**
   - Увек направите `.env` из `.env.example`
   - Не додајте `.env` фајл у комит (налази се у `.gitignore`)

   - Користите свежо виртуелно окружење

   - Неке бележнице могу захтевати додатне пакете наведене у њиховим markdown ћелијама

4. **Azure услуге:**
   - Azure AI услуге захтевају активну претплату
   - Неке функције су специфичне за регију

## Контекст специфичан за пројекат

### Подршка за више језика

Ово складиште користи аутоматизован систем превода:
- Подржано преко 50 језика
- Преводи у директоријумима `/translations/<lang-code>/`
- GitHub Actions workflow рукује ажурирањем превода
- Изворне датотеке су на енглеском у корену репозиторијума

### Структура лекција

Свакa лекција следи конзистентан образац:
1. Видео сличица са линком
2. Писани материјал лекције (README.md)
3. Примери кода у више оквира
4. Циљеви учења и претпоставке
5. Додатни ресурси за учење са линковима

### Именовање примера кода

Формат: `<lesson-number>-python-agent-framework.ipynb`
- `01-python-agent-framework.ipynb` - Лекција 1, MAF Python
- `14-sequential.ipynb` - Лекција 14, напредни MAF обрасци
- `16-python-agent-framework.ipynb` - Лекција 16, продукциони агент корисничке подршке
- `17-local-agent-foundry-local.ipynb` - Лекција 17, локални агент са Foundry Local + Qwen

### Специјални директоријуми

- `translated_images/` - Локализоване слике за преводе
- `images/` - Оригиналне слике за енглески садржај
- `.devcontainer/` - Конфигурација VS Code развојног контејнера
- `.github/` - GitHub Actions workflow-и и шаблони

### Зависности

Кључне пакете из `requirements.txt`:
- `agent-framework` - Microsoft Agent Framework

- `azure-ai-inference`, `azure-ai-projects` - Azure AI услуге
- `azure-identity` - Azure аутентикација (AzureCliCredential)
