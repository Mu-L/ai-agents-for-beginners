# AGENTS.md

## Преглед пројекта

Ово складиште садржи "AI агенте за почетнике" - обједињени образовни курс који подучава све што је потребно за изградњу AI агената. Курс се састоји од 18 лекција (бројаних 00-18) које покривају основе, дизајнерске шаблоне, оквире, производно постављање, локалне/на уређају агенте и безбедност AI агената.

**Кључне технологије:**
- Python 3.12+
- Jupyter ноутбуци за интерактивно учење
- AI оквири: Microsoft Agent Framework (MAF)
- Azure AI услуге: Microsoft Foundry, Microsoft Foundry Agent Service V2

**Архитектура:**
- Структура заснована на лекцијама (директоријуми 00-15+)
- Свака лекција садржи: README документацију, примерке кода (Jupyter ноутбуци) и слике
- Подршка за више језика преко аутоматизованог система превођења
- Један Python ноутбук по лекцији користећи Microsoft Agent Framework

## Команде за подешавање

### Предуслови
- Python 3.12 или новији
- Azure претплата (за Microsoft Foundry)
- Azure CLI инсталиран и аутентификован (`az login`)

### Почетно подешавање

1. **Клонирајте или форкујте складиште:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # ИЛИ
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Креирајте и активирајте Python виртуелно окружење:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # На Виндоус-у: venv\Scripts\activate
   ```

3. **Инсталирајте зависности:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Подесите променљиве окружења:**
   ```bash
   cp .env.example .env
   # Измените .env са вашим API кључевима и крајњим тачкама
   ```

### Потребне променљиве окружења

За **Microsoft Foundry** (потребно):
- `AZURE_AI_PROJECT_ENDPOINT` - крајња тачка пројекта у Microsoft Foundry
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - име распоређеног модела (нпр. gpt-4.1-mini)

За **Azure AI Search** (Лекција 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - крајња тачка Azure AI претраге
- `AZURE_SEARCH_API_KEY` - API кључ за Azure AI претрагу

Аутентификација: Покрените `az login` пре покретања ноутбука (користи `AzureCliCredential`).

Свака лекција садржи више Jupyter ноутбука за различите оквире:

- Образац датотеке: `*-python-agent-framework.ipynb`

## Додатне Напомене

### Чести проблеми

1. **Неслагање верзије Python-а:**
   - Осигурајте коришћење Python 3.12+
   - Неки пакети можда не раде са старијим верзијама
   - Користите `python3 -m venv` да експлицитно одредите верзију Python-а

2. **Променљиве окружења:**
   - Увек креирајте `.env` из `.env.example`
   - Не комитујте `.env` фајл (налази се у `.gitignore`)
   - Пријавите се са `az login` за безкључну Entra ID аутентикацију

3. **Конфликти пакета:**
   - Користите свеже виртуелно окружење
   - Инсталирајте из `requirements.txt`, а не појединачно
   - Неки ноутбуци могу захтевати додатне пакете поменуте у markdown ћелијама

4. **Azure услуге:**
   - Azure AI услуге захтевају активну претплату
   - Неки функционалности су специфичне за регион
   - Обезбедите да ваше Azure OpenAI распоређивање модела подржава Responses API

## Контекст специфичан за пројекат

### Подршка за више језика

Ово складиште користи аутоматизован систем превођења:
- Подржано преко 50 језика
- Преводи у директоријумима `/translations/<код-језика>/`
- GitHub Actions workflow рукова ажурирањима превода
- Изворни фајлови на енглеском у корену складишта

### Структура лекције

Свака лекција следи доследан образац:
1. Преглед видеа са линком
2. Писани садржај лекције (README.md)
3. Примери кода у више оквира
4. Циљеви учења и предуслови
5. Додатни ресурси за учење са линковима

### Именовање примера кода

Формат: `<lesson-number>-python-agent-framework.ipynb`
- `01-python-agent-framework.ipynb` - Лекција 1, MAF Python
- `14-sequential.ipynb` - Лекција 14, MAF напредни шаблони
- `16-python-agent-framework.ipynb` - Лекција 16, производни агент за корисничку подршку
- `17-local-agent-foundry-local.ipynb` - Лекција 17, локални агент са Foundry Local + Qwen

### Специјални директоријуми

- `translated_images/` - Локализоване слике за преводе
- `images/` - Оригиналне слике за енглески садржај
- `.devcontainer/` - Конфигурација развојног контејнера за VS Code
- `.github/` - GitHub Actions радни токови и шаблони

### Зависности

Кључни пакети из `requirements.txt`:
- `agent-framework` - Microsoft Agent Framework
- `a2a-sdk` - Подршка за Agent-to-Agent протокол
- `azure-ai-inference`, `azure-ai-projects` - Azure AI услуге
- `azure-identity` - Azure аутентикација (AzureCliCredential)
