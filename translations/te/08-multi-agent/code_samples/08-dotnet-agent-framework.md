# 🤝 ఎంటర్ప్రైజ్ మల్టీ-ఏజెంట్ వర్క్‌ఫ్లో సిస్టమ్స్ (.NET)

## 📋 నేర్చుకునే లక్ష్యాలు

ఈ నోట్‌బుక్ లో మైక్రోసాఫ్ట్ ఏజెంట్ ఫ్రేమ్‌వర్క్ ఉపయోగించి .NET లో, Azure OpenAI (Responses API) తో అధునాతన ఎంటర్‌ప్రైజ్-గ్రేడ్ మల్టీ-ఏజెంట్ సిస్టమ్స్ ను ఎలా నిర్మించాలో చూపిస్తుంది. మీరు నిర్మాణాత్మక వర్క్‌ఫ్లోల ద్వారా కలిసి పనిచేస్తున్న అనేక ప్రత్యేక ఏజెంట్లను ఆర్కెస్ట్రేట్ చేయలేరని నేర్చుకుంటారు, ఉత్పత్తి సన్నద్ధ పరిష్కారాల కోసం .NET యొక్క ఎంటర్ప్రైజ్ ఫీచర్లను ఉపయోగిస్తారు.

**మీరు నిర్మించే ఎంటర్ప్రైజ్ మల్టీ-ఏజెంట్ సామర్థ్యాలు:**
- 👥 **ఏజెంట్ సహకారం**: కంపైల్-టైమ్ ధృవీకరణతో టైప్-సేఫ్ ఏజెంట్ సమన్వయం
- 🔄 **వర్క్‌ఫ్లో ఆర్కెస్ట్రేషన్**: .NET యొక్క async నమూనాలతో ప్రకటనాత్మక వర్క్‌ఫ్లో నిర్వచనం
- 🎭 **పాత్ర నైపుణ్యం**: బలమైన టైప్ చేసిన ఏజెంట్ వ్యక్తిత్వాలు మరియు నైపుణ్య రంగాలు
- 🏢 **ఎంటర్ప్రైజ్ ఇంటిగ్రేషన్**: ప్రొడక్ష‌న్-సన్నద్ధ నమూనాలు, మానిటరింగ్ మరియు తప్పిద నిర్వహణతో

## ఈ నమూనా ఏమి చూపిస్తుంది

ఈ మల్టీ-ఏజెంట్ వర్క్‌ఫ్లో సిస్టమ్ రెండు ప్రత్యేక ఏజెంట్లతో హోటల్ ట్రావెల్ సిఫార్సు సేవను సృష్టిస్తుంది:

1. **ఫ్రంట్‌డెస్క్ ఏజెంట్**: కార్యకలాపాలు మరియు స్థాన సూచనలను అందించే ట్రావెల్ ఏజెంట్
2. **కాన్సియర్జ్ ఏజెంట్**: సిఫార్సులను తిరిగి సమీక్షించి అసలు, టూరిస్టు కాని అనుభవాలను నిర్ధారిస్తుంది

ఏజెంట్లు ఈ విధమైన వర్క్‌ఫ్లోలో కలిసి పనిచేస్తాయి:
- ప్రాథమిక ప్రయాణ అభ్యర్థనను ఫ్రంట్‌డెస్క్ ఏజెంట్ అందుకుంటుంది
- కాన్సియర్జ్ ఏజెంట్ సిఫార్సులను సమీక్షించి మెరుగుపరుస్తుంది
- వర్క్‌ఫ్లో ప్రత్యుత్తరాలను రియల్-టైమ్‌లో స్ట్రీమ్ చేస్తుంది

## ముఖ్య భావనలు

### ఏజెంట్ సమన్వయం
ఈ నమూనా మైక్రోసాఫ్ట్ ఏజెంట్ ఫ్రేమ్‌వర్క్ ఉపయోగించి టైప్-సేఫ్ ఏజెంట్ సమన్వయాన్ని కంపైల్-టైమ్ ధృవీకరణతో చూపిస్తుంది.

### వర్క్‌ఫ్లో ఆర్కెస్ట్రేషన్
.NET యొక్క async నమూనాలతో ప్రకటనాత్మక వర్క్‌ఫ్లో నిర్వచనం ఉపయోగించి అనేక ఏజెంట్లను పైప్‌లైన్‌లో కలుపుతుంది.

### స్ట్రీమింగ్ ప్రత్యుత్తరాలు
async ఎన్యూమరేబుల్స్ మరియు ఈవెంట్-డ్రైవెన్ నిర్మాణంతో ఏజెంట్ ప్రత్యుత్తరాల రియల్-టైమ్ స్ట్రీమింగ్ అమలు చేస్తుంది.

### ఎంటర్ప్రైజ్ ఇంటిగ్రేషన్
ప్రొడక్షన్-సన్నద్ధ నమూనాలను చూపిస్తుంది:
- పరిసర వేరియబుల్ కాన్ఫిగరేషన్
- భద్రమైన క్రెడెన్షియల్ నిర్వహణ
- తప్పిద నిర్వహణ
- అసింక్రోనస్ ఈవెంట్ ప్రాసెసింగ్
