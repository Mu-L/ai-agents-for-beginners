# AGENTS.md

## ప్రాజెక్ట్ సమీక్ష

ఈ రిపోజిటరీ "ప్రారంభదారుల కోసం AI ఏజెంట్లు"ని కలిగి ఉంది - AI ఏజెంట్లను నిర్మించడానికి అవసరమైన ప్రతిదీ నేర్పించే సమగ్ర విద్యా కోర్సు. కోర్సులో 18 పాఠాలు ఉన్నాయి (00-18 సూచికలతో) ఇది ప్రాథమికాలు, డిజైన్ నమూనాలు, ఫ్రేమ్‌వర్క్‌లు, ఉత్పత్తి సంస్థాపన, స్థానిక/పరికరం-ఆధారిత ఏజెంట్లు, మరియు AI ఏజెంట్ల భద్రతను కవర్ చేస్తుంది.

**ముఖ్య సాంకేతికతలు:**
- Python 3.12+
- ఇంటరాక్టివ్ అభ్యాసం కోసం Jupyter నోటుబుక్స్
- AI ఫ్రేమ్‌వర్క్‌లు: Microsoft Agent Framework (MAF)
- Azure AI సేవలు: Microsoft Foundry, Microsoft Foundry Agent Service V2

**వాస్తవిక రూపకల్పన:**
- పాఠాల ఆధారిత నిర్మాణం (00-15+ డైరెక్టరీలు)
- ప్రతి పాఠంలో: README దస్త్రాలు, కోడ్ నమూనాలు (Jupyter నోటుబుక్స్), మరియు చిత్రాలు ఉంటాయి
- స్వయంచాలక అనువాద వ్యవస్థ ద్వారా బహుభాషా మద్దతు
- ప్రతి పాఠానికి ఒక Python నోటుబుక్ Microsoft Agent Framework ఉపయోగించి

## సెటప్ కమాండ్లు

### ముందస్తు అవసరాలు
- Python 3.12 లేదా అంతకు మించి
- Azure చందా (Microsoft Foundry కోసం)
- Azure CLI సంస్థాపన మరియు ధృవపరిచిన (`az login`)

### ప్రారంభ సెటప్

1. **రిపోజిటరీని క్లోన్ చేయండి లేదా ఫోర్క్ చేయండి:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # లేదా
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python వర్చువల్ ఎన్విరాన్మెంట్ సృష్టించి యాక్టివేట్ చేయండి:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # విండోస్‌పై: venv\Scripts\activate
   ```

3. **అవసరమైన ప్యాకেজెస్ ఇన్‌స్టాల్ చేయండి:**
   ```bash
   pip install -r requirements.txt
   ```

4. **ఎన్విరాన్మెంట్ వేరియబుల్స్ సెట్ చేయండి:**
   ```bash
   cp .env.example .env
   # మీ API కీలు మరియు ఎండ్పాయింట్లతో .envని ఎడిట్ చేయండి
   ```

### అవసరమైన ఎన్విరాన్మెంట్ వేరియబుల్స్

**Microsoft Foundry కోసం** (అవసరం):
- `AZURE_AI_PROJECT_ENDPOINT` - Microsoft Foundry ప్రాజెక్ట్ ఎండ్పాయింట్
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - మోడల్ డిప్లాయ్‌మెంట్ పేరు (ఉదా: gpt-4.1-mini)

**Azure AI Search కోసం** (పాఠం 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search ఎండ్పాయింట్
- `AZURE_SEARCH_API_KEY` - Azure AI Search API కీ

ధృవీకరణ: నోటుబుక్స్ నడుపడముకోసమే `az login` నిర్వహించండి (`AzureCliCredential` ఉపయోగించబడుతుంది).

## అభివృద్ధి వర్క్‌ఫ్లో

### సాధారణ సమస్యలు

1. **Python వెర్షన్ అసమతుల్యత:**
   - Python 3.12+ ఉపయోగిస్తున్నట్లు నిర్ధారించుకోండి
   - కొన్ని ప్యాకేజీలు పాత వెర్షన్లతో పనిచేయవు
   - Python వెర్షన్ స్పష్టం చేయడానికి `python3 -m venv` ఉపయోగించండి

2. **ఎన్విరాన్మెంట్ వేరియబుల్స్:**
   - ఎప్పుడూ `.env.example` నుండి `.env` సృష్టించండి
   - `.env` ఫైల్‌ను కమిట్ చేయవద్దు (అది `.gitignore` లో ఉంటుంది)
   - కీ రహిత Entra ID ధృవీకరణ కోసం `az login` తో సైన్ ఇన్ అవ్వండి

3. **ప్యాకేజీ ఘర్షణలు:**
   - తాజాగా వర్చువల్ ఎన్విరాన్మెంట్ ఉపయోగించండి
   - వ్యక్తిగత ప్యాకేజీల కంటే `requirements.txt` ద్వారా ఇన్‌స్టాల్ చేయండి
   - కొన్ని నోటుబుక్స్ లో మర్కడౌన్ సెల్‌లలో పేర్కొన్న అదనపు ప్యాకేజీలు అవసరం కావచ్చు

4. **Azure సేవలు:**
   - Azure AI సేవలకు సక్రియ చందా అవసరం
   - కొన్ని ఫీచర్లు ప్రాంత-ప్రత్యేకం
   - మీ Azure OpenAI మోడల్ డిప్లాయ్‌మెంట్ Responses APIని మద్దతు ఇస్తున్నదిగా నిర్ధారించుకోండి

### నేర్చుకునే మార్గం

పాఠాల ద్వారా సిఫారసు చేసిన ప్రగతి:
