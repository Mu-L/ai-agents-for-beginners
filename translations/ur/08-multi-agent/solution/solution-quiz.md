# علم کی جانچ کے جوابات

## سوال 1

**جواب: A2۔** متعدد ایجنٹس اس وقت مفید ہوتے ہیں جب کسی کام میں مخصوص ماہر کردار، الگ ٹولز یا اجازتیں، اور واضح ہینڈ آف شامل ہوں جن کی ہم آہنگی ضروری ہو۔ ایک آسان سپورٹ بوٹ عام طور پر ایک ایجنٹ ہی رہ سکتا ہے۔ ایک ہی قسم کی بڑی تعداد میں درخواستیں بنیادی طور پر پیمانے کا مسئلہ ہوتی ہیں اور خود بخود ایک کثیر ایجنٹ ڈیزائن کی ضرورت نہیں ہوتی۔

## سوال 2

**جواب: A1۔** جب ایک سیٹ ہدایات اور ٹولز کسی کام کو سنبھال سکتے ہوں تو عام طور پر ایک ایجنٹ کو آزمانا، مشاہدہ کرنا، اور چلانا آسان ہوتا ہے۔ کئی ٹولز تک رسائی خود بخود کئی ایجنٹس کی ضرورت نہیں بناتی۔ مختلف اجازتوں اور آڈٹ کی ضروریات کے ساتھ الگ کردار ایک کثیر ایجنٹ ڈیزائن کا جواز پیش کر سکتے ہیں۔
