# AGENTS.md

## پروجیکٹ کا جائزہ

یہ ریپوزٹری "نئے آنے والوں کے لئے AI ایجنٹس" پر مشتمل ہے — ایک جامع تعلیمی کورس جو AI ایجنٹس بنانے کے لیے درکار تمام معلومات سکھاتا ہے۔ یہ کورس 18 اسباق (نمبر شدہ 00-18) پر مشتمل ہے جن میں بنیادی باتیں، ڈیزائن پیٹرنز، فریم ورکس، پروڈکشن ڈیپلائمنٹ، لوکل/آن ڈیوائس ایجنٹس، اور AI ایجنٹس کی سیکیورٹی شامل ہیں۔

**اہم ٹیکنالوجیز:**
- Python 3.12+
- انٹرایکٹو سیکھنے کے لیے جیوپیٹر نوٹ بکس
- AI فریم ورکس: مائیکروسافٹ ایجنٹ فریم ورک (MAF)
- Azure AI خدمات: مائیکروسافٹ فاؤنڈری، مائیکروسافٹ فاؤنڈری ایجنٹ سروس V2

**معماری:**
- سبق پر مبنی ساخت (00-15+ ڈائریکٹریز)
- ہر سبق میں شامل ہے: README دستاویزات، کوڈ کے نمونے (جیوپیٹر نوٹ بکس)، اور تصاویر
- خودکار ترجمہ نظام کے ذریعے کثیر اللسانی معاونت
- ہر سبق کے لیے ایک Python نوٹ بک جو مائیکروسافٹ ایجنٹ فریم ورک استعمال کرتی ہے

## سیٹ اپ کمانڈز

### ضروریات
- Python 3.12 یا اس سے اوپر
- Azure سبسکرپشن (مائیکروسافٹ فاؤنڈری کے لیے)
- Azure CLI نصب اور تصدیق شدہ (`az login`)

### ابتدائی سیٹ اپ

1. **ریپوزٹری کلون یا فورک کریں:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # یا
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python وِرچوئل اینوائرنمنٹ بنائیں اور اس کو فعال کریں:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # ونڈوز پر: venv\Scripts\activate
   ```

3. **حوالہ جات انسٹال کریں:**
   ```bash
   pip install -r requirements.txt
   ```

4. **ماحول کے متغیرات سیٹ کریں:**
   ```bash
   cp .env.example .env
   # اپنی API کیز اور اینڈ پوائنٹس کے ساتھ .env کو ترمیم کریں
   ```

### مطلوبہ ماحول کے متغیرات

**مائیکروسافٹ فاؤنڈری** کے لیے (ضروری):
- `AZURE_AI_PROJECT_ENDPOINT` - مائیکروسافٹ فاؤنڈری پروجیکٹ اینڈ پوائنٹ
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - ماڈل کی تعیناتی کا نام (مثلاً gpt-5-mini)

**Azure AI سرچ** کے لیے (سبق 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI سرچ اینڈ پوائنٹ
- `AZURE_SEARCH_API_KEY` - Azure AI سرچ API کلید

تصدیق: نوٹ بکس چلانے سے پہلے `az login` چلائیں (استعمال کرتا ہے `AzureCliCredential`)۔

- بلٹ ان آبزرویبیلٹی کے ساتھ پروڈکشن کے لئے تیار
- فائل پیٹرن: `*-python-agent-framework.ipynb`
