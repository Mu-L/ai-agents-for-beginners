# AGENTS.md

## پروجیکٹ کا جائزہ

یہ ریپوزیٹری "نو آموزوں کے لئے AI ایجنٹس" پر مشتمل ہے - ایک جامع تعلیمی کورس جو AI ایجنٹس بنانے کے لئے ضروری تمام چیزوں کی تعلیم دیتا ہے۔ کورس میں 18 اسباق شامل ہیں (نمبر کردہ 00-18) جو بنیادی باتوں، ڈیزائن پیٹرنز، فریم ورکس، پیداوار کی تعیناتی، مقامی/ڈیوائس پر ایجنٹس، اور AI ایجنٹس کی سیکورٹی کا احاطہ کرتے ہیں۔

**اہم ٹیکنالوجیز:**
- پائتھن 3.12+
- انٹرایکٹو سیکھنے کے لیے جوپیٹر نوٹ بکس
- AI فریم ورکس: Microsoft Agent Framework (MAF)
- Azure AI خدمات: Microsoft Foundry، Microsoft Foundry Agent Service V2

**معماری:**
- سبق پر مبنی ڈھانچہ (00-15+ ڈائریکٹریز)
- ہر سبق میں شامل ہیں: README دستاویزات، کوڈ کے نمونے (جوپیٹر نوٹ بکس)، اور تصاویر
- خودکار ترجمہ نظام کے ذریعے کثیر لسانی معاونت
- ہر سبق کے لیے ایک پائتھن نوٹ بک جو Microsoft Agent Framework استعمال کرتی ہے

## سیٹ اپ کے کمانڈز

### ضروریات
- پائتھن 3.12 یا اس سے جدید
- Azure سبسکرپشن (Microsoft Foundry کے لیے)
- Azure CLI انسٹال اور لاگ ان (`az login`)

### ابتدائی سیٹ اپ

1. **ریپوزیٹری کلون یا فورک کریں:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # یا
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **پائتھن ورچوئل ماحول بنائیں اور فعال کریں:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # ونڈوز پر: venv\Scripts\activate
   ```

3. **ڈپینڈنسیز انسٹال کریں:**
   ```bash
   pip install -r requirements.txt
   ```

4. **ماحول کے متغیرات سیٹ کریں:**
   ```bash
   cp .env.example .env
   # اپنی API کیز اور اینڈپوائنٹس کے ساتھ .env کو ایڈٹ کریں
   ```

### درکار ماحول کے متغیرات

**Microsoft Foundry** کے لیے (لازمی):
- `AZURE_AI_PROJECT_ENDPOINT` - Microsoft Foundry پروجیکٹ اینڈ پوائنٹ
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - ماڈل ڈپلائمنٹ کا نام (مثلاً gpt-4.1-mini)

**Azure AI Search** کے لیے (سبق 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search اینڈ پوائنٹ
- `AZURE_SEARCH_API_KEY` - Azure AI Search API کلید

تصدیق کاری: نوٹ بکس چلانے سے پہلے `az login` چلائیں (استعمال کرتا ہے `AzureCliCredential`)۔

## ترقیاتی ورک فلو

### جوپیٹر نوٹ بکس چلانا

ہر سبق میں مختلف فریم ورکس کے لیے متعدد جوپیٹر نوٹ بکس شامل ہیں:

1. **جوپیٹر شروع کریں:**
   ```bash
   jupyter notebook
   ```

2. **سبق کی ڈائریکٹری میں جائیں** (مثلاً `01-intro-to-ai-agents/code_samples/`)

3. **نوٹ بکس کو کھولیں اور چلائیں:**
   - `*-python-agent-framework.ipynb` - Microsoft Agent Framework (Python) استعمال کرتے ہوئے
   - `*-dotnet-agent-framework.ipynb` - Microsoft Agent Framework (.NET) استعمال کرتے ہوئے

### Microsoft Agent Framework کے ساتھ کام کرنا

**Microsoft Agent Framework + Microsoft Foundry:**
- Azure سبسکرپشن کی ضرورت ہے
- Agent Service V2 کے لیے `FoundryChatClient` استعمال کرتا ہے (ایجنٹس Foundry پورٹل میں نظر آتے ہیں)
- بلٹ ان آبزر ویبیلٹی کے ساتھ پیداوار کے لئے تیار
- فائل پیٹرن: `*-python-agent-framework.ipynb`
