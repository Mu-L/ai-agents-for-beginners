# 🔍 Microsoft Foundry (.NET) کے ساتھ انٹرپرائز RAG

## 📋 سیکھنے کے مقاصد

یہ نوٹ بک دکھاتی ہے کہ Microsoft Foundry کے ساتھ .NET میں Microsoft Agent Framework استعمال کرتے ہوئے انٹرپرائز گریڈ Retrieval-Augmented Generation (RAG) سسٹمز کیسے بنائیں۔ آپ پیداواری تیار ایجنٹس بنانا سیکھیں گے جو دستاویزات کی تلاش کر سکیں اور انٹرپرائز سیکیورٹی اور اسکیل ایبلیٹی کے ساتھ درست، سیاق و سباق سے آگاہ جوابات فراہم کر سکیں۔

**آپ جو انٹرپرائز RAG صلاحیتیں بنائیں گے:**
- 📚 **دستاویز ذہانت**: Azure AI خدمات کے ساتھ جدید دستاویزی پراسیسنگ
- 🔍 **سیمنٹک سرچ**: انٹرپرائز خصوصیات کے ساتھ اعلی کارکردگی والی ویکٹر سرچ
- 🛡️ **سیکیورٹی انضمام**: کردار کی بنیاد پر رسائی اور ڈیٹا تحفظ کے نمونے
- 🏢 **اسکیل ایبل آرکیٹیکچر**: نگرانی کے ساتھ پیداواری تیار RAG سسٹمز

## 🎯 انٹرپرائز RAG آرکیٹیکچر

### بنیادی انٹرپرائز اجزاء
- **Microsoft Foundry**: سیکیورٹی اور تعمیل کے ساتھ منظم انٹرپرائز AI پلیٹ فارم
- **پرسسٹنٹ ایجنٹس**: گفتگو کی تاریخ اور سیاق و سباق کے انتظام کے ساتھ سٹیٹ فل ایجنٹس
- **ویکٹر اسٹور مینجمنٹ**: انٹرپرائز گریڈ دستاویز انڈیکسنگ اور بازیابی
- **شناخت کا انضمام**: Azure AD تصدیق اور کردار کی بنیاد پر رسائی کنٹرول

### .NET انٹرپرائز فوائد
- **ٹائپ سیفٹی**: RAG آپریشنز اور ڈیٹا ڈھانچوں کے لیے کمپائل ٹائم ویلیڈیشن
- **ایسنک کارکردگی**: غیر روکاوٹ والا دستاویز پراسیسنگ اور تلاش کے آپریشنز
- **میموری مینجمنٹ**: بڑی دستاویز مجموعوں کے لیے موثر وسائل کا استعمال
- **انضمام کے نمونے**: انحصار انجیکشن کے ساتھ مقامی Azure سروس انضمام

## 🏗️ تکنیکی آرکیٹیکچر

### انٹرپرائز RAG پائپ لائن
```
Document Upload → Security Validation → Vector Processing → Index Creation
                      ↓                    ↓                  ↓
User Query → Authentication → Semantic Search → Context Ranking → AI Response
```

### بنیادی .NET اجزاء
- **Azure.AI.Agents.Persistent**: سٹیٹ پرسِسٹنس کے ساتھ انٹرپرائز ایجنٹ مینجمنٹ
- **Azure.Identity**: محفوظ Azure سروس تک رسائی کے لیے مربوط تصدیق
- **Microsoft.Agents.AI.AzureAI**: Azure کے لیے بہتر بنایا گیا ایجنٹ فریم ورک امپلیمینٹیشن
- **System.Linq.Async**: اعلی کارکردگی والی غیر ہم آہنگ LINQ آپریشنز

### مینجمنٹ اور نگرانی

- **ایرر ہینڈلنگ**: جامع استثناء مینجمنٹ کے ساتھ ریٹری پالیسیاں
- **کنفیگریشن مینجمنٹ**: ویلیڈیشن کے ساتھ ماحول مخصوص سیٹنگز

**Azure تصدیق سیٹ اپ:**
```bash

az account set --subscription "your-subscription-id"
```

**ماحول کنفیگریشن:**
* Microsoft Foundry کی کنفیگریشن (Azure CLI کے ذریعے خودکار طور پر ہینڈل کی جاتی ہے)
* یقینی بنائیں کہ آپ درست Azure سبسکرپشن کے لیے تصدیق شدہ ہیں
