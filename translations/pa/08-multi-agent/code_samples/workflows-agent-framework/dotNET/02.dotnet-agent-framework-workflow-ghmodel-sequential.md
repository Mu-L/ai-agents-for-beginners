# ⏩ Azure OpenAI (Responses API) (.NET) ਨਾਲ ਲੜੀਵਾਰ ਏਜੰਤ ਵਰਕਫਲੋws

## 📋 ਉੱਨਤ ਲੜੀਵਾਰ ਪ੍ਰਕਿਰਿਆ ਟਿਊਟੋਰਿਅਲ

ਇਸ ਨੋਟਬੁੱਕ ਵਿੱਚ Microsoft Agent Framework for .NET ਅਤੇ Azure OpenAI (Responses API) ਦੀ ਵਰਤੋਂ ਕਰਕੇ **ਲੜੀਵਾਰ ਵਰਕਫਲੋ ਪੈਟਰਨਜ਼** ਦਿਖਾਏ ਗਏ ਹਨ। ਤੁਸੀਂ ਸਿਖੋਗੇ ਕਿ ਕਿਸ ਤਰ੍ਹਾਂ ਕਾਮਪਲੈਕਸ, ਕਦਮ-ਦਰ-कਦਮ ਪ੍ਰਕਿਰਿਆ ਪਾਈਪਲਾਈਨਾਂ ਬਣਾਈਆਂ ਜਾਣ ਜੋ ਕਿ ਇੱਕ ਵਿਸ਼ੇਸ਼ ਕ੍ਰਮ ਵਿੱਚ ਏਜੰਟਸ ਨੂੰ ਚਲਾਉਂਦੀਆਂ ਹਨ, ਜਿੱਥੇ ਹਰ ਪੜਾਅ ਪਿਛਲੇ ਪੜਾਅ ਦੇ ਨਤੀਜਿਆਂ 'ਤੇ ਅਧਾਰਿਤ ਹੁੰਦਾ ਹੈ।

### 🔑 **Azure OpenAI ਵਿਨਿਆਸ**

**ਵਾਤਾਵਰਣ ਸੈਟਅੱਪ (.env ਫਾਈਲ):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-4.1-mini
```

**ਕਨਫਿਗਰੇਸ਼ਨ ਪ੍ਰਬੰਧਨ:**
```csharp
// Load environment variables securely
Env.Load("../../../.env");
var azureEndpoint = Environment.GetEnvironmentVariable("AZURE_OPENAI_ENDPOINT");
var deployment = Environment.GetEnvironmentVariable("AZURE_OPENAI_DEPLOYMENT");
```

2. **ਕੀਮਤ ਏਜੰਟ**: ਬਜਟ ਦਾਸਤਾਵੇਜ਼ਾਂ ਅਤੇ ਖਰੀਦ ਸਿਫਾਰਸ਼ਾਂ ਨਾਲ ਵਿਸਥਾਰਿਤ ਕੀਮਤ ਵਿਸ਼ਲੇਸ਼ਣ ਸ਼ਾਮਲ ਕਰਦਾ ਹੈ
3. **ਕੋਟ ਏਜੰਟ**: ਸਾਰੇ ਜਾਣਕਾਰੀ ਨੂੰ ਸੰਗ੍ਰਹਿਤ ਕਰਦਿਆਂ ਇੱਕ ਫਾਰਮੈਟ ਕੀਤਾ ਹੋਇਆ ਕੋਟ ਦਸਤਾਵੇਜ਼ ਤਿਆਰ ਕਰਦਾ ਹੈ

ਆਖਰੀ ਨਤੀਜਾ ਇਕ ਢੁਕਵਾਂ, ਪ੍ਰੋਫੈਸ਼ਨਲ ਫਰਨੀਚਰ ਕੋਟ ਹੋਵੇਗਾ ਜੋ ਤਸਵੀਰ ਵਿਸ਼ਲੇਸ਼ਣ 'ਤੇ ਅਧਾਰਿਤ ਹੈ।
