# 🔄 ਬੇਸਿਕ ਏਜੰਟ ਵਰਕਫਲੋਜ਼ ਵਿਥ ਅਜ਼ੂਰ ਓਪਨਏਆਈ (Responses API) (.NET)

## 📋 ਵਰਕਫਲੋ ਓਰਕੇਸਟ੍ਰੇਸ਼ਨ ਟਿਊਟੋਰੀਅਲ

ਇਹ ਨੋਟਬੁੱਕ ਮਾਈਕਰੋਸਾਫਟ ਏਜੰਟ ਫਰੇਮਵਰਕ ਫੋਰ .NET ਅਤੇ ਅਜ਼ੂਰ ਓਪਨਏਆਈ (Responses API) ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਸੁਧਰੇ ਹੋਏ **ਏਜੰਟ ਵਰਕਫਲੋਜ਼** ਬਣਾਉਣ ਦੇ ਤਰੀਕੇ ਨੂੰ ਦਰਸਾਉਂਦਾ ਹੈ। ਤੁਸੀਂ ਬਹੁ-ਕਦਮੀ ਕਾਰੋਬਾਰੀ ਪ੍ਰਕਿਰਿਆਵਾਂ ਬਣਾਉਣਾ ਸਿੱਖੋਂਗੇ ਜਿੱਥੇ ਏਆਈ ਏਜੰਟਾਂ ਢਾਂਚਾਗਤ ਓਰਕੇਸਟ੍ਰੇਸ਼ਨ ਪੈਟਰਨਾਂ ਰਾਹੀਂ ਮਿਲ ਕੇ ਜਟਿਲ ਕੰਮ ਮੁਕੰਮਲ ਕਰਦੇ ਹਨ।

## 🎯 ਸਿੱਖਣ ਦੇ ਲਕੜ

### 🏗️ **ਵਰਕਫਲੋ ਆਰਕੀਟੈਕਚਰ ਦੇ ਮੁੱਢਲੇ ਤੱਤ**
- **ਵਰਕਫਲੋ ਬਿਲਡਰ**: ਜਟਿਲ ਬਹੁ-ਕਦਮੀ ਏਆਈ ਪ੍ਰਕਿਰਿਆਵਾਂ ਨੂੰ ਡਿਜ਼ਾਈਨ ਅਤੇ ਓਰਕੇਸਟਰੇਟ ਕਰਨਾ  
- **ਏਜੰਟ ਕੋਆਰਡੀਨੇਸ਼ਨ**: ਵਰਕਫਲੋਜ਼ ਵਿੱਚ ਕਈ ਵਿਸ਼ੇਸ਼ ਕਰਤੂਤਾਂ ਵਾਲੇ ਏਜੰਟਾਂ ਨੂੰ ਸਮਨਵਿਤ ਕਰਨਾ  
- **ਅਜ਼ੂਰ ਓਪਨਏਆਈ (Responses API)**: ਵਰਕਫਲੋਜ਼ ਵਿੱਚ ਅਜ਼ੂਰ ਓਪਨਏਆਈ Responses API ਦੀ ਵਰਤੋਂ ਕਰਨਾ  
- **ਵਿਜ਼ੂਅਲ ਵਰਕਫਲੋ ਡਿਜ਼ਾਈਨ**: ਬਿਹਤਰ ਸਮਝ ਲਈ ਵਰਕਫਲੋ ਢਾਂਚਿਆਂ ਦਾ ਨਿਰਮਾਣ ਅਤੇ ਦਰਸ਼ਨ  

### 🔄 **ਪ੍ਰਕਿਰਿਆ ਓਰਕੇਸਟ੍ਰੇਸ਼ਨ ਪੈਟਰਨ**
- **ਗੱਤਵਾਰ ਪ੍ਰਕਿਰਿਆ**: ਤਰਤੀਬ وار ਕਈ ਏਜੰਟ ਕੰਮਾਂ ਨੂੰ ਜੋੜਨਾ  
- **ਸਟੇਟ ਪ੍ਰਬੰਧਨ**: ਵਰਕਫਲੋ ਅਂਸ਼ਾਂ ਵਿੱਚ ਸੰਦਰਭ ਅਤੇ ਡੇਟਾ ਫਲੋ ਬਰਕਰਾਰ ਰੱਖਣਾ  
- **ਤਰੁੱਟੀ ਸੰਭਾਲ**: ਮਜ਼ਬੂਤ ਤਰੁੱਟੀ ਬਹਾਲੀ ਅਤੇ ਵਰਕਫਲੋ ਲਚਕੀਲੇਪਣ ਨੂੰ ਲਾਗੂ ਕਰਨਾ  
- **ਕਾਰਗੁਜ਼ਾਰੀ ਦਾ ਸੁਧਾਰ**: ਉਦਯੋਗੀ ਪੱਧਰ ਦੇ ਓਪਰੇਸ਼ਨਾਂ ਲਈ ਪ੍ਰਭਾਵੀ ਵਰਕਫਲੋਜ਼ ਤਿਆਰ ਕਰਨਾ  

### 🏢 **ਉਦਯੋਗ ਵਾਲੇ ਵਰਕਫਲੋ ਐਪਲੀਕੇਸ਼ਨਸ**
- **ਕਾਰੋਬਾਰੀ ਪ੍ਰਕਿਰਿਆ ਆਟੋਮੇਸ਼ਨ**: ਜਟਿਲ ਸੰਗਠਨਾਤਮਕ ਵਰਕਫਲੋਜ਼ ਨੂੰ ਆਟੋਮੇਟ ਕਰਨਾ  
- **ਸਮੱਗਰੀ ਉਤਪਾਦਨ ਪਾਈਪਲਾਈਨ**: ਸਮੀਖਿਆ ਅਤੇ ਮਨਜ਼ੂਰੀ ਅੰਸ਼ਾਂ ਨਾਲ ਸੰਪਾਦਕੀ ਵਰਕਫਲੋਜ਼  
- **ਗਾਹਕ ਸੇਵਾ ਆਟੋਮੇਸ਼ਨ**: ਬਹੁ-ਕਦਮੀ ਗਾਹਕ ਪੁੱਛਗਿੱਛ ਦਾ ਸਮਾਧਾਨ  
- **ਡੇਟਾ ਪ੍ਰਕਿਰਿਆ ਵਰਕਫਲੋਜ਼**: ਏ.ਟੀ.ਐੱਲ ਵਰਕਫਲੋਜ਼, ਏਆਈ ਨਾਲ ਸਮਰੱਥਿਤ ਬਦਲਾਅ  

## ⚙️ ਲੋੜੀਂਦੇ ਸਾਜੋ-ਸਾਮਾਨ ਅਤੇ ਸੈੱਟਅੱਪ

### 📦 **ਲੋੜੀਂਦੇ ਨੂਗੇਟ ਪੈਕੇਜ**

ਇਸ ਵਰਕਫਲੋ ਡੈਮੋ ਵਿੱਚ ਕਈ ਮੁੱਖ .NET ਪੈਕੇਜ ਵਰਤੇ ਗਏ ਹਨ:

```xml
<!-- Core AI Framework -->
<PackageReference Include="Microsoft.Extensions.AI" Version="10.*" />

<!-- Azure OpenAI (Responses API) -->
<PackageReference Include="Azure.AI.OpenAI" Version="2.1.0" />
<PackageReference Include="Azure.Identity" Version="1.15.0" />

<!-- Agent Framework (Local Development) -->
<!-- Microsoft.Agents.AI.dll - Core agent abstractions -->
<!-- Microsoft.Agents.AI.OpenAI.dll - Azure OpenAI (Responses API) integration -->

<!-- Configuration and Environment -->
<PackageReference Include="DotNetEnv" Version="3.1.1" />
```

### 🔑 **ਅਜ਼ੂਰ ਓਪਨਏਆਈ ਸੰਰਚਨਾ**

**ਮਾਹੌਲ ਸੈੱਟਅੱਪ (.env ਫਾਇਲ):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-5-mini
```

**ਅਜ਼ੂਰ ਓਪਨਏਆਈ ਪਹੁੰਚ:**
1. ਅਜ਼ੂਰ ਪੋਰਟਲ ਵਿੱਚ ਅਜ਼ੂਰ ਓਪਨਏਆਈ ਰਿਸੋਰਸ ਬਣਾਓ  
2. ਮਾਡਲ ਤੈਆਰ ਕਰੋ (ਉਦਾਹਰਨ ਲਈ, `gpt-5-mini`) ਅਤੇ ਡਿਪਲਾਇਮੈਂਟ ਨਾਮ ਨੋਟ ਕਰੋ  
3. `az login` ਨਾਲ ਸਾਈਨ ਇਨ ਕਰੋ ਅਤੇ ਉਪਰ ਦਿੱਤੇ ਮਾਹੌਲ ਕੀਮਤਾਂ ਸੈੱਟ ਕਰੋ   

### 🏗️ **ਵਰਕਫਲੋ ਆਰਕੀਟੈਕਚਰ ਦਾ ਸਰਵੇਖਣ**

```mermaid
graph TD
    A[ਕਾਰਜ ਪ੍ਰवाह ਬਣਾਉਣ ਵਾਲਾ] --> B[ਏਜੰਟ ਰਜਿਸਟਰੀ]
    B --> C[ਕਾਰਜ ਪ੍ਰवाह ਕਾਰਜਨ ਏੰਜਿਨ]
    C --> D[Agent 1: ਸਮੱਗਰੀ ਜਨਰੇਟਰ]
    C --> E[Agent 2: ਸਮੱਗਰੀ ਸਮੀਖਿਆਕਾਰ] 
    D --> F[ਕਾਰਜ ਪ੍ਰवाह ਨਤੀਜੇ]
    E --> F
    G[ਐਜ਼ਯੂਰ ਓਪਨਏਆਈ (ਜਵਾਬ API)] --> D
    G --> E
```

**ਮੁੱਖ ਹਿੱਸੇ:**
- **WorkflowBuilder**: ਵਰਕਫਲੋਜ਼ ਡਿਜ਼ਾਈਨ ਕਰਨ ਲਈ ਪ੍ਰਧਾਨ ਓਰਕੇਸਟ੍ਰੇਸ਼ਨ ਇੰਜਣ  
- **AIAgent**: ਵੱਖ-ਵੱਖ ਖਾਸ ਯੋਗਤਾਵਾਂ ਵਾਲੇ ਐਜੰਟਾਂ  
- **ਅਜ਼ੂਰ ਓਪਨਏਆਈ ਕਲਾਇੰਟ**: ਅਜ਼ੂਰ ਓਪਨਏਆਈ Responses API ਇੰਟੀਗ੍ਰੇਸ਼ਨ  
- **ਇਕਜ਼ਿਕਿਊਸ਼ਨ ਕਾਂਟੈਕਸਟ**: ਵਰਕਫਲੋ ਮੰਚਾਂ ਵਿੱਚ ਸਟੇਟ ਅਤੇ ਡੇਟਾ ਫਲੋ ਦਾ ਪ੍ਰਬੰਧ  

## 🎨 **ਉਦਯੋਗ ਵਰਕਫਲੋ ਡਿਜ਼ਾਈਨ ਪੈਟਰਨ**

```

```
Customer Inquiry → Classification → Processing → Response Generation → Follow-up

## 🏢 **ਉਦਯੋਗ ਲਾਭ**

### 🎯 **ਭਰੋਸੇਯੋਗਤਾ ਅਤੇ ਸਕੇਲਬਿਲਟੀ**
- **ਨਿਸ਼ਚਿਤ ਕਾਰਜਨਵਾਈ**: ਕਾਇਮ ਅਤੇ ਦੁਹਰਾਏ ਜਾਣ ਯੋਗ ਵਰਕਫਲੋ ਨਤੀਜੇ  
- **ਤਰੁੱਟੀ ਬਹਾਲੀ**: ਵਰਕਫਲੋ ਦੇ ਕਿਸੇ ਵੀ ਪੜਾਅ ’ਤੇ ਨਿੱਜੀ ਤਰੁੱਟੀਆਂ ਸਾਹਮਣੇ ਆਉਣ ਤੇ ਸੁਗਮ ਸੰਭਾਲ  
- **ਕਾਰਗੁਜ਼ਾਰੀ ਦੀ ਨਿਗਰਾਨੀ**: ਕਾਰਜਨਵਾਈ ਮੈਟਰਿਕਸ ਅਤੇ ਸੁਧਾਰ ਮੌਕੇ ਦਾ ਪਤਰਾ  
- **ਸੰਸਾਧਨ ਪ੍ਰਬੰਧਨ**: ਏਆਈ ਮਾਡਲ ਸੰਸਾਧਨਾਂ ਦੀ ਪ੍ਰਭਾਵਸ਼ੀਲ ਵਰਤੋਂ ਅਤੇ ਵੰਡ  

### 🔒 **ਸੁਰੱਖਿਆ ਅਤੇ ਅੰਕੂਲਤਾ**
- **ਸੁਰੱਖਿਅਤ ਪ੍ਰਮਾਣਿਕਤਾ**: `az login` (AzureCliCredential) ਰਾਹੀਂ ਮਾਈਕਰੋਸਾਫਟ Entra ID ਪ੍ਰਮਾਣਿਕਤਾ  
- **ਆਡਿਟ ਟ੍ਰੇਲਸ**: ਵਰਕਫਲੋ ਕਾਰਜਨਵਾਈ ਅਤੇ ਫੈਸਲੇ ਬਿੰਦੂਆਂ ਦੀ ਪੂਰੀ ਲਾਗਿੰਗ  
- **ਪਹੁੰਚ ਨਿਯੰਤਰਣ**: ਵਰਕਫਲੋ ਕਾਰਜਨਵਾਈ ਅਤੇ ਨਿਗਰਾਨੀ ਲਈ ਵਿਸਥਾਰਤ ਅਧਿਕਾਰ  
- **ਡੇਟਾ ਪਰਦੇਦਾਰੀ**: ਸੰਵੇਦਨਸ਼ੀਲ ਜਾਣਕਾਰੀ ਦੀ ਸੁਰੱਖਿਅਤ ਸੰਭਾਲ ਪੂਰੇ ਵਰਕਫਲੋ ਵਿੱਚ  

### 📊 **ਦੇਖ-ਰੇਖ ਅਤੇ ਪ੍ਰਬੰਧਨ**
- **ਵਿਜ਼ੂਅਲ ਵਰਕਫਲੋ ਡਿਜ਼ਾਈਨ**: ਪ੍ਰਕਿਰਿਆ ਫਲੋਅ ਅਤੇ ਨਿਰਭਰਤਾ ਦੀ ਸਾਫ਼-ਸੁਥਰੀ ਨੁਮਾਇੰਦਗੀ  
- **ਕਾਰਜਨਵਾਈ ਨਿਗਰਾਨੀ**: ਵਰਕਫਲੋ ਪ੍ਰਗਤੀ ਅਤੇ ਪਰੇਸ਼ਾਨੀ ਲਈ ਰੀਅਲ-ਟਾਇਮ ਟ੍ਰੈਕਿੰਗ  
- **ਤਰੁੱਟੀ ਰਿਪੋਰਟਿੰਗ**: ਵਿਸਥਾਰਤ ਤਰੁੱਟੀ ਵਿਸ਼ਲੇਸ਼ਣ ਅਤੇ ਡਿਬੱਗਿੰਗ ਸਹੂਲਤਾਂ  
- **ਕਾਰਗੁਜ਼ਾਰੀ ਵਿਸ਼ਲੇਸ਼ਣ**: ਸੁਧਾਰ ਅਤੇ ਸਮਰੱਥਾ ਯੋਜਨਾ ਲਈ ਮੈਟਰਿਕਸ  

ਆਓ ਤੁਹਾਡਾ ਪਹਿਲਾ ਉਦਯੋਗ ਤਿਆਰ ਏਆਈ ਵਰਕਫਲੋ ਬਣਾਈਏ! 🚀

## 💻 ਕੋਡ ਚਲਾਉਣਾ

ਪੂਰਾ ਇੰਪਲਿਮੇਂਟੇਸ਼ਨ `01.dotnet-agent-framework-workflow-ghmodel-basic.cs` ਵਿੱਚ ਉਪਲਬਧ ਹੈ। ਇਸ ਫਾਇਲ ਵਿਚ ਦਿਖਾਇਆ ਗਿਆ ਹੈ:

1. **ਮਾਹੌਲ ਸੰਰਚਨਾ** - `.env` ਫਾਇਲ ਤੋਂ ਅਜ਼ੂਰ ਓਪਨਏਆਈ ਸੰਰਚਨਾ ਲੋਡ ਕਰਨਾ  
2. **ਅਜ਼ੂਰ ਓਪਨਏਆਈ ਕਲਾਇੰਟ ਸੈੱਟਅੱਪ** - ਅਜ਼ੂਰ ਓਪਨਏਆਈ Responses API ਦੀ ਵਰਤੋਂ ਲਈ ਕਲਾਇੰਟ ਸੈਟ ਕਰਨਾ  
3. **ਏਜੰਟ ਬਣਾਉਣਾ** - ਖਾਸ ਯੋਗਤਾਵਾਂ ਵਾਲੇ ਏਜੰਟ (ਫਰੰਟ ਡੈਸਕ ਅਤੇ ਕੰਸੀਅਰਜ) ਨੂੰ ਪਰਿਭਾਸ਼ਿਤ ਕਰਨਾ  
4. **ਵਰਕਫਲੋ ਬਿਲਡਰ** - ਤਰਤੀਬ وار ਬਹੁ-ਏਜੰਟ ਵਰਕਫਲੋ ਬਣਾਉਣਾ  
5. **ਵਰਕਫਲੋ ਚਲਾਉਣਾ** - ਸਟ੍ਰੀਮਿੰਗ ਨਤੀਜਿਆਂ ਨਾਲ ਵਰਕਫਲੋ ਨੂੰ ਚਲਾਉਣਾ
