## 💻 ਕੋਡ ਚਲਾਉਣਾ

ਪੂਰਾ ਇੰਪਲੀਮੇਂਟੇਸ਼ਨ `03.dotnet-agent-framework-workflow-ghmodel-concurrent.cs` ਵਿੱਚ ਉਪਲਬਧ ਹੈ। ਇਹ ਫਾਇਲ ਯਾਤਰਾ ਯੋਜਨਾ ਲਈ ਇੱਕ **Fan-Out/Fan-In ਸਮਕਾਲੀ ਵਰਕਫਲੋ** ਵੇਖਾਉਂਦੀ ਹੈ:

### 🏗️ **ਵਰਕਫਲੋ ਆਰਕੀਟੈਕਚਰ**

```
User Request → ConcurrentStartExecutor → [Researcher Agent || Planner Agent] → ConcurrentAggregationExecutor → Final Output
```

**ਮੁੱਖ ਭਾਗ:**

1. **ConcurrentStartExecutor**: ਯੂਜ਼ਰ ਦੀ ਬੇਨਤੀ ਸਾਰੇ ਏਜੰਟਾਂ ਨੂੰ ਇੱਕ ਸਮੇਂ ਵਿੱਚ ਭੇਜਦਾ ਹੈ
2. **Researcher Agent**: ਗੰਤੀ-ਥਾਵਾਂ ਅਤੇ ਆਕਰਸ਼ਣਾਂ ਨੂੰ ਸਮਕਾਲੀ ਤੌਰ 'ਤੇ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਦਾ ਹੈ
3. **Planner Agent**: ਸਮਕਾਲੀ ਤੌਰ 'ਤੇ ਵਿਸਥਾਰਪੂਰਣ ਯਾਤਰਾ ਯੋਜਨਾਵਾਂ ਬਣਾਉਂਦਾ ਹੈ
4. **ConcurrentAggregationExecutor**: ਦੋਹਾਂ ਏਜੰਟਾਂ ਤੋਂ ਨਤੀਜੇ ਇਕੱਠੇ ਕਰਦਾ ਅਤੇ ਮਿਲਾਉਂਦਾ ਹੈ

### 🎯 **Fan-Out/Fan-In ਪੈਟਰਨ**

ਇਹ ਵਰਕਫਲੋ ਕਲਾਸਿਕ **Fan-Out/Fan-In** ਪੈਟਰਨ ਦਰਸਾਉਂਦਾ ਹੈ:
- **Fan-Out**: ਇੱਕ ਇਨਪੁਟ ਸੁਨੇਹਾ ਕਈ ਏਜੰਟਾਂ ਨੂੰ ਸਮੇਂ ਵਿੱਚ ਭੇਜਿਆ ਜਾਂਦਾ ਹੈ
- **ਸਮਕਾਲੀ ਪ੍ਰੋਸੈਸਿੰਗ**: ਕਈ ਏਜੰਟ ਇੱਕੋ ਕੰਮ 'ਤੇ ਇਕੱਠੇ ਕੰਮ ਕਰਦੇ ਹਨ
- **Fan-In**: ਸਾਰੇ ਏਜੰਟਾਂ ਤੋਂ ਨਤੀਜੇ ਇਕੱਠੇ ਕਰਕੇ ਇੱਕ ਸਰਵ-ਆਊਟਪੁੱਟ ਬਣਾਇਆ ਜਾਂਦਾ ਹੈ

### 🎯 ਅਸਲੀ ਦੁਨੀਆ ਦੀਆਂ ਐਪਲੀਕੇਸ਼ਨਜ਼

- **ਸਮੱਗਰੀ ਸਿਰਜਣਾ**: ਵੱਖ-ਵੱਖ ਲੇਖਕ ਸਮੇਂ ਵਿੱਚ ਵੱਖ-ਵੱਖ ਹਿੱਸੇ ਲਿਖਦੇ ਹਨ
- **ਕੋਡ ਸਮੀਖਿਆ**: ਵੱਖ-ਵੱਖ ਦ੍ਰਿਸ਼ਟਿਕੋਣਾਂ ਤੋਂ ਕੋਡ ਵਿਸ਼ਲੇਸ਼ਣ ਕਰਨਾ
- **ਬਾਜ਼ਾਰ ਰਿਸਰਚ**: ਵੱਖ-ਵੱਖ ਬਜ਼ਾਰ ਖੰਡਾਂ ਦੀ ਪੈਰੱਲਲ ਵਿਸ਼ਲੇਸ਼ਣ
- **ਡੌਕਯੂਮੈਂਟ ਪ੍ਰੋਸੈਸਿੰਗ**: ਸਮਕਾਲੀ ਤੌਰ 'ਤੇ ਡੈਟਾ ਕੱਡਣਾ, ਵਿਸ਼ਲੇਸ਼ਿਤ ਕਰਨਾ ਅਤੇ ਸਹੀ ਕਰਨਾ
- **ਮਲਟੀ-ਦ੍ਰਿਸ਼ਟੀਕੋਣੀ ਵਿਸ਼ਲੇਸ਼ਣ**: ਇੱਕੋ ਇਨਪੁਟ 'ਤੇ ਵੱਖ-ਵੱਖ ਨਜ਼ਰੀਏ ਲੈਣਾ

### 🔍 ਵਿਅਕਤੀਗਤ ਏਗਜ਼ੀਕਿਊਟਰਾਂ ਨੂੰ ਸਮਝਣਾ

**ConcurrentStartExecutor:**
- `IMessageHandler<string>` ਨੂੰ ਲਾਗੂ ਕਰਦਾ ਹੈ ਤਾਂ ਜੋ ਸਤਰ ਇਨਪੁਟ ਲਿਆ ਜਾ ਸਕੇ
- ਸਭ ਜੁੜੇ ਹੋਏ ਏਜੰਟਾਂ ਨੂੰ ਸੁਨੇਹਾ ਭੇਜਦਾ ਹੈ
- ਸਮਕਾਲੀ ਪ੍ਰੋਸੈਸਿੰਗ ਲਈ `TurnToken` ਭੇਜਦਾ ਹੈ

**ConcurrentAggregationExecutor:**
- `IMessageHandler<ChatMessage>` ਲਾਗੂ ਕਰਦਾ ਹੈ ਜੀਹੜਾ ਏਜੰਟਾਂ ਤੋਂ ਜਵਾਬ ਲੈਂਦਾ ਹੈ
- ਧਾਗਾ-ਸੁਰੱਖਿਅਤ ਢੰਗ ਨਾਲ ਸੁਨੇਹਿਆਂ ਨੂੰ ਇਕੱਠਾ ਕਰਦਾ ਹੈ
- ਜਦੋਂ ਸਾਰੇ ਉਮੀਦ ਕੀਤੇ ਜਵਾਬ ਆ ਜਾਂਦੇ ਹਨ ਤਾਂ ਸੰਯੋਜਿਤ ਕਰਦਾ ਹੈ
- `context.YieldOutputAsync()` ਦੀ ਵਰਤੋਂ ਨਾਲ ਅੰਤਿਮ ਆਊਟਪੁੱਟ ਦਿੰਦਾ ਹੈ

### ⚡ ਕਾਰਗਰਤਾ ਦੇ ਲਾਭ

**ਸਮਕਾਲੀ ਵ ਅਨੁਕ੍ਰਮਿਕ ਤਰ੍ਹਾਂ:**
- ਅਨੁਕ੍ਰਮਿਕ: Agent1 (30s) → Agent2 (30s) = **ਕੁੱਲ 60 ਸਕਿੰਟ**
- ਸਮਕਾਲੀ: Agent1 (30s) || Agent2 (30s) = **ਕੁੱਲ 30 ਸਕਿੰਟ**

**ਤਰੰਗਦਾਰਤਾ ਵਿੱਚ ਸੁਧਾਰ**: N ਸਮਕਾਲੀ ਏਜੰਟਾਂ ਲਈ N ਗੁਣਾ ਤੇਜ਼ (ਕੰਮ ਦੇ ਭਾਰ ਅਤੇ ਸੰਸਾਧਨਾਂ ਉੱਤੇ ਨਿਰਭਰ)
