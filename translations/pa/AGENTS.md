## ਸੈਟਅਪ ਕਮਾਂਡਸ

### ਪੂਰਵ ਸ਼ਰਤਾਂ
- ਪਾਇਥਨ 3.12 ਜਾਂ ਵੱਧ
- ਅਜ਼ੂਰ ਸਬਸਕ੍ਰਿਪਸ਼ਨ (ਮਾਈਕ੍ਰੋਸਾਫਟ ਫਾਊਂਡਰੀ ਲਈ)
- ਅਜ਼ੂਰ CLI ਇੰਸਟਾਲ ਅਤੇ ਪ੍ਰਮਾਣਿਕ੍ਰਿਤ ( `az login` )

### ਸ਼ੁਰੂਆਤੀ ਸੈਟਅਪ

1. **ਰਿਪੋਜ਼ਟਰੀ ਨੂੰ ਕਲੋਨ ਜਾਂ ਫੋਰਕ ਕਰੋ:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # ਜਾਂ
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **ਪਾਇਥਨ ਵਰਚੁਅਲ ਐਨਵਾਇਰਨਮੈਂਟ ਬਣਾਓ ਅਤੇ ਚਾਲੂ ਕਰੋ:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # ਵਿਂਡੋਜ਼ ਤੇ: venv\Scripts\activate
   ```

3. **ਡਿਪੈਂਡੈਂਸੀਜ਼ ਇੰਸਟਾਲ ਕਰੋ:**
   ```bash
   pip install -r requirements.txt
   ```

4. **ਐਨਵਾਇਰਨਮੈਂਟ ਵੈਰੀਏਬਲ ਸੈਟ ਕਰੋ:**
   ```bash
   cp .env.example .env
   # ਆਪਣੇ API ਕੁੰਜੀਆਂ ਅਤੇ ਐਂਡਪੁਆਇੰਟਾਂ ਨਾਲ .env ਸੰਪਾਦਿਤ ਕਰੋ
   ```

### ਜਰੂਰੀ ਐਨਵਾਇਰਨਮੈਂਟ ਵੈਰੀਏਬਲ

**ਮਾਈਕ੍ਰੋਸਾਫਟ ਫਾਊਂਡਰੀ** ਲਈ (ਲਾਜ਼ਮੀ):
- `AZURE_AI_PROJECT_ENDPOINT` - ਮਾਈਕ੍ਰੋਸਾਫਟ ਫਾਊਂਡਰੀ ਪਰੋਜੈਕਟ ਐਂਡਪੋਇੰਟ
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - ਮਾਡਲ ਡਿਪਲੋਇਮੈਂਟ ਦਾ ਨਾਮ (ਜਿਵੇਂ gpt-4.1-mini)

**ਅਜ਼ੂਰ ਏਆਈ ਖੋਜ** ਲਈ (ਪਾਠ 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - ਅਜ਼ੂਰ ਏਆਈ ਖੋਜ ਐਂਡਪੋਇੰਟ
- `AZURE_SEARCH_API_KEY` - ਅਜ਼ੂਰ ਏਆਈ ਖੋਜ ਏਪੀਆਈ ਕੁੰਜੀ

ਪ੍ਰਮਾਣਿਕਰਨ: ਨੋਟਬੁੱਕ ਚਲਾਉਣ ਤੋਂ ਪਹਿਲਾਂ `az login` ਚਲਾਓ (`AzureCliCredential` ਵਰਤਦਾ ਹੈ)।

## ਟੈਸਟਿੰਗ ਨਿਰਦੇਸ਼

ਇਹ ਇਕ ਸਿੱਖਿਆਦਾਇਕ ਰਿਪੋਜ਼ਟਰੀ ਹੈ ਜਿਸ ਵਿੱਚ ਪ੍ਰੋਡਕਸ਼ਨ ਕੋਡ ਨਾ ਹੋ ਕੇ ਉਦਾਹਰਨ ਕોડ ਹੈ ਜੋ ਆਟੋਮੈਟਿਕ ਟੈਸਟ ਨਹੀਂ ਰੱਖਦਾ। ਤੁਹਾਡੇ ਸੈਟਅਪ ਅਤੇ ਬਦਲਾਵਾਂ ਦੀ ਜਾਂਚ ਕਰਨ ਲਈ:

### ਮੈਨੂਅਲ ਟੈਸਟਿੰਗ

1. **ਪਾਇਥਨ ਮਾਹੌਲ ਦੀ ਜਾਂਚ ਕਰੋ:**
   ```bash
   python --version  # 3.12+ ਹੋਣਾ ਚਾਹੀਦਾ ਹੈ
   pip list | grep -E "(agent-framework|azure-ai|azure-identity)"
   ```

2. **ਨੋਟਬੁੱਕ ਚਲਾਉਣ ਦੀ ਜਾਂਚ ਕਰੋ:**
   ```bash
   # ਨੋਟਬੁੱਕ ਨੂੰ ਸਕ੍ਰਿਪਟ ਵਿੱਚ ਬਦਲੋ ਅਤੇ ਚਲਾਓ (ਟੈਸਟਾਂ ਇੰਪੋਰਟ ਜਾਂਚਦੇ ਹਨ)
   jupyter nbconvert --to script <lesson-folder>/code_samples/<notebook>.ipynb --stdout | python
   ```

3. **ਐਨਵਾਇਰਨਮੈਂਟ ਵੈਰੀਏਬਲ ਦੀ ਪੁਸ਼ਟੀ ਕਰੋ:**
   ```bash
   python -c "import os; from dotenv import load_dotenv; load_dotenv(); print('✓ AZURE_AI_PROJECT_ENDPOINT' if os.getenv('AZURE_AI_PROJECT_ENDPOINT') else '✗ AZURE_AI_PROJECT_ENDPOINT missing')"
   ```

### ਵਿਅਕਤੀਗਤ ਨੋਟਬੁੱਕ ਚਲਾਉਣਾ

ਜੁਪੀਟਰ ਵਿੱਚ ਨੋਟਬੁੱਕ ਖੋਲ੍ਹੋ ਅਤੇ ਕ੍ਰਮਵਾਰ ਸੈਲ ਚਲਾਓ। ਹਰ ਨੋਟਬੁੱਕ ਖੁਦਮੁਖਤਾਰ ਹੁੰਦਾ ਹੈ ਅਤੇ ਸ਼ਾਮਲ ਕਰਦਾ ਹੈ:
- ਇੰਪੋਰਟ ਕਥਨ
- ਸੰਰਚਨਾ ਲੋਡ ਕਰਨਾ
- ਉਦਾਹਰਨ ਏਜੰਟ ਲਾਗੂਕਾਰ
- ਮਾਰਕਡਾਊਨ ਸੈਲ ਵਿੱਚ ਉਮੀਦ ਕੀਤੇ ਨਤੀਜੇ

### ਡਿਪਲੋਇਡ ਏਜੰਟਸ ਦੀ ਸਮਿੱਜ ਟੈਸਟਿੰਗ

ਉਹ ਪਾਠ ਜਿੱਥੇ ਏਜੰਟ ਮਾਈਕ੍ਰੋਸਾਫਟ ਫਾਊਂਡਰੀ ਮੇਜ਼ਬਾਨ ਏਜੰਟ ਵਜੋਂ ਡਿਪਲੋਇਡ ਕੀਤਾ ਗਿਆ ਹੈ (01, 04, 05, 16), ਰਿਪੋ ਵਿੱਚ `tests/` ਦੇ ਹੇਠਾਂ ਸਮਿੱਕ ਟੈਸਟ ਕੈਟਾਲਾਗ ਹੁੰਦੇ ਹਨ ਜੋ `.github/workflows/smoke-test.yml` ਵਰਕਫਲੋ ਦੁਆਰਾ [AI Smoke Test](https://github.com/marketplace/actions/ai-smoke-test) ਐਕਸ਼ਨ ਰਾਹੀਂ ਚਲਾਏ ਜਾਂਦੇ ਹਨ। ਇਹ ਇਕ ਹਲਕਾ-ਫੁਲਕਾ ਪੋਸਟ-ਡਿਪਲੋਇਮੈਂਟ ਦਰਵਾਜ਼ਾ ਹਨ (ਕੀ ਏਜੰਟ ਪਹੁੰਚਯੋਗ ਹੈ ਅਤੇ ਮੂਲ ਪ੍ਰੰਪਟ ਉਮੀਦਾਂ ਦਾ ਪਾਲਣ ਕਰ ਰਿਹਾ ਹੈ?) ਜੋ ਪਾਠ 10 ਅਤੇ 16 ਵਿੱਚ ਮੁਲਾਂਕਣ ਪਾਈਪਲਾਈਨ ਨੂੰ ਪੂਰਾ ਕਰਦੇ ਹਨ। ਕੈਟਾਲਾਗ-ਤੋਂ-ਪਾਠ-ਤੋਂ-ਏਜੰਟ ਨਕਸ਼ਾ ਲਈ [tests/README.md](./tests/README.md) ਵੇਖੋ। ਪਾਠ 17 ਸਥਾਨਕ ਤੌਰ ਤੇ ਫਾਊਂਡਰੀ ਵਿੱਚ ਚਲਦਾ ਹੈ ਅਤੇ ਇਸਦਾ ਕੋਈ ਮੇਜ਼ਬਾਨ ਐਂਡਪੋਇੰਟ ਨਹੀਂ ਹੈ, ਇਸ ਲਈ ਇਹ ਆਪਣੇ ਨੋਟਬੁੱਕ ਨੂੰ ਸਿੱਧਾ ਚਲਾ ਕੇ ਪਰਖਿਆ ਜਾਂਦਾ ਹੈ।

   - ਕੁਝ ਨੋਟਬੁੱਕਸ ਵਿੱਚ ਅਨੁਵਾਦ ਮਾਰਕਡਾਊਨ ਸੈਲਾਂ ਵਿੱਚ ਕਿਹਾ ਹੋਇਆ ਹੋਰ ਪੈਕੇਜ ਲੋੜੀਂਦੇ ਹੋ ਸਕਦੇ ਹਨ

   - ਆਪਣਾ ਅਜ਼ੂਰ OpenAI ਮਾਡਲ ਡਿਪਲੋਇਮੈਂਟ Responses API ਸਹਾਇਤਾ ਕਰਦਾ ਹੋਵੇ ਇਹ ਯਕੀਨੀ ਬਣਾਓ
