## اسم الحدث: بناء تطبيقك المعتمد على الكود أولاً باستخدام خدمة Microsoft Foundry Agent (عرض EMEA/الولايات المتحدة)

## الوصف

## اسم الحدث: بناء وكلاء الذكاء الاصطناعي النموذجيين مع نماذج GitHub

> **ملاحظة:** نماذج GitHub أصبحت مهجورة (ستتوقف في يوليو 2026). لا تستخدم نماذج GitHub للمشاريع الجديدة — استخدم نماذج **Microsoft Foundry** بدلاً منها.

## اسم الحدث: واجهة برمجة تطبيقات متعددة الوكلاء مع LangGraph و Azure Cosmos DB

## الوصف

إن صعود تطبيقات الذكاء الاصطناعي متعددة الوكلاء يغير طريقة بناء الأنظمة الذكية - لكن كيف تصممها لتكون قابلة للتوسع والأداء في الواقع؟ في هذه الجلسة، سنتناول بعمق تطبيق إنتاجي متعدد الوكلاء مبني باستخدام LangGraph لتنظيم الوكلاء، FastAPI كطبقة API، و Azure Cosmos DB كقاعدة أساسية لإدارة الحالة، وتخزين المتجهات، والبيانات المعاملاتية.

من خلال استعراض مفصل للكود، سترى كيفية تصميم وتنفيذ سير عمل يقوده الوكلاء يدمج بسلاسة التوليد المعزز بالاسترجاع (RAG)، واستمرارية الذاكرة، والتحولات الديناميكية للحالة. سنغطي:

التعاون بين الوكلاء باستخدام LangGraph للتفكير المنظم
تخزين تاريخ المحادثات في الوقت الحقيقي باستخدام Azure Cosmos DB - نفس قاعدة البيانات التي تدعم تاريخ المحادثات في ChatGPT، أسرع تطبيق وكيل ذكاء اصطناعي نمواً في التاريخ
البحث بالمتجهات لاسترجاع المعرفة بدعم الإدخالات الأصلية في Cosmos DB
قدرات FastAPI غير المتزامنة للحفاظ على استجابة وتوسع التفاعلات
بحلول نهاية هذه الجلسة، ستحصل على مخطط واضح لبناء ونشر تطبيقات متعددة الوكلاء قابلة للتوسع والسحابية الأصلية تستفيد من قوة الذكاء الاصطناعي الحديث والبنية التحتية السحابية. سواء كنت مهندس ذكاء اصطناعي، مهندس سحابة، أو مطور بايثون، ستزودك هذه المحاضرة برؤى عملية وأنماط مثبتة لبناء الجيل القادم من التطبيقات المعززة بالذكاء الاصطناعي

## اسم الحدث: التوجيه هو البرمجة الجديدة: تعرف على GenAIScript

## الوصف

جعلت jQuery تطوير الويب أسهل ذات مرة من خلال تجريد التعقيدات، مما سمح للمطورين بالتركيز على البناء بدلاً من التعامل مع شذوذ المتصفحات. اليوم، يواجه تطوير الذكاء الاصطناعي تحدياً مماثلاً. تظهر أنماط جديدة باستمرار، وقد يكون مجاراة ذلك مرهقاً، لا سيما مع تطور أدوات الذكاء الاصطناعي — خاصة الوكلاء — لتصبح أكثر قوة وتعقيداً. ماذا لو استطعت استغلال قدرات الذكاء الاصطناعي المتطورة لأتمتة المهام باستخدام تجريدات جافا سكريبت بسيطة ومألوفة؟ إليك GenAIScript — طريقة لدمج الذكاء الاصطناعي في سير عملك بسهولة، معاملة التوجيهات كقصاصات كود قابلة لإعادة الاستخدام. في هذه المحاضرة، سنستعرض كيف يجعل GenAIScript وكلاء الأتمتة بالذكاء الاصطناعي يشعرون بأنهم بديهيون ككتابة جافا سكريبت، مما يساعدك على تبسيط الأعمال المتكررة دون الحاجة إلى خبرة عميقة في الذكاء الاصطناعي.
