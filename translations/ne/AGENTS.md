# AGENTS.md

## परियोजना अवलोकन

यो रिपोजिटरीमा "शुरुआतीहरूका लागि AI एजेन्टहरू" समावेश छ - AI एजेन्टहरू निर्माण गर्न आवश्यक सबै कुरा सिकाउने व्यापक शैक्षिक कोर्स। कोर्समा आधारहरू, डिजाइन ढाँचा, फ्रेमवर्कहरू, उत्पादन परिनियोजन, स्थानीय/डिभाइस एजेन्टहरू, र AI एजेन्टहरूको सुरक्षा समेटेर १८ वटा पाठहरू (००-१८ नम्बरका) छन्।

**प्रमुख प्रविधिहरू:**
- Python 3.12+
- अन्तरक्रियात्मक सिकाइका लागि Jupyter नोटबुकहरू
- AI फ्रेमवर्कहरू: Microsoft Agent Framework (MAF)
- Azure AI सेवाहरू: Microsoft Foundry, Microsoft Foundry Agent Service V2

**आर्किटेक्चर:**
- पाठ-आधारित संरचना (००-१५+ डाइरेक्टोरीहरू)
- प्रत्येक पाठमा: README डकुमेन्टेसन, कोड नमूनाहरू (Jupyter नोटबुकहरू), र तस्वीरहरू छन्
- स्वचालित अनुवाद प्रणालीमार्फत बहुभाषी समर्थन
- Microsoft Agent Framework प्रयोग गरी प्रत्येक पाठको लागि एउटा Python नोटबुक

## सेटअप आदेशहरू

### पूर्वशर्तहरू
- Python 3.12 वा नयाँ संस्करण
- Azure सदस्यता (Microsoft Foundry का लागि)
- Azure CLI स्थापना र प्रमाणीकरण (`az login`)

### प्रारम्भिक सेटअप

१. **रिपोजिटरी क्लोन वा फोर्क गर्नुहोस्:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # वा
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

२. **Python भर्चुअल वातावरण सिर्जना र सक्रिय गर्नुहोस्:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Windows मा: venv\Scripts\activate
   ```

३. **आवश्यकता स्थापना गर्नुहोस्:**
   ```bash
   pip install -r requirements.txt
   ```

४. **पर्यावरण चरहरू सेटअप गर्नुहोस्:**
   ```bash
   cp .env.example .env
   # तपाईंको API कुञ्जीहरू र endpoints सहित .env सम्पादन गर्नुहोस्
   ```

### आवश्यक वातावरण चरहरू

**Microsoft Foundry** का लागि (आवश्यक):
- `AZURE_AI_PROJECT_ENDPOINT` - Microsoft Foundry परियोजना अन्तबिन्दु
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - मोडेल परिनियोजन नाम (जस्तै, gpt-5-mini)

**Azure AI Search** का लागि (पाठ ०५ - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search अन्तबिन्दु
- `AZURE_SEARCH_API_KEY` - Azure AI Search API कुंजी

प्रमाणीकरण: नोटबुकहरू चलाउनु अघि `az login` चलाउनुहोस् (`AzureCliCredential` प्रयोग हुन्छ)।
