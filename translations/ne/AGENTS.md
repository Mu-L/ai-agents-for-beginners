**मुख्य प्रविधिहरू:**
- Python 3.12+
- अन्तरक्रियात्मक अध्ययनका लागि Jupyter नोटबुकहरू
- AI फ्रेमवर्कहरू: Microsoft Agent Framework (MAF)
- Azure AI सेवा: Microsoft Foundry, Microsoft Foundry Agent Service V2

**वास्तुकला:**
- पाठहरूमा आधारित संरचना (00-15+ डाइरेक्टोरीहरू)
- प्रत्येक पाठमा हुन्छ: README कागजात, कोड नमूनाहरू (Jupyter नोटबुकहरू), र छविहरू
- स्वचालित अनुवाद प्रणाली मार्फत बहुभाषी समर्थन
- प्रत्येक पाठका लागि एक Python नोटबुक जसले Microsoft Agent Framework प्रयोग गर्छ

## सेटअप आदेशहरू

### आवश्यकताहरू
- Python 3.12 वा उच्च संस्करण
- Azure सदस्यता (Microsoft Foundry को लागि)
- Azure CLI स्थापना र प्रमाणीकरण (`az login`)

### प्रारम्भिक सेटअप

1. **संग्रहालय क्लोन वा फोर्क गर्नुहोस्:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # वा
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python भर्चुअल वातावरण सिर्जना र सक्रिय गर्नुहोस्:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # विन्डोजमा: venv\Scripts\activate
   ```

3. **आवश्यकता स्थापना गर्नुहोस्:**
   ```bash
   pip install -r requirements.txt
   ```

4. **पर्यावरण चरहरू सेट गर्नुहोस्:**
   ```bash
   cp .env.example .env
   # आफ्नो API कुञ्जीहरू र अन्त बिन्दुहरूसँग .env सम्पादन गर्नुहोस्
   ```

### आवश्यक पर्यावरण चरहरू

**Microsoft Foundry** को लागि (आवश्यक):
- `AZURE_AI_PROJECT_ENDPOINT` - Microsoft Foundry परियोजना अन्त बिन्दु
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - मोडेल परिनियोजन नाम (उदाहरण: gpt-4.1-mini)

**Azure AI Search** को लागि (पाठ 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI खोज अन्त बिन्दु
- `AZURE_SEARCH_API_KEY` - Azure AI खोज API कुञ्जी

प्रमाणीकरण: नोटबुकहरू चलाउनु अघि `az login` चलाउनुहोस् (`AzureCliCredential` प्रयोग गर्छ)।
