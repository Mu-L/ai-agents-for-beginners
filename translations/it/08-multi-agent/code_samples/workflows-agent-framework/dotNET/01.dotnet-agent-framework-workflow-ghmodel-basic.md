# 🔄 Flussi di Lavoro di Base con Agenti usando Azure OpenAI (Responses API) (.NET)

## 📋 Tutorial di Orchestrazione dei Flussi di Lavoro

Questo notebook dimostra come costruire sofisticati **flussi di lavoro con agenti** utilizzando il Microsoft Agent Framework per .NET e Azure OpenAI (Responses API). Imparerai a creare processi aziendali multi-step in cui agenti AI collaborano per compiere attività complesse tramite modelli di orchestrazione strutturati.

## 🎯 Obiettivi di Apprendimento

### 🏗️ **Fondamenti dell'Architettura dei Flussi di Lavoro**
- **Costruttore di Flussi di Lavoro**: Progetta e orchestra processi AI multi-step complessi

- **Azure OpenAI (Responses API)**: Sfrutta l'API Azure OpenAI Responses nei flussi di lavoro

```xml
<!-- Configuration and Environment -->
<PackageReference Include="DotNetEnv" Version="3.1.1" />
```

### 🔑 **Configurazione Azure OpenAI**

**Configurazione dell'Ambiente (file .env):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-4.1-mini
```

**Accesso ad Azure OpenAI:**
1. Crea una risorsa Azure OpenAI nel portale Azure
2. Distribuisci un modello (per esempio, `gpt-4.1-mini`) e annota il nome della distribuzione
3. Accedi con `az login` e configura le variabili ambiente come mostrato sopra

## 💻 Esecuzione del Codice

L'implementazione completa è disponibile in `01.dotnet-agent-framework-workflow-ghmodel-basic.cs`. Questo file dimostra:

1. **Configurazione dell'Ambiente** - Caricamento della configurazione Azure OpenAI dal file `.env`
2. **Configurazione Client Azure OpenAI** - Configurazione del client per usare l'Azure OpenAI Responses API
3. **Creazione degli Agenti** - Definizione di agenti specializzati (Front Desk e Concierge)
4. **Costruttore del Flusso** - Creazione di un flusso multi-agenti con elaborazione sequenziale
5. **Esecuzione del Flusso** - Esecuzione del flusso con risultati in streaming

### 🚀 Esecuzione dell'Esempio

```bash
# Rendi lo script eseguibile (Unix/Linux/macOS)
chmod +x 01.dotnet-agent-framework-workflow-ghmodel-basic.cs

# Esegui il flusso di lavoro
./01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```

Oppure su Windows:
```powershell
dotnet run 01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```

### 📝 Output Atteso

Il flusso di lavoro:
1. Riceverà la tua richiesta di destinazione di viaggio ("Vorrei andare a Parigi")
2. L'agente Front Desk fornirà una raccomandazione iniziale
3. L'agente Concierge rivedrà e perfezionerà la raccomandazione
4. L'output finale mostrerà l'intero flusso della conversazione
