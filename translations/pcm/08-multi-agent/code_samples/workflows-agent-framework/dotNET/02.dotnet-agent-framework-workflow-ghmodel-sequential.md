### 🔍 How State Flow Dey Work

Each agent for di sequence dey receive:
- **Original Input**: Di first user message (image + text)
- **Previous Agent Outputs**: All di answers wey previous agents don give for di conversation
- **Accumulated Context**: Full state wey di workflow don keep so far

Dis one allow advanced multi-stage processing where each agent go build on top correct context from all previous stages.
