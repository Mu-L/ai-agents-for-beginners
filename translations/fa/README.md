> **ترجیح می‌دهید به صورت محلی کلون کنید؟**
>
> این مخزن شامل ترجمه به بیش از ۵۰ زبان است که حجم دانلود را به طور قابل توجهی افزایش می‌دهد. برای کلون کردن بدون ترجمه، از sparse checkout استفاده کنید:
>
> **Bash / macOS / لینوکس:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (ویندوز):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> این همه چیز لازم برای کامل کردن دوره را با دانلود بسیار سریع‌تر به شما می‌دهد.

## 🌱 شروع به کار

این دوره شامل درس‌هایی درباره اصول ساخت عامل‌های هوش مصنوعی است. هر درس موضوع خاص خود را پوشش می‌دهد، پس از هر جایی که دوست دارید شروع کنید!

پشتیبانی چند زبانه برای این دوره وجود دارد. به [زبان‌های موجود اینجا](#-multi-language-support) مراجعه کنید. 

اگر اولین بار است که با مدل‌های هوش مصنوعی مولد کار می‌کنید، دوره [هوش مصنوعی مولد برای مبتدیان](https://aka.ms/genai-beginners) ما را بررسی کنید، که شامل ۲۱ درس درباره ساخت با GenAI است.

فراموش نکنید که به این مخزن [ستاره (🌟) بدهید](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) و [این مخزن را فورک کنید](https://github.com/microsoft/ai-agents-for-beginners/fork) تا کد را اجرا کنید.

### ملاقات با سایر یادگیرندگان، دریافت پاسخ سوالاتتان

اگر گیر کردید یا سوالی درباره ساخت عامل‌های هوش مصنوعی داشتید، به کانال دیسکورد اختصاصی ما در [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) بپیوندید.

### آنچه نیاز دارید 

هر درس در این دوره شامل نمونه کد است، که در پوشه code_samples یافت می‌شود. می‌توانید [این مخزن را فورک کنید](https://github.com/microsoft/ai-agents-for-beginners/fork) تا نسخه خودتان را بسازید.  

نمونه‌های کد در این تمرین‌ها از Microsoft Agent Framework با Microsoft Foundry Agent Service V2 استفاده می‌کنند:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - نیازمند حساب Azure

این دوره از فریم‌ورک‌ها و سرویس‌های عامل هوش مصنوعی زیر از مایکروسافت استفاده می‌کند:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

برخی نمونه‌های کد همچنین از ارائه‌دهندگان سازگار با OpenAI مانند [MiniMax](https://platform.minimaxi.com/) که مدل‌های با متن طولانی (تا ۲۰۴ هزار توکن) ارائه می‌دهند، پشتیبانی می‌کنند. برای جزئیات پیکربندی به [راه‌اندازی دوره](./00-course-setup/README.md) مراجعه کنید.

برای اطلاعات بیشتر در مورد اجرای کد این دوره، به [راه‌اندازی دوره](./00-course-setup/README.md) مراجعه کنید.
