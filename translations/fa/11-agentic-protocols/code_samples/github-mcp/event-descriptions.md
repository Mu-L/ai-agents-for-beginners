## نام رویداد: API چندعامله با LangGraph و Azure Cosmos DB

## توضیحات

ظهور برنامه‌های هوش مصنوعی چندعامله، نحوه ساخت سیستم‌های هوشمند را متحول کرده است - اما چگونه می‌توان آنها را برای مقیاس‌پذیری و عملکرد واقعی ساختاردهی کرد؟ در این جلسه، به طور عمیق به یک برنامه چندعامله حرفه‌ای ساخته شده با LangGraph برای هماهنگی عوامل، FastAPI برای لایه API و Azure Cosmos DB به عنوان ستون فقرات مدیریت وضعیت، ذخیره‌سازی برداری و داده‌های تراکنشی خواهیم پرداخت.

از طریق مرور دقیق کد، خواهید دید چگونه یک جریان کاری بر اساس عامل طراحی و پیاده‌سازی می‌شود که بدون درز، تولید تقویت‌شده با بازیابی (RAG)، پایداری حافظه و انتقالات پویا وضعیت را ادغام می‌کند. موضوعات پوشش داده شده شامل:

همکاری عامل‌ها با LangGraph برای استدلال ساختارمند
ذخیره تاریخچه چت در زمان واقعی با استفاده از Azure Cosmos DB - همان پایگاه‌داده‌ای که تاریخچه چت را در ChatGPT، سریع‌ترین برنامه عامل هوش مصنوعی در تاریخ، پشتیبانی می‌کند
جستجوی برداری برای بازیابی دانش با پشتیبانی داخلی تعبیه‌های Cosmos DB
قابلیت‌های ناهمگام FastAPI برای حفظ پاسخگویی و مقیاس‌پذیری تعاملات
در پایان این جلسه، نقشه راه واضحی برای ساخت و اجرای برنامه‌های چندعامله مقیاس‌پذیر و بومی ابر خواهید داشت که قدرت هوش مصنوعی مدرن و زیرساخت‌های ابری را به کار می‌گیرند. چه مهندس هوش مصنوعی، معمار ابری یا توسعه‌دهنده پایتون باشید، این گفتار نکات عملی و الگوهای آزمایش شده برای ساخت نسل بعدی برنامه‌های مجهز به هوش مصنوعی را در اختیار شما قرار می‌دهد.
