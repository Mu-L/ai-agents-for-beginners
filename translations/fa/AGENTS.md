# AGENTS.md

## مرور کلی پروژه

این مخزن شامل "عامل‌های هوش مصنوعی برای مبتدیان" است - یک دوره آموزشی جامع که همه چیز لازم برای ساخت عامل‌های هوش مصنوعی را آموزش می‌دهد. این دوره شامل ۱۸ درس (شماره‌گذاری شده از ۰۰ تا ۱۸) است که مبانی، الگوهای طراحی، چارچوب‌ها، استقرار تولید، عامل‌های محلی / در دستگاه و امنیت عامل‌های هوش مصنوعی را پوشش می‌دهد.

**فناوری‌های کلیدی:**
- پایتون ۳.۱۲+
- نوت‌بوک‌های Jupyter برای یادگیری تعاملی
- چارچوب‌های هوش مصنوعی: Microsoft Agent Framework (MAF)
- خدمات Azure AI: Microsoft Foundry، Microsoft Foundry Agent Service V2

**معماری:**
- ساختار مبتنی بر درس‌ها (فهرست‌های ۰۰-۱۵+)
- هر درس شامل: مستندات README، نمونه کدها (نوت‌بوک‌های Jupyter) و تصاویر
- پشتیبانی چندزبانه از طریق سیستم ترجمه خودکار
- یک نوت‌بوک پایتون برای هر درس با استفاده از Microsoft Agent Framework

## دستورات راه‌اندازی

### پیش‌نیازها
- پایتون ۳.۱۲ یا بالاتر
- اشتراک Azure (برای Microsoft Foundry)
- نصب و احراز هویت Azure CLI (`az login`)

### راه‌اندازی اولیه

۱. **کلون یا فورک مخزن:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # یا
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

۲. **ایجاد و فعال‌سازی محیط مجازی پایتون:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # در ویندوز: venv\Scripts\activate
   ```

۳. **نصب وابستگی‌ها:**
   ```bash
   pip install -r requirements.txt
   ```

۴. **تنظیم متغیرهای محیطی:**
   ```bash
   cp .env.example .env
   # فایل .env را با کلیدهای API و نقاط انتهایی خود ویرایش کنید
   ```

### متغیرهای محیطی موردنیاز

برای **Microsoft Foundry** (الزامی):
- `AZURE_AI_PROJECT_ENDPOINT` - نقطه پایانی پروژه Microsoft Foundry
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - نام استقرار مدل (مثلاً gpt-4.1-mini)

برای **جستجوی Azure AI** (درس ۰۵ - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - نقطه پایانی سرویس جستجوی Azure AI
- `AZURE_SEARCH_API_KEY` - کلید API جستجوی Azure AI

احراز هویت: قبل از اجرای نوت‌بوک‌ها، دستور `az login` را اجرا کنید (از `AzureCliCredential` استفاده می‌کند).

## روند توسعه

### اجرای نوت‌بوک‌های Jupyter

هر درس شامل چندین نوت‌بوک Jupyter برای چارچوب‌های مختلف است:

۱. **شروع Jupyter:**
   ```bash
   jupyter notebook
   ```

۲. **رفتن به دایرکتوری درس** (مثلاً `01-intro-to-ai-agents/code_samples/`)

۳. **باز کردن و اجرای نوت‌بوک‌ها:**
   - `*-python-agent-framework.ipynb` - با استفاده از Microsoft Agent Framework (پایتون)
   - `*-dotnet-agent-framework.ipynb` - با استفاده از Microsoft Agent Framework (.NET)

### کار با Microsoft Agent Framework

**Microsoft Agent Framework + Microsoft Foundry:**
- نیاز به اشتراک Azure دارد
- از `FoundryChatClient` برای Agent Service V2 استفاده می‌کند (عامل‌ها در پورتال Foundry قابل مشاهده‌اند)
- آماده تولید با قابلیت مشاهده‌پذیری داخلی
- الگوی فایل: `*-python-agent-framework.ipynb`
