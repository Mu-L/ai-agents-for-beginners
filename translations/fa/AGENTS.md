# AGENTS.md

## مرور پروژه

این مخزن شامل "عامل‌های هوش مصنوعی برای مبتدیان" است - یک دوره آموزشی جامع که همه موارد لازم برای ساخت عامل‌های هوش مصنوعی را آموزش می‌دهد. این دوره شامل ۱۸ درس (شماره‌گذاری شده از ۰۰ تا ۱۸) است که اصول پایه، الگوهای طراحی، چارچوب‌ها، استقرار در تولید، عامل‌های محلی/روی دستگاه، و امنیت عامل‌های هوش مصنوعی را پوشش می‌دهد.

**فناوری‌های کلیدی:**
- پایتون ۳.۱۲ به بالا
- دفترچه‌های Jupyter برای یادگیری تعاملی
- چارچوب‌های هوش مصنوعی: چارچوب عامل مایکروسافت (MAF)
- خدمات هوش مصنوعی آژور: Microsoft Foundry، سرویس عامل Microsoft Foundry نسخه ۲

**معماری:**
- ساختار مبتنی بر درس (پوشه‌های ۰۰-۱۵+)
- هر درس شامل: مستندات README، نمونه کدها (دفترچه‌های Jupyter) و تصاویر
- پشتیبانی چند زبانه از طریق سیستم ترجمه خودکار
- یک دفترچه پایتون برای هر درس با استفاده از چارچوب عامل مایکروسافت

## دستورات راه‌اندازی

### پیش نیازها
- پایتون ۳.۱۲ یا بالاتر
- اشتراک آژور (برای Microsoft Foundry)
- نصب و ورود به Azure CLI (`az login`)

### راه‌اندازی اولیه

۱. **کلون یا فورک کردن مخزن:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # یا
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

۲. **ایجاد و فعال‌سازی محیط مجازی پایتون:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # در ویندوز: venv\Scripts\activate
   ```

۳. **نصب وابستگی‌ها:**
   ```bash
   pip install -r requirements.txt
   ```

۴. **تنظیم متغیرهای محیطی:**
   ```bash
   cp .env.example .env
   # فایل .env را با کلیدهای API و نقاط انتهایی خود ویرایش کنید
   ```

### متغیرهای محیطی مورد نیاز

برای **Microsoft Foundry** (الزامی):
- `AZURE_AI_PROJECT_ENDPOINT` - نقطه انتهایی پروژه Microsoft Foundry
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - نام استقرار مدل (مثلاً gpt-5-mini)

برای **جستجوی Azure AI** (درس ۰۵ - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - نقطه انتهایی جستجوی Azure AI
- `AZURE_SEARCH_API_KEY` - کلید API جستجوی Azure AI

احراز هویت: قبل از اجرای دفترچه‌ها `az login` اجرا کنید (از `AzureCliCredential` استفاده می‌شود).

## جریان کاری توسعه

### اجرای دفترچه‌های Jupyter

هر درس شامل چندین دفترچه Jupyter برای چارچوب‌های مختلف است:

۱. **راه‌اندازی Jupyter:**
   ```bash
   jupyter notebook
   ```

۲. **رفتن به دایرکتوری درس** (مثلاً `01-intro-to-ai-agents/code_samples/`)

۳. **باز کردن و اجرای دفترچه‌ها:**
   - `*-python-agent-framework.ipynb` - استفاده از چارچوب عامل مایکروسافت (پایتون)
   - `*-dotnet-agent-framework.ipynb` - استفاده از چارچوب عامل مایکروسافت (.NET)

### کار با چارچوب عامل مایکروسافت

**چارچوب عامل مایکروسافت + Microsoft Foundry:**
- نیاز به اشتراک Azure
- استفاده از `FoundryChatClient` برای سرویس عامل نسخه ۲ (عامل‌ها در پورتال Foundry قابل مشاهده‌اند)
- آماده تولید با قابلیت مشاهده داخلی
- الگوی فایل: `*-python-agent-framework.ipynb`

   ```bash
   python -c "import os; from dotenv import load_dotenv; load_dotenv(); print('✓ AZURE_AI_PROJECT_ENDPOINT' if os.getenv('AZURE_AI_PROJECT_ENDPOINT') else '✗ AZURE_AI_PROJECT_ENDPOINT missing')"
   ```
