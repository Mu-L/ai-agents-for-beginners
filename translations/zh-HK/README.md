## 🌱 開始吧

本課程包含了構建 AI 代理人的基礎課程內容。每堂課涵蓋各自主題，你可以從任何地方開始學習！

本課程支援多語言版本。請參考我們的[可用語言列表](#-multi-language-support)。

如果這是你第一次使用生成式 AI 模型構建，請參考我們的[生成式 AI 初學者課程](https://aka.ms/genai-beginners)，其中包含 21 個有關使用 GenAI 的課程。

別忘了[點星標記 (🌟) 本倉庫](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst)並且[派生本倉庫](https://github.com/microsoft/ai-agents-for-beginners/fork)以運行程式碼。

### 與其他學習者交流，解決你的問題

如果你在建立 AI 代理人遇到困難或有任何疑問，歡迎加入我們在 [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) 的專屬 Discord 頻道。

### 你需要什麼

本課程的每堂課都附有程式碼範例，可在 code_samples 資料夾找到。你可以[派生本倉庫](https://github.com/microsoft/ai-agents-for-beginners/fork)以創建自己的副本。

這些練習的程式碼範例使用 Microsoft Agent Framework 以及 Microsoft Foundry Agent Service V2：

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - 需要 Azure 帳戶

本課程使用以下來自微軟的 AI 代理人框架和服務：

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Microsoft Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

部分程式碼範例也支援其他 OpenAI 相容的提供者，例如提供大上下文模型（最多 204K tokens）的 [MiniMax](https://platform.minimaxi.com/)。請參閱[課程設置](./00-course-setup/README.md)了解配置詳情。

有關運行本課程程式碼的更多資訊，請參見[課程設置](./00-course-setup/README.md)。
