## Begivenhedsnavn: Multi-Agent API med LangGraph og Azure Cosmos DB

## Beskrivelse

Fremkomsten af multi-agent AI-applikationer forandrer, hvordan vi bygger intelligente systemer – men hvordan arkitekterer man dem til reel skalerbarhed og ydeevne? I denne session går vi i dybden med en produktionsklar multi-agent-applikation bygget med LangGraph til agentorkestrering, FastAPI som API-lag og Azure Cosmos DB som rygsøjle for tilstandsadministration, vektorlager og transaktionsdata.

Gennem en detaljeret gennemgang af koden vil du se, hvordan man designer og implementerer en agentdrevet arbejdsgang, der sømløst integrerer retrieval-augmented generation (RAG), hukommelsespersistence og dynamiske tilstandsovergange. Vi vil dække:

Agent-samarbejde med LangGraph for struktureret ræsonnering
Realtidslagring af chat-historik ved hjælp af Azure Cosmos DB – den samme database, der driver chat-historik i ChatGPT, historiens hurtigst voksende AI-agent-applikation

FastAPIs asynkrone funktioner til at holde interaktionerne responsive og skalerbare
Ved afslutningen af sessionen vil du have en klar plan for at bygge og implementere dine egne skalerbare, cloud-native multi-agent-applikationer, der udnytter kraften i moderne AI og cloud-infrastruktur.
