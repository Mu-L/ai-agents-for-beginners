# 🔄 Grundlæggende Agent Arbejdsgange med Azure OpenAI (Responses API) (.NET)

## 📋 Workflow Orkestrering Tutorial

Denne notesbog demonstrerer, hvordan man bygger sofistikerede **agent arbejdsgange** ved hjælp af Microsoft Agent Framework til .NET og Azure OpenAI (Responses API). Du vil lære at skabe flertrins forretningsprocesser, hvor AI-agenter samarbejder om at udføre komplekse opgaver gennem strukturerede orkestreringsmønstre.

### 🔑 **Azure OpenAI Konfiguration**

**Miljøopsætning (.env-fil):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-5-mini
```

**Azure OpenAI Adgang:**
1. Opret en Azure OpenAI-ressource i Azure-portalen
2. Udrul en model (for eksempel, `gpt-5-mini`) og noter udrulningsnavnet
3. Log ind med `az login` og konfigurer miljøvariabler som vist ovenfor

## 💻 Kørsel af Koden

Den komplette implementering findes i `01.dotnet-agent-framework-workflow-ghmodel-basic.cs`. Denne fil demonstrerer:

1. **Miljøkonfiguration** - Indlæsning af Azure OpenAI konfiguration fra `.env` filen
2. **Azure OpenAI Klient Opsætning** - Konfiguration af klienten til at bruge Azure OpenAI Responses API
3. **Agent Oprettelse** - Definering af specialiserede agenter (Front Desk og Concierge)
4. **Workflow Builder** - Oprettelse af fleragent-arbejdsgang med sekventiel behandling
5. **Workflow Udførelse** - Kørsel af workflow med streaming-resultater

### 🚀 Kørsel af Eksemplet

```bash
# Gør scriptet eksekverbart (Unix/Linux/macOS)
chmod +x 01.dotnet-agent-framework-workflow-ghmodel-basic.cs

# Kør arbejdsgangen
./01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```

Eller på Windows:
```powershell
dotnet run 01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```

### 📝 Forventet Output

Workflowet vil:
1. Modtage din rejsemålsforespørgsel ("Jeg vil gerne til Paris")
2. Front Desk agenten giver en indledende anbefaling
3. Concierge agenten gennemgår og finjusterer anbefalingen
4. Endeligt output viser hele samtalestrømmen
