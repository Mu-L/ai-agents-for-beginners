# 🔄 Các Quy Trình Tác Vụ Cơ Bản cho Agent với Azure OpenAI (Responses API) (.NET)

## 📋 Hướng Dẫn Điều Phối Quy Trình

Tập ghi chú này trình bày cách xây dựng các **quy trình agent** tinh vi sử dụng Microsoft Agent Framework cho .NET và Azure OpenAI (Responses API). Bạn sẽ học cách tạo các quy trình kinh doanh nhiều bước nơi các agent AI hợp tác để hoàn thành các tác vụ phức tạp thông qua các mô hình điều phối có cấu trúc.

## 🎯 Mục Tiêu Học Tập

### 🏗️ **Kiến Trúc Quy Trình Tác Vụ Cơ Bản**
- **Workflow Builder**: Thiết kế và điều phối các quy trình AI nhiều bước phức tạp
- **Agent Coordination**: Phối hợp nhiều agent chuyên môn trong các quy trình
- **Azure OpenAI (Responses API)**: Tận dụng Azure OpenAI Responses API trong quy trình tác vụ
- **Visual Workflow Design**: Tạo và trực quan hóa cấu trúc quy trình để hiểu rõ hơn

### 🔄 **Các Mô Hình Điều Phối Quy Trình**
- **Sequential Processing**: Xâu chuỗi nhiều tác vụ agent theo thứ tự logic
- **State Management**: Duy trì ngữ cảnh và luồng dữ liệu xuyên suốt các giai đoạn quy trình
- **Error Handling**: Triển khai xử lý lỗi và phục hồi mạnh mẽ trong quy trình
- **Performance Optimization**: Thiết kế quy trình hiệu quả cho hoạt động quy mô doanh nghiệp

### 🏢 **Ứng Dụng Quy Trình Doanh Nghiệp**
- **Business Process Automation**: Tự động hóa các quy trình tổ chức phức tạp
- **Content Production Pipeline**: Quy trình biên tập với các bước duyệt và phê duyệt
- **Customer Service Automation**: Giải quyết các yêu cầu khách hàng nhiều bước
- **Data Processing Workflows**: Quy trình ETL với chuyển đổi sử dụng AI

## ⚙️ Yêu Cầu & Cài Đặt

### 📦 **Các Gói NuGet Cần Thiết**

Ví dụ quy trình này sử dụng một số gói .NET chính:

```xml
<!-- Core AI Framework -->
<PackageReference Include="Microsoft.Extensions.AI" Version="10.*" />

<!-- Azure OpenAI (Responses API) -->
<PackageReference Include="Azure.AI.OpenAI" Version="2.1.0" />
<PackageReference Include="Azure.Identity" Version="1.15.0" />

<!-- Agent Framework (Local Development) -->
<!-- Microsoft.Agents.AI.dll - Core agent abstractions -->
<!-- Microsoft.Agents.AI.OpenAI.dll - Azure OpenAI (Responses API) integration -->

<!-- Configuration and Environment -->
<PackageReference Include="DotNetEnv" Version="3.1.1" />
```

### 🔑 **Cấu Hình Azure OpenAI**

**Thiết lập môi trường (tệp .env):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-4.1-mini
```

**Truy cập Azure OpenAI:**
1. Tạo tài nguyên Azure OpenAI trên cổng Azure
2. Triển khai một mô hình (ví dụ, `gpt-4.1-mini`) và ghi lại tên triển khai
3. Đăng nhập bằng `az login` và cấu hình biến môi trường như hướng dẫn trên

### 🏗️ **Tổng Quan Kiến Trúc Quy Trình**

```mermaid
graph TD
    A[Trình tạo luồng công việc] --> B[Đăng ký đại lý]
    B --> C[Bộ máy thực thi luồng công việc]
    C --> D[Agent 1: Bộ tạo nội dung]
    C --> E[Agent 2: Bộ đánh giá nội dung] 
    D --> F[Kết quả luồng công việc]
    E --> F
    G[Azure OpenAI (API Phản hồi)] --> D
    G --> E
```

**Các Thành Phần Chính:**
- **WorkflowBuilder**: Bộ máy điều phối chính để thiết kế quy trình
- **AIAgent**: Các agent chuyên môn riêng biệt với năng lực cụ thể
- **Azure OpenAI Client**: Tích hợp Azure OpenAI Responses API
- **Execution Context**: Quản lý trạng thái và luồng dữ liệu giữa các giai đoạn quy trình

## 💻 Chạy Mã Nguồn

Triển khai đầy đủ có trong `01.dotnet-agent-framework-workflow-ghmodel-basic.cs`. Tệp này minh họa:

1. **Cấu Hình Môi Trường** - Tải cấu hình Azure OpenAI từ tệp `.env`
2. **Cài Đặt Client Azure OpenAI** - Cấu hình client sử dụng Azure OpenAI Responses API
3. **Tạo Agent** - Định nghĩa các agent chuyên môn (Front Desk và Concierge)
4. **Workflow Builder** - Tạo quy trình nhiều agent với xử lý theo thứ tự
5. **Chạy Quy Trình** - Thực thi quy trình với kết quả streaming

### 🚀 Chạy Ví Dụ

```bash
# Đặt script thành có thể thực thi (Unix/Linux/macOS)
chmod +x 01.dotnet-agent-framework-workflow-ghmodel-basic.cs

# Chạy quy trình công việc
./01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```

Hoặc trên Windows:
```powershell
dotnet run 01.dotnet-agent-framework-workflow-ghmodel-basic.cs
```
