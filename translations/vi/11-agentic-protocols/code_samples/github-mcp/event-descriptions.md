## Tên sự kiện: API đa tác nhân với LangGraph và Azure Cosmos DB

## Mô tả

Qua trình bày mã chi tiết, bạn sẽ thấy cách thiết kế và thực hiện quy trình làm việc do agent điều khiển tích hợp liền mạch các công nghệ retrieval-augmented generation (RAG), lưu trữ bộ nhớ và chuyển đổi trạng thái động. Chúng tôi sẽ đề cập tới:

Hợp tác agent với LangGraph cho lập luận có cấu trúc
Lưu trữ lịch sử chat thời gian thực sử dụng Azure Cosmos DB — cùng cơ sở dữ liệu hỗ trợ lịch sử chat trong ChatGPT, ứng dụng AI agent phát triển nhanh nhất lịch sử
Tìm kiếm vector để truy xuất kiến thức với hỗ trợ embeddings gốc của Cosmos DB
Khả năng async của FastAPI để giữ tương tác đáp ứng và mở rộng
Đến cuối buổi này, bạn sẽ có kế hoạch rõ ràng để xây dựng và triển khai ứng dụng đa tác nhân đám mây có thể mở rộng, tận dụng sức mạnh AI hiện đại và hạ tầng đám mây. Dù bạn là kỹ sư AI, kiến trúc sư đám mây hay lập trình viên Python, bài nói chuyện này sẽ trang bị cho bạn những hiểu biết thực tế và mẫu thiết kế đã được thử nghiệm để xây dựng thế hệ ứng dụng dựa trên AI kế tiếp.
