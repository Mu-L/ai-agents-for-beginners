# 🔍 Предприятна RAG с Microsoft Foundry (.NET)

## 📋 Учебни цели

Тази тетрадка демонстрира как да изградите системи за генерация с допълване чрез извличане (Retrieval-Augmented Generation, RAG) от предприятно ниво, използвайки Microsoft Agent Framework в .NET с Microsoft Foundry. Ще научите как да създавате агенти, готови за продукция, които могат да търсят в документи и да предоставят точни, контекстно-съзнателни отговори с ниво на сигурност и мащабируемост за предприятия.

**Предприятни RAG възможности, които ще изградите:**
- 📚 **Документална интелигентност**: Разширена обработка на документи с Azure AI услуги
- 🔍 **Семантично търсене**: Високопроизводително векторно търсене с функции за предприятия
- 🛡️ **Интеграция на сигурността**: Управление на достъпа по роли и модели за защита на данни
- 🏢 **Мащабируема архитектура**: Системи RAG, готови за продукция с мониторинг

## 🎯 Предприятна архитектура на RAG

### Основни компоненти за предприятия
- **Microsoft Foundry**: Управлявана платформа за предприятен AI със сигурност и съответствие
- **Постоянни агенти**: Състояни агенти с история на разговори и управление на контекст
- **Управление на векторни хранилища**: Индексиране и извличане на документи от предприятно ниво
- **Интеграция на идентичност**: Аутентификация Azure AD и контрол на достъпа по роли

### Предимства на .NET за предприятия
- **Типова сигурност**: Валидация по време на компилация за операции RAG и структури от данни
- **Асинхронна производителност**: Незаблокираща обработка на документи и търсене
- **Управление на паметта**: Ефективно използване на ресурси за големи колекции с документи
- **Модели за интеграция**: Родна интеграция на Azure услуги с инжектиране на зависимости

## 🏗️ Техническа архитектура

### Предприятен RAG работен процес
```
Document Upload → Security Validation → Vector Processing → Index Creation
                      ↓                    ↓                  ↓
User Query → Authentication → Semantic Search → Context Ranking → AI Response
```

### Основни .NET компоненти
- **Azure.AI.Agents.Persistent**: Управление на агенти с трайност на състоянието
- **Azure.Identity**: Интегрирана автентикация за защитен достъп до Azure услуги
- **Microsoft.Agents.AI.AzureAI**: Оптимизирана за Azure имплементация на агентската рамка
- **System.Linq.Async**: Високопроизводителни асинхронни LINQ операции

## ⚙️ Изисквания и настройка

**Разработваща среда:**
- .NET 9.0 SDK или по-нова версия
- Visual Studio 2022 или VS Code с разширение за C#
- Абонамент в Azure с достъп до Microsoft Foundry

**Необходими NuGet пакети:**
```xml
<PackageReference Include="Microsoft.Extensions.AI" Version="9.9.0" />
<PackageReference Include="Azure.AI.Agents.Persistent" Version="1.2.0-beta.5" />
<PackageReference Include="Azure.Identity" Version="1.15.0" />
<PackageReference Include="System.Linq.Async" Version="6.0.3" />
<PackageReference Include="DotNetEnv" Version="3.1.1" />
```

**Настройка на Azure автентикация:**
```bash
# Инсталирайте Azure CLI и се удостоверете
az login
az account set --subscription "your-subscription-id"
```

**Конфигурация на средата:**
* Конфигурация на Microsoft Foundry (автоматично управлявана чрез Azure CLI)
* Уверете се, че сте автентикирани в правилния Azure абонамент
