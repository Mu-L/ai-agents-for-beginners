# ⏩ Sekvenčné pracovné toky agentov s Azure OpenAI (Responses API) (.NET)

## 📋 Pokročilý návod na sekvenčné spracovanie

Tento poznámkový blok demonštruje **sekvenčné vzory pracovných tokov** pomocou Microsoft Agent Framework pre .NET a Azure OpenAI (Responses API). Naučíte sa, ako vytvoriť sofistikované, krok za krokom spracovateľské pipeline, kde agenti vykonávajú úlohy v konkrétnom poradí, pričom každá etapa nadväzuje na výsledky predchádzajúcej etapy.

### 🏗️ **Architektúra sekvenčného pracovného toku**


## 💻 Spustenie kódu

Kompletne implementácia je dostupná v súbore `02.dotnet-agent-framework-workflow-ghmodel-sequential.cs`. Tento súbor demonštruje **troj-etapový workflow analýzy nábytku**:

1. **Etapa 1 - Predajný agent**: Analyzuje obrázky nábytku a poskytuje nákupné odporúčania
2. **Etapa 2 - Cenový agent**: Poskytuje detailné cenové rozbory a možnosti rozpočtu
3. **Etapa 3 - Cenová ponuka agent**: Vytvára profesionálny dokument s ponukou vo formáte Markdown

### 🏗️ **Architektúra pracovného toku**

```
Image Input → Sales Analysis → Price Estimation → Quote Generation → Final Output
```

Každý agent:
- Dostáva výstup z predchádzajúcej etapy ako kontext
- Stavia na predchádzajúcej analýze pomocou špecializovaných znalostí
- Udržiava kontinuitu pracovného toku pomocou správy stavu

### 🚀 Spustenie príkladu

**Požiadavky:**
- Umiestnite obrázok nábytku do `../imgs/home.png` (alebo upravte premennú `imgPath`)
- Nakonfigurujte súbor `.env` s koncovým bodom a nasadením Azure OpenAI, potom sa prihláste pomocou `az login`

```bash
# Urobte skript spustiteľným (Unix/Linux/macOS)
chmod +x 02.dotnet-agent-framework-workflow-ghmodel-sequential.cs

# Spustite sekvenčný pracovný postup
./02.dotnet-agent-framework-workflow-ghmodel-sequential.cs
```

Alebo vo Windows:
```powershell
dotnet run 02.dotnet-agent-framework-workflow-ghmodel-sequential.cs
```

### 📝 Očakávaný výstup

Pracovný tok vykoná:
1. **Predajný agent**: Identifikuje predmety nábytku na obrázku a poskytne odporúčania
2. **Cenový agent**: Pridá detailnú analýzu cien s rozpočtovými kategóriami a nákupnými odporúčaniami
3. **Agent cenovej ponuky**: Vygeneruje formátovaný dokument s ponukou so všetkými zosúladenými informáciami

Konečný výstup bude komplexná, profesionálna cenová ponuka na nábytok založená na analýze obrázka.
