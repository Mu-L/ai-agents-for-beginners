## 💻 Spustenie kódu

Kompletná implementácia je dostupná v `02.dotnet-agent-framework-workflow-ghmodel-sequential.cs`. Tento súbor demonštruje **trojfázový workflow analýzy nábytku**:

1. **Fáza 1 - Predajný agent**: Analyzuje obrázky nábytku a poskytuje odporúčania na nákup
2. **Fáza 2 - Cenový agent**: Poskytuje podrobné rozbory cien a možnosti rozpočtu
3. **Fáza 3 - Agent pre ponuky**: Generuje profesionálny dokument ponuky vo formáte Markdown

### 🏗️ **Architektúra workflow**

```
Image Input → Sales Analysis → Price Estimation → Quote Generation → Final Output
```

Každý agent:
- Dostáva výstup z predchádzajúcej fázy ako kontext
- Stavá na predchádzajúcej analýze so špecializovanou expertízou
- Udržiava kontinuitu workflow prostredníctvom správy stavu

### 🚀 Spustenie príkladu

**Požiadavky:**
- Umiestnite obrázok nábytku do `../imgs/home.png` (alebo aktualizujte premennú `imgPath`)

### 📝 Očakávaný výstup

Workflow:
1. **Predajný agent**: Identifikuje položky nábytku z obrázka a poskytne odporúčania
2. **Cenový agent**: Pridá podrobnú analýzu cien s úrovňami rozpočtu a odporúčaniami na nákup
3. **Agent pre ponuky**: Vygeneruje formátovaný dokument ponuky so všetkými zhrnutými informáciami

Finálny výstup bude komplexná profesionálna ponuka nábytku založená na analýze obrázka.
