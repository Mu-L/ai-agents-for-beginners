# AGENTS.md

## திட்டம் கண்ணோட்டம்

இந்த சேமிப்பகம் "ஆரம்பத்திற்கான AI முகவர்கள்" - AI முகவர்கள் கட்டமைப்பதற்கான தேவையான அனைத்தையும் கற்பிக்கும் முழுமையான கல்வி பாடநெறியை கொண்டுள்ளது. இந்த பாடநெறி 18 பாடங்கள் (00-18 எண்கள் கொண்டவை) அடங்கியவை, அடிப்படைகள், வடிவமைப்பு முறைமைகள், கட்டமைப்புகள், உற்பத்தி நாளாடல், உள்ளூர்/சாதன முகவர்கள் மற்றும் AI முகவர்கள் பாதுகாப்பு ஆகியவற்றை உள்ளடக்குகிறது.

**முக்கிய தொழில்நுட்பங்கள்:**
- Python 3.12+
- இடையூறு கற்றலுக்கான Jupyter குறிப்பேடுகள்
- AI கட்டமைப்புகள்: Microsoft Agent Framework (MAF)
- Azure AI சேவைகள்: Microsoft Foundry, Microsoft Foundry Agent Service V2

**வடிவமைப்பு:**
- பாடம் அடிப்படையிலான கட்டமைப்பு (00-15+ அடைவு கோப்புகள்)
- ஒவ்வொரு பாடமும்: README ஆவணங்கள், குறியீடு எடுத்துக்காட்டுகள் (Jupyter குறிப்பேடுகள்), மற்றும் படங்கள் கொண்டுள்ளது
- தானாக மொழிபெயர்ப்பு முறையால் பன்மொழி ஆதரவு
- ஒவ்வொரு பாடத்திற்கும் ஒரு Python குறிப்பேடு Microsoft Agent Framework பயன்படுத்தி

## அமைக்க கட்டளைகள்

### முன்னதிகாரங்கள்
- Python 3.12 அல்லது அதற்கு மேல版本
- Azure சந்தா (Microsoft Foundry க்காக)
- Azure CLI நிறுவப்பட்டு அங்கீகாரம் பெற்றிருப்பது (`az login`)

### ஆரம்ப அமைப்பு

1. **சேமிப்பகத்தை கிளோன் அல்லது ஃபோர்க் செய்யவும்:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # அல்லது
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python மெய்நிகர் சூழலை உருவாக்கி செயல்படுத்தவும்:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # விண்டோஸில்: venv\Scripts\activate
   ```

3. **போன்ற அமைப்புகளை நிறுவவும்:**
   ```bash
   pip install -r requirements.txt
   ```

4. **சூழல் மாறிலிகள் அமைக்கவும்:**
   ```bash
   cp .env.example .env
   # உங்கள் API விசைகள் மற்றும் முடிவுக் குறிப்புகளுடன் .env ஐ தொகுக்கவும்
   ```

### தேவையான சூழல் மாறிலிகள்

**Microsoft Foundryக்காக** (தேவை):
- `AZURE_AI_PROJECT_ENDPOINT` - Microsoft Foundry திட்ட Endpoint
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - மாதிரி நாளாடல் பெயர் (எ.கா., gpt-4.1-mini)

**Azure AI Search**க்காக (பாடம் 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search Endpoint
- `AZURE_SEARCH_API_KEY` - Azure AI Search API திறவுகோல்

அங்கீகாரம்: குறியீடுகள் இயக்குவதற்கு முன் `az login` ஓட்டவும் (`AzureCliCredential` பயன்படுத்தப்படுகிறது).

   - Python 3.12+ பயன்படுத்துவதை உறுதி செய்யவும்
   - பழைய பதிப்புகளுடன் சில தொகுப்புகள் வேலை செய்யாது
   - Python பதிப்பைக் குறிப்பது `python3 -m venv` மூலம்

2. **சூழல் மாறிலிகள்:**
   - `.env.example` இலிருந்து எப்போதும் `.env` உருவாக்கவும்
   - `.env` கோப்பை கமிட் செய்ய வேண்டாம் (`.gitignore` இல் உள்ளது)
   - திறவுகோல் இல்லாத Entra ID அங்கீகரிப்பிற்கு `az login` செய்துகொள்ளவும்

3. **பெக்கேஜ் முரண்பாடுகள்:**
   - புதிய மெய்நிகர் சூழல் பயன்படுத்தவும்
   - தனித்தனி பெக்கேஜ்கள் பதிலாக `requirements.txt` இலிருந்து நிறுவவும்
   - சில குறிப்பேடுகள் மார்க்டவுன் செல் குறிப்புகளை பின்பற்றும் கூடுதல் தொகுப்புகளை வேண்டுபடும்

4. **Azure சேவைகள்:**
   - Azure AI சேவைகள் செயல்பட செயற்கை சந்தா தேவை
   - சில அம்சங்கள் பிராந்திய குறிப்பட்டவை
   - உங்கள் Azure OpenAI மாதிரி நாளாடல் Responses API சார்ந்ததா என உறுதி மூலியுங்கள்
