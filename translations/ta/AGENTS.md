## அமைப்பு கட்டளைகள்

### முன்னிதிகள்
- பைதான் 3.12 அல்லது அதற்கு மேல் பதிப்பு
- Azure சந்தா (Microsoft Foundryக்கு)
- Azure CLI நிறுவப்பட்டு அங்கீகாரம் பெற்றிருத்தல் (`az login`)

### ஆரம்ப அமைப்பு

1. **ரெப்பொசிடரியை கிளோன் செய்யவோ அல்லது ஃபோர்க் செய்யவோ செய்யவும்:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # அல்லது
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **பைதான் மெய்நிகர் சூழலை உருவாக்கி செயல்படுத்தவும்:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # விண்டோசில்: venv\Scripts\activate
   ```

3. **ஆதார கூறுகளை நிறுவவும்:**
   ```bash
   pip install -r requirements.txt
   ```

4. **சூழல் மாறிலிகளைக் கட்டமைக்கவும்:**
   ```bash
   cp .env.example .env
   # உங்கள் API விசைகள் மற்றும் எண்ட்பாயின்ட்களுடன் .env ஐ தொகு
   ```

### தேவையான சூழல் மாறிலிகள்

**Microsoft Foundry** (தேவை):
- `AZURE_AI_PROJECT_ENDPOINT` - Microsoft Foundry திட்டத்தின் முகவரி
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - மாதிரி விண்ணப்பத்தின் பெயர் (உதா., gpt-5-mini)

**Azure AI Search** (பாடம் 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search முகவரி
- `AZURE_SEARCH_API_KEY` - Azure AI Search API விசை

அங்கீகாரம்: நோட்புக்குகளை இயக்கும் முன் `az login` இயக்கவும் (`AzureCliCredential` பயன்படுத்தப்படுகிறது).
