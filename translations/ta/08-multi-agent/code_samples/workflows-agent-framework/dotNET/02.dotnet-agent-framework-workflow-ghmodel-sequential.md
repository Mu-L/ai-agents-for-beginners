# ⏩ Azure OpenAI (Responses API) கொண்டு தொடர் முகவர் வேலைப்பயிற்சிகள் (.NET)

## 📋 மேம்பட்ட தொடர் செயலாக்க பயிற்சி

இந்த நோட்புக் Microsoft Agent Framework for .NET மற்றும் Azure OpenAI (Responses API) பயன்படுத்தி **தொடர் வேலைப்பயிற்சி மாதிரிகள்** ஐப் பிரதிபலிக்கின்றது. முகவர்கள் ஒரு குறிப்பிட்ட வரிசையில் செயல்படுகின்ற, ஒவ்வொரு கட்டமும் முந்தைய கட்டத்தின் முடிவுகளை அடிப்படையாகக் கொண்டு கட்டமைக்கப்பட்டுள்ள, நுட்பமான படித்தடங்கள் செயலாக்க குழாய்களை எப்படி உருவாக்குவது என்பதை நீங்கள் கற்றுக்கொள்ளலாம்.

## 🎯 கற்றல் இலக்குகள்

### 🔄 **தொடர் செயலாக்க கட்டமைப்பு**
- **நேர்கோட்டு வேலைப்பயிற்சி வடிவமைப்பு**: தெளிவான சார்புகளுடன் படித்தட செயலாக்க குழாய்களை உருவாக்கவும்
- **நிலை மேலாண்மை**: தொடர்முறை வேலைப்பயிற்சி கட்டங்களுக்கு இடையே உள்ளடக்கமும் தரவு ஓட்டமும் காப்பாற்றவும்
- **Azure OpenAI (Responses API)**: பல கட்ட .NET வேலைப்பயிற்சிகளில் Azure OpenAI மாதிரிகளை பயன்படுத்தவும்
- **எண்டர்பிரைஸ் குழாய் மாதிரிகள்**: தயாரிப்பு தயாரான தொடர்முறை செயலாக்க அமைப்புகளை கட்டமைக்க

### 🏗️ **மேம்பட்ட தொடர்முறை மாதிரிகள்**
- **கட்டம்-கேட் செயலாக்கம்**: வேலைப்பயிற்சி கட்டங்களுக்கு இடையே சரிபார்ப்பு கட்டுப்பாடுகளை செயல்படுத்தவும்
- **சூழல் பாதுகாப்பு**: அனைத்து கட்டங்களிலும் நிலை மற்றும் சேர்க்கப்பட்ட அறிவை காப்பாற்றவும்
- **பிழை பரப்பல்**: தொடர்முறை செயலாக்க சங்கிலிகளில் தோல்விகளை நுணுக்கமாக கையாளவும்
- **செயல்திறன் மேம்பாடு**: குறைந்த செயல்பாட்டுடன் திறம்பட தொடர்முறை செயல்படுத்தல்

### 🏢 **எண்டர்பிரைஸ் தொடர்முறை பயன்பாடுகள்**
- **கட்டுரை செயலாக்க குழாய்**: பல கட்ட கட்டுரை பகுப்பாய்வு, மாற்றம் மற்றும் சரிபார்ப்பு
- **தர நிர்வாக வேலைப்பயிற்சிகள்**: தொடர்முறை விமர்சனம், சரிபார்ப்பு மற்றும் அனுமதி செயல்முறை
- **உள்ளடக்க உற்பத்தி குழாய்**: ஆய்வு → எழுத்து → திருத்தம் → விமர்சனம் → வெளியீடு
- **வணிக செயல்முறை தானாக்கூலம்**: தெளிவான கட்ட ஆரம்பவிழுப்புகளுடன் பல படி வணிக வேலைப்பயிற்சிகள்

### 🔑 **Azure OpenAI கட்டமைப்பு**

**சுற்றுச்சூழல் அமைப்பு (.env கோப்பு):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-5-mini
```

**கட்டமைப்பு மேலாண்மை:**
```csharp
// Load environment variables securely
Env.Load("../../../.env");
var azureEndpoint = Environment.GetEnvironmentVariable("AZURE_OPENAI_ENDPOINT");
var deployment = Environment.GetEnvironmentVariable("AZURE_OPENAI_DEPLOYMENT");
```

### 🎯 உண்மையிலான பயன்பாடுகள்

இந்த தொடர்முறை மாதிரி மிகவும் பொருத்தமானது:
- **மின் வர்த்தகம்**: தயாரிப்புப் பகுப்பாய்வு → விலை நிர்ணயம் → மேற்கோள் தயாரித்தல்
- **உண்மையான சொத்துக்கள்**: சொத்து பகுப்பாய்வு → மதிப்பீடு → பட்டியல் உருவாக்கம்
- **காப்பீடு**: மனு பகுப்பாய்வு → மதிப்பீடு → மேற்கோள் தயாரித்தல்
- **உள்ளடக்க உருவாக்கம்**: ஆய்வு → எழுதுதல் → திருத்தம் → வெளியீடு
