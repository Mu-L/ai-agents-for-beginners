## 📋 கற்றல் நோக்குகள்

இந்த நோட்புக் .NET இல் மைக்ரோசாஃப்ட் ஏஜென்ட் கடவுச்சொல் கட்டமைப்பைக் கொண்டு Azure OpenAI (Responses API) மூலம் வல்லமைபெற்ற என்டர்ப்ரைஸ் தரமான மல்டி-ஏஜென்ட் முறைமைகளை உருவாக்க எவ்வாறு என்பதை விளக்குகிறது. நீங்கள் கட்டமைக்கப்பட்ட வேலைநடை முறைகளின் மூலம் ஒருங்கிணைந்து செயல்படும் பல சிறப்பு ஏஜென்ட்களை ஒருங்கிணைப்பதை கற்கப்போகிறீர்கள், .NET இன் என்டர்ப்ரைஸ் அம்சங்களை உற்பத்தி தயாராகிய தீர்வுகளுக்கு பயன்படுத்தி.

**நீங்கள் உருவாக்கப்போகும் என்டர்ப்ரைஸ் மல்டி-ஏஜென்ட் திறன்கள்:**
- 👥 **ஏஜென்ட் ஒத்துழைப்பு**: தொகுத்தல் நேர அங்கீகாரத்துடன் வகை-பாதுகாப்பான ஏஜென்ட் கூட்டாண்மை
- 🔄 **வேலைநடை ஒருங்கிணைப்பு**: .NET இன் அசிங்க் மாதிரிகளைப் பயன்படுத்தி அறிவுறுத்தப்படும் வேலைநடை வரையறை
- 🎭 **பங்கு சிறப்பான தன்மை**: வலுவான வகை-மையமான ஏஜென்ட் தனிப்பட்ட பண்புகள் மற்றும் திறன் துறைகள்
- 🏢 **என்டர்ப்ரைஸ் ஒருங்கிணைப்பு**: கண்காணிப்பு மற்றும் பிழை கையாளுதலுடன் உற்பத்தி தயாராகிய மாதிரிகள்

## இந்த உதாரணம் காட்டுவது என்ன

இந்த மல்டி-ஏஜென்ட் வேலைநடை முறைமை இரண்டு சிறப்பு ஏஜென்ட்களுடன் ஒரு ஹோட்டல் பயண பரிந்துரை சேவையை உருவாக்குகிறது:

1. **FrontDesk ஏஜென்ட்**: செயல்பாடுகள் மற்றும் இட பரிந்துரைகள் வழங்கும் பயண ஏஜென்ட்
2. **Concierge ஏஜென்ட்**: பரிந்துரைகளை மதிப்பாய்வு செய்து அசல் மற்றும் சுற்றுலா கட்டாயம் அல்லாத அனுபவங்களை உறுதி செய்கிறது

ஏஜென்ட்கள் ஒருங்கிணைந்து இயங்கும் வேலைநடை:
- FrontDesk ஏஜென்ட் ஆரம்ப பயண கோரிக்கையை பெறுகிறது
- Concierge ஏஜென்ட் பரிந்துரைகளை மதிப்பாய்வு செய்து நுட்பமாக்குகிறது
- வேலைநடை நேரடி பிரதிகளைக் கடத்துகிறது

## முக்கிய கருத்துகள்

### ஏஜென்ட் கூட்டமைப்பு
இந்த உதாரணம் Microsoft Agent Framework கொண்டு வகை-பாதுகாப்பான ஏஜென்ட் கூட்டமைப்பை தொகுத்தல் நேர அங்கீகாரத்துடன் விளக்குகிறது.

### வேலைநடை ஒருங்கிணைப்பு
.NET இன் அசிங்க் மாதிரிகளைப் பயன்படுத்தி அறிவுறுத்தப்படும் வேலைநடை வரையறையை பயன்படுத்தி பல ஏஜென்ட்களைக் குழாயாக்கி இணைக்கிறது.

### நேரடி பிரதிகள்
அசிங்க் எண்ணாக்கள் மற்றும் நிகழ்வு ஆதரவு கட்டமைப்பைப் பயன்படுத்தி ஏஜென்ட் பிரதிகளை நேரடியாக காண்பிக்கும்.

### என்டர்ப்ரைஸ் ஒருங்கிணைப்பு
உற்பத்தி தயாரான மாதிரிகள் காட்டுகிறது, அதில்:
- சுற்றுப்புற மாறி கட்டமைப்பு
- பாதுகாப்பான கடவுச்சொல் முகாமைத்துவம்
- பிழை கையாளுதல்
- அசிங்க் நிகழ்வு செயலாக்கம்
