# အတတ်ပညာ စစ်ဆေးမှု ဖြေကြားချက်များ

## မေးခွန်း ၁

**ဖြေကြားချက်။ A2.** တာဝန်တစ်ခုတွင် ကွဲပြားသော ကျွမ်းကျင်သူအခန်းကဏ္ဍများ၊ သီးခြားကိရိယာများ သို့မဟုတ် ခွင့်ပြုချက်များ၊ နှင့် ပေါင်းစည်းမှု လိုအပ်သည့် သေချာသော လက်တင်များရှိသောအခါ အမှုဆောင်အမြောက်အများ သုံးရန် အသုံးဝင်သည်။ ပုံမှန်ပံ့ပိုးမှုဘော့တစ်ခုမှာ ပုံမှန်အားဖြင့် တစ်ဦးတည်းသော အမှုဆောင်တစ်ဦးနေထိုင်နိုင်သည်။ ညီမျှသော တောင်းဆိုမှုများ အပြားအပြား ဖြစ်ခြင်းမှာ အဓိကအားဖြင့် ပမာဏတိုးရန် စိုးရိမ်မှု ဖြစ်ပြီး မူတော်တစ်ခုအတိုင်း အမှုဆောင်အစုလိုက် ဒီဇိုင်းလိုအပ်မှု မရှိပါ။

## မေးခွန်း ၂

**ဖြေကြားချက်။ A1.** တစ်ခုတည်းသော အမှုဆောင်တစ်ဦးသည် တစ်‌ပုံတစ်ပေါ် အမိန့်များနှင့် ကိရိယာများပါဝင်သည့် အလုပ်ကိုင် အခန်းကဏ္ဍကို လုပ်ဆောင်နိုင်ပါက စမ်းသပ်ရန်၊ ကြည့်ရှုဖို့နှင့် အသုံးပြုရန် ပို၍ လွယ်ကူသည်။ အလယ်တန်းကိရိယာအများအတွက် အမှုဆောင်အများ လိုအပ်ခြင်း မဖြစ်ပါ။ ခွင့်ပြုချက်များကွဲပြားခြားနားပြီး စစ်ဆေးမှု လုပ်ငန်းလိုအပ်ချက်များရှိသော သီးခြား အခန်းကဏ္ဍများသည် အမှုဆောင်အများ ပါဝင်သော ဒီဇိုင်းတစ်ခုကို အတည်ပြုနိုင်ပါသည်။
