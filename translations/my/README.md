## 🌱 စတင်ရန်

ဒီသင်တန်းမှာ AI အေးဂျင့်များကို တည်ဆောက်ခြင်းအခြေခံကို ဖော်ပြထားတဲ့ သင်ခန်းစာများပါဝင်ပါတယ်။ သင်ခန်းစာတိုင်းမှာ ကိုယ်ပိုင်အကြောင်းအရာကို ဖော်ပြထားပြီး သင်စတင်လိုတဲ့နေရာကနေ စတင်နိုင်ပါတယ်။

ဒီသင်တန်းမှာ ဘာသာစကားများစွာကို ပံ့ပိုးထားပါတယ်။ [ဒီမှာ](../..) သွားပြီး ရနိုင်တဲ့ ဘာသာစကားများကို ကြည့်ပါ။

Generative AI မော်ဒယ်များကို ပထမဆုံးအကြိမ်တည်ဆောက်မယ်ဆိုရင် [Generative AI For Beginners](https://aka.ms/genai-beginners) သင်တန်းကို ကြည့်ပါ။ ဒီသင်တန်းမှာ GenAI နဲ့ တည်ဆောက်ခြင်းအကြောင်း 21 ခုသော သင်ခန်းစာများပါဝင်ပါတယ်။

[ဒီ repo ကို 🌟](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) နှင့် [ဒီ repo ကို fork](https://github.com/microsoft/ai-agents-for-beginners/fork) လုပ်ပြီး code ကို run လုပ်ဖို့ မမေ့ပါနဲ့။

### အခြားသော သင်ယူသူများနှင့် တွေ့ဆုံပြီး မေးခွန်းများကို ဖြေရှင်းပါ

AI အေးဂျင့်များကို တည်ဆောက်ရာမှာ အခက်အခဲရှိရင် သို့မဟုတ် မေးခွန်းများရှိရင် [Azure AI Foundry Community Discord](https://aka.ms/ai-agents/discord) မှာ အထူး Discord Channel ကို ဝင်ရောက်ပါ။

### လိုအပ်သော အရာများ

ဒီသင်တန်းရဲ့ သင်ခန်းစာတိုင်းမှာ code နမူနာများပါဝင်ပြီး code_samples folder မှာ ရှာနိုင်ပါတယ်။ [ဒီ repo ကို fork](https://github.com/microsoft/ai-agents-for-beginners/fork) လုပ်ပြီး ကိုယ်ပိုင် copy တစ်ခု ဖန်တီးနိုင်ပါတယ်။

ဒီ exercises တွေမှာပါတဲ့ code နမူနာတွေဟာ Language Models တွေနဲ့ အလုပ်လုပ်ဖို့ Azure AI Foundry နဲ့ GitHub Model Catalogs ကို အသုံးပြုထားပါတယ်-

- [Github Models](https://aka.ms/ai-agents-beginners/github-models) - အခမဲ့ / အကန့်အသတ်ရှိ
- [Azure AI Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure အကောင့်လိုအပ်

ဒီသင်တန်းမှာ Microsoft ရဲ့ AI အေးဂျင့် frameworks နဲ့ services တွေကိုလည်း အသုံးပြုထားပါတယ်-

- [Microsoft Agent Framework (MAF) - New!](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Agent Service](https://aka.ms/ai-agents-beginners/ai-agent-service)
- [Semantic Kernel](https://aka.ms/ai-agents-beginners/semantic-kernel)
- [AutoGen](https://aka.ms/ai-agents/autogen)

ဒီသင်တန်းရဲ့ code ကို run လုပ်ဖို့ အချက်အလက်များကို [Course Setup](./00-course-setup/README.md) မှာ သွားကြည့်ပါ။
