## 💻 コードの実行

完全な実装は `02.dotnet-agent-framework-workflow-ghmodel-sequential.cs` にあります。このファイルは **3段階の家具分析ワークフロー** を示しています：

1. **ステージ 1 - セールスエージェント**：家具画像を分析し購入提案を提供
2. **ステージ 2 - 価格エージェント**：詳細な価格内訳と予算オプションを提示
3. **ステージ 3 - 見積もりエージェント**：Markdown 形式のプロフェッショナルな見積もり文書を作成

### 🏗️ <strong>ワークフローのアーキテクチャ</strong>

```
Image Input → Sales Analysis → Price Estimation → Quote Generation → Final Output
```

各エージェントは：
- 前段階の出力をコンテキストとして受け取る
- 専門知識を活かして前の分析に基づき強化
- 状態管理でワークフローの継続性を維持

### 🔍 状態フローの理解

シーケンス中の各エージェントは以下を受け取ります：
- <strong>元の入力</strong>：初期のユーザーメッセージ（画像＋テキスト）
- <strong>前のエージェント出力</strong>：会話履歴内のすべての前段階エージェント応答
- <strong>蓄積されたコンテキスト</strong>：ワークフロー全体で維持される完全な状態

これにより、各エージェントが前段階の全てのコンテキストを基に構築する高度な多段階処理を可能にします。
