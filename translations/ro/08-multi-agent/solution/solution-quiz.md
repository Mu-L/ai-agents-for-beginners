# Răspunsuri pentru Testul de Cunoștințe

## Întrebarea 1

**Răspuns: A2.** Mai mulți agenți sunt utili atunci când o sarcină are roluri specializate distincte, unelte sau permisiuni separate și predări clare care necesită coordonare. Un bot de suport simplu poate rămâne de obicei un agent unic. Un volum mare de cereri identice este în primul rând o problemă de scalare și nu necesită prin sine un design multi-agent.

## Întrebarea 2

**Răspuns: A1.** Un singur agent este de obicei mai ușor de testat, observat și operat când un set de instrucțiuni și unelte pot gestiona sarcina. Accesul la mai multe unelte nu necesită automat mai mulți agenți. Rolurile separate cu permisiuni diferite și cerințe de audit pot justifica un design multi-agent.
