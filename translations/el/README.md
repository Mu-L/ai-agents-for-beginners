## 🌱 Ξεκινώντας

Αυτό το μάθημα περιλαμβάνει μαθήματα που καλύπτουν τα βασικά για τη δημιουργία Πρακτόρων AI. Κάθε μάθημα καλύπτει ένα ξεχωριστό θέμα, οπότε μπορείτε να ξεκινήσετε από όπου θέλετε!

Υπάρχει υποστήριξη πολλών γλωσσών για αυτό το μάθημα. Δείτε τις [διαθέσιμες γλώσσες εδώ](../..).

Αν είναι η πρώτη σας φορά που δουλεύετε με μοντέλα Γενετικής AI, δείτε το μάθημα μας [Γενετική AI για Αρχάριους](https://aka.ms/genai-beginners), το οποίο περιλαμβάνει 21 μαθήματα για τη δημιουργία με GenAI.

Μην ξεχάσετε να [προσθέσετε αστέρι (🌟) σε αυτό το αποθετήριο](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) και να [κλωνοποιήσετε αυτό το αποθετήριο](https://github.com/microsoft/ai-agents-for-beginners/fork) για να εκτελέσετε τον κώδικα.

### Γνωρίστε Άλλους Μαθητές, Απαντήστε στις Ερωτήσεις σας

Αν κολλήσετε ή έχετε ερωτήσεις σχετικά με τη δημιουργία Πρακτόρων AI, μπείτε στο ειδικό κανάλι Discord στην [Κοινότητα Azure AI Foundry Discord](https://aka.ms/ai-agents/discord).

### Τι Χρειάζεστε

Κάθε μάθημα σε αυτό το μάθημα περιλαμβάνει παραδείγματα κώδικα, τα οποία μπορείτε να βρείτε στον φάκελο code_samples. Μπορείτε να [κλωνοποιήσετε αυτό το αποθετήριο](https://github.com/microsoft/ai-agents-for-beginners/fork) για να δημιουργήσετε το δικό σας αντίγραφο.

Τα παραδείγματα κώδικα σε αυτές τις ασκήσεις χρησιμοποιούν το Azure AI Foundry και τους Καταλόγους Μοντέλων GitHub για την αλληλεπίδραση με Μοντέλα Γλώσσας:

- [Μοντέλα GitHub](https://aka.ms/ai-agents-beginners/github-models) - Δωρεάν / Περιορισμένα  
- [Azure AI Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Απαιτείται Λογαριασμός Azure  

Αυτό το μάθημα χρησιμοποιεί επίσης τα παρακάτω πλαίσια και υπηρεσίες Πρακτόρων AI από τη Microsoft:

- [Microsoft Agent Framework (MAF) - Νέο!](https://aka.ms/ai-agents-beginners/agent-framewrok)  
- [Υπηρεσία Πρακτόρων AI Azure](https://aka.ms/ai-agents-beginners/ai-agent-service)  
- [Semantic Kernel](https://aka.ms/ai-agents-beginners/semantic-kernel)  
- [AutoGen](https://aka.ms/ai-agents/autogen)  

Για περισσότερες πληροφορίες σχετικά με την εκτέλεση του κώδικα για αυτό το μάθημα, δείτε το [Course Setup](./00-course-setup/README.md).
