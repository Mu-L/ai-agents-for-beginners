# AGENTS.md

## Επισκόπηση Έργου

Αυτό το αποθετήριο περιέχει "Πράκτορες Τεχνητής Νοημοσύνης για Αρχάριους" - ένα ολοκληρωμένο εκπαιδευτικό μάθημα που διδάσκει όλα όσα χρειάζονται για να δημιουργήσετε Πράκτορες Τεχνητής Νοημοσύνης. Το μάθημα αποτελείται από 18 μαθήματα (αριθμημένα 00-18) που καλύπτουν βασικές έννοιες, πρότυπα σχεδίασης, πλαίσια εργασίας, ανάπτυξη παραγωγής, τοπικούς/ενσωματωμένους πράκτορες, και ασφάλεια των πράκτορων AI.

**Κύριες Τεχνολογίες:**
- Python 3.12+
- Jupyter Notebooks για διαδραστική μάθηση
- Πλαίσια AI: Microsoft Agent Framework (MAF)
- Azure AI Services: Microsoft Foundry, Microsoft Foundry Agent Service V2

**Αρχιτεκτονική:**
- Δομή με βάση τα μαθήματα (φάκελοι 00-15+)
- Κάθε μάθημα περιέχει: τεκμηρίωση README, δείγματα κώδικα (Jupyter notebooks), και εικόνες
- Υποστήριξη πολλών γλωσσών μέσω αυτόματου συστήματος μετάφρασης
- Ένα Python notebook ανά μάθημα που χρησιμοποιεί το Microsoft Agent Framework

## Εντολές Ρύθμισης

### Προαπαιτούμενα
- Python 3.12 ή νεότερο
- Συνδρομή Azure (για Microsoft Foundry)
- Εγκατεστημένο και πιστοποιημένο Azure CLI (`az login`)

### Αρχική Ρύθμιση

1. **Κλωνοποιήστε ή κάντε fork το αποθετήριο:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # Ή
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Δημιουργήστε και ενεργοποιήστε ένα εικονικό περιβάλλον Python:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # Σε Windows: venv\Scripts\activate
   ```

3. **Εγκαταστήστε τις εξαρτήσεις:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Ορίστε τις μεταβλητές περιβάλλοντος:**
   ```bash
   cp .env.example .env
   # Επεξεργαστείτε το .env με τα κλειδιά API και τα endpoints σας
   ```

### Απαιτούμενες Μεταβλητές Περιβάλλοντος

Για **Microsoft Foundry** (Απαραίτητο):
- `AZURE_AI_PROJECT_ENDPOINT` - το endpoint του έργου Microsoft Foundry
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - Όνομα ανάπτυξης μοντέλου (π.χ., gpt-4.1-mini)

Για **Azure AI Search** (Μάθημα 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - το endpoint του Azure AI Search
- `AZURE_SEARCH_API_KEY` - κλειδί API του Azure AI Search

Πιστοποίηση: Εκτελέστε `az login` πριν τρέξετε τα notebooks (χρησιμοποιεί `AzureCliCredential`).

## Ροή Ανάπτυξης

### Εκτέλεση Jupyter Notebooks

Κάθε μάθημα περιέχει πολλαπλά Jupyter notebooks για διάφορα πλαίσια εργασίας:

1. **Εκκινήστε το Jupyter:**
   ```bash
   jupyter notebook
   ```

2. **Μεταβείτε σε φάκελο μαθήματος** (π.χ., `01-intro-to-ai-agents/code_samples/`)

3. **Ανοίξτε και εκτελέστε τα notebooks:**
   - `*-python-agent-framework.ipynb` - Χρήση Microsoft Agent Framework (Python)
   - `*-dotnet-agent-framework.ipynb` - Χρήση Microsoft Agent Framework (.NET)

### Εργασία με Microsoft Agent Framework

**Microsoft Agent Framework + Microsoft Foundry:**
- Απαιτεί συνδρομή Azure
- Χρησιμοποιεί `FoundryChatClient` για τον Πράκτορα Υπηρεσίας V2 (πράκτορες ορατοί στην πύλη Foundry)
- Έτοιμο για παραγωγή με ενσωματωμένη παρακολούθηση
- Μοτίβο αρχείου: `*-python-agent-framework.ipynb`

### Smoke-Testing Αναπτυγμένων Πρακτόρων

Για μαθήματα όπου ένας πράκτορας αναπτύσσεται ως πράκτορας φιλοξενούμενος Microsoft Foundry (01, 04, 05, 16), το αποθετήριο παρέχει καταλόγους smoke-test στον φάκελο `tests/` που εκτελούνται από το workflow `.github/workflows/smoke-test.yml` μέσω της ενέργειας [AI Smoke Test](https://github.com/marketplace/actions/ai-smoke-test). Αυτά αποτελούν έναν ελαφρύ έλεγχο μετά την ανάπτυξη (είναι ο πράκτορας προσβάσιμος και ακολουθεί βασικές προσδοκίες προτροπής;), συμπληρώνοντας την αξιολόγηση των μαθημάτων 10 και 16. Δείτε το [tests/README.md](./tests/README.md) για τη χαρτογράφηση καταλόγων-μαθημάτων-πρακτόρων. Το μάθημα 17 τρέχει τοπικά με Foundry Local και δεν έχει φιλοξενούμενο endpoint, οπότε επικυρώνεται εκτελώντας το notebook του απευθείας.
