# 🔍 Εταιρικό RAG με το Microsoft Foundry (.NET)

## 📋 Στόχοι Εκμάθησης

Αυτό το σημειωματάριο παρουσιάζει πώς να κατασκευάσετε συστήματα Retrieval-Augmented Generation (RAG) επιχειρηματικού επιπέδου χρησιμοποιώντας το Microsoft Agent Framework σε .NET με το Microsoft Foundry. Θα μάθετε να δημιουργείτε παραγωγικούς πράκτορες που μπορούν να αναζητούν σε έγγραφα και να παρέχουν ακριβείς, συσχετισμένες με το πλαίσιο απαντήσεις με ασφάλεια και επεκτασιμότητα επιπέδου επιχείρησης.

**Ικανότητες Enterprise RAG που θα κατασκευάσετε:**
- 📚 **Νοημοσύνη Εγγράφων**: Προχωρημένη επεξεργασία εγγράφων με υπηρεσίες Azure AI
- 🔍 **Σημασιολογική Αναζήτηση**: Υψηλές επιδόσεις αναζήτησης με διανυσματικό τρόπο και χαρακτηριστικά επιχείρησης
- 🛡️ **Ενσωμάτωση Ασφαλείας**: Πρότυπα πρόσβασης βάσει ρόλων και προστασίας δεδομένων
- 🏢 **Επεκτάσιμη Αρχιτεκτονική**: Παραγωγικά έτοιμα συστήματα RAG με παρακολούθηση

## 🎯 Αρχιτεκτονική Enterprise RAG

### Κύρια Συστατικά Enterprise
- **Microsoft Foundry**: Διαχειριζόμενη επιχειρηματική πλατφόρμα AI με ασφάλεια και συμμόρφωση
- **Επίμονοι Πράκτορες**: Κατάσταση πράκτορες με αποθήκευση ιστορικού συνομιλίας και διαχείριση πλαισίου
- **Διαχείριση Αποθετηρίου Διανυσμάτων**: Επιχειρηματική ευρετηρίαση και ανάκτηση εγγράφων
- **Ενσωμάτωση Ταυτότητας**: Πιστοποίηση Azure AD και έλεγχος πρόσβασης βάσει ρόλων

### Οφέλη .NET για Επιχειρήσεις
- **Ασφάλεια Τύπων**: Επικύρωση κατά τη μεταγλώττιση για λειτουργίες RAG και δομές δεδομένων
- **Ασύγχρονη Απόδοση**: Μη εμπόδιση λειτουργιών επεξεργασίας εγγράφων και αναζήτησης
- **Διαχείριση Μνήμης**: Αποδοτική χρήση πόρων για μεγάλες συλλογές εγγράφων
- **Πρότυπα Ενσωμάτωσης**: Φυσική ενσωμάτωση υπηρεσιών Azure με dependency injection

## 🏗️ Τεχνική Αρχιτεκτονική

### Εταιρικό Pipeline RAG
```
Document Upload → Security Validation → Vector Processing → Index Creation
                      ↓                    ↓                  ↓
User Query → Authentication → Semantic Search → Context Ranking → AI Response
```

### Κύρια Συστατικά .NET
- **Azure.AI.Agents.Persistent**: Διαχείριση επιχειρηματικών πρακτόρων με επιμονή κατάστασης
- **Azure.Identity**: Ενσωματωμένη πιστοποίηση για ασφαλή πρόσβαση σε υπηρεσίες Azure
- **Microsoft.Agents.AI.AzureAI**: Υλοποίηση πλαισίου πρακτόρων βελτιστοποιημένη για Azure
- **System.Linq.Async**: Υψηλής απόδοσης ασύγχρονες λειτουργίες LINQ

**Περιβάλλον Ανάπτυξης:**
- .NET 9.0 SDK ή νεότερο
- Visual Studio 2022 ή VS Code με επέκταση C#
- Συνδρομή Azure με πρόσβαση στο Microsoft Foundry

```xml

<PackageReference Include="DotNetEnv" Version="3.1.1" />
```

**Ρύθμιση Πιστοποίησης Azure:**
```bash

az account set --subscription "your-subscription-id"
```

**Διαμόρφωση Περιβάλλοντος:**
