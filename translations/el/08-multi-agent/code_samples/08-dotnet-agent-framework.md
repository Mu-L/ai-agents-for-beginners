# 🤝 Εταιρικά Συστήματα Πολυ-Πρακτόρων Ροής Εργασίας (.NET)

## 📋 Στόχοι Μάθησης

Αυτό το σημειωματάριο δείχνει πώς να δημιουργήσετε προηγμένα εταιρικού επιπέδου συστήματα πολλαπλών πρακτόρων χρησιμοποιώντας το Microsoft Agent Framework στο .NET με το Azure OpenAI (Responses API). Θα μάθετε πώς να συντονίζετε πολλούς εξειδικευμένους πράκτορες που συνεργάζονται μέσω δομημένων ροών εργασίας, αξιοποιώντας τα εταιρικά χαρακτηριστικά του .NET για λύσεις έτοιμες για παραγωγή.

**Οι Ικανότητες Πολυ-Πρακτόρων Εταιρικού Επιπέδου που Θα Δημιουργήσετε:**
- 👥 **Συνεργασία Πρακτόρων**: Συντονισμός πρακτόρων με type-safe και επαλήθευση σε χρόνο μεταγλώττισης
- 🔄 **Ορχήστρωση Ροών Εργασίας**: Δηλωτικός ορισμός ροών εργασίας με async patterns του .NET
- 🎭 **Εξειδίκευση Ρόλων**: Ισχυρά τυποποιημένες προσωπικότητες πρακτόρων και τομείς εξειδίκευσης
- 🏢 **Εταιρική Ολοκλήρωση**: Πρότυπα έτοιμα για παραγωγή με παρακολούθηση και διαχείριση σφαλμάτων

## Τι Δείχνει Αυτό το Παράδειγμα

Αυτό το σύστημα ροής εργασίας πολλαπλών πρακτόρων δημιουργεί μια υπηρεσία ταξιδιωτικών προτάσεων ξενοδοχείων με δύο εξειδικευμένους πράκτορες:

1. **Πράκτορας FrontDesk**: Ένας ταξιδιωτικός πράκτορας που παρέχει προτάσεις για δραστηριότητες και τοποθεσίες
2. **Πράκτορας Concierge**: Επανεξετάζει τις προτάσεις για να διασφαλίσει αυθεντικές, μη τουριστικές εμπειρίες

Οι πράκτορες συνεργάζονται σε μια ροή εργασίας όπου:
- Ο πράκτορας FrontDesk λαμβάνει το αρχικό αίτημα ταξιδιού
- Ο πράκτορας Concierge επανεξετάζει και βελτιώνει την πρόταση
- Η ροή εργασίας μεταδίδει απαντήσεις σε πραγματικό χρόνο

## Βασικές Έννοιες

### Συντονισμός Πρακτόρων
Το παράδειγμα δείχνει type-safe συντονισμό πρακτόρων χρησιμοποιώντας το Microsoft Agent Framework με επαλήθευση σε χρόνο μεταγλώττισης.

### Ορχήστρωση Ροής Εργασίας
Χρησιμοποιεί δηλωτικό ορισμό ροής εργασίας με async patterns του .NET για να συνδέσει πολλούς πράκτορες σε μια αλυσίδα.

### Μετάδοση Απαντήσεων σε Ροή
Υλοποιεί μετάδοση απαντήσεων πρακτόρων σε πραγματικό χρόνο χρησιμοποιώντας async enumerables και αρχιτεκτονική βασισμένη σε γεγονότα.

### Εταιρική Ολοκλήρωση
Δείχνει πρότυπα έτοιμα για παραγωγή, που περιλαμβάνουν:
- Ρύθμιση μεταβλητών περιβάλλοντος
- Ασφαλή διαχείριση διαπιστευτηρίων
- Διαχείριση σφαλμάτων
- Ασύγχρονη επεξεργασία γεγονότων
