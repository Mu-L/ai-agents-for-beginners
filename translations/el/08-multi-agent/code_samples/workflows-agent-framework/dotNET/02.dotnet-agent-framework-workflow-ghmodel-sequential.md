# ⏩ Αλληλουχίες Εργασιών Πρακτόρων με Azure OpenAI (Responses API) (.NET)

## 📋 Προχωρημένο Σεμινάριο Αλληλουχίας Επεξεργασίας

Αυτό το σημειωματάριο παρουσιάζει **πρότυπα αλληλουχίας ροής εργασιών** χρησιμοποιώντας το Microsoft Agent Framework για .NET και Azure OpenAI (Responses API). Θα μάθετε πώς να δημιουργείτε πολύπλοκες, βήμα προς βήμα γραμμές επεξεργασίας όπου οι πράκτορες εκτελούνται με συγκεκριμένη σειρά, με κάθε στάδιο να βασίζεται στα αποτελέσματα του προηγούμενου.

## 🎯 Στόχοι Μάθησης

### 🔄 **Αρχιτεκτονική Αλληλουχίας Επεξεργασίας**
- **Σχεδιασμός Γραμμικής Ροής Εργασίας**: Δημιουργία γραμμών επεξεργασίας βήμα προς βήμα με σαφείς εξαρτήσεις
- **Διαχείριση Κατάστασης**: Διατήρηση του πλαισίου και της ροής δεδομένων σε διαδοχικά στάδια ροής εργασιών
- **Azure OpenAI (Responses API)**: Αξιοποίηση μοντέλων Azure OpenAI σε πολυεπίπεδες ροές .NET
- **Πρότυπα Επιχειρησιακής Γραμμής Επεξεργασίας**: Δημιουργία συστημάτων αλληλουχίας επεξεργασίας έτοιμων για παραγωγή

### 🏗️ **Προχωρημένα Πρότυπα Αλληλουχίας**
- **Επεξεργασία με Στάδια-Πύλες**: Υλοποίηση σημείων ελέγχου επικύρωσης μεταξύ σταδίων ροής εργασίας
- **Διατήρηση Πλαισίου**: Διατήρηση κατάστασης και συσσωρευμένης γνώσης σε όλα τα στάδια
- **Διαχείριση Σφαλμάτων**: Αντιμετώπιση αποτυχιών με εύγλωττο τρόπο σε αλληλουχίες επεξεργασίας
- **Βελτιστοποίηση Απόδοσης**: Αποδοτική εκτέλεση αλληλουχίας με ελάχιστο overhead

### 🏢 **Επιχειρησιακές Εφαρμογές Αλληλουχίας**
- **Γραμμή Επεξεργασίας Εγγράφων**: Πολυεπίπεδη ανάλυση, μετασχηματισμός και επικύρωση εγγράφων
- **Ροές Εργασίας Ποιοτικού Ελέγχου**: Διαδοχική ανασκόπηση, επικύρωση και έγκριση διαδικασιών
- **Γραμμή Παραγωγής Περιεχομένου**: Έρευνα → Γραφή → Επεξεργασία → Ανασκόπηση → Δημοσίευση
- **Αυτοματοποίηση Επιχειρησιακών Διαδικασιών**: Πολυσταδιακές επιχειρησιακές ροές με σαφείς εξαρτήσεις σταδίων

## ⚙️ Απαιτήσεις & Ρύθμιση

### 📦 **Απαιτούμενα Πακέτα NuGet**

Βασικά πακέτα για αλληλουχίες ροών εργασίας .NET:

```xml
<!-- Core AI Framework -->
<PackageReference Include="Microsoft.Extensions.AI" Version="10.*" />

<!-- Azure OpenAI (Responses API) -->
<PackageReference Include="Azure.AI.OpenAI" Version="2.*" />

<!-- Azure Identity and Async LINQ Support -->
<PackageReference Include="Azure.Identity" Version="1.15.0" />
<PackageReference Include="System.Linq.Async" Version="6.0.3" />

<!-- Local Agent Framework References -->
<!-- Microsoft.Agents.AI.dll - Core agent abstractions -->
<!-- Microsoft.Agents.AI.OpenAI.dll - Azure OpenAI (Responses API) integration -->
```

### 🔑 **Διαμόρφωση Azure OpenAI**

**Ρύθμιση Περιβάλλοντος (.env αρχείο):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-5-mini
```

**Διαχείριση Διαμόρφωσης:**
```csharp
// Load environment variables securely
Env.Load("../../../.env");
var azureEndpoint = Environment.GetEnvironmentVariable("AZURE_OPENAI_ENDPOINT");
var deployment = Environment.GetEnvironmentVariable("AZURE_OPENAI_DEPLOYMENT");
```

- **Πλαίσιο Κατάστασης**: Διατηρεί συσσωρευμένα δεδομένα και αποφάσεις ανά στάδιο
- **Σημεία Ελέγχου**: Σημεία επικύρωσης μεταξύ σταδίων για διασφάλιση ποιότητας και συνέπειας
- **Πελάτης Azure OpenAI**: Σταθερή πρόσβαση σε μοντέλα τεχνητής νοημοσύνης σε όλα τα στάδια ροής εργασίας

## 💻 Εκτέλεση Κώδικα

Η πλήρης υλοποίηση είναι διαθέσιμη στο `02.dotnet-agent-framework-workflow-ghmodel-sequential.cs`. Αυτό το αρχείο παρουσιάζει μια **τρικρηνική ροή ανάλυσης επίπλων**:

1. **Στάδιο 1 - Πράκτορας Πωλήσεων**: Αναλύει εικόνες επίπλων και παρέχει προτάσεις αγοράς
2. **Στάδιο 2 - Πράκτορας Τιμών**: Παρέχει λεπτομερείς αναλύσεις τιμολόγησης και επιλογές προϋπολογισμού
3. **Στάδιο 3 - Πράκτορας Προσφορών**: Δημιουργεί επαγγελματικό έγγραφο προσφοράς σε μορφή Markdown

### 🏗️ **Αρχιτεκτονική Ροής Εργασίας**

```
Image Input → Sales Analysis → Price Estimation → Quote Generation → Final Output
```

Κάθε πράκτορας:
- Λαμβάνει την έξοδο από το προηγούμενο στάδιο ως πλαίσιο
- Βασίζεται στην προηγούμενη ανάλυση με ειδική τεχνογνωσία
- Διατηρεί τη συνέχεια ροής εργασίας μέσω διαχείρισης κατάστασης

- Διαμορφώστε το αρχείο `.env` με το περιβάλλον Azure OpenAI και την ανάπτυξή σας, στη συνέχεια συνδεθείτε με `az login`

```bash
# Κάντε το σενάριο εκτελέσιμο (Unix/Linux/macOS)

```

```powershell
dotnet run 02.dotnet-agent-framework-workflow-ghmodel-sequential.cs
```

### 📝 Αναμενόμενη Έξοδος

Η ροή εργασίας θα:
1. **Πράκτορας Πωλήσεων**: Εντοπίσει είδη επίπλων από την εικόνα και να παρέχει προτάσεις
2. **Πράκτορας Τιμών**: Προσθέσει λεπτομερή ανάλυση τιμών με επίπεδα προϋπολογισμού και προτάσεις αγοράς
3. **Πράκτορας Προσφορών**: Δημιουργήσει μορφοποιημένο έγγραφο προσφοράς με συνθετικές πληροφορίες

Το τελικό αποτέλεσμα θα είναι μια ολοκληρωμένη, επαγγελματική προσφορά επίπλων βασισμένη στην ανάλυση εικόνας.
