# Απαντήσεις Ελέγχου Γνώσης

## Ερώτηση 1

**Απάντηση: A2.** Πολλοί πράκτορες είναι χρήσιμοι όταν μια εργασία έχει διακριτούς ειδικούς ρόλους, ξεχωριστά εργαλεία ή δικαιώματα και σαφείς παραδόσεις που χρειάζονται συντονισμό. Ένας απλός υποστηρικτικός bot συνήθως μπορεί να παραμείνει ένας μόνος πράκτορας. Ένας μεγάλος όγκος πανομοιότυπων αιτήσεων είναι πρωτίστως ζήτημα κλιμάκωσης και από μόνος του δεν απαιτεί σχεδιασμό με πολλούς πράκτορες.

## Ερώτηση 2

**Απάντηση: A1.** Ένας μόνος πράκτορας είναι συνήθως ευκολότερος να δοκιμαστεί, να παρατηρηθεί και να λειτουργήσει όταν ένα σύνολο οδηγιών και εργαλείων μπορεί να χειριστεί την εργασία. Η πρόσβαση σε πολλά εργαλεία δεν απαιτεί αυτόματα πολλούς πράκτορες. Ξεχωριστοί ρόλοι με διαφορετικά δικαιώματα και απαιτήσεις ελέγχου μπορεί να δικαιολογήσουν σχεδιασμό με πολλούς πράκτορες.
