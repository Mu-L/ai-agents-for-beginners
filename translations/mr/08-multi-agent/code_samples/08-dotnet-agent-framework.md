# 🤝 एंटरप्राइझ मल्टि-एजंट वर्कफ्लो सिस्टम्स (.NET)

## 📋 शिकण्याचे उद्दिष्टे

हे नोटबुक .NET मधील Microsoft Agent Framework वापरून Azure OpenAI (Responses API) सह प्रगत एंटरप्राइझ-ग्रेड मल्टि-एजंट सिस्टम्स कसे तयार करायचे हे दर्शविते. तुम्हाला संरचित वर्कफ्लोद्वारे एकत्र कार्य करणाऱ्या अनेक विशेष एजंट्सचे संकलन कसे करायचे याचे शिक्षण मिळेल, उत्पादनासाठी तयार सोल्यूशन्ससाठी .NET चे एंटरप्राइझ वैशिष्ट्ये वापरून.

**तुम्ही तयार कराल अशी एंटरप्राइझ मल्टि-एजंट क्षमता:**
- 👥 **एजंट सहकार्य**: संकलन-वेळ वैधतेसह प्रकार-सुरक्षित एजंट समन्वय
- 🔄 **वर्कफ्लो संचालन**: .NET च्या async पॅटर्नसह घोषणात्मक वर्कफ्लो व्याख्या
- 🎭 **भूमिका विशेषीकरण**: मजबूत-टाईप केलेले एजंट व्यक्तिमत्त्वे आणि कौशल्य क्षेत्रे
- 🏢 **एंटरप्राइझ एकत्रीकरण**: देखरेख आणि त्रुटी हाताळणीसह उत्पादनासाठी तयार पॅटर्न्स

## कोड उदाहरण

या धड्याचा संपूर्ण चालणारा कोड संलग्न C# फाईलमध्ये उपलब्ध आहे: [`08-dotnet-agent-framework.cs`](../../../../08-multi-agent/code_samples/08-dotnet-agent-framework.cs)

नमुना चालवण्यासाठी:

```bash
# फाइल executable बनवा (Linux/macOS)
chmod +x 08-dotnet-agent-framework.cs

# नमुनं चालवा
./08-dotnet-agent-framework.cs
```

किंवा .NET CLI वापरून:

```bash
dotnet run 08-dotnet-agent-framework.cs
```

## हा नमुना काय दाखवितो

हा मल्टि-एजंट वर्कफ्लो सिस्टम दोन विशेष एजंटसह हॉटेल ट्रॅव्हल शिफारस सेवा तयार करतो:

1. **फ्रंटडेस्क एजंट**: एक ट्रॅव्हल एजंट जो उपक्रम आणि स्थान शिफारसी पुरवतो
2. **कॉन्सर्ज एजंट**: शिफारसींची पुनरावलोकने करतो जेणेकरून प्रामाणिक, नॉन-टुरिस्ट अनुभव नक्की होतील

एजंट एकत्र वर्कफ्लोमध्ये काम करतात जिथे:
- फ्रंटडेस्क एजंट आरंभीचा प्रवास विनंती प्राप्त करतो
- कॉन्सर्ज एजंट शिफारस पुनरावलोकन करतो आणि सुधारतो
- वर्कफ्लो प्रतिसाद वेळेवर प्रवाहित करतो

## मुख्य संकल्पना

### एजंट समन्वय
हा नमुना Microsoft Agent Framework वापरून संकलन-वेळ वैधतेसह प्रकार-सुरक्षित एजंट समन्वय दाखवितो.

### वर्कफ्लो संचालन
अनेक एजंट्सना पाइपलाइनमध्ये जोडण्यासाठी .NET च्या async पॅटर्न्ससह घोषणात्मक वर्कफ्लो व्याख्या वापरतो.

### स्ट्रीमिंग प्रतिसाद
async एन्यूमेरेबल्स आणि इव्हेंट-चालित आर्किटेक्चर वापरून एजंट प्रतिसादांचे रिअल-टाइम स्ट्रीमिंग अंमलात आणतो.

### एंटरप्राइझ एकत्रीकरण
उत्पादनासाठी तयार पॅटर्न्स दाखवितो ज्यात समाविष्ट आहे:
- पर्यावरण चल कॉन्फिगरेशन
- सुरक्षित प्रमाणीकरण व्यवस्थापन
- त्रुटी हाताळणी
- असिंक्रोनस इव्हेंट प्रक्रिया
