# ⏩ Mipangilio ya Kazi ya Wakala Mfululizo na Azure OpenAI (API ya Majibu) (.NET)

## 📋 Mafunzo ya Usindikaji wa Mfululizo wa Juu

Kitabu hiki cha kazi kinaonyesha **mifumo ya mfululizo wa kazi** kwa kutumia Microsoft Agent Framework kwa .NET na Azure OpenAI (API ya Majibu). Utajifunza jinsi ya kujenga mitiririko ya usindikaji wa hatua kwa hatua yenye ufanisi, ambapo mawakala hufanya kazi kwa mpangilio maalum, na kila hatua hujengwa juu ya matokeo ya hatua iliyopita.

## 🎯 Malengo ya Kujifunza

### 🔄 **Usanifu wa Usindikaji wa Mfululizo**
- **Muundo wa Mtiririko wa Kazi Mstari**: Unda mitiririko ya hatua kwa hatua na utegemezi wazi
- **Usimamizi wa Hali**: Dumisha muktadha na mtiririko wa data katika hatua za mfululizo
- **Azure OpenAI (API ya Majibu)**: Tumia mifano ya Azure OpenAI katika mitiririko ya .NET yenye hatua nyingi
- **Mifano ya Mitiririko ya Biashara**: Jenga mifumo ya usindikaji ya mfululizo tayari kwa uzalishaji

### 🏗️ **Mifano ya Mfululizo ya Juu**
- **Usindikaji wa Kidhibiti cha Hatua**: Tekeleza pointi za tathmini kati ya hatua za mtiririko wa kazi
- **Uhifadhi wa Muktadha**: Dumisha hali na maarifa yaliyokusanywa kupitia hatua zote
- **Usambazaji wa Makosa**: Shughulikia makosa kwa upole katika minyororo ya usindikaji wa mfululizo
- **Uboreshaji wa Utendaji**: Utekelezaji wa mfululizo kwa ufanisi na mzigo mdogo

### 🏢 **Mifumo ya Mfululizo ya Biashara**
- **Mitiririko ya Usindikaji wa Hati**: Uchambuzi wa hati katika hatua nyingi, uongofu na uthibitishaji
- **Mitiririko ya Uhakikisho wa Ubora**: Ukaguzi wa mfululizo, uthibitishaji, na mchakato wa idhini
- **Mitiririko ya Uzalishaji wa Maudhui**: Utafiti → Kuandika → Kuhariri → Ukaguzi → Kuchapisha
- **Uendeshaji wa Mchakato wa Biashara**: Mitiririko ya biashara yenye hatua nyingi na utegemezi wazi

## ⚙️ Mahitaji & Usanidi

### 📦 **Pakiti Zinazohitajika za NuGet**

Pakiti muhimu kwa mitiririko ya .NET ya mfululizo:

```xml
<!-- Core AI Framework -->
<PackageReference Include="Microsoft.Extensions.AI" Version="10.*" />

<!-- Azure OpenAI (Responses API) -->
<PackageReference Include="Azure.AI.OpenAI" Version="2.*" />

<!-- Azure Identity and Async LINQ Support -->
<PackageReference Include="Azure.Identity" Version="1.15.0" />
<PackageReference Include="System.Linq.Async" Version="6.0.3" />

<!-- Local Agent Framework References -->
<!-- Microsoft.Agents.AI.dll - Core agent abstractions -->
<!-- Microsoft.Agents.AI.OpenAI.dll - Azure OpenAI (Responses API) integration -->
```

### 🔑 **Usanidi wa Azure OpenAI**

**Usanidi wa Mazingira (faili .env):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-4.1-mini
```

**Usimamizi wa Usanidi:**
```csharp
// Load environment variables securely
Env.Load("../../../.env");
var azureEndpoint = Environment.GetEnvironmentVariable("AZURE_OPENAI_ENDPOINT");
var deployment = Environment.GetEnvironmentVariable("AZURE_OPENAI_DEPLOYMENT");
```


- **Pointi za Ukaguzi**: Maeneo ya kuthibitisha kati ya hatua kuhakikisha ubora na muendelezo
- **Mteja wa Azure OpenAI**: Upatikanaji thabiti wa modeli za AI katika hatua zote za mtiririko

- **Kutengwa kwa Makosa**: Matatizo katika hatua moja hayaendelei kwenda hatua zinazofuata

- **Uboreshaji wa Hali**: Uhamishaji mdogo wa hali kati ya hatua kwa utendaji bora
- **Makundi ya Hatua Zenye Mshikamano**: Mitiririko mingi ya mfululizo inaweza kuendeshwa sambamba

- **Usalama wa Kiwango cha Hatua**: Sera tofauti za usalama kwa hatua tofauti za usindikaji

- **Kudhibiti Upatikanaji**: Ruhusa za kina kwa hatua tofauti za mtiririko wa kazi
- **Uzingatiaji wa Kanuni**: Kutosheleza mahitaji ya udhibiti kupitia usindikaji ulio na muundo

### 📊 **Ufuatiliaji & Uchambuzi**
- **Vipimo vya Kiwango cha Hatua**: Ufuatiliaji wa utendaji kwa kila hatua ya mtiririko
- **Utambuzi wa Vizuizi**: Tambua na boresha hatua zinazofanya kazi polepole
- **Vipimo vya Ubora**: Fuatilia ubora na viwango vya mafanikio kwa kila hatua
- **Uboreshaji wa Mchakato**: Kuboresha kila wakati kulingana na uchambuzi wa kiwango cha hatua
