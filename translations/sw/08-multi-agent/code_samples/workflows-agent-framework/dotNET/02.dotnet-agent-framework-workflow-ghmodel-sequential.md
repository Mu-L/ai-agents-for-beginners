# ⏩ Mipangilio ya Mtiririko wa Wakala mfululizo kwa Azure OpenAI (API za Majibu) (.NET)

## 📋 Mafunzo ya Usindikaji mfululizo wa Juu

Kitabu hiki cha majaribio kinaonyesha **mifumo ya mtiririko mfululizo** kutumia Microsoft Agent Framework kwa .NET na Azure OpenAI (API za Majibu). Utajifunza jinsi ya kujenga mifumo ya usindikaji ya hatua kwa hatua yenye akili ambapo mawakala hufanya kazi kwa mpangilio maalum, ambapo kila hatua hujengea juu ya matokeo ya hatua iliyopita.

## 🎯 Malengo ya Kujifunza

### 🔄 **Mimino ya Usindikaji Mfululizo**
- **Ubunifu wa Mtiririko wa Moja kwa Moja**: Tengeneza mifumo ya usindikaji hatua kwa hatua yenye utegemezi wazi
- **Usimamizi wa Hali**: Dumisha muktadha na mtiririko wa data kati ya hatua za mtiririko mfululizo
- **Azure OpenAI (API za Majibu)**: Tumia mifano ya Azure OpenAI katika mitiririko ya .NET yenye hatua nyingi
- **Mifumo ya Ubunifu wa Kiwanda**: Jenga mifumo ya usindikaji mfululizo inayostahili utengenezaji

### 🏗️ **Mifumo ya Mfululizo ya Juu**
- **Usindikaji wa Kizuizi cha Hatua**: Tekeleza vituo vya uhakiki kati ya hatua za mtiririko
- **Utunzaji wa Muktadha**: Dumisha hali na maarifa yaliyokusanywa katika hatua zote
- **Usambazaji wa Makosa**: Shughulikia kushindwa kwa hila katika mnyororo wa usindikaji mfululizo
- **Uboreshaji wa Utendaji**: Utekelezaji mzuri wa mfululizo wenye mzigo mdogo

### 🏢 **Mifumo ya Mtiririko mfululizo ya Biashara**
- **Mtiririko wa Usindikaji wa Nyaraka**: Uchambuzi wa nyaraka yenye hatua nyingi, mabadiliko, na uhakiki
- **Mifumo ya Hakiki ya Ubora**: Mapitio, uhakiki, na michakato ya idhini kwa mfululizo
- **Mtiririko wa Uzalishaji wa Maudhui**: Utafiti → Uandishi → Uhariri → Mapitio → Kuchapisha
- **Automati ya Mchakato wa Biashara**: Mifumo ya mchakato wa biashara yenye hatua nyingi na utegemezi wazi

## ⚙️ Mahitaji na Usanidi

### 📦 **Paketi za Muhtasari**

Paketi muhimu kwa mitiririko ya .NET mfululizo:

```xml
<!-- Core AI Framework -->
<PackageReference Include="Microsoft.Extensions.AI" Version="10.*" />

<!-- Azure OpenAI (Responses API) -->
<PackageReference Include="Azure.AI.OpenAI" Version="2.*" />

<!-- Azure Identity and Async LINQ Support -->
<PackageReference Include="Azure.Identity" Version="1.15.0" />
<PackageReference Include="System.Linq.Async" Version="6.0.3" />

<!-- Local Agent Framework References -->
<!-- Microsoft.Agents.AI.dll - Core agent abstractions -->
<!-- Microsoft.Agents.AI.OpenAI.dll - Azure OpenAI (Responses API) integration -->
```

### 🔑 **Usanidi wa Azure OpenAI**

**Usanidi wa Mazingira (faili .env):**
```env
AZURE_OPENAI_ENDPOINT=https://<your-resource>.openai.azure.com
AZURE_OPENAI_DEPLOYMENT=gpt-5-mini
```

**Usimamizi wa Usanidi:**
```csharp
// Load environment variables securely
Env.Load("../../../.env");
var azureEndpoint = Environment.GetEnvironmentVariable("AZURE_OPENAI_ENDPOINT");
var deployment = Environment.GetEnvironmentVariable("AZURE_OPENAI_DEPLOYMENT");
```
