# 知識檢核答案

## 問題 1

**答案：A2。** 當任務具有明確的專家角色、不同的工具或權限，以及需要協調的明確交接時，多代理人系統才有用。簡單的支援機器人通常可以維持為單一代理人。大量相同請求主要是擴展問題，並不會單獨要求多代理人設計。

## 問題 2

**答案：A1。** 當可用相同的一組指令和工具處理任務時，通常單一代理人較容易測試、觀察與操作。擁有多個工具的使用權並不自動代表需要多個代理人。不同權限與稽核需求的分工角色可能才需要多代理人設計。
