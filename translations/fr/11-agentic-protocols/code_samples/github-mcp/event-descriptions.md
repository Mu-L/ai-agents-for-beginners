## Nom de l'événement : API Multi-Agent avec LangGraph et Azure Cosmos DB

## Description

L'essor des applications IA multi-agent transforme la manière dont nous construisons des systèmes intelligents - mais comment les architecturer pour une scalabilité et performance réelles ? Dans cette session, nous plongerons dans une application multi-agent de niveau production construite avec LangGraph pour l'orchestration des agents, FastAPI pour la couche API, et Azure Cosmos DB comme colonne vertébrale pour la gestion d'état, le stockage vectoriel et les données transactionnelles.

Collaboration des agents avec LangGraph pour un raisonnement structuré
Stockage en temps réel de l'historique de chat avec Azure Cosmos DB - la même base de données qui alimente l'historique de chat dans ChatGPT, l'application agent IA à la croissance la plus rapide de l'histoire
Recherche vectorielle pour la récupération de connaissances avec le support natif des embeddings de Cosmos DB
Capacités async de FastAPI pour maintenir des interactions réactives et scalables
À la fin de cette session, vous aurez un plan clair pour construire et déployer vos propres applications multi-agent cloud-natives scalables exploitant la puissance de l'IA moderne et de l'infrastructure cloud. Que vous soyez ingénieur IA, architecte cloud ou développeur Python, cette présentation vous dotera d'enseignements pratiques et de patterns éprouvés pour construire la prochaine génération d'applications alimentées par l'IA.
